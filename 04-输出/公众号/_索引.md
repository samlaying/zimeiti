# 20-公众号 — 目录索引

## 定位

公众号文章的**终稿存放区**。这里的每一篇都应该是定稿。

## 接受的内容

| 状态 | 说明 |
|------|------|
| 定稿 | 完整打磨、可直接发布或已发布的文章 |

## 不放这里

- 写到一半的文章 → 放 `05-内容输出/国内/公众号/`
- 原始素材 → 放 `输入/`

## frontmatter 要求

公众号终稿必须有完整的 frontmatter：

```yaml
---
title: 文章标题
author: Sam
digest: 一句话摘要
status: 定稿
type: 教程          # 或 观点 / 个人故事 / 其他
date: 2025-05-24
---
```

## 分类示例

> 一篇反复打磨过的 Vibe Coding 入门指南。
> → status: `定稿`，type: `教程`
