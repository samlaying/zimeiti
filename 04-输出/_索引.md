# 05-内容输出 — 目录索引

## 定位

面向各平台的内容**草稿区**。从素材加工出来的、准备发布的内容。

## 接受的内容

| 状态 | 说明 | 例子 |
|------|------|------|
| 半成品 | 有了框架和部分内容，还没写完 | 大纲 + 部分段落 |
| 定稿 | 完整可发布，等待或已经发布 | 成品文章 |

## 子目录

| 子目录 | 用途 |
|--------|------|
| `国内/公众号/` | 公众号内容 |
| `国内/小红书/` | 小红书内容 |
| `国内/朋友圈/` | 朋友圈内容 |
| `出海/Twitter/` | Twitter 内容 |
| `出海/Reddit/` | Reddit 内容 |
| `出海/YouTube/` | YouTube 内容 |

## 与 `20-公众号` 的关系

- `05-内容输出/国内/公众号/` — 公众号的**草稿和迭代过程**
- `20-公众号/` — 公众号的**终稿**，已打磨完成

## 分类示例

> 你基于一篇课程笔记，开始写公众号文章，写了一半。
> → status: `半成品`，type: `教程`，放在 `05-内容输出/国内/公众号/`

> 文章写完了，反复改了几遍，可以直接发了。
> → status: `定稿`，可以移到 `20-公众号/`
