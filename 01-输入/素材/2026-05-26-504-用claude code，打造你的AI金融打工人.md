# 用claude code，打造你的AI金融打工人

> [罗勇-量化投资](https://space.bilibili.com/385259693) | [BV1dNzXBTEPJ](https://www.bilibili.com/video/BV1dNzXBTEPJ) | 时长 18:09

## 笔记：使用Claude Code打造AI金融分析工具

### 一、 Claude Code 初印象与产品定位
*   **工具定位**：Claude Code 是一个强大的AI编程助手（“AI金融打工人”），可以用于全程自动生成代码、脚本、安装包和说明文档。
*   **演示案例**：UP主在电脑上安装了Claude Code，并命名为“老罗打工人”。展示了由该工具在前一天独立生成的一个金融分析小产品的完整文件夹。

### 二、 工具成果展示与价值评估
#### 1. 工作成果与效率
*   **生成内容**：产品文件夹包含所有原始文件，最终产品是一个可执行文件及配套的大模型密钥、技术文档（如`借个T文档`和`MT文档`）。
*   **效率评估**：
    *   询问Claude Code如果由普通程序员完成此中等量级开发工作需要多久。
    *   **AI评估结论**：
        *   熟练老手：需要3-4天。
        *   初级程序员：可能需要一周。
    *   **核心价值**：AI将用户从繁琐的代码实现中解放出来，使其能专注于更高层次的架构设计和方法论思考，实现了“把时间的厚度换成洞察的深度”。

#### 2. 产品价值与定价
*   **功能演示**：运行可执行文件后，工具自动开始工作，通过大模型进行实时金融市场综合判断。
*   **价值评估**：
    *   **初步评估**：该工具能帮助一个管理100万资金的客户每年减少约6.75万元的亏损。建议每月订阅价为800-1500元。
    *   **修正评估**：发现该工具并非每日查看一次的简单工具，而是**盘中实时决策引擎**。因此对标市场上的专业付费社群（如`问的轻浪版`），定价调整为**年费8000至12000元**，认为完全合理。

### 三、 产品深度解析
#### 1. 核心价值并非代码本身
Claude Code在回答中指出，真正有价值的是：
1.  **方法论沉淀**：UP主独创的“情绪量化+板块矩阵+场景推演”分析框架。
2.  **AI产品工程化**：让AI学会了可复用、可规模化的“老罗风格”。
3.  **工具思维**：不空谈洞见，而是将判断力封装成可直接使用的工具。

#### 2. 工具实际工作过程
*   **数据抓取**：大模型实时抓取了**84条快讯、20条行业新闻、23条有效市场评论**，并进行高频词统计。
*   **输出成果**：自动生成一份Markdown格式的深度分析文档，为后续决策提供强大数据支撑。

### 四、 实操步骤：如何用Claude Code打造产品
使用Claude Code实现上述金融产品主要分为三步：
1.  **第一步：学习习惯** —— 让Claude Code学习你（开发者）之前写过的代码风格。如果没有，可以跳过，但开发过程可能更波折。
2.  **第二步：定义技能** —— 给它一个“Skill”（技能）。如果自己不会写，可以上网找一个类似的，然后通过修改提示词（Prompt）来完成你自己的专属Skill。
3.  **第三步：迭代产出** —— 通过与Claude Code聊天、修改、调试、迭代代码，逐步打包，形成最终产品。

### 五、 核心总结与展望
**当代码实现本身不再是壁垒时，提示词工程（Prompt Engineering）就成为了关键。** 无论是产品需求文档（PRD）还是技能描述（Skill），本质都是对提示词的精准掌控。通过Claude Code这类AI工具，个人能将自己的创意、方法论和专业洞察快速产品化，实现能力的规模化延伸。UP主罗勇在其著作《AI量化交易》及续集中，详细介绍了所使用的第二代提示词，正是视频中产品生成效果的底层支撑。行动起来，运用AI工具，可以解锁人生的更多可能性。
