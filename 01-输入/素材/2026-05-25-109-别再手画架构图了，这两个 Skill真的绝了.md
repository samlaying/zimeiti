# 别再手画架构图了，这两个 Skill真的绝了

> [AI产品经理养成记](https://space.bilibili.com/47333744) | [BV1QxoPBgEUp](https://www.bilibili.com/video/BV1QxoPBgEUp) | 时长 2:50

## 视频笔记：两款自动生成可编辑架构图/流程图的Skill
视频分享了两款能通过描述自动生成可编辑架构图和流程图的AI Skill，旨在替代手绘，提升工作效率。

### 一、核心推荐的两款Skill
视频重点介绍了UP主日常使用的两个Skill：

1.  **Jetting Canvas (来自 Obsidian Skills)**
    *   **来源**：Obsidian Skills 项目。
    *   **特点**：除了此Skill，该项目还有其他实用技能可供探索。

2.  **Excel Draw Diagram Generator (来自 GitHub Awesome Corporate)**
    *   **来源**：GitHub上的开源项目 **Awesome Corporate**，该项目下拥有**超过300个Skill**。
    *   **功能**：能根据文本描述生成多种图表，例如：
        *   基本流程图
        *   业务流程图
        *   功能流程图
        *   页面流程图

### 二、Skill使用方法
这两款Skill的使用逻辑相似，核心步骤如下：

**通用步骤：**
1.  **引用Skill**：在AI编程工具中引用对应的Skill。
2.  **输入描述**：用自然语言描述你想要的图表内容。
3.  **生成与编辑**：Skill会生成可编辑的图表。如果不满意，可以删除、新增或修改其中的内容，按需调整。

**示例描述**：“给我生成一个常见场景的基本流程图，页面流程图，目的是展示XXX流程。”

### 三、通过Momen语法转换图表
除了直接生成，还支持将已有的 **Momen语法** 转换为图表：
1.  点击“更多” -> 选择“Momen G-Exceledron”。
2.  在弹出的输入框中**粘贴你的Momen语法**。
3.  点击“插入”，即可将语法转换为可视化的流程图。

### 四、其他相关Skill资源
视频还提及了其他可探索的Skill资源：
*   **Austin Obsidian Visual Skills**：其中包含类似“Excel Draw”的skill，以及“Maman的Vualizer Skill”和“Canvas Creator Skill”。

### 五、总结
视频核心是推荐使用 **Jetting Canvas** 和 **Excel Draw Diagram Generator** 这两个AI Skill，通过输入描述快速生成专业、可编辑的架构图与流程图，能有效替代手绘，显著提升工作效率。对于经常需要绘制此类图表的人员，值得尝试。此外，还可以探索文中提到的其他相关Skill以丰富工具库。
