## 二级导航菜单实现笔记

### 1. 项目概述与准备
- **项目名称**：二级导航菜单。
- **场景说明**：非常实用且普遍的网页组件，当鼠标悬停在一级菜单上时，会滑出显示对应的二级菜单列表。
- **代码获取**：原码可通过微信公众号 **“笑谈科技”** 回复 **“45071”** 获取。
- **开发环境**：使用 VS Code 进行编码。

### 2. HTML 结构搭建
基本HTML结构如下：
```html
<div class="navbar">
    <div class="nav">
        <ul>
            <li>
                <a href="#">一级菜单</a>
                <ol>
                    <li><a href="#">二级菜单1</a></li>
                    <li><a href="#">二级菜单2</a></li>
                    <!-- 更多二级菜单项... -->
                </ol>
            </li>
            <!-- 更多一级菜单项... -->
            <li class="underline"></li> <!-- 用于滑动的线条元素 -->
        </ul>
    </div>
</div>
```
**关键点**：
- 外层容器 `.navbar` 包裹 `.nav` 导航栏。
- 导航栏内部使用 `<ul>` 定义列表。
- 每个一级菜单项 (`<li>`) 内包含一个 `<a>` 标签和一个 `<ol>` 列表来存放二级菜单。
- 在一级菜单列表的最后，添加一个 `<li class="underline">` 元素，用于后期实现滑动线条效果。

### 3. CSS 样式实现
#### 3.1 页面初始化
```css
* {
    padding: 0;
    margin: 0; /* 清除元素内外边距 */
}
```

#### 3.2 导航栏样式
```css
body {
    width: 100vw; /* 宽度占满浏览器可视区域 */
    background-color: #f2f2f2;
}
.navbar {
    width: 100%;
    height: 70px;
    background-color: #ffffff;
    box-shadow: 0 0 5px rgba(0, 0, 0, 0.3); /* 盒子阴影 */
}
.navbar .nav {
    width: 1200px;
    height: 100%;
    margin: 0 auto; /* 水平居中 */
}
```

#### 3.3 一级菜单样式
```css
.navbar .nav ul {
    position: relative;
    display: flex; /* 弹性布局 */
    justify-content: space-around; /* 子元素平均分配宽度 */
    width: 100%;
    height: 100%;
}
.navbar .nav ul > li {
    width: 100%;
    height: 100%;
}
.navbar .nav ul > li a {
    display: block; /* 转换为块级元素以设置宽高 */
    width: 100%;
    height: 100%;
    line-height: 70px; /* 行高与高度一致实现垂直居中 */
    text-align: center;
    text-decoration: none; /* 去除下划线 */
    color: #000;
}
```

#### 3.4 二级菜单样式与动画
```css
.navbar .nav ul > li ol {
    width: 100%;
    background-color: #ffffff;
    box-shadow: 0 2px 3px rgba(0, 0, 0, 0.2);
    transform: scaleY(0); /* 沿Y轴缩放至0，隐藏 */
    transform-origin: 50% 0; /* 缩放中心点为顶部中间 */
    transition: all 0.6s; /* 过渡动画 */
}
.navbar .nav ul > li ol li {
    height: 70px;
    border-bottom: 1px solid rgb(245, 245, 245);
}
.navbar .nav ul > li ol li a {
    display: block;
    width: 100%;
    height: 100%;
    line-height: 70px;
    text-align: center;
}
/* 鼠标悬停一级菜单时，显示对应的二级菜单 */
.navbar .nav ul > li:hover ol {
    transform: scaleY(1);
}
/* 鼠标悬停二级菜单项时，添加背景色 */
.navbar .nav ul > li:hover {
    background-color: rgba(0, 0, 0, 0.03);
}
```

#### 3.5 滑动线条 (`.underline`) 样式
```css
.navbar .nav ul .underline {
    position: absolute; /* 绝对定位 */
    bottom: 0;
    left: 0; /* 默认位置 */
    width: 240px; /* 每个一级菜单项的宽度 */
    height: 6px;
    border-top-left-radius: 6px; /* 左上角圆角 */
    border-top-right-radius: 6px; /* 右上角圆角 */
    background-color: #cc3333;
    transition: left 0.5s, background-color 0.5s; /* 过渡动画 */
    pointer-events: none; /* 避免鼠标事件干扰 */
}
```

#### 3.6 滑动线条的三角形指示器
使用伪元素 `.underline::before` 绘制：
```css
.navbar .nav ul .underline::before {
    content: '';
    position: absolute;
    top: -10px;
    left: calc(50% - 9px); /* 水平居中 */
    width: 18px;
    height: 10px;
    background-color: inherit; /* 继承父元素颜色 */
    /* 使用 clip-path 绘制三角形 */
    clip-path: polygon(0% 100%, 50% 0%, 100% 100%);
}
```

#### 3.7 为不同菜单项定制滑动线条
通过 `:nth-child` 选择器为每个一级菜单项设置不同的线条位置和颜色：
```css
/* 第2个菜单项 */
.navbar .nav ul > li:nth-child(2):hover ~ .underline {
    left: 240px; /* 移动到第二个菜单下方 */
    background-color: #339933;
}
/* 第3个菜单项 */
.navbar .nav ul > li:nth-child(3):hover ~ .underline {
    left: 480px;
    background-color: #0099cc;
}
/* 第4个菜单项 */
.navbar .nav ul > li:nth-child(4):hover ~ .underline {
    left: 720px;
    background-color: #9966cc;
}
/* 第5个菜单项 */
.navbar .nav ul > li:nth-child(5):hover ~ .underline {
    left: 960px;
    background-color: #ff9933; /* 示例颜色，可根据需要调整 */
}
```

### 4. 核心实现技巧总结
- **二级菜单隐藏/显示**：通过 `transform: scaleY(0)` 隐藏，`scaleY(1)` 显示，配合 `transform-origin: 50% 0` 使其从顶部中心展开。
- **滑动线条效果**：使用一个绝对定位的元素 `.underline`，在鼠标悬停不同一级菜单项时，通过 `:nth-child` 选择器改变其 `left` 位置和 `background-color`。
- **三角形绘制**：利用伪元素和 `clip-path: polygon()` 绘制三角形指示器。
- **交互优化**：使用 `pointer-events: none` 防止滑动线条阻挡鼠标对下层菜单的交互。
- **自定义**：可以轻松修改菜单文字、跳转链接（修改 `<a>` 标签的 `href` 属性）以及颜色主题。

### 5. 总结
本笔记详细记录了使用纯 HTML 和 CSS 构建一个带有滑动指示条和下拉动画的二级导航菜单的完整过程。核心在于利用 CSS 的弹性布局 (`flex`) 实现菜单项水平分布，使用 `transform` 和 `transition` 实现平滑的下拉动画，并通过一个绝对定位的线条元素和 `:nth-child` 选择器动态改变其位置和颜色，从而实现跟随鼠标悬停菜单项滑动的视觉效果。整个实现过程代码结构清晰，交互效果自然，适合直接应用于网页项目中。
