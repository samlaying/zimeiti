# 一行命令从话题到4K成片：Video Podcast Maker 全新功能介绍 (V2 Update)

> [探索未至之境](https://space.bilibili.com/441831884) | [BV1s9ABzrEeA](https://www.bilibili.com/video/BV1s9ABzrEeA) | 时长 3:57

## 视频概述
Video Podcast Maker 是一个基于 Cloud Code 或 Open Code 的技能，用于实现从话题输入到 4K 视频成片的全自动流程。它通过 14 个步骤的自动化工作流，帮助用户快速制作视频播客，无需编写代码或剪辑技能，旨在提高视频产出效率。该工具开源免费，适用于多种话题类型，如 GitHub 开源项目介绍、论文解读、AI 行业动态、财经投资知识科普、产品对比评测、软件使用教程或热点新闻分析。

## 工作流程
### 14 个自动化步骤
整个视频制作流程分为以下 14 个步骤，用户只需输入话题，Cloud 会交互式引导完成：
1. **定义话题方向**：用户输入任何感兴趣的话题。
2. **自动上网调研**：系统自动收集相关信息。
3. **设计章节结构**：规划视频的章节布局。
4. **撰写解说脚本**：生成视频的解说文案。
5. **收集素材资源**：获取视频所需的图像、视频等素材。
6. **生成发布信息**：创建视频的元数据和发布详情。
7. **制作视频封面**：自动生成视频封面。
8. **生成 TTS 配音和字幕**：使用文本转语音技术生成配音，并制作字幕。
9. **创建 Remotion 组件**：构建视频的可视化组件。
10. **实时预览调试**：在渲染前预览和调整视频效果。
11. **渲染 4K 视频**：生成 3840x2160 分辨率的 4K 视频。
12. **混合背景音乐**：添加和调整背景音乐。
13. **可选烧录字幕**：可选择将字幕直接嵌入视频。
14. **补全章节信息**：完善章节的细节和时间戳。

用户可以在任何步骤中随时叫停，并使用自然语言提出具体要求，例如调整章节顺序、修改文案或更换表达风格，系统会立即响应。

## 最新功能
### Remotion Studio 可视化编辑
Video Podcast Maker 最新加入了 Remotion Studio 功能，允许用户在渲染前进行可视化编辑，实现所见即所得的调整：
- **实时预览**：打开 Studio 后，可实时预览视频效果。
- **面板调整**：右侧面板支持调整颜色、字体大小、进度条样式、背景音乐音量等参数。
- **用户友好**：无需修改代码，不懂代码的用户也能轻松定制视频风格。
- **交互式优化**：如果效果不满意，可在 Cloud Code 或 Open Code 中用自然语言进一步调整，一边修改一边预览，直到满意后再开启 4K 渲染，从而节约时间。

## 六大核心功能
### 1. Azure TTS 高质量配音
- 使用 Azure 文本转语音服务，提供高质量配音。
- 支持中英文混读：脚本中中文和英文可随意混排，系统会自动切换对应语音。

### 2. 多音字智能校正
- 内置常用多音字词典，自动识别并纠正读音。
- 如果遇到读错的字，用户可以用拼音标注来手动调教，确保发音准确。

### 3. 4K 输出
- 支持 4K 分辨率（3840x2160），输出高清视频。

### 4. 章节进度条
- 在视频播放时显示当前章节信息，提升观看体验。

### 5. B 站深度优化
- 自动生成章节时间戳，适配 B 站平台。
- 提供两种比例（如 16:9 和 4:3）的封面生成选项。

### 6. 内置背景音乐库
- 提供内置的背景音乐库，用户可以通过一行命令进行混音。
- 用户还可以直接修改音色和背景乐，使用自然语言在 Cloud Code 或 Open Code 中交互调整。

## 使用方法
### 前提条件
- 需要 Cloud Code 或 Open Code（UP 主已尝试过两者）。
- 需要一个 Remotion 项目。
- 需要 Azure Speech 的 API 密钥。

### 启动方式
- 安装好前提工具后，直接输入命令，例如：“帮我制作一个关于某某话题的视频”。
- Cloud 会引导用户走完全部 14 个步骤的流程，全程无需编写代码或剪辑技能。
- 据 UP 主介绍，整个流程约 10 分钟即可触片（生成视频）。

## 总结核心要点
Video Podcast Maker 是一款开源免费的自动化视频制作工具，通过 14 步全自动流程实现从话题到 4K 成片的快速生成，最新加入了 Remotion Studio 可视化编辑功能，让用户无需代码即可轻松定制视频风格。它支持 Azure TTS 配音、多音字校正、4K 输出、B 站优化等核心功能，适用于多种话题类型，显著提高视频产出效率，适合希望提升内容创作效率的用户使用。
