# 【全网首发】Gemini网页会员也能当API用?教你接入小龙虾等工具，国内网络也可用

> [康旭_](https://space.bilibili.com/511154971) | [BV1q8QwBwE1t](https://www.bilibili.com/video/BV1q8QwBwE1t) | 时长 4:03

## 概述
本教程介绍如何将Gemini网页会员订阅转换为API接口，通过低成本方式实现不限量Token调用。配合GMI会员白条教程，可以畅用Gemini Pro。工具涉及OpenClaw或ClawedCode（可能指OpenAI兼容接口），适用于国内网络环境，帮助用户提升AI应用效率并降低成本。

## 准备工作
- **购买VPS**：选择资源较少的VPS，例如512MB内存即可满足个人使用，确保成本低廉。
- **系统选择**：推荐使用Debian 12操作系统，兼容性好且稳定。
- **支付与创建**：通过扫码支付备用，等待服务器创建完成；所有相关网站链接会在评论区提供。

## 服务器连接与配置
1. **连接服务器**：
   - 使用SH软件（如SSH客户端，如PuTTY或终端）连接服务器。
   - 服务器名称随意填写，IP地址从VPS控制台复制。
   - 修改服务器密码以增强安全性，并在SSH软件中填入连接信息后点击创建并连接。
2. **克隆项目**：
   - 前往Github项目地址（具体地址见评论区）。
   - 将项目克隆到服务器：使用命令 `git clone [项目地址]`，例如 `git clone https://github.com/example/project`。
3. **配置环境变量**：
   - 转到项目根目录：使用命令 `cd [项目目录]`。
   - 使用nano编辑器编辑env文件：运行 `nano .env`（如果文件为.env.example，需先复制或重命名）。
   - env文件变量说明（从上到下顺序）：
     - **是否开启思考模式**：设置为true（开启）或false（关闭），用于控制模型思考行为。
     - **临时模式**：自动删除绘画等功能，设置为true或false，根据需求调整。
     - **Cookie**：用于Gemini网页登录认证，需要从浏览器获取。
     - **调用TI key**：API密钥，可以使用默认值（如GMI-default-key）或自定义；通常为倒数二三个变量。
   - 填写变量：如果需要开启思考模式，将第一个变量设为true。填写Cookie和API key（从倒数二三个变量中获取）。
   - 保存并退出：在nano编辑器中，按Ctrl+O保存，回车确认；按Ctrl+X退出。

## 获取Cookie与启动服务
1. **获取Cookie**：
   - 新开一个无痕浏览器窗口（如Chrome无痕模式），登录Gemini账号。
   - 在页面上按F12（或右键选择“检查”）打开开发者工具。
   - 选择“应用”标签（或“Application”），在左侧点开Cookie。
   - 找到以GMI开头的网页Cookie条目，复制两个相关变量下的值（例如sessionid和token）。
   - 将这些值填入env文件中的相应Cookie变量。
2. **启动服务**：
   - 重命名env文件：将.env.example重命名为.env（删除后缀“Example”）。
   - 启动服务：运行项目提供的启动命令（例如 `python app.py` 或 `node server.js`，具体见项目说明）。
   - 服务默认运行在IP地址的8000端口：访问地址为 `http://[服务器IP]:8000`。
   - API接口路径为：`/v1/chat/completions`，符合OpenAI API标准。
   - 认证使用env文件中设置的API key（即调用TI key变量的值）。

## 接入测试
- **接入New API**：
  - 演示接入New API（如其他兼容接口），测试是否接通。
  - 在接入工具中，填写URL为 `http://[服务器IP]:8000/v1/chat/completions`，并指定API key。
  - 测试成功标志：如果返回正常响应（如模型回复），表示接口接通。
- **接入OpenClaw或ClawedCode**：
  - 将你的URL（即服务地址加接口路径）和API key提供给小龙虾等工具。
  - 指定接口类型为OpenAI兼容接口（因为路径为/v1/chat/completions）。
  - 例如，在工具配置中，输入URL和key，选择“OpenAI API”模式即可。

## 总结
本教程通过购买低成本VPS（512MB内存）、配置Gemini网页订阅转API的服务器，实现了国内网络下不限量Token调用Gemini模型的方案。核心步骤包括VPS设置、环境变量配置、Cookie获取和API服务启动，最终可接入OpenClaw等工具，提升AI应用效率并降低成本，适合个人或小规模使用。
