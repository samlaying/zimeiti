> [AI技术投降派](https://space.bilibili.com/54052968) | [BV1f3RcBKEmV](https://www.bilibili.com/video/BV1f3RcBKEmV) | 时长 4:15

## AngelSlim：腾讯开源的全流程模型压缩工具箱

### 工具概览
*   **开发者**：腾讯会员AI基础设施团队。
*   **核心痛点**：解决大模型（动辄数百亿至上千亿参数）因推理成本过高而难以部署的问题。硬件（如A100、H100显卡）成本高昂是主要障碍。
*   **定位**：**不是单一的量化工具**，而是一个覆盖模型压缩全流程的**工具箱**。
*   **支持模型类型**：覆盖生产环境中四大类主流模型：
    1.  大语言模型
    2.  多模态模型
    3.  图像生成模型
    4.  语音模型
*   **压缩手段**：提供丰富的组合拳，包括：
    *   **量化算法**：超过十几种，精度从高到低覆盖 FP8, Int8, Int4 乃至 1.25 比特。
    *   **推测解码加速**
    *   **Token压缩**（主要面向多模态模型）

### 核心技术板块
#### 1. 量化（主力方向）
支持从粗到细的多种精度与算法，针对不同硬件和场景优化：
*   **FP8**：支持静态和动态量化。
*   **Int8**：动态量化。
*   **Int4**：支持 GPQ、AWQ、GPTQ 三种算法。
*   **NVFP4 精度**、**TEPILA 三值量化**、**Sky 1.25 比特量化**（极限压缩）。
*   **DAQ 动态自适应量化**。

#### 2. 推测解码
*   采用 **EGO3 方案**。
*   **实测效果**：可带来 **1.4 倍至 1.9 倍** 的推理加速。

#### 3. Token压缩
*   主要面向多模态模型（如视觉语言模型）。
*   使用 **ID Pruner** 等方法，砍掉冗余的视觉 Token，以降低计算量。

### 主要亮点
1.  **覆盖面广**：支持主流模型，如 HunYuan、Qwen2、DeepSeek-V3、GLM4.6（大语言模型）、Flux（图像模型）、Whisper（语音模型）等，用户无需担心兼容性问题。
2.  **兼容性好**：压缩后的模型可直接部署到 **vLLM** 和 **SGLang** 两个主流推理框架，无需额外适配。
3.  **上手简单**：
    *   安装：一行命令 `pip install AngelSlim`。
    *   调用：通过 Python 代码直观调用。
4.  **数据详实**：项目仓库提供了大量精度与速度的基准测试对比表格，便于用户评估效果。

### 典型使用场景
1.  **云端大模型部署降本**：
    *   将数百亿参数的模型进行 INT4 量化后，显存占用可降至原来的 **1/4**。
    *   **效果示例**：原来需要4张卡才能运行的模型，现在1张卡即可搞定。

2.  **边缘设备部署**：
    *   在手机、嵌入式设备上部署大模型，可利用 FP8 甚至 INT4 量化来满足资源限制。
    *   AngelSlim 的多精度支持正好匹配此类需求。

3.  **图像和语音模型的生产化**：
    *   **图像生成模型**（如Flux）：量化后推理速度可翻倍。
    *   **语音模型**（如Whisper）：量化后可以在低配设备上实现**实时运行**。
    *   这些都是可直接投入生产的工具，而非实验室Demo。

### 总结
AngelSlim 是一个由腾讯开源的、覆盖大语言模型、多模态模型、图像模型、语音模型四大类的**全流程模型压缩工具箱**。它提供了从 FP8 到 1.25 比特的全精度量化算法库，辅以推测解码和 Token 压缩技术。该工具具备广泛的模型兼容性（支持众多主流模型）、优秀的推理框架兼容性（vLLM/SGLang），并且通过简单的 `pip install` 和 Python 调用即可上手。对于任何面临大模型推理成本高、部署难问题的开发者和工程师而言，AngelSlim 是一个非常实用且值得收藏的开源工具。
