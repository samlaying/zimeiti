# 6个UI设计技巧：从平庸到高级

> 来源：设计师凯文 CreativeKev · 2024-09-08

---

## 技巧一：善用高级灰（颜色）

**颜色体系构成：** 主颜色 + 中性色 + 功能颜色（大型设计系统还会有辅助色）

**核心观点：** 灰色是中性色，不承担品牌视觉任务，但在 Web/App 设计中极其关键，是使用面积最大的颜色类型。

**实操要点：**
- 不同灰度构成页面基础轮廓结构，划分功能区域，帮助视觉导航
- **多用两端色值**（白色、浅灰、深灰），**少用中间段色值**——中间段会加重视觉负担，打破颜色平衡
- **小窍门：** 建立灰色色阶时往里加一点蓝色，使中性色更透气、高级、有商务感
- 实际项目中**先从灰色入手**，用灰色完成页面基础构建，再加入主颜色进行强调
- **不要执着于添加多种颜色**——绝大部分情况下一个主颜色就够了，多种颜色难以驾驭，也会让品牌记忆点模糊
- 只有黑白灰的网站一样可以很高级
- 对于品牌服装、艺术展览等本身含大量产品图片/摄影作品的网站，使用黑白灰之外的颜色反而会产生干扰

## 技巧二：字体优先保证易读性

**核心观点：** 对于字体，相比好看，**易读性更重要**。文字是用来传递信息的，任何信息的不清楚都是用户体验的失败。

**涉及概念：无障碍设计（Accessibility）**
- 鼓励产品设计为残障人士考虑，尤其是视力障碍、色盲
- 全世界至少有 **20 亿人**有视觉障碍，这个数量不可忽略
- 无障碍设计不仅服务残障人士，也为普通人提供更好的体验

**文字易读性的具体要求：**

| 要点 | 说明 |
|------|------|
| **正文字号不要过小** | Web/App 正文大多以 **14–16px** 为基准 |
| **颜色与对比度** | 文字和背景之间要有足够对比度，必须进行对比度检查 |
| **W3C 2A 标准** | 普通文字对比度 ≥ **4.5:1**；字号 > 18px 的文字 ≥ **3:1** |
| **3A 标准** | 用户群体更广泛的产品，需采用更严格的 3A 标准 |

**工具推荐：**
- 使用在线**对比度检查工具**，输入文字和背景色色值即可验证
- W3C 官方提供的网站检查工具：输入网址即可检查各项无障碍指标，给出整体评分及问题定位

## 技巧三：合理使用间距与留白

**基础：8 点网格系统**
- 所有数值为 **8 的倍数**
- 8 符合偶数原则，可被 4 和 2 整除
- 能适应大多数屏幕设备，保证输出各种倍数图时不出现奇数或半个像素

**留白的价值：**
> "页面上的留白并不是无用的背景，而是有影响力的视觉元素。"——齐希尔德

- 留白更多的页面看起来更加**舒适**
- 留白带来**高级感**，与"少即是多"同理
- 留白是**强调信息**的方式：元素周围留白增加 → 用户注意力集中在该元素上

**元素关联性原则：**
- 通过间距和样式，把**相关联的元素放在一起**，不相关的加以区分
- 例：标题与所含内容间距较近，不同段落之间间距较远
- 不相关的内容用**间距、排版、颜色**区分，创造层次感和清晰的信息布局

## 技巧四：优先使用已有设计模式（不要过度发明创造）

**核心观点：** 如果能用市面上已有的视觉元素（组件、模式、图标），就不要发明创造新的事物。

**原因：**
- 产品设计中，相比视觉创意，更需考虑**用户体验**
- 用户使用数字产品是为了解决问题、完成任务，希望用**最短时间、最容易的方式**完成
- 全新 UI 元素需要用户额外花时间**认知和学习**，可能影响耐心和信心
- **减少认知负荷** → 提升用户满意度 → 增加转化率

**典型示例：**
- 导航栏放在页面顶部 → 用户的普遍认知
- Hamburger 菜单图标 → 通用语言，点击可看更多信息
- 股票交易中红绿代表涨跌 → 通用语言，不应随意替换

**总结：** 除非有足够的理由，否则不要发明用户不熟悉的元素。

---

## 技巧五：保持设计简洁

**核心原则：** UI 设计始终以**用户体验和功能性**为主。

**文字层面：**
- 用最简洁的语言说明问题，做到信息明确
- 但也不要低估用户的理解能力和学习能力

**视觉图形层面：**
- 图标、插画、图片、视频——更多的是**辅助内容说明**
- 装饰性元素不要过多
- **少即是多**

**动画层面：**
- 合理使用动画/交互可辅助文字说明、增加趣味性、提升交互体验
- 但要谨慎，需考虑：
  - 制作和**开发成本**
  - 是否影响**页面加载时间**
  - 能否在**不同尺寸**下流畅显示
- 如果不能确保动画的质量和目的，**就让页面保持简洁**

---

## 技巧六：珍惜每一寸空间

> 在快节奏和缺少耐心的时代，要珍惜页面上的每一点空间，以留住客户。

- 任何元素如果**没有实际价值和作用，绝不添加**
- 每一个留下来的元素都应该有它存在的理由

## 总结速览

| # | 技巧 | 一句话概括 |
|---|------|-----------|
| 1 | 善用高级灰 | 灰色系统打底，一个主色点缀，克制用色 |
| 2 | 字体重易读性 | 字号不小、对比度达标、符合无障碍标准 |
| 3 | 间距与留白 | 8 点网格为基础，留白即高级，用间距表达信息层级 |
| 4 | 用已有设计模式 | 不为创新而创新，降低用户认知负荷 |
| 5 | 保持简洁 | 少即是多，文字、视觉、动画都要克制 |
| 6 | 珍惜空间 | 没有存在理由的元素一律移除 |
