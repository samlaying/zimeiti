# 为 Memos 打造的移动端 — MemoFlow

> [秋刀橘咯](https://space.bilibili.com/370787927) | [BV1Zuk3BPECM](https://www.bilibili.com/video/BV1Zuk3BPECM) | 时长 2:13

## MemoFlow：为自托管笔记系统 Memos 打造的移动端应用

### 一、 应用概述与核心价值
*   **定位**：一个专为自托管笔记系统 **Memos** 设计的移动端客户端。
*   **解决的核心痛点**：许多笔记应用数据不存储在用户自己手中，缺乏安全感。MemoFlow 允许用户连接到自己搭建的 Memos 服务器，真正实现**数据自主**。
*   **目标用户**：喜欢折腾自托管服务，或单纯想拥有一个安静、无干扰的碎碎念空间的记录者。

### 二、 功能详解
#### 1. 界面与设计
*   **界面风格**：**非常干净**，无广告和复杂按钮，专注于内容本身。
*   **笔记展示**：记录的生活碎片以**小卡片**形式呈现，视觉清晰。
*   **个性化设置**：支持**自选语言**和**调整外观**。

#### 2. 核心功能
*   **连接服务器**：只需输入自建的 Memos 服务器地址和 **Token** 即可登录。
*   **Markdown 支持**：对 Markdown 的支持**相对丰富**，包括列表、加粗、代码块等，排版清晰，阅读舒适。
*   **随机漫步**：一个特色功能，可随机浏览以往的笔记，回顾“胡思乱想”，带来趣味体验。
*   **强大筛选器**：解决“记了找不到”的问题。可按**时间段**或**标签**快速筛选出目标笔记。
*   **热力图**：**最具激励性的功能**。通过填满每日记录的小格子，提供持续记录的成就感，帮助用户养成记录习惯。

#### 3. 输入与创作体验
*   **流畅输入**：输入体验流畅。
*   **快捷工具栏**：键盘上方提供一排小工具，可一键插入：
    *   **Checkbox**：用于创建待办清单。
    *   **图片**：无需记忆复杂 Markdown 语法。
*   **语音输入**：在走路、开车等不便打字的场景下，可**长按录音**，记录音频和文字，回听时能保留**当时的情绪**。

#### 4. 安全与隐私
*   **应用锁**：可为应用额外设置**保护密码**，为私密笔记提供多一层安全屏障，让使用更安心。

### 三、 使用场景与体验
*   **记录内容**：主要用于记录生活碎片、胡思乱想、学习笔记等。
*   **回顾方式**：通过“随机漫步”或精准筛选进行回顾。
*   **激励方式**：通过“热力图”的视觉反馈获得成就感。
*   **适用环境**：适用于任何想安静记录的场景，尤其语音功能方便了移动中记录。

### 四、 总结
MemoFlow 是一个设计简洁、体验流畅的移动端笔记应用，其核心价值在于为自建的 Memos 服务器提供了优秀的移动端入口。它通过干净的卡片界面、丰富的 Markdown 支持、便捷的输入工具、强大的筛选功能以及激励人心的热力图，解决了自托管用户“数据自主”与“便捷使用”的双重需求，让用户能更纯粹、更高效、更安心地坚持记录。
