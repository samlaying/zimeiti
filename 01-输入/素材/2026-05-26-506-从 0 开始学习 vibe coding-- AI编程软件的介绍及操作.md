# 从 0 开始学习 vibe coding-- AI编程软件的介绍及操作

> [杰瑞Jerry学AI](https://space.bilibili.com/3706932525271542) | [BV1snZ3BDEeS](https://www.bilibili.com/video/BV1snZ3BDEeS) | 时长 5:17

## 一、Vibe Coding 概述与学习准备
### 1.1 什么是 Vibe Coding
- **核心概念**：通过与AI编程助手（Agent）对话，让AI生成、修改和解释代码，从而快速完成Web应用开发的学习与实践方式。
- **学习资源**：UP主的学习参考了GitHub上的教程 **WebVive**，建议访问其官网获取同步学习内容。

### 1.2 如何选择AI编程工具
- **当前主流工具**：Glockode、Codex、Cursor、Tree等，种类繁多。
- **选择建议**：
    - **核心原则**：**不用纠结**，选择最容易获取的工具即可。
    - **原因**：对于初学者，几乎任何AI编程工具都能满足基础学习需求。
- **UP主示例**：
    - **工具**：选择 **Tree**。
    - **关键数据**：首月订阅费用仅 **3美元**，可使用 **Gimni3 Pro** 大模型，性价比高。
    - **备注**：工具可能涨价，UP主表示若涨价会考虑更换其他工具。

## 二、AI编程IDE（集成开发环境）基础操作
### 2.1 界面核心区域介绍
创建一个新项目后，界面通常包含以下三个核心区域：
1.  **文件区域（最左侧）**：显示项目中的所有文件结构。
2.  **代码展示区（中间）**：显示当前编辑或AI生成的代码。
3.  **Agent对话区（右侧）**：与AI助手进行自然语言交互，下达指令（即“教”AI完成编程）。

### 2.2 实操步骤：创建第一个项目
1.  **新建项目**：打开AI编程软件，选择新建项目。
2.  **命名项目**：以最经典的“待办事项（To-Do List）APP”为例，将项目命名为 `to-do list`。
3.  **开始对话**：在Agent对话区输入你的第一个指令，例如：“帮我做一个来办事项的APP，用HTML实现”。
4.  **AI生成与预览**：
    - AI会在代码区**自动生成代码**。
    - 通常会提供一个可本地预览的**App展示页面**。
    - AI会告知任务完成情况，并提供功能说明。

## 三、实践演示：快速生成待办事项APP
### 3.1 AI生成过程演示
- **输入指令**：`帮我做一个来办事项的APP，用HTML实现`。
- **AI响应**：基于海量数据学习，AI能为这类常见项目快速生成**功能全面**的代码，项目几乎开箱即用。
- **演示功能**：
    - **添加任务**：在应用中输入“学习webcoding”，成功添加。
    - **完成与删除**：任务前打勾标记为完成，点击斜杠可删除任务。
- **结论**：对于高度模板化的项目（如待办事项），AI只需一句话指令即可生成功能完善的APP。

### 3.2 学习意义与进阶方向
- **当前演示的价值**：让初学者**快速熟悉AI编程软件的使用流程和交互方式**。
- **未来学习重点**：当需要开发**个性化应用**时，AI不会自动理解全部需求。因此后续学习将从以下三个维度展开：
    1.  **前端**：用户界面与交互逻辑。
    2.  **后端**：服务器端业务逻辑与API。
    3.  **存储**：数据持久化方案。
- **视频目标**：本视频旨在**熟悉工具**，后续视频将逐步详解如何一步步完善一个完整应用。

## 四、总结
本次学习介绍了Vibe Coding这一新兴AI辅助编程范式，强调了入门时**无需纠结工具选择**，应优先利用手头最易得的资源（如Tree的首月优惠）。通过一个“待办事项APP”的快速生成演示，展示了AI编程IDE的**核心界面（文件、代码、对话三区）** 与**基础对话生成代码的流程**。关键点在于，虽然AI能轻松完成通用项目，但真正的学习目标是理解**前端、后端、存储**的完整开发逻辑，为后续打造个性化应用打下基础。本视频为整个系列教程的操作入门篇。
