# Obsidian Excalidraw 脚本重构升级：按功能划分、多语言、UI、描述 -- 以思维导图功能为例演示

> [一鸣惊人_001](https://space.bilibili.com/39231346) | [BV1WjRRBaEJE](https://www.bilibili.com/video/BV1WjRRBaEJE) | 时长 11:14

## Obsidian Excalidraw 插件脚本重构与思维导图功能详解

### 一、 重构概述与准备
- **核心变化**：脚本架构从**以代码为中心**转变为**以功能为中心**。
  - **好处1（功能解耦）**：用户可按需安装单个功能脚本，避免了以往全耦合、大而全的安装方式。
  - **好处2（底层稳定）**：代码基础更加稳定。
- **使用前提**：您的 Excalidraw 插件版本需要 **≥ 2.220-4**。
- **分发与语言**：
  - 脚本在 **GitHub** 和 **Gitee** 同步更新（方便国内用户）。
  - 默认脚本市场链接已指向 Gitee。
  - **全面支持多国语言**：默认中英双语，自动跟随 Obsidian 界面语言切换。

### 二、 脚本市场与管理
- **界面革新**：脚本市场界面焕然一新，开头的表格按功能将脚本分门别类。
- **浏览与搜索**：
  - 可点击分类跳转，也可在搜索框直接输入（如 “MindMap”）进行快速定位。
- **信息展示**：点击任一脚本，下方会显示其**详细描述**和**安装按钮**。这些描述信息由系统自动从脚本的 `FrontMatter` 中抓取。
- **自动运行**：脚本包含 `autorun` 属性，意味着**第一次打开画布后会自动在后台静默运行**，无需手动干预。但初次安装后，画布已打开状态下需**手动运行一次 Engine 脚本**进行激活。

### 三、 思维导图核心操作
#### 3.1 创建思维导图
1.  在画布中绘制一个矩形并选中。
2.  运行 `Action Create Mind Map` 脚本。
3.  在弹出的可视化UI界面中配置：
    - **主题配色**：如彩虹色、莫兰迪色等。
    - **排版方向**：向右、向左、向下展开。
    - **连线样式**：平滑曲线（Smooth）、折线箭头（Elbow）、S型曲线（Curve）。
    - **其他**：箭头固定点、节点间距等。
    - （注意：UI语言随Obsidian设置变化，支持中文）
4.  采用默认设置，点击确认，即可将所选矩形生成为思维导图根节点。

#### 3.2 节点编辑与快捷键
- **创建子节点**：选中一个节点，按 `Tab` 键。
- **创建兄弟节点**：选中一个节点，按 `Enter` 键。
- **删除节点**：选中叶子节点，按 `Delete` 键，连接它的箭头也会被自动清理。
- **“万物皆可为节点”**：可直接将**文字、数学公式、图片、包含文字的容器、复杂组合图形**等拖拽到节点的**右侧**（成为子节点）或**下方**（成为兄弟节点），即可自动吸附连接。
- **修改层级**：直接拖拽现有节点，放到其他节点的右侧或下侧，即可自由改变其位置和层级关系。

#### 3.3 节点折叠与展开
- **折叠/展开手柄**：包含子元素的节点右侧有一个**橙色交互手柄**。
  - 显示 **`-` 号**：表示子节点已展开。
  - 显示**数字**：表示折叠状态下包含的子节点数量。
- **操作**：
  - 点击 `-` 号可折叠。
  - 点击 `+` 号可逐级展开。
  - **按住 `Ctrl` 键再点击 `+` 号**，可一次性展开所有嵌套层级。

### 四、 高级节点操作
#### 4.1 脱离与重组
- **脱离节点**：选中不想放入脑图的元素，运行 `Action Detach Mind Map` 脚本，该元素即从脑图结构中脱离，连线自动删除，可作为普通元素移动或删除。
- **重组逻辑**：通过拖拽节点，可随时重组思维导图的结构。

#### 4.2 嵌入可交互文档
- Excalidraw 的独特优势：可以直接将 Obsidian 中的**可交互 Markdown 文档（Eframe）拖入脑图**，使其成为脑图的一个分支，并且可以编辑。

### 五、 样式定制与导出导入
#### 5.1 全局样式修改
- 运行 `Action Change Mind Map Dell` 脚本，可快速切换：
  - **配色方案**：如从彩虹色切换为莫兰迪色。
  - **连线样式**：平滑曲线、折线箭头、S型曲线。
  - **生长方向**：向右、向左、向下等。
- **自由定制**：在 Excalidraw 内，可以框选所有节点，利用其**自带的属性面板**任意更改字体、线条风格（如手绘粗糙风格）。

#### 5.2 与Markdown的互转
- **导出到Markdown文件**：
  1.  选中脑图，运行 `Action Export Mind Map to MD`。
  2.  系统弹窗询问是否用根节点做文件名（可选“否”自定义）。
  3.  导出后，文件结构为：**根节点变为二级标题，子节点变为无序列表**。
- **从剪贴板导入**：
  1.  复制一段包含层级关系的列表文本。
  2.  运行 `Action Import Mind Map from Clipboard` 脚本。
  3.  在弹出的配置UI中确认，即可生成思维导图。
- **导出到剪贴板**：运行 `Action Export Mind Map to Clipboard`，可将脑图内容以Markdown格式写入系统剪贴板，便于粘贴到其他应用。

### 总结
本次更新是对 Excalidraw 插件脚本系统的**重大重构**，核心是将其转变为**以功能为中心、可独立安装**的模式，提升了灵活性和稳定性。同时引入了**多语言支持**。以**思维导图**功能为例，新版脚本提供了从创建、编辑（支持多种元素作为节点、折叠展开）、重组、样式一键切换到与Markdown文件**双向互导**的完整、流畅体验，充分发挥了 Excalidraw “万物皆可绘”的优势，显著提升了在 Obsidian 中进行可视化思考和知识管理的效率。
