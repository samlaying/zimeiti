# 挑战不可能,如何把4000+份的PPT变身成个人独享知识库

> [绿皮火车Roadtrip](https://space.bilibili.com/565619114) | [BV15gFTzfEcL](https://www.bilibili.com/video/BV15gFTzfEcL) | 时长 17:30

## 挑战不可能：将海量PPT变身个人知识库
视频展示了如何利用 **Claude Code** 和三个免费的 **Skills**（技能/插件），以零代码、会说人话的方式，高效完成三项“不可能任务”：为Mac释放超过120G磁盘空间、整理上千个混乱文件、并将数年积累的PPT/PDF转化为可随时检索的个人知识库。

## 三个核心Skills概览
视频演示了三个核心技能，它们像给AI助手加载的“知识胶囊”，让它能解决具体问题。
1.  **磁盘瘦身**：自动安全清理系统缓存、垃圾文件。
2.  **文件收纳**：智能分类和整理下载文件夹等杂乱目录。
3.  **文档提炼**：批量处理PPT/PDF，提取摘要并构建知识库。

## 详细操作与效果
### 1. 磁盘瘦身：清理数字垃圾，释放100G+空间
- **问题**：磁盘空间严重不足。
- **Skill功能**：自动扫描可安全清理的缓存（应用缓存、浏览器缓存、系统日志等），并提供清理方案和白名单保护。
- **实操步骤与数据**：
    1.  告知Claude磁盘空间不足，触发“磁盘清理”技能。
    2.  扫描后生成报告，显示可清理项（如：用户应用缓存近80G，浏览器缓存37G，系统日志22G）。
    3.  提供5个选项：**清理方案**、**生成详细CSV清单**、**配置白名单**。用户选择先生成清单查看，再配置“办公”白名单模式以保护文档和桌面文件夹。
    4.  确认清理后，完成清理。
- **成果**：磁盘可用空间从 **31G** 激增至 **137G**，安全释放 **106G** 空间。相当于节省了一次Mac系统升级的费用。

### 2. 文件收纳：整理下载文件夹的1600+个文件
- **问题**：下载文件夹极度混乱，有1600多个文件。
- **Skill功能**：分析文件分布，通过创建“智能文件夹”视图进行预览和筛选，并自动清理指定类型的冗余文件。
- **实操步骤**：
    1.  扫描下载文件夹，分析出文件类型和数量（办公文档、图片、音视频等）。
    2.  提供“手动”和“自动”两种整理模式。选择手动模式后，在桌面创建智能文件夹供预览和备份重要文件。
    3.  确认备份后，选择自动清除指定类型文件（如音视频）。
- **成果**：自动删除了171个音视频文件，又释放了 **11G** 空间。结合磁盘清理，总计释放 **约120G** 空间。

### 3. 文档提炼：将PPT/PDF变为结构化知识库
- **问题**：积累了4000多份PPT和14000多份PDF，因工作量巨大无法整理。
- **Skill功能**：批量转换文档格式、用本地AI生成摘要、并从多个维度自动分类归档。
- **实操步骤（以30份PPT为例）**：
    1.  扫描桌面PPT文件夹，发现并删除3组重复文件。
    2.  **批量转换**：自动将PPT转换为纯文本（Markdown格式）。
    3.  **本地生成摘要**：使用本地运行的AI模型（如通过Ollama）为每份文档生成摘要，全程文件不上传云端，保护隐私。
    4.  **智能分类**：从 **主题、用途、客户** 三个维度分析每份文档内容，并进行自动分类（如按主题分为产品管理、技术改进、数据治理等12类；按用途分为内部参考、培训材料、市场营销等）。
    5.  **优化文件名**：AI建议更具描述性的文件名。
    6.  **生成总览**：汇总所有摘要，生成一份思维导图。
- **成果**：在桌面生成了按“主题”、“用途”、“客户”分类的文件夹体系，并附有分类方案。所有文档有了AI摘要，实现了秒级检索。一个**易用、免装特殊软件、不过时**（普通文件夹+纯文本）的个人知识库就此建成。

## Skills背后的工作原理
Skills本身是预设好的指令集，它包装并调用了已有的强大开源工具，让普通人能通过对话使用。
- **磁盘清理**：核心调用了开源项目 [**Mall**](https://github.com)（MacOS清理工具），将命令行操作转化为对话。
- **文件整理**：利用 **macOS 自带的智能文件夹、文件元数据和 Spotlight 搜索**能力，通过脚本自动化分析与创建。
- **文档提炼**：
    - **格式转换**：使用开源工具（如微软的Markdown转换器）。
    - **本地摘要**：利用 **Ollama** 在本地运行一个中小型大模型，保障数据隐私。
    - **智能分类**：基于生成的摘要，由Claude进行理解和分类。

## Claude Code vs 网页版Claude
视频强调，**Claude Code**（安装在电脑上的版本）具备网页版无法比拟的优势：
1.  **具有上下文和动手能力**：能直接操作你电脑上的文件（读、写、删），而网页版只能提供建议。
2.  **可扩展性**：通过安装Skills，如同给系统打“技能补丁”，让AI助手能力无限增强。安装越多，能力越强。
3.  **与开源生态结合**：Claude Code + Skills 的模式，与更前沿的自动操控电脑的AI项目（如Open Interpreter）路线一致，是更可控的实践。
