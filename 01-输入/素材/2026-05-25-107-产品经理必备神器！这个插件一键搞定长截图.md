# 产品经理必备神器！这个插件一键搞定长截图

> [AI产品经理养成记](https://space.bilibili.com/47333744) | [BV11eQAB7Emr](https://www.bilibili.com/video/BV11eQAB7Emr) | 时长 1:56

## 插件简介
- **名称**：Awesome Screenshot（字幕中亦提及为Awesome Scream Shot，可能为口误，实际指同一插件）。
- **核心功能**：一款浏览器插件，专注于长截图和录屏，可一键捕获完整网页内容，提升工作效率。
- **适用场景**：特别适合产品经理、文档撰写者等需要处理长页面的情况，例如撰写产品帮助文档时截取整个页面。

## 获取与安装
1. 打开谷歌应用商店（Google Chrome Web Store）。
2. 在搜索框中输入“Awesome Screenshot”。
3. 找到插件并点击安装。
4. 安装完成后，浏览器右侧工具栏会出现该插件的图标，方便快速访问。

## 功能详解
### 截图功能
- **模式选择**：
  - **可视区域**：截取当前浏览器窗口中可见的页面部分。
  - **区域**：手动选择网页中的特定区域进行截图。
  - **整个页面**：自动截取网页的完整长图，适用于长内容捕获。
- **操作步骤**：
  1. 点击浏览器工具栏中的插件图标。
  2. 在弹出菜单中选择“截图”选项。
  3. 选择所需模式（如“整个页面”以截取长图）。
  4. 插件自动执行截图，完成后提供输出选项。
- **输出选项**：
  - 下载截图文件到本地。
  - 复制截图链接，可直接粘贴到文档中。
  - 将截图转换为PDF格式，便于分享和存档。

### 录屏功能
- **录制类型**：
  - **桌面录制**：录制整个电脑桌面活动。
  - **摄像头录制**：录制摄像头视频，适用于演示或讲解。
  - **页签录制**：录制当前浏览器页签的内容，聚焦于网页操作。
- **操作步骤**：
  1. 点击插件图标，选择“录屏”功能。
  2. 根据需要选择录制类型（如桌面、摄像头或页签）。
  3. 点击“开始录制”按钮。
  4. 系统会提示登录账户；若不登录，录屏功能无法使用。
- **登录后额度**：登录后提供20次免费录屏机会，超出后需考虑升级或付费。

## 免费额度与版本
- **不登录状态**：截图功能可免费无限次使用，无需账户登录。
- **登录后状态**：
  - 免费额度：包含20次录屏和100次截屏。
  - 额度用尽后，可能需要使用专业版（字幕未详细说明专业版内容，建议根据需求查看插件详情）。
- **使用建议**：
  - 对于偶尔需要截长图的用户，不登录即可免费使用截图功能。
  - 对于需要录屏或频繁截图的用户，建议登录以利用免费额度，并考虑专业版以满足更高需求。

## 总结核心要点
Awesome Screenshot是一款高效的浏览器插件，专为产品经理和文档工作者设计，通过简单操作实现长截图和录屏功能，支持多种输出格式并提供免费额度，能显著提升内容捕获和分享的效率，是日常工作中的实用工具。
