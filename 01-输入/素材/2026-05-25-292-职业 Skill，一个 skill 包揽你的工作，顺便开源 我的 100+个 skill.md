# 职业 Skill，一个 skill 包揽你的工作，顺便开源 我的 100+个 skill

> [空格的键盘](https://space.bilibili.com/14762998) | [BV1BADPBeEoZ](https://www.bilibili.com/video/BV1BADPBeEoZ) | 时长 2:55

## 职业 Skill：用 AI 一键生成你的专属工作技能包

### 1. 核心理念：职业点 Skill
UP主提出了一种基于“职业”自动生成 Skill 的新方法。用户只需在 AI 对话中（如 Cloud Code）输入“使用职业 skill 我是 [职业名]”，AI 就能自动为你创建一系列与该职业高度相关的实用 Skill。
*   **示例**：输入“使用职业 skill 我是产品经理”，AI 会自动生成如下的 Skill 制作清单：
    *   需求文档撰写
    *   多角色评审需求
    *   优先级排序
    *   竞品分析
    *   ...等

### 2. 开源成果与实用 Skill 包
UP主已将自己的职业 Skill 以及其他常用 Skill 开源在 GitHub 仓库中，供所有人免费使用。
*   **产品经理必备 13 个 Skill 包**：
    1.  需求文档编写
    2.  需求评审团
    3.  需求优先级划分
    4.  需求路线图撰写
    5.  需求数据分析
    6.  **原型图绘制（三种方法，强烈推荐）**：
        *   方法一：发送截图 → 输出可运行的 HTML 原型。
        *   方法二：发送截图 → 输出 Pencil 设计稿（特别推荐“Image to Pencil”插件，效果很好）。
        *   方法三：发送 URL → 输出一个 Next.js 的本地项目。
*   **其他开源 Skill**：
    *   **个人信息汇总 Skill**：用于总结 RSS 信息流、播客内容，并能提取播客音频为文字摘要。
    *   **绘图 Skill 包**：包含根据文章输出插图的 Skill，以及宝玉老师的绘图 Skill。

### 3. 四种制作 Skill 的方法
除了“职业点 Skill”，UP主回顾并分享了之前总结的四种制作 Skill 的经典方法：

#### 3.1 样板法
*   **定义**：将你最满意的交付物（如需求文档、技术文档、运营方案）提供给 AI。
*   **步骤**：让 AI 学习并提炼该交付物的风格、结构和核心要素，从而制作成可复用的 Skill。

#### 3.2 流程法
*   **定义**：清晰地写下你的工作步骤。
*   **步骤**：将步骤中可以串联、自动化的部分提炼出来，制作成 Skill。

#### 3.3 归纳法
*   **定义**：回顾你和 AI 的聊天记录。
*   **步骤**：导出记录，找出你反复让 AI 做的同类型任务（例如总是让 AI 帮文章起标题），然后将这些高频任务补全为正式的 Skill。

#### 3.4 经验法
*   **定义**：将隐性知识（如你个人的设计方法、写文档的技巧）显性化。
*   **步骤**：静心思考并总结这些方法，写成文档。也可以去网上搜索通用的优秀方法，然后将它们封装成 Skill。

**关于 Skill 的本质**：一个 Skill 本质上是“一个提示词加上一个脚本”的文件夹。

## 总结
本文介绍了一种基于职业自动生成相关 Skill 的创新方法，并分享了 UP主开源的大量实用 Skill 资源（重点推荐产品经理的 13 个技能包和多种原型图生成方法）。同时，系统性地回顾了制作 Skill 的四种核心方法：样板法、流程法、归纳法和经验法，帮助用户不仅能直接使用现成 Skill，还能掌握自己创造 Skill 的能力，从而大幅提升工作效率。
