# AI Agent 操作浏览器：Playwright-CLI 自动抓取 B站评论实战

> [码力全开666](https://space.bilibili.com/702042127) | [BV1AS5i62EwM](https://www.bilibili.com/video/BV1AS5i62EwM) | 时长 8:53

## 一、 工具介绍与工作原理
### 1. 什么是 Playwright-CLI？
- **定义**：一个**命令行工具**，用于让 AI Agent（如视频中提到的 “Cloud”）自动控制和操作浏览器。
- **核心功能**：通过与 AI 交互，驱动浏览器执行自动化任务，如页面导航、点击、填写表单、滚动、截图以及**提取网页信息**。
- **支持浏览器**：谷歌浏览器（Chrome）、火狐浏览器（Firefox）、Edge 浏览器。

### 2. 完整工作流程
1.  **安装工具**：在电脑上安装 Playwright-CLI 命令行工具。
2.  **人机交互**：用户向 AI Agent（如 Cloud Code）描述任务需求。
3.  **指令生成与执行**：AI 会阅读 CLI 命令，生成操作指令，自动驱动浏览器执行一系列操作（如打开页面、滚动、点击）。
4.  **数据提取**：AI 识别并抓取浏览器页面中的目标信息（如评论）。
5.  **流程固化**：完成一次成功操作后，AI 可以将整个**操作流程总结成一个 “Skill”**（可复用的操作模板或脚本）。
6.  **重复执行**：使用总结好的 Skill，可以对其他同类页面进行重复的自动化操作。

## 二、 安装前提与方法
### 1. 前置条件
- 需要安装 **Node.js 18** 环境。
- 安装教程可参考 UP 主的上一期关于 Cloud Code 安装的视频。

### 2. 安装命令
- 通过 **npm** 进行安装。
- **具体命令**：`npm init playwright-cli`（根据视频内容推断的典型安装流程，用户需根据实际情况与AI交互执行安装）。

## 三、 功能详解与操作概念
### 1. 工具核心能力（无需人工记忆，由 AI 调度）
- **页面导航**：前进、后退、跳转到指定 URL。
- **页面交互**：点击元素、填写输入框、模拟键盘按键（如回车）。
- **页面控制**：滚动页面（用于触发懒加载）、截图。
- **数据获取**：解析页面 DOM 结构，提取所需文本、链接等信息。

## 四、 实战演示：自动抓取 B站视频评论
### 1. 准备工作
- 找到一个目标 B站视频页面（示例：与“中美关系”相关的视频）。
- 与 AI 交互，告知需要爬取该视频的评论信息。

### 2. 详细操作步骤
1.  **启动浏览器**：AI 会首先执行命令，打开一个**有头浏览器**（即可见的浏览器窗口）。
2.  **处理登录**：
    - 新打开的浏览器通常**未登录**。
    - 用户需要**手动扫码登录**（使用 B站 App）。
    - 登录成功后，告知 AI “已登录”。
3.  **自动滚动加载**：
    - B站评论区采用**懒加载**机制，需要滚动才会加载更多。
    - AI 会自动驱动浏览器**向上滑动再向下滑动**，重复两三次，以模拟人工滚动，确保评论全部加载。
    - 示例中成功加载了 **61 条**评论。
4.  **展开所有回复**：
    - 评论下的回复默认是折叠的（显示“共 X 条回复”）。
    - AI 会自动点击所有“展开回复”的按钮，以获取完整的讨论内容。
5.  **提取数据**：
    - AI 解析页面，**提取所有评论和回复的文本内容**。
    - 最终将抓取到的数据整理并呈现。

### 3. 注意事项
- 即使有了预设的 Skill 流程，在实际运行中**仍可能需要调试和交互**，以应对页面结构变化等情况。
- 该工具是一个持续发展中的自动化方案，其稳定性会逐步提高。

## 五、 工具价值与总结
- **降低技术门槛**：特别适合**不擅长编写传统爬虫代码（如 Python、Pash）**，但有数据抓取需求的用户。
- **交互式开发**：用户只需通过自然语言描述“看到什么页面”、“需要点击哪里”、“要抓取什么数据”，AI 即可理解并执行。
- **效率提升**：AI 可以自动将一次成功的交互**总结成可复用的脚本文件（Skill）**，实现批量、重复操作，极大提升效率。
- **应用场景广泛**：除了示例中的 B站评论，还可应用于抓取知乎回答、电商商品信息等各类网站数据。

**核心要点总结**：
本视频介绍了一种利用 AI Agent 驱动 Playwright-CLI 工具实现浏览器自动化与数据抓取的创新方案。其核心优势在于用户无需编写代码，仅通过与 AI 进行自然语言交互，就能完成包括登录、滚动、点击、解析在内的复杂网页操作，并成功将操作流程固化为可重复执行的“Skill”。该工具为非技术人员提供了一个强大、高效的数据获取途径，显著降低了自动化操作与数据抓取的技术门槛。
