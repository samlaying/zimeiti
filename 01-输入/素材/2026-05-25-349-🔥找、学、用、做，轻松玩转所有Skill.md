# 🔥找、学、用、做，轻松玩转所有Skill

> [AI产品经理养成记](https://space.bilibili.com/47333744) | [BV1mSw3zvEYR](https://www.bilibili.com/video/BV1mSw3zvEYR) | 时长 13:35

## 找、学、用、做，轻松玩转所有Skill

本文档基于视频内容，系统梳理了如何高效查找、学习、应用和创建AI编程工具中的Skill（技能/插件），旨在帮助产品经理及开发者快速上手。

### 一、找Skill
寻找合适的Skill是第一步，主要推荐以下两个网站和一个技能：
1. **SkillsMP**：一个收录Skill非常全面的网站。
    - **使用方法**：在搜索框输入关键词（如“产品经理”），即可查看相关Skill列表。
    - **信息详情**：点击某个Skill可查看其介绍、安装命令、下载压缩包或跳转到GitHub仓库。
2. **Skills.sh**：一个包含热榜、下载榜和24小时趋势的Skill发现平台。
    - **特色功能**：可根据下载量排序、查看Skill的流行趋势，帮助发现热门技能。
3. **FindSkill技能**（下载量第一的技能）：
    - **用途**：直接通过自然语言让AI帮你寻找所需的Skill。
    - **示例**：告诉它“我要写一个商业分析报告”，它能为你推荐相关Skill。

### 二、学Skill
找到Skill后，高效学习是关键，推荐两种方法：
1. **借助DeepWiki网站学习**：
    - **步骤**：将Skill的GitHub链接粘贴到DeepWiki，可查看其介绍、安装设置、入门指南，并支持持续对话追问。
2. **拆解提示词，借助AI编程工具学习**：
    - **步骤**：将Skill的提示词下载下来，导入如Cursor、Tree等AI编程工具。
    - **操作示例**：在工具中先让AI介绍项目，再要求其将`skills`文件夹内容翻译成中文并保存，通过对话逐步深入理解。
    - **进阶——固化学习流程**：多次学习后，可将分析Skill的流程固化成自己的**Skill Creator**技能，实现自动化翻译和拆解。

### 三、用Skill
安装和使用Skill的步骤通常大同小异，以在**Cloud Code**中为例：
1. **安装Skill**：
    - **方法一**：使用`mix`命令安装。
    - **方法二**：用自然语言指令，如“请帮我安装XX文件夹下的技能”。
    - **方法三**：将Skill文件夹复制到`Cloud Code Skill`目录下。
    - **方法四**：上传Skill压缩包（支持在Tree等工具中）。
2. **使用Skill**：
    - **步骤**：
        1.  在终端中进入项目路径（`CD`到项目路径）。
        2.  启动Cloud Code（如希望减少确认操作，可在命令后添加特定参数）。
        3.  执行Skill：输入`/技能名`（例如`/SkillTranslator`），即可运行该技能完成特定任务。

### 四、做Skill
使用**Skill Creator**技能可以创建自己的Skill。
1. **创建流程**：
    - **描述需求**：清晰说明Skill的输入、输出、处理过程。
    - **提供参考**：提供格式、示例文件等要求。
    - **生成与优化**：先在Cloud Code中生成初稿，再用其他AI编程工具（如Cursor、Tree）进行优化。可要求AI分析技能功能并提出优化建议。
    - **测试与调试**：在Cloud Code中通过**软链接**方式安装开发中的Skill，方便在项目路径下直接进行测试。
2. **Skill输出文档结构示例**：
    - 生成一个以`Skill名.z`命名的文件夹。
    - 包含**整体技能介绍**、技能总数、定义场景、如何使用。
    - 提供**应用演示**：清晰展示用户输入示例和模型预期输出，便于理解用法。

### 五、推荐Skill示例
- **Anthopic官方技能**：
    - `Skill Creator`：用于构建其他技能。
    - `Fronted Design`：用于前端UI设计，减少AI生成的“AI味”。
- **`FindSkill`**：帮助发现新技能。
- **`Knowledge Worker Plugin`**：通过封装专业角色、流程、工具来提升效率。
- **`Enhanced Prompt`**（Stitch领域）：将模糊的UI想法转化为详细丰富的提示词，也适用于优化原型设计输入。
- **`Product Manager Skill`**：产品经理专用技能包（含46个技能），涵盖组件、交互、工作流等分类。
- **`UX Pro Max`**：去除页面AI生成痕迹。
- **`Notebook LM Skill`**：查询Notebook LM笔记，获取记忆源内容。
- **`Superpower`**：基于Cloud Code的开发指导技能，包含头脑风暴、制定计划、执行计划等子技能。

### 总结核心要点
掌握AI编程工具中的Skill能极大提升工作效率。核心方法论是 **“找、学、用、做”四步法**：通过特定网站（如SkillsMP, Skills.sh）或技能（FindSkill）高效**找**到所需Skill；利用DeepWiki或AI编程工具对话拆解来快速**学**习；在IDE（如Cloud Code）中通过标准化命令安装和**用**；最终借助Skill Creator将自身流程固化，**做**出专属技能。善用官方推荐及社区热门技能，可以快速构建起强大的个人AI工具箱。
