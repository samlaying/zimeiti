# Obsidian插件-公众号、视频封面AI生成器-免费分享

## 一、插件概览
### 1.1 核心功能
- **根据文章内容自动生成**美观的封面图。
- 支持生成**微信公众号、B站、小红书等多平台**的封面。
- 可**一次性生成多张不同风格**的封面供选择。
- 生成的封面为**HTML代码**，所有内容和样式均可编辑。

### 1.2 支持的风格
- 插件**内置多种预设风格**，如“快按下注方”、“极客工具方”、“复古学生处报告方法”等。
- 支持在设置页面**自定义添加风格**（本质是优化后的AI提示词）。
- 建议**参考优秀封面截图**，让豆包等视觉大模型帮你生成对应的风格提示词。

### 1.3 推荐模型
- **首推 DeepSeek V4 Flash 模型**，生成速度快，效果好。
- 也可使用 DeepSeek Pro 等更复杂模型，效果可能更好，但生成时间更长。
- **平均生成一张封面约需 2-3 分钟**，复杂模型耗时更久。

## 二、安装与配置
### 2.1 安装步骤（需借助 BRAT 插件）
1.  在 Obsidian **设置** 中，进入 **第三方插件** 选项。
2.  **关闭安全模式**。
3.  先安装 **BRAT 插件**（该插件用于从GitHub安装Beta版插件）。
4.  安装并启用 BRAT 后，点击左侧新增的 **BRAT 图标（笑脸）**。
5.  点击 **Add Plugin** 按钮。
6.  **粘贴** AirPostMaker 插件的 GitHub 仓库地址（地址见评论区置顶或简介）。
7.  选择版本（建议选 latest 最新版），点击 **Add Plugin**。
8.  等待安装完成，左侧出现 **AirPostMaker 图标** 即表示安装成功。

### 2.2 核心配置
1.  进入 Obsidian **设置** -> **第三方插件** -> 找到 **AirPost** 并点击设置。
2.  **添加 API**：建议前往 DeepSeek 官网获取 **V4 Flash 模型**的 API Key 并填入。
3.  **平台管理**：可添加或修改生成封面的目标平台（如微信公众号、B站、小红书），并设置对应的封面尺寸。
4.  **风格管理**：查看、添加或删除预设风格。自定义风格需按格式编写提示词。

## 三、使用流程
### 3.1 生成封面
1.  在 Obsidian 中打开一篇写好的文章。
2.  点击左侧 **AirPostMaker 图标** 进入插件界面。
3.  **选择 API 模型**（如 DeepSeek V4 Flash）。
4.  **选择目标平台**（如微信公众号）。
5.  **选择风格**（可多选），例如“快按下注方”和“极客工具方”。
6.  点击 **“导入当前文章”**。
7.  点击 **“生成封面”**，开始等待。生成过程中可进行其他操作。

### 3.2 编辑与修改
- **修改文字**：直接在生成的封面预览界面中**点击文字进行修改**，然后点击“应用”。
- **精细修复**：点击封面下方的 **“请求修改”按钮**，输入具体指令（如“修复第三个标签的遮挡问题”），AI将重新生成。
- **编辑代码**：点击 **“编辑”** 按钮，可直接修改生成的 **HTML/CSS 代码**，实现更自由的样式调整。

### 3.3 导出与使用
- 对生成的封面满意后，点击 **“复制”** 按钮。
- 将复制的 HTML 代码**粘贴到微信公众号编辑器**中，即可作为文章封面使用。

## 四、进阶功能与搭配建议
### 4.1 配合公众号转换器插件
- 推荐使用另一个B站UP主开发的 **“微信公众号转换器”插件**。
- **功能**：在 Obsidian 中将 Markdown 文章直接转换为公众号排版样式。
- **AI排版**：更新后支持 AI 智能排版，可选择风格和配色。
- **一键同步**：配置公众号开发信息后，可将文章**直接同步到公众号草稿箱**，并自动使用第一张图作为封面，便于手机端快速发布。

### 4.2 自定义风格技巧
1.  找一张你喜欢的封面图并截图。
2.  将截图发给支持视觉模型的AI（如豆包）。
3.  指示AI：“请按照这个格式（指插件内置的风格提示词格式），帮我为这个图片生成对应的风格提示词。”
4.  将生成的提示词添加到插件的 **“风格管理”** 中即可使用。
