# 一条命令把YouTube视频变成B站双语字幕视频！yt2bb技能详解

> [探索未至之境](https://space.bilibili.com/441831884) | [BV1daDNBYEkx](https://www.bilibili.com/video/BV1daDNBYEkx) | 时长 3:45

## 一、 工具简介与核心价值
- **工具名称**：YT-2BB（YouTube to Bilibili）
- **本质**：是一个 **Cloud Code 技能插件**。Cloud Code技能是给AI编程助手（如Cursor等）安装的扩展包，赋予其新能力。
- **核心价值**：**一条命令**，**全自动**完成从YouTube视频到可发布B站双语字幕视频的全流程，解决手动加字幕、翻译、调格式耗时过长的问题。

## 二、 核心功能与亮点
1.  **全自动流程**：输入一个YouTube链接，自动执行以下六个步骤：
    -   下载视频
    -   语音识别
    -   生成字幕
    -   字幕质量检查与修复
    -   翻译成中文
    -   合并双语字幕并烧录进视频
2.  **高质量双语字幕**：
    -   语音识别使用 **OpenAI Whisper** 大模型，准确率高。
    -   翻译由 **Claude**（推测）完成，质量远超普通机器翻译。
    -   字幕格式：**英文在上，中文在下**，排版干净专业。
3.  **智能中文排版**：
    -   中文字幕自动在**标点符号处换行**。
    -   每行最多 **20个字**，确保在手机上观看体验良好，不会拥挤。
4.  **自动质量检查与修复**：
    -   自动检测并修复常见问题，如：时间轴重叠、字幕时长过短、单行字幕过长等。
5.  **一键生成B站发布信息**：
    -   自动生成包含 **五种风格标题**（选面式、数据式、争议式、教程式、情感式）的发布文档。
    -   提供 **10个以上混合中英文关键词的标签**。
    -   包含**简介**和**章节时间戳**，可直接复制粘贴到B站发布。

## 三、 详细工作流程
在Cloud Code中输入YouTube链接后，自动执行：
1.  **下载视频**：使用 **YT-DLP** 工具下载原始视频。
2.  **语音转文字**：调用 **Whisper** 模型进行语音识别，生成**英文字幕文件**。
3.  **字幕质检与修复**：自动检查时间轴等问题并进行修复。
4.  **翻译**：将英文字幕翻译为中文，**严格保持中英文字幕一一对应**，避免错位。
5.  **中文排版优化**：智能分割中文长句，确保每行不超过20个字。
6.  **烧录字幕**：使用 **FFmpeg** 将双语字幕（外挂或内嵌）烧录进视频，输出最终成品。

## 四、 最终输出成果
运行结束后，会得到一个完整的项目文件夹，内含：
- 原始视频文件
- 英文字幕文件（.srt等格式）
- 中文字幕文件
- 双语合并字幕文件
- **烧录了双语字幕的成品视频**（可直接上传B站）
- **B站发布信息文档**（包含标题、标签、简介、时间戳）

## 五、 其他特性
- **多语言支持**：不仅限于英文，支持**日语、韩语、西班牙语**等任何Whisper能识别的语言。
- **开源免费**：基于 **MIT协议** 开源，代码托管在GitHub，可免费使用和修改。

## 六、 总结核心要点
YT-2BB 是一款强大的自动化视频处理工具，它通过集成下载、语音识别、AI翻译和视频处理等多个环节，**用一条命令实现了从YouTube外文视频到B站双语字幕视频的全自动转换**。其核心优势在于**高质量的双语字幕生成**、**智能的中文排版优化**、**自动的质量检查**以及**一键生成完善的B站发布信息**，极大地节省了内容创作者在视频搬运、翻译和格式处理上的时间成本，是提升视频搬运和本地化效率的利器。
