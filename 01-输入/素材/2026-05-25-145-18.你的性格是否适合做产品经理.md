# 18.你的性格是否适合做产品经理

> [多多量化_产品经理](https://space.bilibili.com/380479775) | [BV19DoTBDEWC](https://www.bilibili.com/video/BV19DoTBDEWC) | 时长 15:10

## 性格对产品经理的重要性
- 性格是做产品经理的**核心要素**之一。
- **负面案例**：
    1.  **格局小，自私**：同事在公司下午茶时拿取远超份额的食物（一人一两个，他拿七八个），显得没有格局。
    2.  **职场PUA与高压管理**：研发经理对下属进行精神打压（PUA），以扣绩效、打低分威胁下属加班，导致所有下属都讨厌他。
    3.  **说话得罪人**：在职场中，得罪人意味着沟通的基础失败。喜欢得罪人、人身攻击、威胁说教的人，性格存在缺陷，在职场难以长久发展。
- **核心原则**：作为产品经理，沟通的第一要素是**不得罪人**。

## 好性格与坏性格特征对比
视频通过对比阐述了健康与缺陷性格的特质：

| **需要避免的性格缺陷** | **值得培养的好性格特质** |
| :--- | :--- |
| 自卑、胆怯 | 自信、落落大方 |
| 自我为中心 | 有同理心 |
| 对别人人身攻击、贬低、威胁、说教 | 尊重他人 |
| 不信任别人 | 信任他人 |
| 爱发脾气、受不了气 | 情绪管理能力强 |
| （受）原生家庭差（影响） | 家庭幸福（带来的积极影响） |
| 自私 | （隐含的）慷慨、有格局 |

## 性格缺陷的成因分析
性格深受**原生家庭**和**外部环境**的影响。
### 1. 原生家庭的影响
- **故事一（贬低与封闭）**：
    - **表现**：家长对内向的孩子进行打、骂、吼、贬低、羞辱，导致孩子将这种沟通模式复制到自己孩子身上。
    - **影响**：当事人将自己封闭起来，不愿接受新事物和不同观点。
    - **启示**：如果性格封闭、不愿Open（开放），就不适合做产品经理。
- **故事二（成绩不好与挨打）**：
    - **表现**：邻居将孩子成绩差简单归因于其本人，并进行残酷体罚（打到满身是血）。
    - **真相**：孩子成绩差的原因可能是**外部**的，如学校差、老师差、被欺负、生病、失眠、生活条件差、自卑等。
    - **启示**：如同孩子是“一张白纸”，员工在工作中出问题（如Bug、延期），管理者不应第一时间指责，而应去了解背后的真实原因（资源不足、需求问题等）。
- **故事三（催婚与贬低）**：
    - **表现**：父母用“你不结婚我就去死”等说教、贬低、威胁的方式逼婚。
    - **职场映射**：研发经理以“项目出问题要你负责”来威胁产品经理。正确做法是就事论事，了解延期原因，帮助同事解决问题、争取资源。
    - **环境因素**：年轻人不结婚是社会环境（996、高房价、高彩礼）导致，不应一味指责个人。
- **故事四（骚扰与抑郁）**：
    - **表现**：女生高中时被男同学骚扰导致抑郁和精神分裂，父母不仅不理解，反而骂她成绩不好。
    - **启示**：这种指责极其残忍，可能导致更严重的后果。

### 2. 性格缺陷的职场表现与根源
- **说话伤人**：有些人生气时专挑伤害对方自尊心的话说。
- **告黑状、甩锅**：同事喜欢通过向领导告状、反问推卸责任（“那你觉得怎么办？我们找领导”），令人反感。
- **敢于“横行”的原因**：除了个人性格缺陷，还因为其**所在部门在公司地位高、有话语权，且有领导撑腰**，使得其他部门难以制约他。

## 改变性格的建议与路径
如果发现自己有性格问题（如左边列的缺陷），又立志做产品经理，可以通过以下路径进行“救赎”：

1.  **第一步：自我学习与认知**
    - 阅读《爱的五种能力》。
    - 阅读《社会心理学》。
    - 阅读《道歉的五种语言》。
2.  **第二步：寻求专业帮助**
    - 如果看书后感觉无效，说明无法通过自我成长解决问题，需要借助外力。
    - **进行心理咨询**：优先选择**公立医院**的心理咨询。
3.  **核心逻辑**：
    - 好的性格（右边特质）会让人沟通顺畅、事事顺利。
    - 如果性格不好，生硬地模仿别人的友好言行（东施效颦），会显得奇怪、做作。根本之道是**改变性格本身**。
    - 如果通过学习和咨询都无法改变，或不愿改变，那么**就不适合做产品经理**，建议放弃。

## 总结
本节内容的核心是强调**性格对于产品经理职业的极端重要性**。通过大量真实的正反面案例，说明了自私、傲慢、不尊重人、说话伤人等性格缺陷会直接破坏职场协作与个人发展，并深入剖析了这些缺陷常源于原生家庭的错误教育和外部环境的压力。最终指出了改变性格的可行路径（学习与咨询），并明确告诫：如果无法或不愿培养出一个开放、尊重、善于沟通的好性格，就不应该选择产品经理这条道路。产品经理的成功，始于良好的自我认知与性格修炼。
