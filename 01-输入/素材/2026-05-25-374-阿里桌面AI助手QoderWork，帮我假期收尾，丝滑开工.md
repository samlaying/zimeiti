## 视频笔记：阿里桌面AI助手 CodeWork 助力假期收尾与丝滑开工

### 一、CodeWork 简介与优势
- **产品定位**：国产版的桌面AI助手，对标 Anthropic 的 Cloud Cowork。
- **核心优势**：
    1.  **简单易用，开箱即用**：拥有简洁UI界面，无需复杂配置，国内可直连。
    2.  **本地文件操作**：最大优势是能直接读写电脑本地文件，这是传统网页对话式AI工具不具备的。
    3.  **技能（Skills）与MCP**：通过“Skills”和“MCP”等机制，能直接调用和操作外部专业工具（如Word、Excel、PPT处理软件）。
    4.  **任务自动化拆解**：只需用自然语言描述需求，CodeWork即可将任务拆解为计划，分步执行并交付完整结果。

### 二、核心功能与操作方式
- **交互方式**：通过自然语言在指定的工作目录下下达指令。
- **任务执行逻辑**：理解需求 -> 拆解为可执行计划 -> 调用内置技能（Skills）或编写脚本 -> 分步执行 -> 交付结果。

### 三、具体应用场景与案例
#### 场景1：照片整理（假期收尾）
- **需求**：将散乱的旅行照片，按拍摄地点或日期分文件夹整理。
- **实操与效果**：
    - **按地点分类**：
        1.  输入指令：“把文件夹里的所有照片按照拍摄的旅游景点整理到不同的文件夹里”。
        2.  **CodeWork执行过程**：调用EXIF工具读取照片GPS坐标 -> 将坐标相近的照片分组 -> 综合分析场景推测地点 -> 对不确定地点进行网络搜索确认 -> 创建文件夹并分类。
        3.  **效果**：分类百分百准确。
    - **按日期分类**：
        1.  输入指令：“先把照片拷贝一份，然后按照照片的拍摄日期移动到不同的子文件夹里面”。
        2.  **效果**：快速完成按日期的文件夹分组。

#### 场景2：办公文件处理（Word/Excel/PPT）
- **核心机制**：通过“Skills”技能包（如DocX、XLSX、PPTX）获得处理特定文件格式的专业能力。
- **案例1：Word合同对比与审阅**
    - **需求**：对比两份合同（原始版与修改版），找出修改内容并审阅潜在问题。
    - **实操步骤**：
        1.  打开合同文件夹，输入对比提示词。
        2.  **CodeWork执行过程**：调用DocX技能 -> 安装Pandoc将Word转换为Markdown格式 -> 执行对比 -> 总结修改点。
        3.  进一步输入审阅指令，要求在文件上批注潜在问题。
        4.  **效果**：快速准确完成对比和问题标记，找到十几条潜在问题。
- **案例2：Excel发票识别与汇总**
    - **需求**：将多张发票图片识别并汇总成带图表的Excel文件。
    - **实操步骤**：
        1.  输入指令，要求汇总发票并生成统计图表，包含指定字段（员工、日期、销售方等）。
        2.  **CodeWork执行过程**：读取文件结构（6位员工，28张发票）-> 识别每张发票图像 -> 调用XLSXSkill -> 创建Excel文件（包含4个工作表与5个图表）。
        3.  **效果**：生成了包含明细表、按员工统计柱状图、年度趋势折线图等的格式完善的Excel文件，将一天的工作压缩到几分钟。
- **案例3：PPT信息补全生成**
    - **需求**：搜索假期期间计算机和AI领域热点，生成一个总结PPT。
    - **实操步骤**：
        1.  输入指令要求搜索热点并做成PPT。
        2.  **CodeWork执行过程**：调用PPT技能 -> 搜索信息 -> 生成PPT。
        3.  **效果**：生成了采用暗色科技风、内容全面、重点突出、结构完整的PPT，达到可汇报演讲的水准。

#### 场景3：账单数据整理
- **需求**：从手机账单截图中提取数据，分类汇总成Excel。
- **实操步骤**：
    1.  将4张账单截图粘贴给CodeWork，要求汇总成Excel并分为红包、年货采购、旅游、生活开销四大类。
    2.  **CodeWork执行过程**：读取并识别图片 -> 调用XLSXSkill -> 编写专用Python脚本 -> 创建Excel文件。
    3.  **效果**：生成了格式规范、带颜色分类标记、支持筛选排序并包含详细分类汇总的Excel账单表。

#### 场景4：深度信息整合（安装外部技能）
- **需求**：安装并使用Notebook LM技能，进行深度研究并生成多种形式的内容。
- **实操步骤（安装与使用）**：
    1.  输入指令及Notebook LM Skill的GitHub地址。
    2.  **CodeWork执行过程**：检查Python版本 -> 安装Python 3.12 -> 创建虚拟环境 -> 完成安装 -> 提示并引导用户登录谷歌账户。
    3.  **使用过程**：
        - 指令：创建笔记本，使用深度研究功能搜索AI及计算机热点。
        - **效果**：成功创建笔记本，启动研究任务，最终导入了131个信息源。
        - 指令：基于笔记内容生成PPT -> **效果**：生成了炫酷的PPT。
        - 指令：生成文本摘要 -> **效果**：交付了内容丰富的完整摘要。
        - 指令：生成音频播客 -> **效果**：交付了长达半小时的双人对话式音频播客，方便通勤收听。

### 四、核心要点总结
本期视频展示了阿里桌面AI助手CodeWork作为一款强大生产力工具的实际应用。它通过**自然语言交互**和独特的**Skills技能扩展机制**，不仅能高效处理本地文件（如照片整理），还能深度整合并操作专业办公软件（Word, Excel, PPT），完成从合同审阅、数据汇总、图表生成到PPT制作等一系列复杂任务。此外，通过安装外部技能（如Notebook LM），它能进一步整合深度研究并生成报告、摘要、播客等多种成果。CodeWork显著提升了从假期过渡到工作状态的效率，将繁琐的日常任务自动化、智能化，是一款值得体验的国产AI助手工具。
