# Vibe Coding 第3课：Gemini + Figma联动，输入一句话直接生成高颜值UI、Figma Make UI

> [下班学AI](https://space.bilibili.com/3546599351388553) | [BV1VGoXB5Emg](https://www.bilibili.com/video/BV1VGoXB5Emg) | 时长 5:20

## Vibe Coding 第3课：Gemini + Figma Make 一键生成高保真UI设计

### 一、背景与问题
- **核心痛点**：拥有产品需求文档后，非设计背景的用户难以绘制出专业级的UI界面。
- **2026年的答案**：借助AI工具，个人无需掌握设计技能即可生成高保真度的设计稿。
- **课程目标**：将一份枯燥的产品文档，通过AI工具链转化为精美的、像素级可交付的UI设计稿。

### 二、工具介绍
- **核心AI工具**：Gemini（用于处理文本生成提示词）、Figma Make（用于根据提示词生成UI界面）。
- **Figma平台简介**：
    - 主要用途：设计师进行UI界面设计的主流工具。
    - 资源：拥有庞大的公共社区和大量共享资源，可供参考借鉴。

### 三、实操步骤：从产品文档到UI设计稿
整个流程分为三步：**生成提示词 -> 生成设计 -> 调整设计**。

#### 步骤1：使用Gemini生成UI设计提示词
1.  **输入源**：打开已准备好的产品需求文档。
2.  **复制内容**：将产品文档的全部内容复制。
3.  **与Gemini交互**：将文档内容发送给Gemini，并添加核心指令：“帮我生成UI界面设计的提示词，我用Figma Make来制作UI”。
4.  **输出结果**：Gemini会为产品文档中的每个页面（如首页、心情记录列表页、设置页）分别生成详细的UI设计提示词。
5.  **保存**：将所有生成的提示词保存到文档中备用。

#### 步骤2：使用Figma Make生成UI界面
1.  **访问工具**：打开Figma网站，登录或注册账号。
2.  **生成第一个页面**：
    - 在Figma Make中，粘贴第一个页面的提示词。
    - 点击“生成”按钮。
    - 等待片刻，Figma Make会在右侧自动生成对应的高保真UI页面。
3.  **切换视图**：生成后可在“网页版本”和“手机版本”之间切换预览。
4.  **生成后续页面**：依次复制并粘贴第二、第三个页面的提示词，重复生成操作，直至所有页面设计稿完成。

#### 步骤3：调整与优化设计稿
AI生成的设计稿可能需要微调，有两种调整方式：
1.  **整体调整（色调、风格不符预期）**：
    - **方法**：返回到Gemini对话界面。
    - **操作**：指出需要修改的部分（如“主色调需要调整”），让Gemini重新生成优化后的提示词。
    - **结果**：使用新提示词在Figma Make中重新生成整个设计稿。
2.  **局部调整（某个组件不满意）**：
    - **方法**：在Figma Make的设计稿中直接操作。
    - **操作示例**：
        1. 选中不满意的UI组件（如一个按钮Button）。
        2. 在出现的AI助手窗口中，输入调整指令，例如：“UI，给我重新调整一下颜色，简单一点”。
        3. 发送指令，AI将只对选中的组件进行局部修改（如将深色按钮改为浅色）。

### 四、核心总结与下一步
- **本节课核心**：演示了如何利用 **Gemini（生成精准提示词）** 和 **Figma Make（生成高保真UI）** 这一AI工具链，实现从产品文档到精美UI设计稿的自动化转换。用户只需具备基础审美和沟通能力，即可完成专业设计。
- **课后行动**：视频评论区提供了提示词模板，用户可下载练习。
- **课程前瞻**：**下一讲**将演示如何使用 **Web Coding** 工具，一行代码不写，直接读取生成的UI设计稿，自动生成可在手机上运行的前端代码，最终完成从需求到成品的闭环。
