# 从夯到拉，前端网页设计skills，4款模型十个skills做首页小程序后台UI

> [侃侃我的](https://space.bilibili.com/21687610) | [BV147dYBLEqf](https://www.bilibili.com/video/BV147dYBLEqf) | 时长 50:17

## 视频笔记：前端AI设计技能测评

### 一、 测评概述
本次测评旨在对当前主流AI前端设计技能（Skills）进行横向对比。**Skills** 指的是提供给AI的“说明书”，指导其进行设计和网页生成工作。
- **测评模型**：Kimi、小米（Mimo）、GLM（智谱）、Gemini（谷歌）。
- **测评场景**：**官网**（首页）、**小程序**、**后台管理系统**。
- **测评技能**：共10个热门前端设计技能。
- **测评方法**：为每个场景生成约3000字的详细提示词（例如“充电桩官网”、“共享电动车调度后台”），然后由不同模型搭配不同技能进行批量生成，评估其美观度、完成度和风格稳定性。

### 二、 核心发现
1.  **原生效果**：不使用任何技能，让模型直接生成，通常效果较为普通和呆板。
2.  **技能加成**：安装合适的技能后，能显著提升设计质量和风格一致性。
3.  **模型差异**：不同模型在同一技能下的表现有优有劣，且可能受到上下文污染（如提示词中的“高端”导致普遍使用黑金配色）。
4.  **技能差异**：技能各有所长，需根据设计场景（官网、落地页、后台）和风格（暗色、毛玻璃、SVG等）选择。

### 三、 模型表现总结
- **Kimi（2.5/2.6）**：前端能力在国内领先，本次测评中综合表现**稳定且出色**。
- **小米（Mimo）**：模型较大，免费额度高（8-9亿Token），设计能力**中规中矩**，但颜色控制（如金色）常出问题。
- **GLM（5.1）**：能力很强，但模型**昂贵且难抢**。视觉能力可能5.1版更优，但成本高。
- **Gemini（Flash）**：本次表现**一般**，且存在将设计做成报告、以及中英文混杂（在后台界面中尤其明显）的问题。

### 四、 重点技能点评与排名
以下技能按在本次测评中从高到低的印象进行归类：

#### **顶级（视觉效果出色，风格鲜明）**
1.  **UI UX Pro Max**：**最适合普通人的首选**。风格全面，下限高，在前台、小程序、后台表现均**稳定优秀**，成功率很高。
2.  **Frontend Design Pro**（Anthropic官方出品）：**顶级视觉专家**，尤其擅长**暗色模式、液态玻璃、毛玻璃**等现代风格，效果精致，极具设计感。但暗色模式应用场景相对受限。
3.  **视觉风格突出类**：
    - **黑色专家/暗色专家**：擅长**黑色、紫色渐变、膨胀/粘土**等风格，视觉冲击力强，但可读性和通用性可能不足。
    - **SVG绘制专家**：擅长用SVG创造独特视觉效果，风格独特，但通用性较窄。

#### **中上等（稳健可靠，各有所长）**
1.  **Web Design Guidelines**：**偏向后台和设计规范**。具备自动更新指南的能力，**懒人必备**，上限可能较低，但下限和稳定性好。
2.  **Material Design风格类**：如谷歌相关技能，卡片感强，**非常适合做后台管理系统**，组件化风格稳定。
3.  **模板驱动类**（如Frontend）：包含大量预制模板（约100个）。**保质保量，下限非常高**，不易出错，但上限也较低，适合快速出图。

#### **中等及一般（存在明显短板或被取代）**
1.  **Nuxt.js相关技能**：功能与Vue/React技能类似，但被更优秀的技能取代。
2.  **规范型技能**（如Design.txt）：仅提供设计规范，而非直接生成能力，不如直接使用顶级技能。
3.  **专用型技能**：如仅适用于**制作落地页**或**绘制特定SVG形状**的技能，场景局限。

#### **拉胯（不推荐）**
- 多个技能因**被上下文污染**，导致所有生成页面风格雷同（如全是黑金）。
- 部分技能效果**抽象、怪异**，或无法正确处理组件和排版。
- 注：视频中“NPC”类技能并非全无用处，可用于配合其他工具绘制SVG等。

### 五、 实操建议与步骤
1.  **技能安装策略**：
    - **切忌同时安装多个同类技能**！这会导致AI调用混乱，设计风格不稳定。建议**单场景安装一个最合适**的技能。
    - 例如：做**官网**可选 `Frontend Design Pro`；做**后台**可选 `Web Design Guidelines` 或组件库类技能；需要**暗色特效**可选 `Frontend Design Pro`。

2.  **提示词撰写**：
    - 必须提供**详细的需求文档**（约3000字），明确说明网站类型、功能、风格要求。
    - 避免模糊词汇（如“高端”），以防AI做出错误的风格联想（如黑金）。

3.  **模型与技能搭配**：
    - **追求综合稳定与美观**：`Kimi` + `UI UX Pro Max`。
    - **追求极致视觉与暗色风格**：`Kimi` + `Frontend Design Pro`。
    - **快速出图、求稳**：任意模型 + `模板驱动类技能`。
    - **免费额度优先**：`小米` + 合适技能。

4.  **技能获取**：
    - 尽量从**官方或权威镜像站**下载，确保是最新版本。注意同名技能很多，需认准发布者（如Anthropic、Google官方出品）。

### 六、 总结
本次测评的核心结论是：**没有一款模型或技能是万能的，但存在明显更优的组合**。对于大多数用户，`Kimi`模型搭配`UI UX Pro Max`技能是平衡效果与稳定性的**首选方案**。若追求特定高级视觉风格，`Frontend Design Pro`技能是顶尖之选。关键是**根据设计场景（官网/后台/小程序）选择最匹配的技能，并坚持“一次只装一个”原则**，通过详细的提示词来引导AI，才能获得最佳且稳定的AI设计产出。
