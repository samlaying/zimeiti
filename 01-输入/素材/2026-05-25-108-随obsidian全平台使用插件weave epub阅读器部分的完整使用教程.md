# Weave插件之EPUB阅读器完整使用教程

## 一、 概述与基础
1.  **插件性质**：Weave插件中的EPUB阅读器是其**基础功能**，用户可**免费使用**。
2.  **核心定位**：用于在Obsidian中构建**刺激信息单元网络**，实现书籍阅读、摘录、笔记关联和知识管理。

## 二、 主界面布局
点击Obsidian侧边栏的**书架图标**，进入EPUB阅读器。界面主要分为四部分：
- **左侧侧边栏**：在书籍阅读模式下，会自动切换为包含**目录、摘录和书签**三个功能的导航栏。
- **中间内容区**：当前书籍的**正文阅读界面**。
- **顶部工具栏**：包含一系列辅助阅读和操作的**功能按钮**。
- **右侧笔记区**：用于保存和关联阅读笔记的区域，可显示**Modal文件、Converse文件**或**卡片管理网格视图**。

## 三、 核心功能与实操

### 1. 书籍导入
- **步骤**：
  1.  在书架界面，点击顶部**设置（齿轮）图标**。
  2.  选择**扫描仓库中的书籍**。
  3.  勾选**未添加**的书籍（支持多选），点击“加入”。
  4.  返回书架，若未显示，可点击**刷新**按钮。

### 2. 阅读与导航
- **正文阅读**：直接在内容区阅读。
- **源链接跳转**：点击笔记中的链接，可**自动定位**回书籍中的原文位置。反之，在书籍中点击笔记图标，也能跳转到关联的笔记。
- **阅读进度**：使用“**保存最后阅读点**”功能计算和记录当前阅读百分比，避免因跳转丢失阅读位置。

### 3. 摘录与笔记（核心）
- **基础摘录**：
  1.  在正文中**选中一段文本**。
  2.  点击工具栏的**“摘录”** 按钮。
  3.  在弹出的卡片中，可选择**记忆牌组**并添加**标签**。
  4.  点击“创建”，卡片会自动添加。
- **自动摘录模式（推荐）**：
  - 在右侧笔记区激活“自动摘录”模式。
  - 此时在正文中选中文字，文本会**自动添加**到右侧光标所在的笔记文件末尾。
- **高亮与笔记存储逻辑**：
  - 插件中EPUB阅读器的**高亮摘录笔记是保存在各个文档中**的，没有一个单独的集中文件。
  - **此逻辑与Obsidian PDF++插件一致**。

### 4. 自动化与卡片管理
- **Canvas自动化布局**：
  1.  点击自动化按钮，可**新建Canvas文件**或**关联已有文档**。
  2.  在阅读器中选中文本后，点击工具栏按钮，文本会自动添加为Canvas中的一个**新卡片节点**。
  3.  可调整节点添加方向（**向下**或**向右**）。
  4.  **截图功能**：使用截图工具框选后，可自动添加为图片卡片。
  5.  所有添加的笔记卡片都支持**源链接跳转**。
- **卡片管理视图**：
  - **网格卡片视图**（高级功能）：以网格形式展示所有摘录卡片。
  - **时间线视图**：以时间顺序展示摘录卡片。
  - 在Canvas视图中，可**右键点击卡片**，选择“**添加到记忆牌组**”用于后续的**回顾与记忆**。

## 四、 其他实用功能
1.  **章节导出**：点击“**导出为Markdown文件**”，可将当前章节转换为MD格式，并**支持图片提取**和语法转换，方便发送给AI进行分析或自行编辑。
2.  **增量阅读**：
    - 使用“**新增阅读点**”功能可以拆分章节，创建独立的阅读笔记。
    - 使用“**隐藏**”功能可以暂时屏蔽部分信息，专注于其他内容。
    - 阅读点和历史会记录在插件的增量阅读视图中。
3.  **搜索**：选中文本后点击“搜索”，可**直接调用Obsidian的全局搜索**功能。
4.  **全平台支持**：插件在**移动端（手机/平板）同样可用**，并针对移动设备进行了优化。

## 五、 总结
Weave插件的EPUB阅读器是一款深度集成于Obsidian的免费阅读工具，其核心价值在于将**书籍阅读与笔记系统无缝结合**。它支持从导入书籍、高亮摘录、自动化卡片布局到增量阅读的全流程，并通过源链接实现笔记与原文的双向跳转。其摘录逻辑与Obsidian生态内的其他工具（如PDF++）保持一致，且支持全平台同步，非常适合需要将阅读内容转化为知识网络、进行深度学习的用户使用。
