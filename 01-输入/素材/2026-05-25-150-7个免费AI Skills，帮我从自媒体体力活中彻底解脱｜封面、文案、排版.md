# 7个免费AI Skills，帮我从自媒体体力活中彻底解脱｜封面、文案、排版

> [Shiki实验室](https://space.bilibili.com/301388587) | [BV1J1o3B6Emz](https://www.bilibili.com/video/BV1J1o3B6Emz) | 时长 19:19

## 视频主题与背景
- **问题描述**：一个人做自媒体账号时，最耗时的不是拍视频和剪视频，而是琐碎的体力活，如字幕整理、B-roll配置、封面设计、标题生成、文案撰写和公众号排版。这些工作每期重复，占用大量时间。
- **解决方案**：通过AI Skills接管重复、机械、格式化的步骤，解放生产力。UP主分享7个免费AI Skills，基于真实使用经验，旨在提高效率而非生成低质量内容。

## 什么是AI Skills
- **定义**：Skills是执行类AI（如Code Codex、腾讯Workbody、OpenClaw小龙虾）的操作手册，让AI按预设步骤执行任务。
- **作用**：将流程文档化，使AI从普通对话机器人变为自动化工具，适用于每次操作相似的工作。
- **适合场景**：自媒体内容创作中的重复性任务，如脚本润色、字幕校对、封面生成等。

## 7个AI Skills详解
### 1. 老板大纲的打磨和Fact Check
- **功能**：修改视频脚本的开头、结尾，提供三个版本选项，并进行事实检查（联网搜索信息，纠正不严谨处）。
- **例子**：UP主拍摄视频时只写大纲，用此Skills检查脚本准确性，降低被观众质疑的风险。

### 2. 字幕整理捉虫
- **功能**：校对剪映自动生成的字幕，纠正专有名词错误（如Obsidian、Code Codex），生成SRT文件。
- **实操步骤**：
  1. 在剪映中生成自动字幕并导出。
  2. 将字幕发送给AI，AI先列出问题点并询问确认。
  3. 确认后AI修改字幕，生成SRT文件。
  4. 在剪映中导入本地字幕替换原字幕。
- **优势**：AI联网搜索，提高专业词汇准确性。

### 3. 视频配套的动画（B-roll）
- **功能**：根据视频脚本生成动画效果、图标和样式，作为B-roll插图。
- **实操步骤**：
  1. 视频初剪完成后，将脚本扔给AI。
  2. AI建议B-roll画面，经确认后生成网页形式的动画。
  3. 截图动画用作视频插图，提升视觉效果。
- **例子**：UP主视频过门变得更好看，节省手动制作时间。

### 4. 封面生成
- **功能**：自动生成横屏和竖屏封面设计，支持导出PNG格式。
- **实操步骤**：
  1. 提供视频标题或脚本，AI设计封面。
  2. 选择横屏或竖屏格式，点击导出PNG。
  3. 下载后可添加头像或logo完成封面。
- **优势**：避免手动调整尺寸，排版更美观。

### 5. 小红书文案生成
- **功能**：从视频脚本提取核心内容，生成小红书标题和正文。
- **实操步骤**：
  1. 将视频脚本发送给AI。
  2. AI提取内容并生成标题和正文。
  3. 用户确认后，可多次修改优化。
- **建议**：提前有标题草稿，减轻AI负担。

### 6. 公众号文章生成与上传
- **功能**：生成markdown格式的公众号文章，并自动上传到公众号草稿箱。
- **实操步骤**：
  1. AI根据脚本生成公众号文章（markdown格式）。
  2. AI生成预览，用户确认后调整。
  3. AI直接上传到公众号，设置标题、封面和签名。
- **注意**：需要配置公众号App ID、App Secret和白名单，UP主有单独教程。

### 7. 播客节目素材生成
- **功能**：为播客生成标题、show notes、时间轴和豆瓣书籍链接。
- **实操步骤**：
  1. 将剪映中的时间轴字幕发送给AI。
  2. AI自动生成标题、简介、时间轴，并搜索豆瓣链接。
- **例子**：快速解决播物料的制作问题。

## 如何获得和安装AI Skills
### 获取Skills
- **步骤**：
  1. 访问UP主网站（主页链接）。
  2. 在网站下方“人工智能指令”处留下邮箱（避免QQ邮箱）。
  3. 确认接收邮件后，获得第二封邮件中的资料页面。
  4. 从资料页面复制各Skills的提示词。
- **注意**：Skills免费分享，通过邮箱获取。

### 安装Skills
- **适用工具**：执行类AI软件，如Workbody、Codex、OpenClaw小龙虾（不支持DeepSeek）。
- **实操步骤（以Workbody为例）**：
  1. 在Workbody中点击“新建任务”或“New Chat”。
  2. 粘贴复制的提示词。
  3. AI会询问个性化问题（如Skills名称、输出格式、颜色主题等），根据需求回答。
  4. 确认后AI自动创建Skills。
  5. 在软件的“技能”按钮中查看和使用Skills，通过斜杠命令激活。
- **推荐**：Workbody最适合新手，免费额度多。
- **特殊配置**：公众号上传需单独设置，参考UP主的教程视频。

## 工具推荐与管理
- **管理工具**：Obsidian，用于管理视频脚本、生成内容，支持markdown格式和本地存储。
  - **优势**：AI可直接操作Obsidian文件路径，减少复制粘贴。
  - **推荐资源**：UP主的“上头Obsidian基础入门手册”。
- **Skills优化**：使用过程中不满意可截图标注，让AI修改并更新Skills，使其更符合个人需求。

## 总结核心要点
视频核心在于通过AI Skills自动化自媒体中的重复性体力活，如脚本校对、封面设计、文案生成等，从而大幅提升效率。UP主分享了7个实用Skills，涵盖从内容制作到发布的全流程，并提供了获取、安装和使用Skills的详细步骤。关键在于将AI作为执行工具，通过定制化Skills接管机械工作，让创作者专注于内容本身。推荐使用Obsidian管理内容，以进一步优化工作流。
