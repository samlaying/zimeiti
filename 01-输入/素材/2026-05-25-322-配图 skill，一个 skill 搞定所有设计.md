# 配图 skill，一个 skill 搞定所有设计

> [空格的键盘](https://space.bilibili.com/14762998) | [BV11aSRBXE2p](https://www.bilibili.com/video/BV11aSRBXE2p) | 时长 2:08

## 一、Skill 简介与核心优势
Skill 是一种能根据指令批量生成配图的工具，尤其适用于为文章、logo、海报、故事版等设计场景生成图片。
- **核心优势**：与文章逻辑紧密结合，生成手绘风格配图（含图片、icon和文字描述），提升内容可理解性与美观度。
- **传统配图 vs. Skill 配图**：
  - 传统方式：需发送“提示词 + 文章段落”逐一生成。
  - Skill 方式：完成文章后，仅需发送一句 **“帮我配图”** 即可批量生成匹配段落的图片。

## 二、实际应用案例
### 1. 文章配图实例
- **场景**：为专栏文章《AI工具的使用和安装教程》配图。
- **操作**：使用 Skill 自动生成 **10 张图片**。
- **效果**：
  - 图片风格简约、美观，可直接插入文章。
  - 图片与文章各段落逻辑匹配，支持一键推送到公众号或飞书。

### 2. 故事版（漫画图解）实例
- **场景**：制作“元宝签问与豆包商战”漫画。
- **效果**：通过 Skill 生成连续图解，呈现“元宝发红包、签问发奶茶、豆包打电话”等场景，画面生动且逻辑连贯。

## 三、创建自定义 Skill 的步骤
UP主分享了一个 **“创建生图 Skill 的 Skill”**，允许用户根据需求自定义设计模板。
1. **启动自定义创建**：调用该 Skill，系统会引导回答以下问题：
   - 图片类型：文章配图、Logo、分镜、社交媒体海报、数据可视化等。
   - 图片主要用途。
   - 每次批量生成数量。
   - 偏好的风格。
   - 是否有参考图。
2. **提供关键信息**：
   - **API Key** 和 **API URL**：必须使用 **Gemini 3 的图片模型**（即 “Nano of Nano”）。
3. **资源获取**：
   - 在 **ASQ6** 平台免费申请该模型额度，每月提供 **10 美金 API 额度**，基本满足个人使用。

## 四、工具与资源
- **模型**：Gemini 3 图片模型（Nano of Nano）。
- **平台**：ASQ6（免费申请额度）。
- **开源资源**：UP主已开源 Skill 代码，可从其代码仓库下载。

## 核心要点总结
Skill 是一款强大的 AI 配图工具，能通过一句指令批量生成与内容逻辑紧密匹配的手绘风格图片，显著提升设计效率。它不仅适用于文章配图，还可扩展至 Logo、海报、故事版等多种设计场景。用户可通过自定义 Skill 定制化设计需求，并结合 Gemini 3 模型低成本实现。所有资源已开源，便于快速上手应用。
