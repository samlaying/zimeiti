# 【重磅发布】工程团队编码神器✨ Code Xpert 正式上线！

> [XpertAI_元数信息技术](https://space.bilibili.com/3493144465246852) | [BV1zpZtBWEDS](https://www.bilibili.com/video/BV1zpZtBWEDS) | 时长 6:27

## 一、CodeXpert 简介
CodeXpert是面向工程团队的多整体编码平台，融合了对话式协作与任务化工作流。核心功能包括：创建编码任务、观察智能体执行过程、查看文件变更，并一键生成或打开PR（Pull Request），实现从推理到代码生成的闭环协作。

## 二、技术架构
CodeXpert采用分层解耦架构，主要包括以下组件：
### 1. 前后端系统
- **CodeExport Web**：负责用户交互界面。
- **CodeExport Server**：负责会话管理和任务调度。
### 2. AI Agent Server
- 作为智能体编排平台，管理编码智能体、维护线程上下文，并调用第三方或本地大模型进行推理与代码生成。
### 3. 隔离执行环境
- 所有代码执行与测试任务均在**Sinbox（沙箱）环境**中安全执行，确保环境隔离和代码安全。
### 4. 代码仓库集成
- 系统可连接**GitHub**及其他代码仓库平台（如GitLab、Gitee），实现代码拉取、提交与PR交互，形成完整闭环：智能推理 → 代码生成 → 安全执行 → 仓库协作。

## 三、使用流程
### 1. 登录与账户设置
- 通过**iSport AI智能体平台**进行单点登录：注册新账号或使用现有账号登录，登录后自动跳回CodeXpert系统。
### 2. 连接代码仓库
- **步骤**：
  1. 在设置中点击“连接代码仓库”按钮，选择代码提供商（如Github、GitLab、Gitee）。
  2. 认证方式为**App安装**：系统提供Webhook地址用于接收GitHub事件通知。
  3. 点击连接后跳转到GitHub，安装CodeExportAI的App到个人账号，配置仓库授权（可增减授权仓库），保存配置后返回系统。
  4. 连接成功后，在首页可看到已授权的仓库列表。
### 3. 提交任务
- 在首页输入框提交任务描述，选择已连接的仓库和分支。
- **示例**：任务为“参考Lunchin PI Type的智能体中间键逻辑，编写新的PI Midware插件”，并提供参考代码网址。
### 4. 任务执行与监控
- 系统自动创建**sandbox环境**，拉取代码仓库，新建任务并执行。
- **任务页面**：左侧为对话模式，智能体逐步调研代码结构；右侧为终端执行情况，可实时监控。
- **环境管理**：如需从新环境开始，可删除现有环境后重新提交任务，系统会创建全新环境并拉取最新代码。
### 5. 创建PR
- 任务执行完成后，点击“创建PR”按钮，系统自动生成PR到代码仓库。
- PR包含**总结性信息**、功能描述和文件更改链接，可直接在GitHub查看。

## 四、示例：编写智能体中间键插件
### 1. 任务描述
- 参考Lunchin PI Type的智能体中间键逻辑，在Community目录中新建PI Midware插件。
- 提供参考代码逻辑和网址，指导智能体开发。
### 2. 执行步骤
- 智能体读取scale文件，调研代码结构，逐步编写新插件代码。
- 用户在右侧终端查看执行过程，例如代码拉取、编译、测试等步骤。
### 3. 结果查看
- 任务完成后，查看**文件变更**：创建了PII Plugin文件，包含插件代码逻辑、规则校验、中间键策略实现，以及README文档和package配置。
- PR由CodeExportAI智能体作为作者，总结了功能和更改细节。
### 4. Token消耗与优化建议
- **数据**：该任务消耗了**245万5000个token**，对应一定金额。
- **优化建议**：智能体可能在某些细节（如package名称）上不足，可通过添加提示词或逐步与智能体沟通来修复和优化。

## 五、资源与限制
### 1. Token消耗
- 用户可在设置中查看token用量，具体消耗取决于任务复杂度（例如示例中的245万5000token）。
### 2. 使用政策
- 系统**免费使用**，但有限额：**五小时的使用限额**和**每周的使用限额**。
- 所有token免费提供，适合团队试用和学习。

## 六、总结
CodeXpert是一个AI驱动的工程编码助手，通过智能体平台和沙箱环境，实现了从任务提交到PR创建的自动化编码流程。它支持GitHub等多仓库集成，提供对话式协作和任务化工作流，旨在提升工程团队的编码效率和协作体验，同时具备免费试用限额，便于用户快速上手。
