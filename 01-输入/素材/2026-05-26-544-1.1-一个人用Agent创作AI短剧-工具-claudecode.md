# 1.1-一个人用Agent创作AI短剧-工具-claudecode

> [jeff车](https://space.bilibili.com/350355984) | [BV1ct6FBiEir](https://www.bilibili.com/video/BV1ct6FBiEir) | 时长 26:40

## 一、 工具准备与概览
制作AI短剧前，需要准备以下三个核心工具，它们将贯穿整个创作流程。
1.  **文件夹（电脑系统）**：用于结构化管理所有项目素材。Mac或Windows系统均可。
2.  **Claude Code（命令行AI Agent工具）**：核心创作工具，集成了AI与智能体功能，可通过自然语言指令在指定文件夹内执行复杂任务，如生成文件、代码、调用API等。
3.  **Obsidian（笔记工具）**：用于更方便、直接地管理电脑文件夹，替代系统自带文件管理器。

## 二、 文件夹结构设计（以“三国短剧”项目为例）
清晰的文件夹结构利于人工查找，也利于AI理解内容。一个推荐的结构如下：
-   **根目录**：`1.1-一个人用Agent创作AI短剧-工具-claudecode` (课程项目)
    -   `三国` (短剧项目根目录)
        -   `人物`：存放所有角色相关的素材，如角色描述、图片等。
        -   `场景`：存放可重复使用的场景图片（如“桃园结义”的桃园、“蜀国宫殿”），并做好命名。
        -   `道具`：存放可重复使用的道具图片（如关羽的“青龙偃月刀”），确保剧情视觉连贯性。
        -   `第一集_桃园结义`：存放第一集的具体内容。
            -   `剧本`
            -   `分镜`
            -   `关键帧图片`
            -   `生成的视频`
            -   (其他相关资源)

## 三、 Claude Code 详解与使用
### 3.1 安装与启动
1.  **安装**：参考Claude Code官方文档进行安装。它是一个命令行工具。
2.  **启动**：
    -   Mac：使用自带终端或安装`iTerm2`，在终端中进入项目文件夹后，输入`claude`命令启动。
    -   Windows：使用`CMD`或`PowerShell`，同样在项目目录下输入命令启动。
3.  **权限**：首次启动会请求读取/修改文件夹内容及执行命令的权限。为方便后续操作，建议选择“永远允许”。

### 3.2 核心功能与优势
-   **自然语言交互**：像聊天一样，用中文向Claude Code下达指令，它能理解并在指定目录下执行。
-   **批量操作**：能高效处理批量任务（如一次性创建100个人物档案），远超手动操作。
-   **上下文理解**：基于当前文件夹内容进行理解和操作。

### 3.3 项目配置文件 `cloud.md`
1.  **作用**：让AI（Claude Code）在每次进入项目时首先读取此文件，从而快速理解项目整体结构、风格和规范。
2.  **创建**：可以通过指令让Claude Code自动生成。例如：“帮我创建一个cloud.md文件，描述这是一个三国动画短剧项目，并列出目录结构。”
3.  **内容示例**：文件会包含项目描述、整体风格（如“中国风动漫，参考《魔道祖师》”）、目录结构说明、人物信息摘要等。后续可随时通过指令优化。

## 四、 创建和使用“技能”（Skills）
### 4.1 Skills 概念
Skills是Claude Code中可自定义的工具包。通过编写提示词（或提供API调用示例），可以创建一个“技能”，之后通过简单指令就能调用该技能执行复杂任务（如调用文生图API）。

### 4.2 创建“豆包文生图”技能示例
1.  **准备工作**：
    -   在火山引擎平台注册账号，申请文生图模型的API使用权限，获取`API Key`。
    -   （也可为其他文生图服务如Midjourney、海螺创建类似技能）。
2.  **创建步骤**：
    -   向Claude Code提供从该API官网复制的调用代码示例（不含个人Key）。
    -   指令示例：“请帮我创建一个调用豆包API的文生图技能。”
    -   Claude Code会在`.claude/skills`目录下创建一个配置文件，包含描述、调用模型、API密钥占位符、图片比例、风格等说明。
3.  **调用技能**：
    -   创建后，直接向Claude Code下达指令即可。
    -   示例：“参考刘备的人物描述文件，生成一张他的全身肖像，保存到刘备文件夹。”
    -   Claude Code会调用“豆包文生图”技能执行，并返回结果图片。

### 4.3 使用技巧
-   **并行工作**：可以同时在多个终端窗口运行Claude Code，分别执行不同任务（如一个生成图片，一个写剧本）。
-   **权限管理**：为提升效率，可在`Plan Mode`（按`Shift+Tab`切换）下操作，减少确认步骤。
-   **API密钥安全**：示例中的API密钥仅为演示，实际使用时应保护好自己的密钥，录制教程后应作删除处理。

## 五、 实操演示回顾
1.  **创建项目与人物**：在Claude Code中，通过指令“创建三国短剧的桃园结义情节，为刘备、关羽、张飞各创建一个描述文件”，快速生成人物目录和提示词文件。
2.  **配置项目**：生成`cloud.md`文件，定义项目风格和结构，并可在此修改人物提示词（如将风格从“中国风动漫”改为其他）。
3.  **生成素材**：使用自建的“豆包文生图”技能，根据人物提示词批量生成了刘备、关羽、张飞的角色形象图，并自动保存至对应目录。

## 总结
本视频介绍了以**文件夹结构化管理、Claude Code AI智能体为核心、Obsidian辅助管理**的高效AI短剧创作工作流。核心在于利用Claude Code这个强大的AI Agent，通过**自然语言指令**完成项目初始化、文件管理、调用外部API生成素材等繁琐工作，实现一个人高效推进短剧创作。关键技巧包括设计清晰的项目结构、配置`cloud.md`上下文文件，以及创建可复用的“技能”（Skills）来封装复杂操作，从而将创意快速转化为实际素材。
