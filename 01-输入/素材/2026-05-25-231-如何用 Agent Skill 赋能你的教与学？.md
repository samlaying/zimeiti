## 如何用 Agent Skill 赋能你的教与学？

### 一、概述
- **主题**：探讨如何利用 Agent Skill（智能体技能）辅助教学与学习。
- **背景**：王树义老师基于参加“教创赛”的反思与心得，结合最新技术进行分享。
- **核心场景**：
    1.  **制作教学动画**。
    2.  **创建个性化助教**。

### 二、场景一：用 Agent Skill 制作教学动画
#### 1. 工具介绍：Hagent 公司的 Hyper Frames
- **性质**：一系列专注于视频生成的 Skills。
- **产出形式**：并非传统视频，而是 **HTML 加上音频**（音频可克隆教师声音）。

#### 2. 教学应用示例：机器学习课程（以 DNN 分类泰坦尼克数据集为例）
- **动画示例一：前向传播过程**
    - **内容**：展示从输入乘客特征（头等舱、女性、年龄28、独自一人、票价82.17英镑、南安普顿登船）到输出分类概率（生还0.91，罹难0.09）的完整计算流程。
    - **过程**：分类特征过嵌入层 -> 与连续特征拼接 -> 经过隐藏层（加权求和、激活函数） -> 最终输出层过Softmax。
- **动画示例二：优化函数对比**
    - **内容**：在“泰坦尼克损失地貌”中，对比三个小球（SGD、Momentum、Adam）从同一起点下山的路径。
    - **结论**：Adam（自适应学习率，估计一阶和二阶动量）收敛最快，SGD最慢。直观展示“大多数泰坦尼克这类表格任务直接用Adam或AdamW”的原因。

#### 3. 使用 Agent Skill 制作动画的优势
- **优势一：修改极其简便**
    - **传统痛点**：视频渲染耗时，修改成本高、心智负担重。
    - **新方式**：发现错误（如将“200,100”误读为“200,1000”）后，可直接要求AI修改脚本对应部分，重新生成该片段的MP3并替换，过程快速。
- **优势二：高度个性化**
    - 动画中的所有讲解和指标都**围绕当前具体的数据集（泰坦尼克号）**展开，而非通用描述。
    - **好处**：学生听起来不“跳戏”，学习体验更连贯、贴合实际。

### 三、场景二：创建个性化学习助教
#### 1. 技能来源与初始设计
- **前身**：一个用于数据分析的 Agent Skill，曾用于北大讲座。
- **初始功能**：扮演“AI蓝军”，结合领域专家与数据分析技能，提供分析服务，会反思过程并给出置信度。

#### 2. 迭代为机器学习课程学习助教
- **迭代目的**：应对学生在使用AI辅助编程时可能出现的“认知依赖”和“认知卸载”问题（直接交出AI生成结果而不学习）。
- **核心设计**：Skill**不会**直接给出端到端的最终报告和代码，而是在**每一个关键步骤设置问题**，要求学生回答，以此考察学生对任务和知识点的理解。
- **工作流程示例（以DNN分类泰坦尼克为例）**：
    1.  准备数据后，提问：“任务设定为二分类预测建模，判定的依据是什么？”
    2.  分析特征后，提问：“部分字段有缺失值，你觉得这些缺失属于哪种类型？”
    3.  要求学生逐步回答，并对回答进行评价（如“抓住了关键点...回答正确”）。

#### 3. 助教 Skill 的三样核心产出
1.  **IPython Notebook**：
    - 一个可运行的文档，包含清晰的步骤、关键发现、代码、运行结果，以及之前所有问答卡点的记录和反思。
2.  **专属学习报告**：
    - 面向人类阅读，包含：
        - **任务摘要**与**Checkpoint**（记录了哪些问题）。
        - 每个步骤需掌握的知识点及学生表现。
        - 可分享给教师的**学习画像**与**课上补讲建议**。
3.  **个性化教程**：
    - 针对学生在问答中暴露的卡点和错误答案，提供针对性的讲解、代码示例和练习题。

#### 4. 获取与使用
- 王树义老师分享了该 Skill 的 Zip 文件（通过二维码获取）。
- 可在支持 Agent/Skill 的客户端（如 Tray 国内版、HermesCloat Code 等）安装使用。

### 四、总结与启示
#### 1. 技术赋能教师
- AI Agent Skill（如动画生成、智能助教）能极大提升教学材料制作效率和教学交互质量。
- 教师需要敏锐抓住技术发展的机遇。

#### 2. **核心：赋能学生，实现真正个性化**
- **技能可修改性**：学生获得 Skill 文件后，可以根据自己的需求，用自然语言对其进行添加、修改和删除。
- **激发创造力**：王树义老师在课程中让学生（分组）在15分钟内创建属于自己的 Skill，学生做出了如“菜谱生成”等多样化的模块。
- **个性化学习的定义**：不是所有学生用同一个 Agent，而是**每个学生拥有自己定制的、运行着独特 Skill 的 Agent**，以此处理自己的问题，并可举一反三扩展到其他课程和学科。这是引领学生适应AI时代的关键途径。
