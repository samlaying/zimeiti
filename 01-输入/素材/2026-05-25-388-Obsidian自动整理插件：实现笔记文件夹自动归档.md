# Obsidian自动整理插件：实现笔记/文件夹自动归档

> [极拓工坊](https://space.bilibili.com/128962829) | [BV1CzcbzGEC5](https://www.bilibili.com/video/BV1CzcbzGEC5) | 时长 2:29

## 介绍
AutoMove File 插件已根据用户反馈进行更新，新增了多个功能模式，以增强笔记和文件夹的自动归档能力。更新后的插件支持复制模式、文件名监控和文件夹归档模式，提升了 Obsidian 的文件管理效率。

## 功能模式详解
### 复制模式
- **定义**：在移动文件时，除了将原文件移动到目标文件夹，还会额外复制一份副本到目标位置，原笔记保留在原处。
- **作用**：实现笔记的归档备份，方便用户同时保留原始文件和归档版本。
- **示例**：打开复制模式后，触发归档规则时，目标文件夹会生成一份副本，而原笔记仍在原文件夹中。

### 文件名监控模式
- **定义**：基于文件名中的关键字触发归档规则。当文件名包含指定关键字（如“易发”）时，自动执行归档操作。
- **设置方法**：在插件设置中输入关键字（例如“易发”），插件会监控文件名变化。
- **示例**：如果文件名中包含“易发”，且复制模式打开，则会触发复制归档；如果复制模式关闭，则直接移动归档。

### 文件夹归档模式
- **定义**：允许将整个文件夹归档到目标文件夹，而不仅仅是单个文件。
- **设置方法**：选择要归档的源文件夹（例如“笔记/学习笔记”），设置关键字，并指定目标文件夹（例如“知识地图”）。
- **示例**：当源文件夹中的文件触发关键字（如“易发”）时，整个文件夹会被移动或复制到目标文件夹中。

## 实操步骤
以下是从字幕中提取的实操步骤，用于设置和使用插件：
1. **安装或更新插件**：确保使用最新版本的 AutoMove File 插件。
2. **设置关键字**：在插件配置中输入触发关键字（例如“易发”）。
3. **选择模式**：
   - 打开或关闭复制模式：根据需要选择是否在归档时保留原文件副本。
   - 启用文件名监控：基于文件名中的关键字自动触发归档。
   - 启用文件夹归档模式：选择源文件夹（如“笔记/学习笔记”）和目标文件夹（如“知识地图”）。
4. **保存设置**：完成配置后保存，插件将自动监控并执行归档。
5. **测试触发**：修改文件或文件夹名称以包含关键字，观察归档效果。例如，将文件名添加“易发”以触发规则。
