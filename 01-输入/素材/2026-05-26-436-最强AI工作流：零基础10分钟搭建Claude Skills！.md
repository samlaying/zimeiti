# 最强AI工作流：零基础10分钟搭建Claude Skills！

> [李厂长来了](https://space.bilibili.com/474455919) | [BV1wRF7zzEDg](https://www.bilibili.com/video/BV1wRF7zzEDg) | 时长 10:37

## Claude Skills 笔记

### 一、 Claude Skills 是什么？
- **一句话概括**：Skills 是 Claude 的“存放工作流和数据的文件夹”。
- **核心功能**：用于搭建工作流。只需向 Claude 描述任务，它就能自动调用后台相关的“技能包”和数据来完成任务。
- **优势**：实现任务自动化，提升工作效率。

### 二、 快速开始：使用官方内置 Skills
#### 1. 官方 Skills 仓库
- **获取方式**：SRobin（视频中指代Claude官方）开源了一个 Skills 仓库。
- **用途**：官方已做好的 Skills，几乎所有人都能用得上。
- **推荐**：视频中列出了几个推荐安装的 Skills（如用于制作幻灯片、报告和网页的 `Sync Factory`）。

#### 2. 操作演示（以 `Sync Factory` 为例）
- **场景**：作为房地产员工，需将一份关于澳洲利息预测的调研报告转换成客户可查看的网页。
- **步骤**：
    1.  **启用技能**：进入 Claude 设置 -> 功能 -> 往下滑，打开 `Sync Factory` Skills。
    2.  **上传文件**：在聊天框中上传调研报告文件。
    3.  **下达指令**：要求 Claude “根据文件内容创建一个直观的网页”。
    4.  **自动匹配**：Claude 会自动评估并选择最合适的 Skills（`Sync Factory`）执行任务。
- **结果**：约2分钟生成一个专业网页，主题色专业，重点内容已加粗变色，并包含策略调整建议。
- **关键点**：此功能依赖于在“设置 -> 通用 -> 个人偏好”中设置的自定义指令（让Claude每次任务前评估最合适的Skills）。

#### 3. 进阶演示：生成常见问题清单
- **指令**：基于同一份报告，生成20个常见问题清单，并提出格式和风格要求。
- **过程**：Claude 查阅文档和前端设计 Skills，生成问题与回复。
- **输出**：可下载为 Markdown 或 PDF 格式，便于分享。

### 三、 创建自定义 Skills
#### 方法一：扩展官方 Skills（推荐新手）
- **适用场景**：将现有模板（如公司PPT模板）转化为可复用的 Skill。
- **操作步骤**：
    1.  **上传模板**：将你的模板文件（如PPT）上传给 Claude。
    2.  **下达创建指令**：让 Claude 读取官方的 `PPT Skills`，然后基于你的模板创建一个新 Skills（如 `SRLPPT`）。
    3.  **提供详细上下文**（强烈建议）：
        - 触发 Skills 的时机。
        - 使用案例。
        - 保持不变的信息。
        - 具体工作流程。
        - 要求参考官方原技能以避免重复。
        - 输出格式。
    4.  **等待生成**：Claude 会读取官方技能和你的模板，抓取视觉元素（颜色、字体），约5分钟生成新 Skills 文件。
    5.  **下载并启用**：下载生成的文件，然后在 Claude 设置的“功能”里上传该文件以激活。
- **应用示例**：上传工作流程文件，让 Claude 用自定义的 `SRLPPT` Skills 生成一份12页的内部培训幻灯片。结果风格统一，与公司模板一致。
- **优化**：若输出有问题，可继续让 Claude 调整并更新 Skills。

#### 方法二：从零创建并结合 MCP（进阶）
- **适用场景**：完全定制化，用于特定、复杂的工作流。
- **操作步骤**：
    1.  **连接数据源**：在 Claude 项目中，通过连接器（如连接谷歌云端硬盘）提供知识库。
    2.  **详细定义 Skills**：
        - 描述背景和目标（如：基于40万字历史视频文稿，为任一AI主题生成文案）。
        - 定义触发机制、知识库调用方式、参考风格、完整工作流程（从引文到总结）。
        - 指定输出形式（视频逐字稿、标题、简介、标签）。
    3.  **生成与下载**：约7分钟生成新 Skills 文件并下载。
    4.  **在项目中使用**：在 Claude 项目中，通过个人指令指定输出形式，并补充额外要求。调用时只需说明“用哪个Skills做什么任务”。
- **应用示例**：创建视频文案写作 Skills。要求 Claude 撰写一篇关于 `Motabot` 的文案。结果：Claude 严格按流程生成了一篇6800字、结构完整的专业文案。
- **重要提醒**：
    - 调用项目和MCP工具时，Skills 需频繁调整。
    - 保持流程简单：一个流程中避免使用过多的 MCP 工具。

### 四、 总结与核心建议
- **核心价值**：Skills 的价值在于**固化和调用重复性工作流**，从而提升效率。
- **行动建议**：
    1.  **立即开始**：打开 `Skill Creator`，将你日常最重复的一个工作（如筛选热点新闻、摘要链接）固化下来。
    2.  **体验价值**：当你完成并运行第一个 Skills 时，你就会理解其价值，从而激发你将更多流程自动化。
    3.  **循序渐进**：从利用官方 Skills 扩展开始，再尝试从零创建，逐步将工作全部“搬进去”。
- **资源指引**：如不熟悉 Skills 基本概念，可参考UP主之前的介绍视频。

**核心要点总结**：本期视频详细介绍了如何零基础利用 Claude Skills 提升工作效率。核心是通过两种路径——“使用并扩展官方内置Skills”和“从零创建自定义Skills结合MCP”——来搭建个人自动化工作流。关键是提供详尽的上下文指令，并通过实际案例（如报告转网页、自动生成PPT和文案）展示了 Skills 如何将重复性任务流程化、自动化，从而实现效率的质的飞跃。建议观众从固化一个最简单的重复任务开始，亲身体验 Skills 的强大价值。
