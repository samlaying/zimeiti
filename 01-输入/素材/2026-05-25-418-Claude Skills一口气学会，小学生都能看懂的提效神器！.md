# Claude Skills一口气学会，小学生都能看懂的提效神器！

> [来真的](https://space.bilibili.com/590784254) | [BV1WK6vBqEqu](https://www.bilibili.com/video/BV1WK6vBqEqu) | 时长 26:28

## 一、引言
- **Agent Skills概念**：最近非常火热，能快速设计网页、PPT、整理表格、做动画等，但使用门槛较高。
- **视频目的**：从没有编程经验的视角分享经验，帮助零基础用户学会使用甚至创建Agent Skills。
- **推荐工具**：Claude Code（Cloud Code），是最早提出Agent Skills概念的工具，支持最完善，学会了它，其他AI编程工具也能触类旁通。

## 二、安装Claude Code
### 1. 前置要求
- **网络要求**：确保电脑网络通畅，能连接国外网站。
- **前置软件**：先安装NodeJS和Git，安装过程一路点击默认即可。
### 2. 安装步骤
- **方法一**（推荐）：访问Claude Code的Github地址，复制对应系统的安装代码（如macOS或Windows）。
- **示例**（Windows系统）：
  - 打开PowerShell（在搜索框输入“PowerShell”）。
  - 粘贴复制的代码，按Enter执行。
- **启动方式**：
  - **方式一**：在PowerShell中输入“cloud code”命令。
  - **方式二**：打开CMD（命令提示符），输入“cloud code”。
  - **方式三**：在文件夹中右键选择“在终端中打开”，输入“cloud code”。
  - 启动后按1确认信任文件，进入Claude Code界面。

## 三、配置AI模型
### 1. 使用CC Switch插件（推荐管理多个模型）
- **作用**：方便切换和管理多个AI模型，支持故障转移。
- **安装**：从Github的Release页面下载对应系统的安装包（如Windows的.msi文件）。
- **配置步骤**：
  - 打开CC Switch界面，点击加号添加模型。
  - **自定义配置示例**（如使用中转站）：
    - 填写API密钥（从购买渠道获取）。
    - 填写请求地址（Base URL），例如AWS特价CC的地址。
    - 自定义名称，然后点添加。
  - **国产模型配置**（如自谱GLM）：
    - 选择模型平台（如字谱GLM），只需填写官方API密钥，请求地址已预填。
  - **使用**：在界面中点击模型启动切换，支持自动故障转移（如一个模型失败，自动切换到备用模型）。
### 2. 直接修改配置文件（适用于固定使用一个模型）
- **步骤**：
  - 找到用户目录下的“.cloud”文件夹。
  - 在文件夹中创建或编辑“setting.json”文件，复制并修改以下代码：
    ```json
    {
      "baseUrl": "你的请求地址",
      "model": "你的模型名称"
    }
    ```
  - 在用户目录外找到“.cloud.json”文件，将“skipLogin”参数改为true（用于跳过官方登录）。
- **注意**：新手不推荐通过命令行设置，以免难以恢复。

## 四、获取和安装Skills
### 1. 官方仓库
- 访问Claude Code的Github仓库，下载官方提供的Skills（如处理Word、PDF、PPT的技能）。
- 将下载的Skills文件夹放到用户目录下的“.cloud/Skills”文件夹中。
### 2. 第三方网站（如Skills.sh）
- 该网站收录大量Skills，显示全球下载量。
- **安装特定Skills**：
  - 在网站找到需要的Skills，复制安装代码（如“cl install <skill-name>”）。
  - 在终端中粘贴代码并按Enter执行，一路确认即可完成安装。
- **示例**：安装“find skill”技能，在CMD中执行安装命令。

## 五、使用Skills
### 1. 自动调用
- 在Claude Code中描述任务，工具会自动分析并调用相关Skills（如生成PPT时自动使用PPT Skills）。
- **示例**：输入“帮我将这个网页重新编排成多页的PPT”，Claude Code会自动调用PPT Skills生成PPT文件。
### 2. 手动调用
- 使用斜杠命令（/）强制调用特定Skills。
- **步骤**：输入“/”后，输入Skills名称前缀（如“fr” for frontend design），按Tab键补全。
- **示例**：输入“/frontend”然后描述任务，强制使用前端设计Skills重新设计网页。

## 六、实操演示
### 1. 生成网页示例
- **无Skills**：直接生成网页，效果普通（如蓝紫色风格，较丑陋）。
- **使用Skills**：
  - 手动调用“frontend”Skills，输入提示词如“重新设计的具有科技的美学风格并且简约大气”。
  - 结果：生成高级感网页，带有动态光效和交互元素。
### 2. 生成PPT示例
- **自动调用**：输入“帮我用这个风格重新编排成多页的PPT”，Claude Code自动使用PPT Skills。
- **结果**：生成与网页风格一致的PPT文件，可编辑，支持进一步转换为PDF等。

## 七、总结与注意事项
- **危险模式**：使用“Claude Code --dangerous-skip-permission”可跳过所有手动确认，实现全自动执行，但需谨慎，避免误删文件（仅适用于生成新内容）。
- **进阶建议**：可同时开多个Claude Code实例执行任务，提高效率。
- **资源获取**：笔记和学习资料可参考UP主的“Leader AI xLiDentor的AEI机器的资质库”。

## 八、核心要点总结
本视频详细介绍了如何从零开始使用Claude Code和Agent Skills，包括安装、配置AI模型、获取Skills以及实操演示生成网页和PPT。核心是让用户即使没有编程经验，也能通过简单的步骤利用AI工具提升工作效率，关键点在于正确配置模型、灵活使用Skills的自动或手动调用模式，并注意安全风险。整体上，Agent Skills结合Claude Code是一个强大的提效神器，适用于前端设计、文档处理等任务。
