# vibe coding了一个教你 vibe coding 的网站

> [数字游民Holiday](https://space.bilibili.com/341646510) | [BV14YDMB8Eqw](https://www.bilibili.com/video/BV14YDMB8Eqw) | 时长 7:27

## Vibe Coding 一个教你 Vibe Coding 的网站：笔记

### 一、 项目背景与初衷
*   **UP主痛点**：Holiday 在同时进行多个全栈App和网站项目时，虽然使用AI进行编程，但作为缺乏系统编程基础的独立开发者，遇到了学习效率低、新工具迭代快导致的“卡点”问题。
*   **决策**：决定暂停现有项目，花时间系统性地补齐知识短板，目标是不仅掌握代码，更要理解**UI/UX设计规范、项目上线、后端数据维护**等独立开发者所需的关键能力。
*   **AI辅助设计**：将需求告知Claude（AI助手），让其设计适合自己的学习清单。Claude没有给出简单列表，而是用**视觉化板块**呈现了课程雏形。
*   **最终目标**：将个人学习计划转化为一个**公开、系统、视觉化**的网站，帮助所有零基础想学习“vibe coding”的人。

### 二、 网站设计详解
1.  **整体风格与体验**：
    *   **视觉风格**：赛博朋克与像素风结合。
    *   **交互细节**：点击按钮有类似游戏的音效，旨在让学习过程像游戏一样有趣。
2.  **核心功能系统**：
    *   **语言切换**：新增了中英文切换功能，方便更多用户。
    *   **侧边导航栏**：用于切换不同的标签页（学习模块）。
3.  **课程结构**：
    *   **三大阶段**：工具与基础 -> 进阶（全栈开发） -> 整合上线。
    *   **每日学习单元**：每个阶段包含多天课程，点击可进入当天详细内容。
        *   **学习大纲**：以“待办清单”形式呈现，完成可打勾。
        *   **学习内容**：当天的具体知识。
        *   **实践作业**：结合学员**正在进行的项目**布置任务，实现“学做结合”。
        *   **参考资料**：提供官方文档等外部链接。
        *   **学习笔记**：记录心得、卡点，并可保存。笔记按日期标签管理，便于回顾。
4.  **项目展示**：
    *   列出具体实战项目（iOS App、网页等），每个项目对应需要提升的不同技能（如后端同步、交互UX）。

### 三、 核心巧思与学习工具设计
1.  **倒计时日历与请假机制**：
    *   **60天倒计时**：追踪学习进度。
    *   **人性化设计**：**允许请假**。打开日历可以将某一天标记为“请假”，该天不计入进度。第二天重新登录时，显示的剩余天数不变，只有实际学习的那天才会递减。这解决了“一天接一天不间断”不现实的问题。
2.  **学习热力图**：
    *   可视化学习投入程度，颜色深度由每日学习**时长**决定。
3.  **专注计时器**：
    *   记录每日学习时间，支持暂停。
    *   集成**番茄钟休息模式**（默认10分钟倒计时）。
4.  **课程阶段指示器**：
    *   光标悬停在页面上方可显示当前所处的**阶段和天数**。
    *   点击可**快速跳转**至对应天的课程。
5.  **项目快速启动**：
    *   在项目列表旁添加“继续”按钮，点击可直接在本地编辑器中打开对应项目，提升效率。
    *   状态指示：学习中时按钮对应的“Holiday”小灯亮起，休息时熄灭。
6.  **课程规划**：
    *   课程内容（60天）将随Holiday本人的学习进度**持续填充和优化**。网站当前框架完整，但内容需长期更新，全部完成后才是初步完成态。

### 四、 总结核心要点
UP主Holiday分享了他如何从一个零基础开发者的实际需求出发，将个人“边学边做”的痛点转化为一个公开项目。他利用AI（Claude）设计了系统化的60天vibe coding课程框架，并巧妙地将游戏化元素（像素风、音效、进度追踪）、人性化设计（请假机制）与实用学习工具（热力图、计时器、项目快速跳转）相结合，构建了一个既服务于自己持续学习，也希望能帮助其他初学者的、互动性强的视觉化教学网站。这展示了在AI辅助编程时代，个人如何高效、有创造性地规划自己的学习路径并将其产品化。
