# 成功使用Skill实现智能分析小说并生成分镜网站

> [糯然](https://space.bilibili.com/14198451) | [BV1vzfBBjEYA](https://www.bilibili.com/video/BV1vzfBBjEYA) | 时长 2:31

## 引言
视频主题介绍了一个名为“Agent Skill”的实践，用于智能分析小说并生成分镜网站，以优化动漫制作流程。UP主糯然分享了这一工作流，旨在通过AI工具提高效率，减少策划分镜的步骤，并确保生成内容符合小说描述。

## 关键工具和模型
### Agent Skill概述
Agent Skill是一个基于网页的工具，用于加载小说内容并进行分析。它通过集成AI模型，实现自动化的小说解析和分镜生成。
- **使用方式**：在网页中加载小说后，使用Cloud Opus 4.6模型加载Skill。
- **处理范围**：可指定章节或全部处理，灵活适应不同需求。

### Cloud Opus 4.6模型
这是核心AI模型，负责加载Skill并执行小说分析任务。它支持高效推理和内容生成，确保输出质量。

## 分析流程步骤
实操步骤完整保留自字幕：
1. **加载小说**：在网页中导入小说文本。
2. **使用Cloud Opus 4.6模型加载Skill**：激活AI分析能力。
3. **指定处理范围**：选择特定章节或全部内容进行处理。
4. **生成分析内容**：自动生成剧情、人物、场景、世界、情节线、世界观以及章节片段分镜。
5. **后续处理**：基于分析结果，生成资产并结合分镜脚本制作动漫视频。

## 生成内容的特点和质量
### 关联性和一致性
生成内容具备超高关联性和一致性，能精确合理地推理小说中的各种元素，确保逻辑连贯。

### 情绪标记和关键事件分析
- **情绪带入**：像人类一样对每个章节进行情绪分析，增强叙事感染力。
- **关键事件标记**：精确分析每个章节中的关键事件点，并进行标记、解释和深度解析，为后续分镜制作提供基础。

### 深度解析
每个章节片段都经过深度解析，以方便实现高质量的小说分镜生成，提升动漫制作精度。

## 资产生成和视频制作
### 人物、场景、道具库生成
- **C-C分析小说**：先进行小说分析，然后生成资产，包括人物库、场景库和道具库。
- **资产生成**：结合增强的分镜头描述，生成符合小说内容的AI资产（如图片素材）。

### 结合分镜脚本生成视频
- **视频生成**：将资产与分镜脚本结合，生成对应的动漫视频。可根据个人想法调整内容，自定义视频分镜。
- **最终目的**：优化动漫制作流程，减少策划分镜步骤，保证生成内容与小说描述一致。

## 工具投入和工作流
### 购买的工具列表
UP主投入资金购买了多个工具进行测试，最终确定了动漫制作工作流。工具包括：
- 集末年会员
- Copilot Pro
- CAM9 3 Pro
- Tapno Pro Me Journey
- 简易年卡
每个工具都经过测试，以验证工作流的可行性。

### 工作流设计
工作流以Copilot系统为核心，整合了小说分析、资产生成和视频制作步骤。目前，提示词和底词已设计好，只需“抽卡”即可生成资产。

## 当前进展和挑战
### 制作一集动漫视频的尝试
UP主正在尝试制作一集动漫视频，时长约5分钟。虽然投入了成本，但学到了很多知识，且流程上暂无难点。

### 资产创建的麻烦点
资产创建是当前工作流中最麻烦的环节，但通过现有系统已尽可能优化，符合小说描述的提示词已设计完成。

## 总结核心要点
视频核心展示了如何利用Agent Skill和AI模型（如Cloud Opus 4.6）智能分析小说，自动生成高质量分镜内容和资产，从而优化动漫制作流程。关键步骤包括加载小说、指定处理范围、生成关联性内容，并结合资产与分镜脚本制作视频。UP主通过工具投入和工作流设计，实现了效率提升，尽管资产创建仍有挑战，但整体流程已无重大难点，为动漫制作提供了可行AI解决方案。
