# 【番外】安卓手机自动化神器：Automate vs MacroDroid，彻底解放你的双手！

> [机械小鸽](https://space.bilibili.com/40097653) | [BV1GqCPYjE1v](https://www.bilibili.com/video/BV1GqCPYjE1v) | 时长 5:45

## 引言
视频介绍了安卓手机上的自动化工具，重点比较了 Automate 和 MacroDroid，帮助用户解放双手，实现自动化操作。背景是上一期介绍了 Mac 和 iPhone 的自动化工具快捷指令，本期转向安卓平台。

## Automate 工具介绍
- **功能特点**：功能非常丰富，支持多种传感器操作和应用控制，功能可能超过 Power Automate（安卓客户端）。示例包括传感器操作和应用操作。
- **界面语言**：仅支持英文界面，没有中文选项。
- **操作体验**：在手机小屏幕上创建自动化流程比较困难，画流程的方式操作不便。

## MacroDroid 工具介绍
- **界面语言**：提供中文界面，使用方便。
- **费用模式**：是付费软件，但免费版本足够一般用户使用。
- **模板功能**：支持模板功能，用户可以访问其网站获取其他用户（尤其是海外用户）分享的自动化脚本。
- **触发条件多样**：
  - 支持多种触发方式，包括传感器触发、短信、呼叫、时间日期、应用程序触发、用户手动输入等。
  - 触发条件类似于 Power Automate 的三种方式：自动触发、手动触发和按照时间周期触发。
  - 示例：当设备插入充电器时触发（选择设备 > 电池 > 充电状态）。
- **动作功能丰富**：
  - 包括变量操作、电话功能、宏调用、媒体播放、屏幕操作、时间日期设置、设备控制（如 UI 交互，可用于抢票等）、通知弹出、网络操作（HTTP 请求、JSON 解析和输出）、位置设置、文件操作、相机功能（拍照或截图）、短信和电子邮件发送、应用程序控制等。
  - 高频使用功能示例：通过 HTTP 请求发送数据或以邮件形式发送信息。
- **自定义功能**：支持手动编写自定义脚本，满足更复杂的自动化需求。

## 实操步骤：在 MacroDroid 中创建自动化流程
以下是在 MacroDroid 中创建一个“充电时杀死后台应用”自动化流程的完整步骤：
1. **打开应用**：启动 MacroDroid 应用。
2. **创建空白宏**：点击主屏幕的加号，选择创建空白宏。
3. **命名流程**：输入宏的名称，例如“充电时杀死后台”。
4. **添加触发器**：
   - 点击触发器区域的加号。
   - 导航到设备 > 电池 > 充电状态。
   - 选择“已插入充电器”作为触发事件，并确定。
5. **添加动作**：
   - 点击动作区域的加号。
   - 在功能列表中找到杀死后台应用的功能（通常在设备或 UI 交互选项中）。
   - 选择“杀死所有应用”并确认。
6. **添加约束条件（可选）**：
   - 设置约束为时间条件，例如选择“日落之后”。
   - 系统需要定位功能来确定日落时间，需授权定位。
7. **保存或测试**：保存流程后，当插入充电器时，自动触发动作杀死所有后台应用，实现安静充电。
- **测试方法**：插入充电器，观察动作是否执行。

## 扩展应用和建议
- **其他自动化想法**：
  - 当短信收到时，读取短信内容，通过 HTTP 请求将内容发送到指定服务。
  - 或以邮件形式自动转发短信内容。
  - 更多功能探索：使用 UI 交互进行抢票工具设置、利用相机拍照或截图等。
- **工具扩展**：Automate 也可以在电脑端使用，视频中提到未来可能探讨在车上运行的可能性。

## 总结
核心要点：安卓手机上有两款强大的自动化工具——Automate 和 MacroDroid。Automate 功能全面但界面为英文且操作不便；MacroDroid 提供中文界面、免费版可用，支持丰富的触发条件和动作，适合在手机上轻松创建自动化流程。通过实操示例（如充电时杀死后台应用），用户可以解放双手提升效率。视频还展示了自动化工具在短信处理、网络请求等场景下的应用潜力，鼓励用户探索更多自动化可能性。
