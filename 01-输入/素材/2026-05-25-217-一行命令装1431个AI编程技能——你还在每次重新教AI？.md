# 一行命令装1431个AI编程技能——你还在每次重新教AI？

> [老汤的碳基突围](https://space.bilibili.com/384111845) | [BV142dmBQEV9](https://www.bilibili.com/video/BV142dmBQEV9) | 时长 5:41

## 一、 项目概述与核心价值
### 1.1 核心问题
- **痛点**：每次开启新项目，都需要向AI（如Claude Code）重复解释代码风格、测试标准、安全规范等，效率低下。
### 1.2 解决方案：Anti-Gravity Awesome Skills
- **一句话概括**：一个将全网最佳AI编程技能收集、整理、打包成一个**可安装**仓库的项目。
- **关键数据**：
  - 技能数量：**1431个**。
  - GitHub热度：创建于2026年1月14日，3个多月内获得**34000颗星**，全球排名前830。
  - 更新频率：几乎每天更新，版本号已迭代至VC05.0。

## 二、 技术基础与标准
### 2.1 Agent Skills 标准
- **来源**：由Anthropic在Claude Code中正式支持（2025年10月），Google的AntiGravity IDE同样支持。
- **技能文件结构**：
  - **文件格式**：`.md` 文件。
  - **上层**：YAML元数据，定义技能的触发条件。
  - **下层**：Markdown内容，描述技能的具体操作步骤。
- **价值**：能让AI助手在特定任务上的表现**提升一个量级**。

## 三、 实操指南：安装与使用
### 3.1 三种安装姿势
1.  **全量安装**（一次性安装所有技能）：
    ```bash
    npx anti-gravity-awesome-skills
    ```
2.  **工具定向安装**（指定AI工具）：
    - 仅安装到Claude Code：`npx anti-gravity-awesome-skills --tool claude`
    - 仅安装到Cursor：`npx anti-gravity-awesome-skills --tool cursor`
3.  **精准筛选安装**（使用过滤器）：
    - 使用 `--category` 过滤类别。
    - 使用 `--risk` 过滤风险等级。

### 3.2 入门推荐：8个万能技能
- 项目推荐从以下8个基础技能开始，适用于任何项目：
  1.  **Brainstorming**：用于规划。
  2.  **TDD**：测试驱动开发。
  3.  **Debugging**：系统排查。
  4.  **Security Auditor**：安全审计。
  5.  **Front-end Design**：管理UI。
  6.  **API Design**：管理接口。
  7.  **Link-end Validate**：质量检查。
  8.  **Create PR**：打包提交PR。

### 3.3 Bundle：按角色打包的解决方案
- **作用**：解决“1431个技能从哪里开始”的选择困难症。
- **示例**：
  - **做全栈产品**：推荐 `Essentials` + `Fullstack Developer` + `QA Testing` 三个Bundle。
  - **加固安全**：使用 `Security三件套`。
  - **做开源项目**：使用 `Essentials` + `OSS Maintainer` Bundle。

### 3.4 Workflow：技能的有序执行流程
- **比喻**：不仅给你“工具箱”，还给了你“施工图纸”。
- **示例**：“Shipping Assess MVP” Workflow会按顺序调用：头脑风暴 → 架构设计 → 前端开发 → 测试 → 安全审计 → 提交PR。

## 四、 核心特性与注意事项
### 4.1 跨平台通用性
- **结论**：同一套 `.md` 技能文件可在 **Claude Code、Cursor、Codex、Gemini、Google AntiGravity、GitHub Copilot** 等多种工具中通用。
- **唯一区别**：安装目录路径不同。

### 4.2 资源管理：上下文窗口占用问题
- **社区测试数据**：
  - **50个技能**：约占 **5000个token**，约为上下文窗口的 **4%**。
  - **300个技能**：约占 **3万个token**，约占 **23%**。
- **建议**：
  - 从 **5到10个技能** 起步。
  - 使用 **Bundle** 按需扩展，避免一次性全量安装。
  - 技能采用**渐进式加载**，仅被触发的技能会完整加载。

## 五、 项目资源与竞品对比
### 5.1 在线技能目录
- 项目提供在线外部目录，可按分类浏览、搜索，并查看每个技能的详细说明（地址见视频简介）。

### 5.2 竞品对比
- **Voltagen的Awesome Agent Skills**：
  - **路线**：精选路线，只收官方团队技能。
  - **适用场景**：想要**少而精、最靠谱**的那一小撮。
- **Anti-Gravity Awesome Skills**：
  - **路线**：广度路线，1431个技能，包罗万象。
  - **适用场景**：想要**大而全的军火库**。

## 总结
视频介绍了Anti-Gravity Awesome Skills项目，它通过一个NPX命令，将1431个基于Agent Skills标准的AI编程最佳实践（涵盖开发、测试、安全等）快速安装到Claude Code、Cursor等工具中，解决了用户重复向AI解释规范的低效问题。项目提供了入门技能推荐、角色化Bundle包和有序的Workflow流程，并支持跨平台通用。其核心价值在于，它像为AI编程助手提供了宝贵的“临床经验”，让AI从“刚毕业的医学生”快速成长为“老主任医师”。建议用户从少量技能和Bundle入手，以避免上下文窗口过载。
