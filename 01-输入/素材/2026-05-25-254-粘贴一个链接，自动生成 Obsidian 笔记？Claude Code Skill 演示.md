# 粘贴一个链接，自动生成 Obsidian 笔记？Claude Code Skill 演示

> [Yanonk](https://space.bilibili.com/15203937) | [BV1J8DCBJEda](https://www.bilibili.com/video/BV1J8DCBJEda) | 时长 4:44

## 概述
视频演示了两个Claude Code自定义Skill，用于自动化生成Obsidian笔记：一个将文章链接转为笔记，另一个将视频链接转为笔记。这些技能展示了AI在内容提取和笔记整理方面的提效应用，能快速处理网页文章和视频内容，输出结构化的笔记文件。

### Skill 1: Article to Note（文章转笔记）
**功能描述**：粘贴文章链接，自动生成结构化的Obsidian笔记，适用于微信公众号等网页文章。
**实操步骤**：
1. 粘贴文章链接（如微信公众号文章URL）。
2. Skill自动调用DeFado工具抓取网页正文内容，同时提取标题、作者等元数据。
3. 分析文章主题，按照预设模板生成结构化笔记。
4. 笔记生成完毕，自动在Obsidian中打开。

**具体例子和数据**：
- 演示中使用了一个微信公众号文章链接。
- 整个过程只用了几秒钟。
- 输出笔记特点：
  - 属性区：包含标题、作者、来源、标签、阅读时间、核心观点等元数据，格式规范。
  - 正文：按主题分段整理，每段有小标题和要点总结，完整保留文章核心内容，阅读清晰。

### Skill 2: Video to Note（视频转笔记）
**功能描述**：粘贴视频链接，自动生成结构化的Obsidian笔记，适用于B站等平台视频。
**实操步骤**：
1. 输入视频链接（如B站视频URL）。
2. 使用YT-DLP获取视频元数据（如标题、时长、作者）。
3. 下载音频文件。
4. 由于视频没有字幕，Skill自动使用Whisper ASR进行语音识别：将音频切分成多个片段，并行转录以提高效率。
5. 转录完成，得到约9000多字的文本。
6. 基于转录内容生成结构化视频笔记。
7. 笔记生成完毕，自动在Obsidian中打开。

**具体例子和数据**：
- 演示中使用了一个B站视频链接。
- 整个过程用时1分46秒，包括音频下载、语音转录和笔记生成三个阶段。
- 输出笔记特点：
  - 属性区：包含视频来源、作者、时长、平台等完整信息。
  - 正文：每个视频章节整理成独立小节，包含核心观点提炼和个人思考。
  - 原始转录文本完整保留在底部，方便随时查阅。

## 总结核心要点
这两个Claude Code Skill通过自动化抓取、分析和生成，极大地提高了从文章和视频创建Obsidian笔记的效率。Article to Note技能能快速处理文字内容，生成带元数据和结构化正文的笔记；Video to Note技能则利用Whisper ASR语音转录技术，将视频内容转化为可读笔记并保留原始转录。整个过程体现了AI在知识管理中的强大能力，为用户提供了便捷的提效工具，适用于内容整理和学习场景。
