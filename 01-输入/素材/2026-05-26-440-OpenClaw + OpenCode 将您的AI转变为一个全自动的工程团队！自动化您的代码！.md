# OpenClaw + OpenCode 将您的AI转变为一个全自动的工程团队！自动化您的代码！

> [攒钱换房车的福叔](https://space.bilibili.com/662992226) | [BV1VUc3zVEmM](https://www.bilibili.com/video/BV1VUc3zVEmM) | 时长 15:04

## 核心概念
### OpenClaw 介绍
- OpenClaw 是一个自主人工智能智能体，原名 Cloud博，集成了自托管智能体运行室和消息路由器，可作为直接在本地设备上运行的个人AI助手。
- 功能：能根据Twitter推文自动构建全栈应用，包含功能、数据库、认证系统等组件，并通过多个子智能体持续优化迭代。
- 例如：本地部署多个子智能体用于清理和整理文件，处理迅速。

### OpenCode 介绍
- OpenCode 是一个开源人工智能编码智能体，可直接运行在终端中。
- 特点：能够阅读、编辑和推理实际的代码库，执行任务如使用Grad命令跨文件操作，像一个真正的编程伙伴。
- 支持免费模型使用。

## 集成工作流程
### 优势与应用
- 最佳工作流程：将OpenClaw与OpenCode集成，OpenCode作为指挥中心，自动决策并通过OpenCode智能体路由任务。
- OpenClaw的子智能体由OpenCode驱动，支持更强大的免费模型如Gemini 2.5和GPT-4.6。
- OpenClaw负责规划和协调，OpenCode执行任务，可集成Crew作业或专用命令，实现自主运行。
- 结果：有人应用此工作流程构建信用应用，通过审核智能体优化输出，提升质量。

### 安全考虑
- OpenClaw本地部署存在风险：可控制并执行本地工具，恶意提示或错误可能导致文件修改、删除或泄漏。
- 问题：Claw Hub等仓库的社区技能不安全，8000个实例暴露在互联网默认端口上，近50%含恶意指令，可能窃取加密货币钱包、信用卡信息等。
- 解决方案：使用Agent Trust Hub（Argent Asha）技能扫描器，自动检测并阻止关键威胁，提供信任保障。

## 安装与配置
### 前提条件
- 确保安装Node.js 18以上版本。
- 准备Google账户用于免费认证。

### 实操步骤
1. **安装OpenClaw**：
   - 复制并粘贴NPM安装命令到命令提示符：`npm install openclaw`。
2. **配置Google认证**：
   - 启用OpenCode插件的Google反重力认证：使用命令`opencode plugin enable google`。
   - 登录Google账户：粘贴登录命令，选择提供商（如反重力功能），使用Google账户登录。
3. **安装OpenCode**：
   - 前往OpenCode官网，使用NPM安装到电脑：`npm install opencode`。
4. **添加OpenCode Controller技能**：
   - 访问Claw Hub，找到OpenCode Controller技能（用于控制OpenCode）。
   - 使用Agent Trust Hub扫描技能链接以确保安全（例如，扫描结果显示某技能恶意，需避免）。
   - 在终端输入技能添加命令，粘贴技能链接完成添加。
5. **验证与运行**：
   - 安装后，在OpenCode工作区的技能部分查看OpenCode Controller线是否启用。
   - 可安排多个智能体在不同时间执行任务，如设置审核智能体自动部署。

## 示例演示
- 例子：使用OpenClaw控制OpenCode构建CRM表板。
  - 流程：OpenClaw通过OpenCode Controller技能，自动搭建Node.js + React项目，安装依赖，启动开发服务器。
  - 结果：生成自定义脚本，功能齐全的CRM表板运行正常，配色方案符合预设。
  - 意义：演示自动化编码，利用预设生成内容，通过消息路由系统执行重复任务。

## 总结核心要点
OpenClaw和OpenCode的集成创造了一个全自动工程团队，OpenClaw作为核心智能体负责协调，OpenCode执行编码任务，通过Google认证免费访问先进模型，同时使用Agent Trust Hub确保安全，避免恶意技能威胁。这一工作流程提升了开发效率，实现了代码自动化、迭代优化和任务调度，代表了基于开源技术的最佳协作方式。
