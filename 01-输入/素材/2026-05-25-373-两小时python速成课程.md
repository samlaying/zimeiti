# 两小时python速成课程

> [](https://space.bilibili.com/) | [BV16A411i7W7](https://www.bilibili.com/video/BV16A411i7W7) | 时长 0:00

## 介绍模块
模块是预先定义好的代码文件（.py格式），包含函数、变量等，可以在其他Python脚本中导入和使用，便于代码复用和项目管理。模块分为内置模块（如math、sys）和外部库（需安装，如pygame）。

## 内置模块示例
- **math模块**：提供数学运算函数，如`math.pow(2, 2)`计算2的平方，结果为4。
- **sys模块**：包含系统信息，例如`sys.path`输出Python查找模块的路径列表，第一个路径是当前文件夹，后续是系统模块路径。

## 自定义模块的创建和使用
### 创建模块
1. 新建文件夹（如Modern Tutorial），在其中创建两个Python脚本文件（如MainCode.py和SetCode.py）。
2. 在SetCode.py中定义函数，例如：
   - `getSum(a, b)`返回a+b的和。
   - `getSub(a, b)`返回a-b的差。

### 导入和使用模块
- **直接导入**：在MainCode.py中使用`import SetCode`，然后调用函数如`SetCode.getSum(2, 2)`，结果为4。
- **from import导入特定函数**：使用`from SetCode import getSum`，可直接调用`getSum(2, 2)`，无需模块前缀。
- **as别名导入**：使用`import SetCode as cc`，然后调用`cc.getSum(2, 2)`，方便重命名模块引用。
- **导入全部内容**：使用`from SetCode import *`，导入模块所有函数和变量，可直接使用。

## 导入方式详解
- `import module`：导入整个模块，使用时需加模块名前缀。
- `from module import function`：仅导入指定函数，直接使用。
- `import module as alias`：为模块设置别名，简化调用。
- `from module import *`：导入模块所有内容，谨慎使用以避免命名冲突。

## __name__变量
- 用于判断脚本是否作为模块被导入。
- 当脚本直接运行时，`__name__`等于`"__main__"`；当作为模块导入时，`__name__`不等于`"__main__"`。
- 示例：在SetCode.py中添加代码：
  ```python
  if __name__ == "__main__":
      print("我是主代码")
  else:
      print("我不是主代码")
  ```
  - 直接运行SetCode.py，输出“我是主代码”。
  - 在MainCode.py中导入SetCode并运行，输出“我不是主代码”。

## 外部库安装
- **安装工具**：使用Python内置的包管理工具`pip`。
- **安装步骤**：
  1. 打开命令行：按Windows + R，输入`cmd`打开命令行窗口。
  2. 使用命令`pip install package_name`安装库，例如`pip install pygame`安装游戏库。
  3. 安装后，库会被添加到`sys.path`的路径中，可直接导入使用。
- **示例**：
  - 安装numpy库：`pip install numpy`。
  - 在代码中使用`import numpy`导入。
- **建议**：查阅官方文档（如numpy官网）获取安装和使用指南。Anaconda等版本管理工具也可用于管理多个Python版本。

## 总结
本课介绍了Python模块的核心概念：模块是代码复用的基本单位，通过import关键字导入；自定义模块可提升项目结构清晰度；掌握不同导入方式（如from import、as别名）能灵活使用模块；__name__变量帮助区分脚本运行模式；外部库通过pip安装扩展功能。学习Python时，善用文档和教程（如菜鸟教程）加速掌握。
