# 详细笔记：用 Hyper Frames + 编程智能体制作精美短视频

> 来源：哔哩哔哩 2026-05-20 08:09 | 整理时间：2026-05-20



## 核心思路

不用 Premiere、剪映等专业剪辑软件，仅依靠：

1. **Hyper Frames**（免费开源视频处理工具）
2. **编程智能体**（如 Codex、Open Code 等）

通过**自然语言指令**驱动整个流程，最终产出带字幕、频道标签、图标展示的精美短视频。

---

## 一、所需工具清单

| 工具 | 用途 | 备注 |
|---|---|---|
| Hyper Frames | 开源视频合成引擎 | 官方仓库提供安装命令，也可通过插件市场安装 |
| Codex / Open Code 等编程智能体 | 接收自然语言指令，调用 Hyper Frames API 完成自动化操作 | 二选一即可 |
| 素材文件 | 自己准备的视频/音频素材 | 放入项目根目录 |

---

## 二、完整操作流程

### 步骤 1：安装 Hyper Frames

两种方式：

- **方式 A（手动）**：前往 Hyper Frames 官方仓库，复制安装命令，在终端中执行。
- **方式 B（省事）**：直接将安装命令发送给编程智能体（如 Codex），让它帮你执行安装。
- **方式 C（Codex 插件）**：在 Codex 的插件市场中搜索 `Hyper Frames`，点击安装即可。

> 视频演示者使用的是 Codex + 插件安装方式。

### 步骤 2：创建项目并导入素材

1. 在 Codex 中新建一个项目
2. 右键打开项目目录
3. 将自己的视频/音频素材文件放入项目根目录

### 步骤 3：获取 API 密钥

1. 前往 Hyper Frames **官网**注册并登录账号
2. 进入 **API 密钥** 页面
3. 点击「新建 API 密钥」（名称随意填写）
4. 新建完成后**复制该密钥**

### 步骤 4：配置环境变量文件

在项目根目录新建一个 `.env` 文件，内容格式如下：

```
API_KEY=你复制的API密钥
MODEL=模型名称
```

- 将 `你复制的API密钥` 替换为步骤 3 中实际复制的密钥
- `MODEL` 填写 Hyper Frames 支持的模型名称

### 步骤 5：编写提示词并提交任务

1. 在 Codex 中粘贴一段**提示词**（Prompt），关键提示内容包括：
   - 指示智能体读取 `.env` 中的 API 信息用于**语音转录**
   - 指示智能体处理项目中的**核心素材文件**
   - 明确告知："你可以使用我放在项目目录里的环境变量文件，获取语音转录的 API 接口和密钥"
2. 点击发送/提交

系统将自动执行：
- **语音识别与转录**（基于 API）
- **视频自动合成**

> 首次合成耗时约 **11 分钟**（实际时间取决于素材大小和复杂度）

### 步骤 6：预览与迭代修复

合成完成后，点击链接即可在线预览视频。

**常见问题及修复方式：**

| 问题 | 解决方式 |
|---|---|
| 组件未正常显示 | 用自然语言告诉 Codex："组件未显示，请修复" |
| 文字重叠 | 截图 + 文字描述问题，发送给 Codex |
| 布局错位等其他问题 | 截图辅助定位 + 自然语言描述 |

每次修复后 Codex 会重新生成视频，再次预览，反复迭代直到成片满意为止。

---

## 三、最终成片效果

成品视频包含以下元素：
- **字幕**（自动生成）
- **频道名称标签**（品牌标识）
- **图标展示**
- 整体视觉效果精美，无需任何手动剪辑操作

---

## 四、关键总结

```
素材准备 → 安装工具 → 创建项目 → 配置 API → 自然语言提示 → 自动合成 → 迭代修复 → 成片
```

**核心优势：**
- 全程零专业剪辑软件
- 全程自然语言交互，无需编写代码
- 工具完全免费开源
- 通过截图 + 文字描述即可精准定位和修复问题

**一句话概括：** 用编程智能体（Codex / Open Code）作为"中间人"，把自然语言指令翻译成 Hyper Frames 能理解的操作，实现从素材到成片的全自动流水线。
