# GPT-SoVITS V2 Pro 完整教学：复刻自己的声音！结合 OpenClaw 实现 AI 自动化短影音口播

> [院长G大](https://space.bilibili.com/439305065) | [BV1AyArzLEnX](https://www.bilibili.com/video/BV1AyArzLEnX) | 时长 14:58

## 一、简介
本视频介绍了如何使用 GPT-SoVITS V2 Pro 工具复刻个人声音，并通过 API 结合 OpenClaw（或 OpenCode）的 skill 实现 AI 自动化短影音口播。背景是 UP 主之前分享过使用 OpenCode skill 将长影片剪辑成短影片，并由语言模型生成口播稿；如果不想自己录制声音，就可以用 GPT-SoVITS 复刻声音，然后通过 API 将字幕转换为语音，最后烧录到短影片中。

## 二、工具介绍
- **GPT-SoVITS V2 Pro**：一个声音克隆工具，用于基于文本生成语音。UP 主强调，V2 Pro 是当前最佳版本，V3 和 V4 已被弃用。
- **GitHub 仓库**：提供详细英文教程，但有中文页面。下载整合包模型时，根据操作系统选择 Windows 或 macOS 版本。
- **API 服务**：启动后，通过 OpenClaw 的 skill 调用本地 API，实现自动化语音生成。

## 三、硬件要求
- **显卡必要性**：训练声音模型需要显卡；如果没有，训练可能失败或效果差。建议至少 NVIDIA 3080 或 4090 显卡。
- **云服务器替代方案**：如果本地无显卡，可租用云 GPU 服务器。价格示例：3080 显卡约 1 美元/小时，4090 显卡约 2 美元/小时。UP 主推荐使用云平台，因为便宜且方便。

## 四、实操步骤
### 4.1 安装与启动
1. 从 GitHub 下载 GPT-SoVITS V2 Pro 整合包。
2. 安装后，首次启动点击“instar”进行初始化。
3. 点击“go web UI”启动用户界面，同时双击打开 API 服务终端（不要关闭终端窗口，否则服务停止）。

### 4.2 训练声音模型（以云服务器为例）
1. **租用云服务器**：
   - 选择提供 GPT-SoVITS 镜像的云平台，选择 V2 Pro 版本。
   - 创建并开机实例，通过 Jupyter Lab 访问。
2. **上传音频**：
   - 准备音频文件（例如从影片中提取的 7 分钟声音）。
   - 上传到云服务器文件夹，复制路径并添加前缀 `/root/`（避免路径错误）。
3. **语音切分**：自动将长音频切分成小段。
4. **语音识别**：生成字幕，但可能不准确，需手动修正。
5. **音频标注**：修正字幕后保存，确保文本准确（中文或英文）。
6. **微调模型**：
   - 选择 V2P 版本，默认参数或调整（如训练轮数 20-24，频率 5）。
   - 点击“一键三连”或分别点击按钮进行训练。
7. **训练完成**：下载训练好的模型文件（通常存放在 `GPT_WAI` 和 `SOVIWAI` 文件夹中，或版本特定文件夹如 `v2po`）。

### 4.3 本地使用与推理
1. **复制模型到本地**：将云服务器下载的模型文件放入本地 GPT-SoVITS 安装目录的对应文件夹。
2. **启动本地 UI**：打开 GPT-SoVITS Web UI，进入 TTS 推理界面。
3. **生成语音**：
   - 上传参考音频（建议 10 秒以内，实测 16-17 秒可能无误）。
   - 填写要合成的文本。
   - 调整参数，如 top_k（推荐 34，可根据电脑情况调整）。
   - 点击“合成语音”生成音频。
4. **API 集成**：保持 API 服务运行，通过 OpenClaw 的 skill 自动化调用，生成口播音频并烧录到短影片。

## 五、参数与建议
- **训练参数**：轮数建议 8-12 或 20-24，频率 5。可使用 AI（如 ChatGPT）查询优化参数。
- **推理参数**：top_k 值推荐 34，但需个人尝试调整。
- **音频质量**：确保训练音频清晰，字幕准确，以提升模型效果。
- **云服务器使用**：选择下载量高的镜像，简化安装流程。

## 六、总结
核心要点是：通过 GPT-SoVITS V2 Pro 复刻个人声音，利用云服务器训练模型（解决本地硬件限制），然后在本地通过 UI 或 API 生成语音。结合 OpenClaw 的 skill，可实现 AI 自动化短影音口播，将字幕转换为逼真语音并集成到视频中，极大提升内容创作效率。整个过程涉及工具安装、模型训练、参数优化和自动化集成，适合有基础硬件或愿使用云服务的用户。
