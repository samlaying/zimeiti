## Claude Code + PPT Skill 制作高级感 HTML 演示文稿笔记

### 一、PPT Skill 简介与特点
- **来源与热度**：该 Skill 来源于小红书博主张杂拉，其 GitHub 项目已获得 1.4 万星标，反响很好。
- **核心优势**：
    - **设计高级感**：整体风格简洁、有高级感。
    - **交互与动效**：交互酷炫，动效丰富。
    - **便捷部署**：支持一键部署至 Vercel 或 GitHub Pages，生成长期可分享链接。
    - **格式新颖**：生成 HTML 格式演示文稿，区别于传统 PPT，体验有新鲜感。
    - **兼容与可改**：支持将传统 PPT 格式转换为 HTML 格式，并可在生成后的基础上进行修改。
- **实例效果**：UP主展示了已部署的演示文稿，包含鼠标动效、图片展示、页面翻转、元素凸显效果，且支持在线编辑修改。

### 二、下载与安装 PPT Skill
#### 方法一：通过 Claude Code 对话安装（推荐）
1. 在 GitHub 上复制该 Skill 的仓库链接。
2. 在 Claude Code 对话框中，将链接发送给它，并指示：“帮我下载并安装这个 Skill”。

#### 方法二：手动安装
1. 在 GitHub 页面点击 “Code” -> “Download ZIP” 下载压缩包。
2. 解压缩下载的文件。
3. 在 VS Code 中打开你的 Claude Code 项目。
4. 找到项目的 `Skill` 文件夹。
5. 将解压后的文件夹（通常为 `frontend slides`）拖拽到 `Skill` 文件夹内。
    - **注意**：下载的文件夹可能带有后缀（如 `-main`），需去掉后缀，以确保在 VS Code 中调用指令时名称一致。

#### 安装成功验证
- 在 Claude Code 对话框中输入斜杠 `/`，然后输入 `frontend slides`。如果能自动补全并出现相关说明，则表示安装成功。

### 三、使用 PPT Skill 制作演示文稿步骤
1. **提出需求**：向 Claude Code 描述你的制作目标。示例：“我想为我的人工智能初创公司制作一份推荐演示文稿”。
2. **需求澄清**：Skill 会主动询问一系列问题以明确需求，例如：
    - 演示文稿的使用场景（路演、教学等）。
    - 希望的幻灯片页数。
    - 是否已有框架或内容。
    - 希望最终网页是可编辑的还是仅供展示。
3. **选择风格**：根据你的需求，它会生成 **3 个独特的风格预览文件** 供你选择。挑选后告知它你喜欢的风格编号。
    - **注意**：UP主在实际操作中发现，选择的风格与最终生成效果可能不一致。例如，选择了暖色调，但得到的是蓝紫色科技感风格。此时可以要求它修改，但需注意效果可能不理想，有时最终会回到初始风格。
4. **微调与优化**：在生成的基础上，可以进行进一步调整，例如：
    - 通过截图或文字描述，告诉它需要调整的图标位置、Logo 大小等。
    - 直接指出乱码等错误让其修改。
5. **添加高级功能**：
    - **鼠标动效**：可以要求添加鼠标动效，并会提供多种选择（如光标效果、跟随光运、卡片放大旋转、光标轨迹、旋转秒边填充等）。选择喜欢的效果序号即可应用。
    - **图片与链接**：要求添加图片展示，并可链接到外部网站（如小红书）。需要将图片文件放到项目工作区目录，并按其识别的格式命名，它会自动替换。

### 四、部署上线分享
1. **一键部署**：在 Skill 内，可以直接告诉它要一键部署到 Vercel 或 GitHub Pages。
2. **使用终端**：如果插件模式下能力受限（如配额用完），推荐在 Claude Code 的终端中进行部署操作。UP主使用的是自己的 API（如 OneAPI）。
3. **获取链接**：部署完成后，它会提供一个链接，即可访问已上线的演示文稿。
4. **推荐插件**：安装 `Cline TokenLine` 状态栏插件，可在终端底部实时显示调用的模型、项目名称、Token 使用量等信息。

### 五、总结
- **核心要点**：本教程展示了如何在 Claude Code 中集成并使用一个功能强大的 PPT 制作 Skill，以一键式、对话交互的方式快速生成具有高级设计感、丰富动效的 HTML 格式演示文稿，并实现在线分享。流程涵盖了从技能安装、需求沟通、内容生成、自定义调整到最终部署上线的完整环节。
- **关键经验**：注意风格选择与生成结果可能存在差异，需耐心调整；善用终端进行操作以获取完整能力；利用状态栏插件监控资源使用。
