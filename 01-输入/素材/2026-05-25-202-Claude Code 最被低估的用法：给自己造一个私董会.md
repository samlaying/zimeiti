# Claude Code 最被低估的用法：给自己造一个私董会

> [Agent黄叔](https://space.bilibili.com/25591280) | [BV1NSDxBkEXx](https://www.bilibili.com/video/BV1NSDxBkEXx) | 时长 4:58

## 私董会技能概述
- **核心概念**：利用Claude Code模拟一个由12位顶级思想家组成的虚拟私董会，基于用户提供的商业议题，进行多轮辩论并提供决策建议。
- **核心价值**：将个人决策转化为集体智慧交锋，获得多元视角和深度洞察，突破个人思维局限。

## 案例演示：从100万到1000万的决策
- **议题背景**：UP主“黄叔”作为个人运营商（OPC），年收入100万，考虑组团队扩大到年收入1000万。
- **虚拟顾问团构成**：选取了6位思想家作为示例，包括南怀瑾、纳瓦尔、老毛（毛泽东）、张一鸣、乔布斯、芒格。
- **三轮讨论交锋**：
  1.  **第一轮观点**：
      - **南怀瑾**：先厘清100万的利润结构和时间投入，确定需求真实性。
      - **纳瓦尔**：带条件反对，认为管理团队可能不是黄叔的“特定知识”（Specific Knowledge），需反思是否享受管理过程。
      - **老毛（毛泽东）**：主张抓住核心矛盾（类似“农村包围城市”），明确游击任务（当前阶段核心目标）再行动。
      - **张一鸣**：将工作分为“需要我”和“不需要我”两类，只有“不需要我”的工作才可能委托，关键在于评估这部分工作的比例。
      - **乔布斯**：坚决反对扩张，建议“先专注，后扩展”（Focus first, Expand later）。
      - **芒格**：从成本收益角度分析，招人的收入确定性必须超过其成本。
  2.  **关键分歧点**：
      - AI杠杆能否替代团队。
      - 发展策略应先聚焦还是先找主要矛盾。
      - 从百万到千万，是量的增长还是质的改变（加法问题还是乘法问题）。
  3.  **决议与建议**：
      - **共识**：100万到1000万不是同一件事做更多，而是要换一件事做。
      - **风险地图**：指出扩张可能面临的关键风险。
      - **行动三步走**：
        1.  **诊断现状**：深入分析当前业务结构。
        2.  **聚焦并应用AI杠杆**：优先利用AI提升效率，而非直接组建团队。
        3.  **验证后再考虑团队**：只有当业务模式被验证可规模化后，才谨慎引入团队。
      - **核心比喻（蓝天×芒格）**：团队是放大器，不是发动机。发动机（商业模式/个人核心能力）没变，放大器加再多也没用。

## 私董会技能的创建过程
- **灵感来源**：将流行的“人物技能”（如纳瓦尔、芒格的思维模型蒸馏而成的技能）组合成一个互动辩论体。
- **创建步骤**：
  1.  **收集素材**：先制作或获取一系列“蒸馏技能”合集，这些技能封装了不同思想家的核心思维模型。
  2.  **多Agent分析**：Claude Code派出11个Agent去研究每个技能（思想家），解析其思维图谱、潜在分歧点。
  3.  **构建辩论流程**：设计一套多阶段工作流，确保顾问们能基于分歧充分辩论。
  4.  **技能整合与开源**：最终创建一个完整的Skill，并发布到GitHub供下载使用。

## 技能的工作流程
1.  **接收议题**：用户提供需要决策的商业问题或背景。
2.  **信息补充**：从不同视角提出3-5个关键问题，让用户补充细节，使议题更完整。
3.  **顾问遴选**：从12位顾问中，选择5-7位与议题最相关的专家。
4.  **多轮辩论**：顾问们基于各自思维模型给出判断，并在关键分歧点上进行辩论交锋。
5.  **生成决议**：汇总辩论内容，输出结构化的决议报告，包括共识、分歧、风险分析和行动建议。
6.  **输出形式**：可生成为HTML网页，方便阅读和存档。

## 实操使用示例
- **简单调用**：输入“私董会”并附上你的问题。
- **交互流程**：系统会先向你提出关键问题要求补充信息；你回答后，系统召集相关顾问进行多轮发言辩论。
- **结果获取**：最后可要求生成格式清晰的HTML报告，一目了然。

## 总结核心要点
黄叔展示的“私董会”技能，是利用Claude Code将多个知名人物的思维模型封装为可交互、可辩论的AI代理团，通过模拟多角色、多轮次的深度辩论，为个人复杂的商业决策提供结构化、多元化的分析与建议。其核心在于**利用AI杠杆放大个人思考的广度与深度，而非简单地寻求唯一答案**，通过虚拟的思想交锋来揭示盲点、评估风险、厘清行动路径，最终实现“给自己造一个私董会”的高效决策支持。该技能构建流程清晰，使用简单，旨在将顶尖思想家的集体智慧赋能给每一个需要做重大决策的个体。
