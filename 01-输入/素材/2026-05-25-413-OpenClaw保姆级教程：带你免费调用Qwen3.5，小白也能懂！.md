# OpenClaw保姆级教程：带你免费调用Qwen3.5，小白也能懂！

> [磊哥聊AI](https://space.bilibili.com/432174503) | [BV19CZeBPEfa](https://www.bilibili.com/video/BV19CZeBPEfa) | 时长 5:48

## 一、 模型简介与核心优势
1.  **Qwen3.5（通义千问3.5）**：
    *   国产重量级开源大模型，于除夕夜正式发布。
    *   号称“最强的开源模型”，能力对标GPT-5.2、Claude 4.5，可与国际顶级模型抗衡。
2.  **Ollama Cloud模式**：
    *   Ollama平台新推出的功能，模型名称后带有 `:cloud` 标识（如 `qwen3.5:cloud`、`deepseek-r1:cloud`、`minimax-m1.5:cloud`）。
    *   **核心优势**：无需本地部署满血版大模型，**突破本地硬件限制**。模型由Ollama的服务器远程部署和运行，用户通过客户端即可免费调用。

## 二、 从零开始：四步接入Qwen3.5
### 第一步：安装Ollama客户端
*   **前提**：确保安装的是**最新版**Ollama。若已安装但版本过旧，请先更新。
*   **操作**：从官网下载安装包，执行常规安装（一路“下一步”）。

### 第二步：在Ollama中部署Qwen3.5模型
*   由于使用Cloud模式，此步实质是“获取/接入”云端模型，而非本地下载运行。
*   **操作**：
    1.  以管理员身份运行CMD（命令提示符）。
    2.  复制并执行以下命令，等待模型信息拉取完成：
        ```bash
        ollama run qwen3.5:cloud
        ```

### 第三步：登录Ollama账号
*   必须登录才能使用Cloud模型服务。
*   **操作**：
    1.  执行以下命令，系统会自动打开浏览器登录页面（或手动访问页面地址）：
        ```bash
        ollama login
        ```
    2.  在页面上完成以下流程：
        *   **注册**：若无账号，使用邮箱注册，并完成手机验证。
        *   **登录**：输入账号信息登录。
    3.  登录成功后，在命令行会显示登录成功信息。

### 第四步：启动OpenClaw并连接Qwen3.5
*   **操作**：
    1.  执行启动命令，指定使用Ollama的Qwen3.5 Cloud模型：
        ```bash
        openclaw --model ollama/qwen3.5:cloud
        ```
    2.  等待OpenClaw启动完成。

## 三、 验证与使用
1.  打开命令窗口或WebUI（如`飞速碗`）。
2.  新建一个会话，输入问题进行测试。
3.  **示例**：提问“告诉我你使用的大模型是啥”，模型应回复确认自己是`Qwen3.5`模型。
4.  至此，成功通过OpenClaw免费调用了Qwen3.5模型。

## 四、 重要注意事项：免费额度限制
*   Ollama的云端模型调用是**有成本**的，因此为用户提供了**免费额度**。
*   **额度查看**：在Ollama账号页面的“账号设置”中，可看到“云端模型使用限制”的百分比进度条。
*   **额度规则**：
    *   **每周**有总限额。
    *   **每四个小时**也有调用次数限制（大约可供调用100次左右，具体消耗与对话上下文长度相关）。
*   **建议**：合理规划使用，避免额度耗尽。该免费额度相比其他平台更为实在。

## 总结
本教程的核心是利用Ollama平台的**Cloud模式**，结合**OpenClaw**客户端，实现了对国产强开源模型**Qwen3.5的免费调用**。关键点在于通过`ollama run qwen3.5:cloud`和`ollama login`命令完成云端模型的接入与身份验证，从而突破本地硬件限制。用户需注意Ollama提供的**每周及每四小时的免费调用额度限制**，以便高效使用。整个过程仅需四步，适合小白用户操作。
