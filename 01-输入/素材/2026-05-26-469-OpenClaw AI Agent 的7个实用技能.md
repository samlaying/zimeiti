# OpenClaw AI Agent 的7个实用技能

> [辣条自由](https://space.bilibili.com/403414695) | [BV1M6Z2BREYA](https://www.bilibili.com/video/BV1M6Z2BREYA) | 时长 41:58

## OpenClaw AI Agent 的7个实用技能笔记

### 核心理念与概述
- **AI Agent的应用场景**：作者使用运行在Mac mini上的OpenClaw AI Agent来管理其庞大的**内容运营（Content Operation）**。该运营体系拥有**超过250万粉丝**，分布在**15个不同账号**上。
- **Agent的作用**：该AI Agent负责起草所有推文、管理项目、完全掌控Notion和Linear、提取竞争对手视频、生成缩略图等。作者只需通过对话即可创建新技能，赋予Agent执行不同任务的能力。
- **视频核心**：分享作者花费一整周时间亲手构建此AI Agent后，总结出的**七个最实用的技能**及其设置方法。

### 一、环境设置与基础认知
1.  **安装与连接**：
    - 从OpenClaw AI网站下载并安装。
    - 推荐连接**Telegram**作为专用聊天应用，因其设置简单、功能齐全，支持重置、终止等命令。
    - 建议为AI Agent设立**专用应用**，避免与个人聊天混杂。
2.  **理解文件结构**：
    - AI Agent运行时，会在电脑特定文件夹中更新小文件。
    - 通过询问Agent或使用Finder（显示隐藏文件：`Command + Shift + .`）找到文件路径。
    - **核心文件夹**：`Workspace`，其中包含：
        - `Agent.md` / `Identity.md`：定义AI助手的**核心个性、执念和兴趣**（例如，对内容、营销细节的执着）。
        - `Mission.md`：使命文件。
        - **技能（Skills）文件夹**：存放各种技能文件，是赋予Agent能力的关键。

### 二、七个实用技能详解

#### 技能一：Notion技能
- **功能**：让AI Agent能够**完全访问和操作你的Notion工作区**。
- **用途**：创建、移动、重命名文档和页面，自动将生成的内容放入指定笔记本。
- **设置步骤**：
    1.  在Notion中创建一个**集成（Integration）**。
    2.  将生成的**API密钥**提供给AI Agent。
    3.  在Notion的对应页面/数据库中，**为该集成启用访问权限**。
    4.  Agent会自动创建技能文件。
- **实操示例**：
    - 创建视频脚本文档并获取链接。
    - 提取竞争对手视频转录并存入Notion，用于后续内容创作参考。
    - 通过对话**设定偏好**（如“普通文档放Webclaw笔记本，内容脚本放内容笔记本”），Agent会将其记忆到内部文件（如`ToseM.md`）中。

#### 技能二：转录提取器技能
- **功能**：利用**SuperData API**，从Instagram、YouTube等主流社交媒体平台**提取视频的完整文字转录**。
- **用途**：为内容创作提供上下文，分析竞争对手。
- **设置步骤**：
    1.  研究SuperData API并获取**API密钥**。
    2.  告诉Agent使用该API创建技能。
- **成本**：每月约15美元可获得3000次调用。

#### 技能三：Twitter/X技能
- **功能**：通过**Tapful API**，实现推文的**起草、安排和发布**。
- **用途**：自动生成贴合作者风格和受众的推文描述。
- **设置步骤**：
    1.  在Tapful设置页面注册并找到**API密钥**。
    2.  可选择为Agent分配特定账号的访问权限。
- **工作流程示例**：
    1.  上传视频到Notion。
    2.  Agent分析视频内容，并参考历史上表现好的推文。
    3.  自动生成多个推文描述选项。
    4.  用户选择后，Agent将草稿创建在Tapful中或直接发布。

#### 技能四：Linear技能
- **功能**：读取或写入**Linear**项目管理平台中的任务。
- **用途**：查询任务分配、总结待办事项、规划未来内容日历。
- **设置**：通过对话引导Agent获取Linear的API密钥，并可精细控制读写权限。

#### 技能五：缩略图情绪版技能
- **功能**：结合**Cirp API（用于抓取高质量缩略图）** 和 **Nano Banana（用于人脸替换）**，自动生成带有作者面部的缩略图灵感集。
- **用途**：快速创建个性化缩略图情绪板。
- **实操示例**：
    1.  命令Agent搜索关于某主题的YouTube热门视频，抓取缩略图。
    2.  Agent使用提供的作者自拍，将缩略图中的人物面部替换为作者。
    3.  最终生成一份包含原图和编辑后图片的PDF文档。

#### 技能六：Google图片技能
- **功能**：利用**Suppie API**快速搜索Google图片。
- **用途**：为创作的脚本或文档**自动添加相关图片**。
- **特点**：简单但实用，能增强内容的可视性。

#### 技能七：接管方向盘技能
- **功能**：**扭转AI Agent的被动角色**，使其变得积极、紧迫，主动提问以推动任务完成。
- **用途**：用于需要快速推进、高质量输出的任务，如长视频脚本、广告脚本批量创作。
- **核心机制**：
    1.  用户说“**接管方向盘**”触发技能。
    2.  Agent识别任务上下文，**一次只问一个最关键的问题**，并用大写字母突出该问题。
    3.  用户回答后，Agent会根据回答**爆发式地给出下一步行动或内容**，营造紧迫感和心流状态。
    4.  最终自动生成高质量文档（如Notion脚本）。
- **价值**：将AI从工具转变为“主动的合作伙伴”，极大提升完成任务的效率和确定性。

### 三、总结
- **核心要点**：OpenClaw AI Agent通过集成各种外部服务（如Notion, Twitter/Tapful, Linear, SuperData, Google图片）的API，并将其封装成“技能”，从而成为了一个强大的内容运营自动化中枢。七个实用技能覆盖了**内容管理、创作、发布、项目管理到灵感生成**的全流程。其成功的关键在于：1）精心设计Agent的**核心人格与上下文**；2）通过对话**设定清晰的偏好和规则**；3）利用“**接管方向盘**”等技巧将Agent转变为主动的执行伙伴，最终实现**高效、个性化且规模化**的内容运营。建议从单一、具体的技能开始构建，逐步扩展，避免技能过多导致Agent混乱。
