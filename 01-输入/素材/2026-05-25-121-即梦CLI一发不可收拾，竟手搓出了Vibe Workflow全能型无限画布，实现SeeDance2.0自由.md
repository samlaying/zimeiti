# 即梦CLI一发不可收拾，竟手搓出了Vibe Workflow全能型无限画布，实现SeeDance2.0自由

> [地质大学博士说AI](https://space.bilibili.com/43149384) | [BV1bdDRBcE8M](https://www.bilibili.com/video/BV1bdDRBcE8M) | 时长 4:59

## 介绍
即梦CLI（一个AI工具）近期推出，能够创建高自由度的无线画布，名为WebWalk Pro。该画布类似CompUI界面，允许节点自由组合，专注于AI生成内容，如图像和视频。UP主“地质大学博士说AI”在视频中演示了如何利用即梦CLI实现SeeDance2.0的自由应用，展示了其强大的功能。

## 功能特点
### 模型选项
- **图片模型**：支持多种版本，包括5.0、4.0、2K、4K，提供不同分辨率和质量选择。
- **视频模型**：支持参考生视频，允许上传最多9张图片和3个视频作为参考，包括SeeDance2.0的满血版，用于高质量视频生成。
- 所有模型均集成了即梦的满血版功能，确保性能和效果。

### 账号关联
- 画布直接关联用户自己的即梦账号，无需额外进行API相关操作。
- 积分和登录信息与即梦账号完全匹配，简化了使用流程。

### Workflow生成
- 通过豆包（一个agent）以对话方式自动创建workflow节点，无需手动搭建。
- 豆包能智能理解用户输入，自动生成相应的图像或视频生成流程，提升效率。

## 实操示例
### 示例1：生成猫图
1. 在画布的对话框中输入指令：“生成一张图，内容是一只猫”。
2. 豆包自动创建workflow，并加载到画布中。
3. 在workflow中选择模型版本（如4.6），点击生成按钮。
4. 等待图像生成完成，图像会显示在对应节点中。

### 示例2：生成猫图视频
1. 输入指令：“生成猫图之后还需要让它动起来”。
2. 加载生成的workflow，包含图像生成节点和视频生成节点。
3. 选择视频模型版本（如SeeDance 2.0），设置视频时长为8秒。
4. 先生成猫图：在图像节点选择模型4.6，点击生成。
5. 图像生成后，在视频节点点击生成按钮，等待视频完成。
6. 视频生成后，可在画布中直接播放查看。

### 示例3：多模态应用（脱口秀舞台定装照）
1. 上传两张角色图片到画布。
2. 输入描述：“生成这两个角色在一个脱口秀舞台的定装照”。
3. 豆包创建workflow，包含上传的图片作为参考。
4. 加载workflow，点击创建按钮，生成舞台图像。
5. 可选：连接视频节点，进一步生成视频或进行视频延长。

## 总结
即梦CLI作为一个桥接工具，能够连接任何AI平台或服务，通过WebWalk Pro无限画布实现自动化workflow生成。它支持图像和视频的多模态创作，提供丰富的模型选项和用户友好的对话式操作，极大地提升了AI内容生成的效率和自由度。核心要点在于：利用CLI自动化搭建workflow，结合即梦模型完成从图像到视频的完整创作流程，无需手动编程或API配置。
