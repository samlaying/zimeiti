# 20+ skills，打造跨境电商AI超级选品大师

> [飞哥Flynn](https://space.bilibili.com/3546790685051581) | [BV1d2XqBcEDj](https://www.bilibili.com/video/BV1d2XqBcEDj) | 时长 1:39

## 用AI打造跨境电商超级选品流程：核心方法论与实操解析

### 一、 核心工具与目标
*   **核心工具**: Fone Cloud / Cloud Code 的 Skills 功能。
*   **核心目标**: 通过AI进行用户导向的品类拓展和深度市场调研，生成详尽的选品分析报告，辅助决策。
*   **核心产出**: 超长、结构化的深度市场调研报告。

### 二、 具体操作步骤详解
1.  **建立个性化Skills（技能库）**
    *   **目的**: 教AI理解你的业务逻辑和需求模式。
    *   **内容示例**:
        *   预设用户身份：例如“我是一名AI矛盾玩具卖家”。
        *   预设拓展方向：希望AI思考“还能做什么产品”。
        *   **关键功能**: Skills中可设定AI必须主动询问用户进行信息补充，例如询问目标用户群体、产品故事等细节。

2.  **执行调研任务**
    *   **流程**: 用户输入初始指令（如“我想拓展卡牌包或Steam教育产品市场”） -> AI读取记忆与预设的Skills -> AI与用户交互，确认细节 -> AI执行深度调研。
    *   **信息输入**: 用户提供的目标品类、用户画像需求、产品故事等。

3.  **生成并分析报告**
    *   AI执行完毕后，自动生成全面的市场调研报告。
    *   用户需在报告中进一步筛选，例如选择“七分FPU”（可能为具体品类或数据指标）进行更深度的调研，如“卡牌包”或“Steam编程教育”。

### 三、 AI生成的深度市场分析报告内容
以下为报告涵盖的关键模块（示例为并行分析的“卡牌包”与“Steam编程教育”市场）：

1.  **市场总览与宏观分析**
    *   提供思维导图，总览产品组合拓展方向。
    *   详述整个业务链路（从用户认知到售后）各阶段可做的产品。

2.  **具体市场深度调研（以“卡牌包”和“Steam教育”为例）**
    *   **数据来源**: 基于类似Sorftime（卖家精灵）或Mallin的市场数据。
    *   **报告核心内容**:
        *   **宏观市场数据**: 整体市场规模、趋势。
        *   **价格分布**: 分析市场价格带分布情况。
        *   **竞争格局分析**:
            *   **统治者**: 谁是该市场的领导者（如亚马逊霸主）。
            *   **高潜黑马**: 有哪些快速增长的新兴竞争对手。
            *   **不同动态（动向）**: 市场竞争的侧重点是什么。
        *   **销量与价格分析**: 具体销量数据与价格分布的结合分析。
        *   **用户需求深度挖掘**:
            *   **用户评论数据**: 分析现有产品的用户评价。
            *   **未被满足的需求**: 从评论中识别市场机会和产品改进点。
            *   **用户统治**: 可能指用户偏好或主流用户画像的统治性趋势。
    *   **可视化呈现**: 包括玫瑰图（可能是雷达图或饼图的变体）等图表，直观展示数据。

### 四、 总结：核心要点
本次分享的核心在于，利用AI工具（如具备定制Skills的Cloud Code）将选品工作系统化、智能化。它通过预设技能和交互式调研，能够自动从用户需求出发，生成涵盖市场宏观数据、竞争格局、用户痛点和价格策略的深度分析报告，从而高效、全面地评估新品类机会，实现数据驱动的智能选品。
