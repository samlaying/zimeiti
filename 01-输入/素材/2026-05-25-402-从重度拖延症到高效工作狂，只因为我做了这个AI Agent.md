# 从重度拖延症到高效工作狂，只因为我做了这个AI Agent

> [技术蛋老师](https://space.bilibili.com/327247876) | [BV1pofFBnEqB](https://www.bilibili.com/video/BV1pofFBnEqB) | 时长 9:40

## 一、 项目目标与整体架构
- **核心需求**：创建一个能治疗拖延症、防止摸鱼的AI Agent。
- **Agent能力模型（有脑袋、眼睛、嘴巴、手、能隐身）**：
    - **眼睛**：通过屏幕截图识别用户行为（是否在摸鱼）。
    - **脑袋**：利用多模态大模型深度思考并判断状态。
    - **记忆**：记住多次摸鱼行为。
    - **嘴巴**：通过语音发出警告或讽刺提醒。
    - **手**：能自动关闭摸鱼使用的软件。
    - **隐身**：能在后台静默运行。

## 二、 技术选型与实现过程
### 2.1 模型与平台选择
- **多模态模型选择**：豆包Seed 2.0 Pro。
    - 优势：具备深度思考的视觉语言能力，不仅是简单OCR，能理解复杂场景（如区分弹幕、歌词、特定游戏/影视内容）。
    - 调用平台：通过**火山引擎**的**火山方舟**平台。
- **代码辅助模型**：豆包Seed 2.0 Code，用于生成和优化Python代码。

### 2.2 环境配置与项目初始化
1. **创建项目**：在火山方舟聊天框切换至“豆包Seed 2.0 Code”模型。
2. **明确提示词**：指示AI使用Python和**UV**（推荐的环境配置工具）来生成代码。
3. **生成代码**：AI生成用于调用火山方舟API、系统截图、图片转Base64的代码。
4. **初始化项目**：
    - 创建并进入指定目录。
    - 运行 `uv init` 初始化项目。
    - 运行 `uv install` 安装依赖。

### 2.3 API接入与配置
1. **获取模型ID**：在火山方舟平台，确保“豆包Seed 2.0 Pro”已开通，并记录模型ID。
2. **创建API Key**：在火山方舟控制台的“API Key管理”中创建API Key。
3. **配置环境文件**：将API Key和模型ID填入项目根目录的`.env`文件中。

### 2.4 代码实现与功能迭代
- **基础版（一次运行）**：
    - 在`main.py`中粘贴AI生成的代码。
    - **关键步骤**：需提前为运行终端的应用（如WezTerm）授予系统截屏权限。
    - 功能：截屏 -> 发送给模型分析 -> 模型返回结论（是否摸鱼及具体内容）。
- **记忆版（定时监控）**：
    - 修改代码，让Agent每半分钟自动截屏分析一次。
    - 记录历史行为，并在被发现摸鱼时进行语音提醒。
- **语音警告版**：
    - 调用MacOS系统自带的人声合成功能。
    - 让Agent的提醒话语每次不重复，且具有“阴阳怪气”的讽刺风格。
- **终极版（自动执行）**：
    - **白名单机制**：设置不被关闭的软件。
    - **自动关闭**：当累积摸鱼次数达到阈值，Agent自动关闭指定的摸鱼软件。
    - **后台运行**：
        - 创建启动和停止脚本，并配置权限。
        - 后台运行后，可通过`tail`命令查看日志。
        - 建议调整截屏间隔以节省Token。

## 三、 Agent核心功能与能力展示
- **精准视觉识别**：
    - 能区分：看漫画、玩游戏、看视频（B站、电视剧、动画）、看剧的具体内容（如《鱿鱼游戏》、《疯狂动物城》）。
    - 能理解状态：区分“准备玩游戏”和“正在玩游戏”。
    - 能忽略干扰：在视频全屏、界面重叠、有弹幕/歌词的情况下仍能准确判断。
- **深度思考过程**：模型会进行内部推理和反思（如“左边看漫画，右边有终端，不对…”），而非简单匹配关键词。
- **动态语言交互**：警告语句富有变化和情绪（从提醒到讽刺再到威胁），增强了用户体验和“威慑力”。
- **自动执行能力**：成功实现自动关闭摸鱼软件（如播放器）的功能，且一次完成，无需调试。

## 四、 实践心得与建议
1. **模型实用性**：跑分高的模型未必落地好，豆包Seed 2.0系列在**视觉理解（VLM Think）、工具调用和执行**方面表现出了出色的实用性。
2. **开发效率**：借助AI代码生成，两轮对话即可实现一个可用程序。
3. **Token优化**：后台运行的Agent建议采用**定时或动态间隔**截屏，避免不必要的Token消耗。
4. **重要提醒**：在他人电脑上运行此类监控Agent前，务必告知并获得许可。

## 五、 总结
本视频完整展示了一个基于豆包Seed 2.0多模态大模型的“防摸鱼”AI Agent从构想到落地的全过程。其核心是通过定时截屏，利用模型强大的视觉理解和深度思考能力来精准识别用户的各类摸鱼行为，并结合语音提醒和自动关闭软件等功能进行干预。整个实践不仅解决了个人效率问题，更是一次出色的AI Agent开发实战，展示了如何利用先进的VLM模型构建具备感知、思考、记忆和执行能力的实用自动化工具。
