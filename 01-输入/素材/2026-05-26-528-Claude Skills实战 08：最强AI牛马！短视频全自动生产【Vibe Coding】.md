# Claude Skills实战 08：最强AI牛马！短视频全自动生产【Vibe Coding】

> [ai798Lab](https://space.bilibili.com/2094023919) | [BV1x8zeBME6t](https://www.bilibili.com/video/BV1x8zeBME6t) | 时长 3:01

## Claude Skills 实战：短视频全自动生产工作流
### 核心理念
利用 AI 工具链（以 Claude Skills 为核心）自动化完成从创意到成片的短视频全流程，无需人工后期处理，可快速、批量地生产内容。

### 实际效果演示
- **示例视频**：一段关于海洋生物的科普短视频。
- **质量表现**：视频质量尚可，包含问答对话、插画和连贯的画面。

### 自动化工作流详细步骤
#### 1. 生成对话脚本
- **输入**：一个科普问题（如“天空为什么是蓝的”）。
- **AI处理**：
    - 根据问题设计两轮问答对话。
    - 将对话切分为**四个视频切片**。
    - 为每个切片生成结构化 **JSON 文件**，包含：
        - 说话者
        - 台词
        - 时长
        - 场景描述（元素、肢体/表情变化）
        - 所需插画描述

#### 2. 生成关键帧图片（核心步骤）
- **目的**：确保多个短片段拼接后视频连贯。
- **原理**：
    - 当前主流AI视频生成工具（如Sora, Kling等）单次生成时长上限通常在 **8-12秒**。
    - 生成长视频需先分片段，再拼接。
    - **关键帧**：前一片段的**尾帧**与后一片段的**首帧**必须是同一张图片。
- **操作**：
    - 读取第一步生成的JSON中的场景描述Prompt。
    - 调用 **GPT-4.5模型** 生成关键帧图片。

#### 3. 生成插画
- 根据每个切片的对白内容，生成对应的解释性插画。

#### 4. 生成视频片段
- **生成方式**：使用支持**首尾帧固定**的视频生成模型（如提到的 **Sedans 1.5 Pro**）。
- **特点**：该模型还支持**音画同步**，可直接输出 **1080P 竖屏视频**。
- **替代方案（对音频控制要求更高时）**：
    1. 使用其他工具（如可灵）生成**无声视频**。
    2. 使用 **ElevenLabs** 或 **MiniMax** 的 TTS 模型单独生成配音。
    3. 手动合成画面与音频。

#### 5. 合成与后处理（通过命令行调用 Final Cut Pro）
- **步骤一**：将插画置入每个视频切片的指定位置。
- **步骤二**：将所有切片片段合成为一个完整视频。
- **步骤三**：为视频自动添加字幕。

### 核心要点总结
通过 Claude Skills 搭建的这一自动化工作流，将短视频生产拆解为“脚本生成 → 关键帧设计 → 插画创作 → 分片段视频生成 → 智能合成”的标准化流程。它核心解决了长视频生成中“片段衔接”的连贯性问题，并利用命令行工具自动完成后期合成。这套方法显著降低了科普、故事类短视频的创作门槛，使得批量、快速地验证爆款内容创意成为可能。
