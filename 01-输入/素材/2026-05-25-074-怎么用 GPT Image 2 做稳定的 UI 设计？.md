# 怎么用 GPT Image 2 做稳定的 UI 设计？

> [oil欧呦](https://space.bilibili.com/13030183) | [BV1LnR5BqEdb](https://www.bilibili.com/video/BV1LnR5BqEdb) | 时长 8:04

## 如何用 GPT Image 2 做稳定的 UI 设计？

### 一、 核心挑战
1.  **创造力**：如何激发 GPT Image 2 的创造力，使其设计出美观、有设计感的 UI，而不是局限于现有风格。
2.  **一致性**：如何确保生成的新页面 UI 与现有系统的整体设计风格（如侧边栏、顶部栏）保持一致，避免破坏现有框架。

### 二、 解决“一致性”问题
#### 方法尝试
1.  **提示词控制（失败）**：
    *   使用相同、统一的提示词生成 UI。
    *   **结论**：两次生成的结果差异明显，说明仅靠提示词无法保证稳定性。
2.  **参考图控制（成功）**：
    *   提供一张包含系统现有元素（如侧边栏、配色、图标风格）的截图作为参考图。
    *   **效果**：
        *   **高度一致**：生成的多张图中，参考图里已有的元素（如左侧栏）保持完全不变。
        *   **风格延续**：新生成的部分（如右侧内容区）会自动沿用参考图中的设计风格（如图标样式、配色、标签样式）。
        *   **提示词简化**：只需在提示词中强调“保持侧边栏与参考图完全一致”，并描述需要的新页面内容即可。

#### 参考图的两种使用策略
1.  **丰富参考图**：
    *   提供包含侧边栏和右侧部分元素的完整页面截图。
    *   **优点**：生成的 UI 风格与原系统高度一致，稳定性强。
    *   **缺点**：可能会限制 AI 的创造力，设计相对保守。
2.  **精简参考图（推荐）**：
    *   仅提供需要保留的框架元素截图（如只包含一个干净的侧边栏），右侧内容留白。
    *   **优点**：在保证框架一致的前提下，给予 AI 最大的发挥空间。
    *   **效果**：生成的 UI 设计感更强，排版更精致，配色更舒服，能带来全新灵感。例如，某次生成的结果被评价为“排版紧凑又不过密，配色舒服”。

### 三、 激发“创造力”的方法
#### 1. 参考图策略
*   使用上述的**精简参考图策略**是激发创造力的关键一步。

#### 2. 提示词编写策略（五种角度）
AI 会根据场景自动选择合适的策略。以下为五种测试过的策略：
1.  **叙事视角**：将用户的使用过程（用户旅程）写入提示词，而非直接罗列页面信息。
2.  **信息布局法**：列出页面需要展示的核心信息和资源，但不指定排版方式，让 AI 自行设计布局。
3.  **类比参考法**：要求 AI 参考特定产品的风格（如“做成像 Notion 一样的风格”）进行设计。
4.  **用户目标法**：描述用户浏览该页面后期望获得的结论或完成的目标。
5.  **情绪描述法（不推荐）**：描述页面希望带给用户的情绪。**测试发现此方法效果不佳，易导致 AI 幻觉，生成类似海报的功能页面。**

### 四、 自动化实现：Draw UI Skill
UP主已将上述最佳实践打包成一个名为 `Draw UI` 的 Skill。
#### 安装与使用步骤
1.  **安装 Skill**：
    *   通过 Git 仓库链接（仓库名：OilOil）或在终端中执行相应命令进行安装。
2.  **配置 API Key**：
    *   需要配置一个第三方图像生成服务的 API Key（如 ZenMax）。用户可自行更换为 OpenAI 官方或其他服务商，通过 AI Agent 辅助调整即可。
3.  **自动化工作流**：
    *   **自动学习**：Skill 会自动查看当前代码仓库中已有的设计资源。
    *   **参考图引导**：如需更佳效果，AI 会主动引导用户提供一张系统内具有代表性的页面截图（如 Dashboard 页面）。
    *   **智能生成**：AI 结合参考图、代码仓库中的设计规范，以及内置的多种提示词策略（叙事、信息布局、类比参考、用户目标），自动分析和生成最优提示词，最终输出 UI 设计稿。
    *   **迭代优化**：若对设计不满意，可让 AI 重新调整。

### 五、 总结核心要点
本视频系统性地解决了用 GPT Image 2 进行 UI 设计时面临的**创造力与一致性**两大矛盾。核心方法是采用**精简参考图策略**（仅提供必须保留的框架元素），在确保与现有系统风格一致的基础上，最大化激发 AI 的设计潜力。同时，通过多样化的**提示词策略**（如叙事、类比参考等）引导 AI 进行创意设计。最终，UP主将这些最佳实践封装成自动化的 `Draw UI` Skill，实现了从手动调试到一键智能生成的工作流提效。
