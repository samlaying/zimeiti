# 热门AI技巧分享：如何做高级感的网页UI

> [机器之心官方](https://space.bilibili.com/73414544) | [BV1LcAYzkEPZ](https://www.bilibili.com/video/BV1LcAYzkEPZ) | 时长 1:39

## **如何用AI制作高级感网页UI：保姆级教程**

### **核心理念与准备**
- **核心目标**：使用AI工具，快速将一份简单的简历转化为具有“多巴胺风”或高级感的专业求职网站。
- **主要工具**：
    - **Cursor**：用于编写和优化网页代码的核心AI编程工具。
    - **MidJourney**：生成视觉风格统一、具有审美一致性的参考图片和设计元素。
- **关键认知**：
    1. AI的复刻能力远强于其创造能力。先提供明确的框架和参考，再让AI执行，效果更好。
    2. 用视觉信息（截图）与AI沟通，比用文字描述（Prompt）更精准、高效。

### **实操步骤详解**

#### **第一步：利用AI生成网站初版**
1.  打开 **Cursor**。
2.  上传你的**MID简历**（或任何文本简历）。
3.  向AI下达指令：“根据这份简历，帮我生成一个求职网站”。
4.  AI将生成一个基础网站，但通常配色单一（如“经典蓝”），缺乏设计感和一致性。

#### **第二步：定义框架与视觉风格（核心原则1）**
- **原则**：**先统一视觉风格和框架，再写代码。**
- **操作**：
    1.  **搭框架**：在开始写代码前，先确定页面的布局结构。明确导航栏、按钮、图片、文字区域的位置。
    2.  **找参考**：上传一张你期望的**网站设计风格参考图**（例如，你喜欢的“多巴胺风”网页截图）。告诉AI：“按照这个风格来写”。
    3.  **统一配色**：
        - 使用 **Nano Banna** 等工具，根据参考图自动生成一套和谐的**色彩调色板**。
        - 让AI**提前在代码中定义好**这套配色方案、字体和排版层级，确保整个页面风格一致。
    4.  **使用模板**：可以上传一个包含“骨架”布局的HTML模板文件给AI，让它基于此模板填充内容和样式，效率更高。

#### **第三步：高效沟通与细节微调（核心原则2）**
- **原则**：**用截图作为与AI沟通的主要方式。**
- **操作**：
    1.  当需要修改具体细节时（例如，“让文字有圆角风格”），**直接截图标注**需要修改的部分。
    2.  上传截图给AI，并附上简单指令（如“优化这里”）。AI能通过视觉信息快速理解你的意图，比用文字描述“让文字边角更圆润”等Prompt更准确。
    3.  重复“截图-反馈-AI修改”的循环，进行多轮细节微调，直至达到满意效果。

#### **第四步：总结与优化循环**
- 通过以上步骤（**框架/风格先行 -> AI生成初版 -> 截图式微调**），你就能从一份简单简历迭代出一个设计感强、风格统一的高级感网页。
- 整个过程是循环的：用参考图设定方向 -> AI执行 -> 用截图反馈修正 -> AI再优化。

### **核心要点总结**
使用AI制作高级网页UI的核心在于：**先明确设计框架和视觉风格，再利用AI的强大复刻能力执行**。关键技巧是**善于使用截图与AI沟通**，这能大幅提升理解和执行的准确性。整个过程是“指导-生成-反馈”的迭代循环，结合Cursor、MidJourney等工具，能高效地将创意转化为专业设计。
