# Excalidraw MCP一键生成手绘风格架构图

> [去飞GoFly](https://space.bilibili.com/52846118) | [BV1XvZtBLEDa](https://www.bilibili.com/video/BV1XvZtBLEDa) | 时长 4:08

## 概述
Excalidraw MCP 是一款基于 MCP 协议的开源工具，旨在让大语言模型（如 Claude 或 ChatGPT）通过标准协议控制 Excalidraw 画布，从而一键生成手绘风格的流程图、架构图或思维导图。该工具实现了 AI 与画布的实时交互，用户可通过自然语言指令绘制和修改图形。

## 核心概念
### Excalidraw
- **定义**：一个手绘风格的在线白板，专用于绘制流程图、架构图和思维导图。
- **特点**：支持元素的实时修改、删除和调整，提供交互式绘图体验。

### MCP 协议
- **定义**：上下文协议（Model Context Protocol），充当工具与 AI 之间的桥梁。
- **作用**：实现 AI 与外部工具（如 Excalidraw）的实时通讯，使画布成为 AI 可操作的可编程平台。

## 功能和目的
- **核心功能**：
  - 使用自然语言指令绘制手绘风格图形。
  - 实时修改、删除或查询画布状态，支持交互式字图。
  - 让大语言模型直接控制 Excalidraw 画布，生成如流程图、架构图等视觉内容。
- **目的**：通过 AI 自动化绘图，提升效率，降低绘图门槛，实现“一键生成”。

## 安装方式与步骤
Excalidraw MCP 提供两种安装方式：
1. **下载 .mcp 文件并运行**（推荐，适合不熟悉 CLI 的用户）：
   - 步骤：
     - 访问 MCP 官网，下载 `.mcpb` 文件（如 `excalidrawmcp-app.mcpb`）。
     - 双击该文件，在 Cloud 桌面端弹出安装界面。
     - 点击“install”按钮完成安装。
     - 配置后，点击“configure”并允许工具进行绘图和读取操作（选择“总是允许”）。
2. **通过 CLI 命令行安装**：
   - 视频未详细演示，适用于习惯命令行的用户。

## 实际应用演示
通过两个例子展示 Excalidraw MCP 的实际效果：
1. **绘制 OpenCloud 运行原理图**：
   - 用户指令：使用 Excalidraw MCP 画一个 OpenCloud 运行原理图。
   - AI 行为：首先搜索 OpenCloud 的运行原理，识别其为开源 AI 代理框架，然后展示核心架构工作流程，并在画布上实时绘制手绘风格原理图。
   - 结果：生成可编辑的手绘风格架构图，所有元素（如形状、文字）均可修改、删除或调整。
2. **绘制树莓派 5 图片**：
   - 用户指令：画一个树莓派 5 的图片使用 Excalidraw。
   - AI 行为：自动生成树莓派 5 的手绘风格图片。
   - 结果：成功输出视觉元素，体现了工具对复杂硬件图的处理能力。

## 总结
Excalidraw MCP 通过 MCP 协议将 Excalidraw 白板与 AI 大模型连接，使用户能以自然语言快速生成和编辑手绘风格图形，如架构图和流程图。该工具支持实时交互和修改，提升了绘图效率和便捷性，是 AI 与前端工具结合的典型案例，适用于需要快速可视化设计的场景。
