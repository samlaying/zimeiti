# 一条视频教你用Codex做动态图表和视频！除了惊艳还是惊艳

> [是子鱼AI](https://space.bilibili.com/50984680) | [BV1UtR9BNEKw](https://www.bilibili.com/video/BV1UtR9BNEKw) | 时长 4:46

## 视频制作教程：使用 Codex 和 HaperFrames 创建动态图表和视频

### 工具介绍
- **Codex**：主工具，用于生成内容，支持动态图表和视频制作。
- **HaperFrames**：插件，用于增强视频制作功能，提供动态效果和视觉元素。

### 安装与设置步骤
1. **下载并安装 Codex**：按照正常软件安装流程完成。
2. **登录 Codex**：使用 GBA 账号登录。
3. **安装 HaperFrames 插件**：
   - 在 Codex 左侧栏点击“插件”，搜索“HaperFrames”。
   - 进入插件详情页面，直接安装。
   - 安装后，在对话中使用，确保添加项目并确定模型智能等级的速度。

### 使用流程与示例
1. **输入提示词**：在 Codex 中输入详细提示词，控制视频内容。示例提示词：“帮我做一个 Codex 和 Cloud Code 功能对比的分析视频时长15秒尺寸是横板16比9视觉风格是苹果简约高级风需要增加一些动态效果的展示”。
   - 注意：如使用 AI 语音工具，可能需手动修正识别错误的词语。
2. **发送生成**：点击发送后，Codex 自动执行：
   - 搜索官方资料确保信息最新。
   - 查看 HaperFrames 要求，确定视觉身份。
   - 生成 HTML 内容，并在后台网页搜索资料，思考过程类似人类。
   - 创建单独文件夹以避免干扰其他项目。
3. **生成过程**：
   - 自动产出内容，自行检查和更正问题。
   - 耗时约十分钟，完成后显示文件位置和参考资料。
4. **效果查看**：第一版视频即可用，质量惊艳，无需预先模板，适用于社交媒体发布或工作汇报。

### 案例展示
- **动态图表视频**：如 Codex 和 Cloud Code 功能对比。
- **产品介绍视频**：例如蜡条产品视频，或品牌介绍。
- **业绩表**：如 OpenAI 的业绩图表，可嵌入 PPT。
- **动画效果**：logo 散落视频，或宋代深水风格 AI 发展史。
- **其他应用**：给一段提示词，即可生成多样内容，如风格化视频。

### 进阶技巧与建议
1. **图片素材处理**：
   - 涉及 logo 或人像时，建议自行上传图片。
   - 或先用 Cloud Code 整理图片，将文件夹打包到项目中以减少错误。
2. **风格指定**：
   - 必须提前指定风格，避免大众化（如科技风易做成蓝指色）。
   - 示例风格：苹果简约高级风；可参考列出的风格建议直接使用。
3. **音乐与音效**：
   - 音效：可由 Codex 生成简单音效。
   - 背景音乐：使用 Suno 生成原创音乐，或在剪辑工具中添加音乐库素材。

### 其他插件推荐：Bell Render
- **用途**：专门用于科研、生物科学等领域的复杂概念可视化。
- **结合使用**：与 HaperFrames 结合，将专业晦涩信息以动态形式表达。
- **示例**：制作科普视频，如谚语“An apple a day keeps the doctor away”的背后科学解释，生成动态可视化内容。

### 总结核心要点
本视频核心展示了使用 Codex 和 HaperFrames 插件简化动态视频制作的过程：通过简单提示词，无需专业软件如 AE 或 PR，即可在十分钟左右生成高质量视频，适用于多种场景；进阶技巧如指定风格、处理素材和音乐可进一步提升效果；配合 Bell Render 插件，还能扩展至专业领域可视化，凸显了 AI 工具在提效和设计方面的强大能力。
