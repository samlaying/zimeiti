# 用html做出高级感ppt！从0到1全网最完整教程（附免费编辑工具！）

> [栗氪聊AI](https://space.bilibili.com/67389029) | [BV1JFLV6yE1a](https://www.bilibili.com/video/BV1JFLV6yE1a) | 时长 9:53

## 一、 准备工作与工具安装
### 1. 所需环境与Agent
*   **核心Agent**：需要一个AI编程Agent，如 **Cline**、**Codex** 或国内同类工具（如DeepSeek、GLM等）。本教程主要以 Cline 为例。
*   **基础准备**：如果尚未安装Cline，可参考UP主上期视频《零基础安装Cline》。

### 2. 安装两个关键Skill（技能）
1.  **Frontend Slides Skill**：
    *   **用途**：教会你的AI Agent如何使用HTML制作优秀的PPT。
    *   **来源**：GitHub项目，已有超过17,000星。
    *   **安装方式**：将该项目的GitHub链接直接发送给你的AI Agent，Agent会自动完成安装。
2.  **Beautiful HTML Templates Skill**：
    *   **用途**：集合了大量精美、有设计感的HTML PPT模板，由博主“Zandara”制作。
    *   **安装方式**：同上，将链接发给AI Agent即可自动安装。

### 3. 修改Skill以实现模板调用
*   **目的**：为了让`Frontend Slides`在生成PPT时能直接调用`Beautiful HTML Templates`中的精美模板，避免后续手动替换的麻烦。
*   **操作步骤**：
    1.  下载`Beautiful HTML Templates`项目文件（如果无法访问GitHub，可通过UP主提供的备用链接下载并解压）。
    2.  将解压后的项目文件夹拖入Cline窗口。
    3.  向AI Agent发送指令：`帮我安装这个skill`。Agent会自动完成安装。
    4.  **关键修改**：将`Beautiful HTML Templates`文件夹中的`agents.md`文件拖入Cline，并附上类似指令：“生成PPT时，请直接调用这个文件夹里的模板”。
    5.  AI Agent会自动修改`Frontend Slides` Skill，使其能集成美观模板。

## 二、 创建与生成HTML PPT
### 1. 建立专用工作目录
*   在终端中，使用`cd`命令并拖入你创建的文件夹，将其设置为当前工作目录。
*   随后输入`claude`或`c1`启动Cline，确保后续所有生成文件都在此目录下。

### 2. 使用Skill生成PPT
1.  **触发技能**：在Cline对话框中输入`/`，选择并调用`Frontend Slides` Skill。
2.  **提供内容**：将你的文案或演讲稿内容粘贴给AI。例如，发送短视频文案并附上指令：“帮我做一个演示的PPT”。
3.  **回答风格问题**：AI会提问以定制PPT，通常包括：
    *   **页数**：选择期望的页数范围（如8-10页）。
    *   **编辑需求**：询问是否需要在浏览器中直接编辑文字（可选择“不需要”，因为后续有更专业的工具）。
    *   **核心感受**：希望观众看完PPT后的感受（如“清晰易懂”）。
4.  **选择风格**：AI会先生成三个不同风格的标题页示例供你选择，选定后将基于该风格生成全部内容。

## 三、 PPT的编辑与调整
### 1. 使用专业HTML编辑工具
*   **工具介绍**：UP主自行开发的网页端可视化编辑工具，可轻松调整HTML PPT的字体、图片和视频。
*   **基本操作**：
    *   打开工具网页，选择PPT所在的文件夹。
    *   **调整文本**：点击选中任何文字，即可使用工具栏放大、缩小字号或修改内容。
    *   **插入图片**：
        1.  **向AI提需求**：例如，“我想在P3插入一张图片，内容是‘Token和自学习数的变化图’”。
        2.  **分情况处理**：
            *   若AI模型（如Claude、Codex）支持读图：可直接将图片拖入Cline，AI会自动添加。
            *   若AI模型不支持读图（如DeepSeek、GLM）：则让AI使用“图片占位服务”，后续手动替换。
        3.  **在编辑工具中替换**：在工具中打开HTML文件，找到图片占位符，点击“插入图片”上传本地素材，并调整大小。
    *   **插入视频**：流程与插入图片类似。向AI提需求，使用“视频占位服务”，然后在编辑工具中点击“插入视频”上传本地视频文件并调整大小。
*   **保存**：编辑完成后，点击“覆盖原文件”保存更改。

## 四、 分享与部署
### 1. 文件分享方式
*   **纯HTML文件**：若PPT未使用外部图片/视频，直接分享`.html`文件即可，用浏览器打开。
*   **包含素材的文件夹**：若使用了图片/视频等外部资源，需将整个项目文件夹打包（如压缩为.zip）发送。接收者需解压后，再打开其中的HTML文件。

### 2. 在线链接分享方式（推荐）
*   **推荐平台**：**Netlify**。免费，部署后生成的链接在国内可访问，流程简单。
*   **部署步骤**：
    1.  访问Netlify官网并注册登录。
    2.  点击上传选项。如果PPT包含多个素材文件，选择“Upload a folder”。
    3.  选择打包好的整个文件夹进行上传。
    4.  上传部署完成后，为项目设置一个易于记忆的网址名称。
    5.  生成的链接可直接复制分享，他人通过浏览器即可在线观看你的HTML PPT。

## 核心要点总结
本教程完整演示了如何利用AI Agent（如Cline）和特定技能（Frontend Slides、Beautiful HTML Templates）从零开始制作具有高级感的HTML演示文稿。核心流程包括：安装和配置技能、提供内容让AI生成初稿、使用可视化编辑工具微调文本并替换图片视频等素材，最后通过文件打包或免费平台Netlify部署为在线链接进行分享。这种方法结合了AI的高效生成能力和手动精细调整的灵活性，是AI时代提升演示文稿制作效率与质量的高级技能。
