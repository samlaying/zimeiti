# 笔记：Claude Code + Higsfield MCP 打造内容自动化机器

## 核心价值：MCP Server的重要性
### 解决工具碎片化问题
- **过去痛点**：需要将每个最佳的AI内容创作工具（图像、视频等）单独连接到Claude Code，过程繁琐。
- **工具快速迭代**：最好的工具（如图像生成）频繁更换（例如：Denobanana Pro → GPT Images → VO3 → Clean → C-Dance），单独接入导致无法及时使用最优工具。
- **MCP Server解决方案**：通过Higsfield的MCP Server，可以**统一接入所有AI内容创作工具**，实现程序化调用。
- **规模优势**：可访问**17个图像模型**、**14个视频模型**及Higsfield的专有模型。

### 实现自动化流程
- **突破性意义**：关键在于其作为MCP服务器的本质，这允许通过Claude Code**将多个步骤自动化**。
- **自动化示例**：设置一个流程，让Claude Code每天自动抓取GitHub上的热门AI仓库信息，将其整理成社交媒体发布格式（如轮播图），并为图片生成提示词，然后发送给Higsfield调用GPT Images等模型生成内容，最后自动带回结果。
- **最终形态**：结合Claude Code与Higsfield MCP，可以将Claude Code变成一个**全自动化的营销内容生产机器**。

## 安装与设置
### 方法一：在ClaudeAI Web应用中设置连接器
1.  进入 **ClaudeAI** 的 **Settings**。
2.  选择 **Connector**，然后 **Add Custom Connector**。
3.  复制并粘贴Higsfield提供的连接器代码。
4.  点击 **Connect** 并完成身份验证登录。
5.  设置完成后，即可在Claude Web或桌面应用中直接调用Higsfield中的任何音频、视频或图像工具。

### 方法二：在终端（Claude Code）中设置MCP Server
1.  访问Higsfield MCP设置页面，选择 **Custom Connector**。
2.  复制代码，粘贴到Claude Code终端中，程序会自动开始设置。
3.  根据提示，完成与Web应用中相同的身份验证流程。
4.  设置完成后，在Claude Code中运行 `For NCP`，应看到“Hickfield已连接”。
5.  如未成功，尝试退出并重启Claude Code，它会引导你完成连接。

## 实操案例：24小时10万播放量轮播图制作
### 流程概述
这是一个完整的自动化内容生成案例，目标是为社交媒体（如Instagram）创建热门AI仓库推荐的轮播图帖子。

### 步骤详解
1.  **信息抓取（自动化起点）**：
    *   创建一个每日运行的自动化脚本，用于从GitHub抓取信息。
    *   脚本会获取**过去七天内创建的、按Star数排名的前10个热门GitHub仓库**的简短描述。
    *   同时也会获取**过去一个月的前5名趋势项目**（新冒头的项目）。

2.  **内容构思与准备**：
    *   明确轮播图的主题，例如“本月五大热门AI仓库”。
    *   准备**参考图片**：提供一个理想的封面页和几张正文幻灯片的样式作为参考，以指导AI生成风格统一的图片。
    *   向Claude Code给出自然语言提示，结合抓取的GitHub数据和参考图片，要求生成轮播图。

3.  **AI生成内容（核心步骤）**：
    *   Claude Code会分析GitHub数据，结合参考图片和指令，**为每一张幻灯片（封面和内容页）生成详细的图像生成提示词**。
    *   通过Higsfield MCP Server，将提示词发送给指定的AI模型（如GPT Image 2），并设置参数（如高质量、2K分辨率、生成4个变体）。
    *   **重要机制**：MCP在请求完成后不会主动通知，需要指令Claude Code**每隔60-90秒去查询**Higsfield以获取生成结果，并将图片文件下载回来。

4.  **审查、编辑与批量处理**：
    *   Claude Code会返回生成的图片变体，用户可进行审查。
    *   如果需要编辑，可以指示Claude Code发送编辑指令（如同手动操作，会关联参考图像）。
    *   一旦封面页和内容页的样式确定，就可以**指示Claude Code对剩余的所有幻灯片进行批量生成**，极大提升效率。

5.  **最终产出与发布**：
    *   最终得到一套完整、风格统一的轮播图，内容基于最新的GitHub趋势数据。
    *   整个流程可以打包成一个“Scale”（技能），实现**每天早上自动运行，生成一篇“今日十大AI仓库”轮播图帖子**。

### 案例关键点
- **内容来源**：数据驱动，基于真实的、不断更新的GitHub热门项目（“常青内容”）。
- **质量控制**：通过提供参考图来严格控制视觉风格和美感。
- **混合方法**：不必全部依赖AI生成，例如封面图可用AI生成，正文幻灯片可用HTML代码生成以节省成本。
- **案例成果**：该自动化生成的内容在不到24小时内获得了**100,000次播放量**。

## 核心要点总结
Claude Code与Higsfield MCP Server的结合，解决了AI内容创作工具碎片化和难以程序化调用的问题。它通过统一的接口（MCP Server）接入众多顶尖的AI生成模型，并将Claude Code的强大推理和规划能力与之结合，从而实现了从外部数据抓取、内容构思、多模态内容生成到自动化发布的全流程内容生产。这不仅将Claude Code转变为一个高效的营销自动化机器，更重要的是为创作者提供了前所未有的灵活性和可能性，能够快速响应热点，批量生产高质量、风格统一的社交媒体内容，显著提升创作效率和内容影响力。
