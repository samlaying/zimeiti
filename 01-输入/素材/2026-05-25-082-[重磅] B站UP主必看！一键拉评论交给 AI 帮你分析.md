# [重磅] B站UP主必看！一键拉评论交给 AI 帮你分析

> [探索未至之境](https://space.bilibili.com/441831884) | [BV1FcdZBPEAT](https://www.bilibili.com/video/BV1FcdZBPEAT) | 时长 3:04

## BBC Skill：一键拉取并AI分析B站视频评论的工具

### 1. 工具概述与核心价值
*   **核心痛点**：B站UP主面对海量视频评论（数百至上千条），人工阅读、筛选和分析效率极低，难以快速识别关键建议、潜在铁粉、核心吐槽点及观众活跃地域分布。
*   **解决方案**：BBC Skill 是一个开源的 Python 命令行工具，专门用于**一键拉取视频下的全部评论**，并将其整理为结构化数据，便于交给 AI Agent 进行深度分析。
*   **工具特性**：
    *   **零依赖**：仅使用 Python 3.9 标准库，无需 pip install。
    *   **全面抓取**：抓取内容包括顶级评论、楼中楼（回复）、置顶评论，同时打包视频的标题、播放、点赞、投币、收藏等元数据。
    *   **标准化输出**：输出为 `comments.csv`（结构化评论数据）和 `summary.json`（统计数据），专门为 AI Agent 读取分析而设计。
    *   **开放兼容**：遵循开源 Agent Skills 规范，原生兼容六大主流 AI 编程助手（如 Claude Code, OpenAI Codex 等）及 SkillsMP 包管理器。

### 2. 安装与准备
*   **安装方式**：通过 `git clone` 将项目克隆到对应 AI Agent 的 Skills 目录下即可使用。
*   **前期准备（关键步骤）**：
    1.  **导出 Cookie**：在 Chrome 浏览器安装开源插件 **Get cookies.txt LOCALLY**，登录 B站 后，一键导出 Cookie 文件。
    2.  **验证 Cookie**：使用命令 `bbcskill.cookie_check` 验证导出的 Cookie 是否可用。

### 3. 完整使用流程（三步）
1.  **第一步：导出 B站 Cookie**
    *   安装并使用 Chrome 插件 **Get cookies.txt LOCALLY**。
    *   登录 B站 账号，使用插件一键导出 Cookie 文件。
2.  **第二步：验证 Cookie 可用性**
    *   运行命令：`bbcskill.cookie_check`
3.  **第三步：拉取视频评论与数据**
    *   运行命令：`bbcskill.fetch <BV号> <输出目录>`
    *   **输出结果**：
        *   `comments.csv`：结构化评论数据。
        *   `summary.json`：原数据统计（如总评论数、点赞数等）。
        *   `raw/` 目录：归档原始接口响应数据。
    *   **容错机制**：整个过程支持**断点续传**，运行中断后可继续。

### 4. 交给 AI 分析的常见场景
将拉取的数据交给 AI Agent 后，可以一句话指令实现多种数据分析：
*   **情感倾向分析**：统计正面、负面、中性评论占比。
*   **关键词提取与词云**：从评论中挖掘高频关键词并生成词云。
*   **创作者互动分析**：统计创作者本人的回复分布情况。
*   **观众地域分析**：通过 IP 属地生成热力图。
*   **铁粉识别**：基于互动行为识别高粘性用户。
*   **差评筛查**：快速定位和分类负面反馈。
*   **反馈演变分析**：分析评论情感或主题随时间变化的曲线。

### 5. 合理使用准则与注意事项
*   **权限范围**：本工具仅限用于分析**你自己的视频**或已获得**明确授权**的视频。
*   **爬取节流**：内置每秒一次的请求限制。批量处理多个视频时，视频间会随机休眠 5-10 秒。
*   **禁止行为**：
    *   禁止大规模爬取陌生创作者的评论。
    *   禁止基于抓取数据构建二级数据产品并对外出售。
*   **商业场景**：请通过 B站 开放平台的官方接口。
*   **数据安全**：所有命令为**只读**操作，不会发布、删除或修改任何数据。

### 核心要点总结
BBC Skill 是一个为B站UP主设计的Python开源工具，它能通过一条命令快速、完整地抓取指定视频的所有评论及元数据，并输出为标准化文件（CSV和JSON），旨在无缝对接AI编程助手，实现情感分析、关键词提取、铁粉识别等深度数据洞察。其工作流为“导出Cookie → 验证Cookie → 拉取数据”，最后将结构化数据交给AI一句话分析。工具强调合法合规使用，仅限分析自有或已授权内容，并内置了严格的请求节流和只读机制，旨在提升创作者分析观众反馈的效率，而非用于非法数据爬取或商业倒卖。
