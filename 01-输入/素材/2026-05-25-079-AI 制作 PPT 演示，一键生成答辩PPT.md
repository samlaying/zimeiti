# AI 制作 PPT 演示，一键生成答辩PPT

> [小钟同学hz](https://space.bilibili.com/449048571) | [BV1qiR1BoEj3](https://www.bilibili.com/video/BV1qiR1BoEj3) | 时长 35:48

## **AI 制作答辩 PPT 全流程指南**

本指南基于视频内容，整理了从环境搭建到最终生成一份可编辑答辩 PPT 的完整步骤和关键信息。

### **第一部分：环境安装与配置**
#### **1. 安装 Python 环境**
- **推荐版本**：Python 3.1.2.10
- **安装步骤**：
    1.  访问视频提供的链接，下载对应版本的安装包。
    2.  运行安装程序，**勾选 “Add Python to PATH”**，点击安装。
    3.  安装完成后验证：
        - 右键在文件夹打开命令行窗口。
        - 输入 `python --version`，应显示安装的 Python 版本。
        - 输入 `pip --version`，应显示 pip 版本。

#### **2. 更新 pip 并配置国内镜像源**
- **更新 pip**：
    - 由于命令较长，建议在文本文档中编辑好再复制到命令行执行。
- **配置清华镜像源（解决无梯子访问慢问题）**：
    - 执行命令后，使用 `pip config list` 验证，输出中应显示 `index-url` 为清华源地址。

#### **3. 安装 Node.js 环境**
- **安装步骤**：
    1.  访问官网下载安装包。
    2.  运行安装程序，一路默认（**不要勾选任何额外选项**）直至完成。
- **验证与问题解决**：
    - 安装后，在命令行验证 `node -v` 和 `npm -v`。
    - 若出现“禁止运行脚本”错误，需以管理员身份运行 PowerShell，执行 `Set-ExecutionPolicy RemoteSigned` 并选择 `Y` 确认。
- **配置国内镜像源**：执行 `npm config set registry https://registry.npmmirror.com` 并验证。

### **第二部分：AI 工具安装与配置**
#### **1. 安装 Cursor IDE**
- **介绍**：一个集成 AI 的代码编辑器，用于本次演示。
- **安装与登录**：
    1.  从官网下载 IDE 版本（**不要下载 Solo 版**）。
    2.  安装完成后，使用个人账号登录。
- **核心功能**：
    - 集成多种 AI 模型（如图片理解、推理模型）。
    - 本次实操选用 **Kimi K2.5** 模型（无需排队）。
    - 通过创建项目文件夹（如 `PPT Maker`）开始工作。

#### **2. 安装 Draw I.O. SKILL (绘图技能)**
- **用途**：用于绘制流程图、架构图等图表，推荐有需求的用户安装。
- **安装步骤**：
    1.  从官网下载 Draw I.O. 桌面版并安装。
    2.  从对应的 GitHub 仓库下载 `skill-drawio` 的压缩包。
    3.  将解压后的文件夹复制到用户目录下的 `skills` 文件夹中（如 `C:\Users\你的用户名\skills`）。
    4.  在 Cursor 中刷新 SKILL 列表，并通过 AI 对话验证其配置（如询问“你能够使用 drawio SKILL 吗？”）。
- **关键建议**：**不要修改 SKILL 的安装路径**，否则可能导致验证失败。

#### **3. 安装 PPT-master SKILL (核心技能)**
- **用途**：一键将文本、图片等内容转化为美观的 PPT。
- **安装步骤**：
    1.  从对应的 GitHub 仓库下载 `skill-ppt-master` 的压缩包。
    2.  将解压后包含 `skill.md` 文件的文件夹，同样复制到用户目录的 `skills` 文件夹中。
    3.  在该 SKILL 文件夹内，右键打开终端，执行依赖安装命令。
    4.  在 Cursor 中通过 AI 对话验证其配置（如“帮我验证一下 PPT-master SKILL 是否已经配置完成”）。
- **功能亮点**：支持动画、音频和旁白导出。

### **第三部分：实操演示：一键生成答辩 PPT**
#### **1. 准备素材与模板**
- **获取模板**：可让 AI 根据参考图片（如学院 PPT 模板）设计一个新模板。
- **准备内容**：使用报告文档（本例中由 ChatGPT 生成），通过 OCR 工具（如视频中提到的网站）将其转换为可编辑的 Markdown 文件。

#### **2. 生成与完善 PPT**
- **生成**：
    1.  在 Cursor 的 AI 对话中，引用准备好的 Markdown 文件。
    2.  输入提示词，如要求“根据内容制作一个答辩 PPT”。
    3.  确认信息后，AI 开始自动生成。
- **后续完善与文件结构**：
    - 生成后，可在文件夹中找到 `export` 文件夹，内含可编辑的 `.pptx` 文件。
    - 若出现元素缺失（如 logo），可继续通过对话指令让 AI 补充。
    - 文件夹中会保留每个版本的 `svg` 源文件和 `.pptx` 文件，方便回溯。

### **核心要点总结**
本教程完整演示了如何利用 AI 工具链（Cursor + SKILLs）高效制作答辩 PPT 的全流程。核心在于：**1. 正确搭建 Python 和 Node.js 环境并配置镜像源；2. 安装 Cursor IDE 及配套的绘图（Draw I.O.）和PPT生成（PPT-master）技能；3. 将报告文本等内容转化为结构化数据，通过与 AI 的自然语言交互，一键生成、并迭代完善最终的专业 PPT。** 整个过程强调自动化和可配置性，大幅提升了文档和演示文稿的制作效率。
