# [HTML+CSS] 3D展开效果天花板 拿捏啦

> [总监日记](https://space.bilibili.com/606308219) | [BV1ZL411y7YK](https://www.bilibili.com/video/BV1ZL411y7YK) | 时长 13:18

## 一、 核心目标与知识点
本视频旨在通过一个3D卡片展开效果的实战Demo，彻底掌握CSS3中与3D相关的核心属性。这些属性包括：
1.  **透视深度** (`perspective`): 控制3D效果的深度感。
2.  **旋转** (`rotate`): 控制元素在X、Y、Z轴上的旋转角度。
3.  **扭曲/倾斜** (`skew`): 控制元素在X、Y轴上的扭曲程度。
4.  **缩放** (`scale`): 控制元素的大小缩放。
5.  **变换原点** (`transform-origin`): 设置元素进行变换时所依据的基准点。

## 二、 实战步骤详解

### 2.1 HTML结构搭建
1.  创建最外层容器 `div.wrap`。
2.  在 `wrap` 内创建卡片容器 `div.card`。
3.  在 `card` 内创建三个部分：
    *   `div.card-bg`：用于放置游戏背景图。
    *   `div.role`：用于放置游戏角色图。
    *   `div.content`：用于放置标题和描述文字。

### 2.2 CSS基础样式与布局
1.  **全局初始化**：`margin: 0; padding: 0; box-sizing: border-box;`。
2.  **容器 `wrap` 样式**：
    *   宽度：`100%`；高度：`100vh`。
    *   设置主背景图片，`background-size: no-repeat`。
    *   使用Flex布局实现居中：`display: flex; justify-content: center; align-items: center;`。
3.  **卡片 `card` 样式**：
    *   宽度：`440px`；高度：`250px`。
    *   设置 `position: relative`，为内部绝对定位元素提供参考。
4.  **背景 `card-bg` 样式**：
    *   宽高：与卡片一致 (`440px` x `250px`)。
    *   圆角：`border-radius: 20px`。
    *   内部图片样式：`width: 100%; height: 100%; object-fit: cover`。
    *   裁切溢出：`overflow: hidden`。
    *   添加柔和阴影以融合背景：`box-shadow: 0 0 30px 20px rgba(255, 255, 255, 0.05);`。
5.  **角色 `role` 样式**：
    *   宽高：`440px` x `230px`。
    *   内部图片：`width: 100%; height: 100%;`。
    *   定位：`position: absolute; bottom: 0;`，使其底部与卡片底部对齐。
6.  **内容 `content` 样式**：
    *   初始状态隐藏：`opacity: 0`。
    *   宽度：`440px`；内边距：`padding: 10px`。
    *   文字颜色：白色。
    *   定位：`position: absolute; bottom: -50px;`，初始位置在卡片下方。
    *   设置 `h1` 和 `p` 标签的字体粗细 (`font-weight: 200`) 和大小。

### 2.3 鼠标悬停（Hover）动画实现
这是实现3D展开效果的核心部分。
1.  **背景旋转与倾斜** (`.card:hover .card-bg`)：
    *   应用3D变换：`transform: perspective(1000px) rotateX(30deg) skewX(-20deg);`。
        *   `perspective(1000px)`：设置透视深度为1000像素，创造近大远小的立体感。
        *   `rotateX(30deg)`：绕X轴旋转30度，实现“翻起”效果。
        *   `skewX(-20deg)`：在X轴方向扭曲-20度，增加倾斜感。
    *   设置变换原点：`transform-origin: bottom;`，确保旋转和扭曲以底部为支点。
    *   设置过渡动画：`transition: 2s;`，使动画平滑。
2.  **角色缩放** (`.card:hover .role`)：
    *   应用缩放变换：`transform: scale(1.3);`。
    *   设置变换原点：`transform-origin: bottom;`。
    *   设置过渡动画：`transition: 2s;`。
3.  **内容显示与移动** (`.card:hover .content`)：
    *   显示内容：`opacity: 1;`。
    *   向下移动：`transform: translateY(120px);`。
    *   设置过渡动画：`transition: 2s;`。

## 三、 关键技巧与注意事项
*   **透视 (`perspective`)**：通常设置在父元素上或直接应用于变换中，值越大，3D效果越微妙；值越小，效果越夸张。本例使用 `1000px`。
*   **变换原点 (`transform-origin`)**：至关重要。本例通过设置 `bottom` 关键字，确保所有3D变换都以元素的**底部**为基准点，从而模拟出“从底部翻开”的自然效果。
*   **过渡 (`transition`)**：为所有在Hover状态下改变的属性（`transform`, `opacity`）设置相同的过渡时间，保证动画同步且流畅。
*   **图层与定位**：合理使用 `position: relative` 和 `position: absolute` 进行图层叠加和精确定位，是构建复合视觉效果的基础。

## 四、 总结
本Demo通过精心组织的HTML结构和分层的CSS样式，结合对CSS3核心3D属性（透视、旋转、扭曲、缩放）的综合运用，并重点强调了**变换原点 (`transform-origin`)** 和**过渡动画 (`transition`)** 的关键作用，最终实现了一个视觉上极具冲击力的3D卡片展开交互效果。掌握这个案例，对于理解前端3D变换的原理和实战应用大有裨益。
