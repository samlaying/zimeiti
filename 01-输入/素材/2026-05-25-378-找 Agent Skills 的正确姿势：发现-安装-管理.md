## 找 Agent Skills 的正确姿势：发现->安装->管理

### 引言
- 视频推荐两个工具来帮助发现和管理 Agent Skills：Skills CLI 和 Skills.sh 网站。
- 这两个工具由 Versalab 发布，博客发布于1月20号，介绍了 Skills CLI 作为命令行工具用于安装和管理 Skill 包，Skills.sh 作为目录和排行榜网站用于发现新 Skill。

### 工具一：Skills CLI
- **介绍**：一个命令行工具（CLI），用于安装和管理 Skill 包，类似于包管理器（如 npm），已被多个工具使用，如 AMP、Cloud Code、Code Boot、CodeX、Cursor、Gemina 等。
- **主要命令**：
  - `npx skills add`：安装 Skill 包，支持选择安装范围和平台。
  - `npx skills list`：查看当前安装的 Skill，添加 `-g` 选项可查看全局安装。
  - `npx skills find`：通过关键词查找 Skill，例如 `npx skills find vue3` 来寻找 Vue 3 相关的 Skill。
  - `npx skills remove`：删除 Skill，支持删除指定 Skill 或全部删除。
  - `npx skills check`：检查 Skill 的更新状态，显示是否需要更新。
  - `npx skills update`：更新 Skill 到最新版本。
  - `npx skills init`：创建新的 Skill 模板，快速生成 skill.md 文件。
- **用途**：简化 Skill 的安装、管理和更新过程，支持软链接以便于维护。

### 工具二：Skills.sh 网站
- **介绍**：一个目录和排行榜网站，用于发现新 Skill，提供分类和流行程度浏览，跟踪使用统计和安装情况。
- **功能与特性**：
  - **排行榜**：显示所有时间使用量、24小时内热度、当前热度排名，帮助用户发现热门 Skill。
  - **详细页面**：每个 Skill 页面包含安装命令、介绍文件（skill.md）、使用量数据（如周下载量）、使用分布（例如在 Cloud Code 和 OpenCode 中的使用量）、首次发布时间和仓库链接。
  - **示例**：FundSkill 是最火的 Skill，通过自然语言描述（如“我该怎么做，需要一个什么样的 Skill”）帮助用户寻找和安装 Skill。

### Skill 示例详解
- **FundSkill**：
  - **目的**：通过自然语言查询帮助用户发现和安装合适的 Skill。
  - **数据**：周使用量高，24小时内热度排名第一，首次发布于9天前。
- **Versalab 官方 Skill**：
  - **方向**：偏向前端和 React 开发。
  - **内容**：包含 React 最佳实践、Cloud AI 使用指南和 Web 设计指南。
- **Vue 相关 Skill**：
  - **官方 Skill**：VoE、NUXT、PINION 等 Vue 生态库的 Skill。
  - **非官方 Skill**：如 Anson Foo 的个人 Skill（以名字命名，包含个人喜好和最佳实践）、Sleedip、TSDUN、Turbo Repo、VoUZ。
  - **备注**：一些 Skill（如 Anson Foo 的 Vue Skill）发布后迅速被官方收录，显示了生态系统的活跃性。

### 实操步骤
1. **安装 Skill**：
   - 复制安装命令，例如 `npx skills add ansonfoo-vue`。
   - 在终端粘贴并运行，系统提示选择安装范围（全部或部分）。
   - 选择平台，如 OpenCode（默认选项）。
   - 选择安装方式：推荐使用软链接（第一个选项），便于后续更新。
   - 等待安装完成，系统显示安装成功信息。
2. **查看已安装 Skill**：
   - 运行 `npx skills list` 列出当前项目安装的 Skill。
   - 要查看全局安装的 Skill，使用 `npx skills list -g`。
3. **检查更新**：
   - 运行 `npx skills check` 检查所有 Skill 的更新状态，系统显示是否可更新。
4. **更新 Skill**：
   - 如果检查到更新可用，运行 `npx skills update` 将 Skill 更新到最新版本。
5. **删除 Skill**：
   - 运行 `npx skills remove` 删除指定 Skill，例如删除全部 Skill 使用命令 `npx skills remove --all`。
   - 删除后再次运行 `npx skills list` 确认删除。
6. **创建新 Skill 模板**：
   - 在项目目录运行 `npx skills init`。
   - 系统生成 Skill 模板文件（如 skill.md），用于自定义 Skill 开发。

### 建议和数据
- **安装建议**：在安装时选择软链接选项，便于 Skill 的更新和维护；根据开发平台（如 OpenCode、Cloud Code）选择合适选项。
- **数据参考**：Skills.sh 网站提供使用量数据和热度排名，可帮助选择高质量 Skill；FundSkill 等热门 Skill 有高使用量，表明其流行度和实用性。
- **生态提示**：Versalab 官方和社区持续发布新 Skill，用户可通过 Skills.sh 跟踪最新动态。

### 总结
视频介绍了使用 Versalab 发布的 Skills CLI 和 Skills.sh 网站来发现、安装和管理 Agent Skills 的正确姿势。Skills CLI 作为命令行工具，提供了完整的命令集用于安装、查找、更新和删除 Skill，简化了管理流程；Skills.sh 网站则作为目录和排行榜，帮助用户通过分类、热度和自然语言描述（如 FundSkill）发现新 Skill。通过实操演示，展示了从安装到管理的完整步骤，强调了遵循最佳实践（如 React 规范）和利用工具提升开发效率的重要性。
