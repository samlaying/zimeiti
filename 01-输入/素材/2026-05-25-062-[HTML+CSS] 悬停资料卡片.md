# [HTML+CSS] 悬停资料卡片

> [大猫是小白](https://space.bilibili.com/172008592) | [BV1sx4y197MP](https://www.bilibili.com/video/BV1sx4y197MP) | 时长 7:27

## 笔记正文

### 1. 项目概述与核心目标
- **项目名称**：创建一个悬停（Hover）资料卡片。
- **技术栈**：HTML + CSS。
- **核心效果**：当用户将鼠标悬停在卡片上时，卡片会产生视觉上的变化（如展开、翻转、显示更多信息等），以增强交互性和用户体验。
- **适用场景**：个人主页、作品集、团队成员介绍等。

### 2. HTML 结构搭建
- 使用语义化标签构建卡片的基本骨架。
- **典型结构示例**：
  ```html
  <div class="card">
    <div class="card-inner">
      <div class="card-front">
        <img src="avatar.jpg" alt="头像" />
        <h3>姓名</h3>
        <p>职位/简介</p>
      </div>
      <div class="card-back">
        <h3>详细信息</h3>
        <p>这里可以放更详细的介绍、技能、联系方式等。</p>
        <a href="#">了解更多</a>
      </div>
    </div>
  </div>
  ```
- **要点**：
  - 使用一个容器（如 `div.card`）包裹整个卡片。
  - 内部通常包含正反两面（`.card-front` 和 `.card-back`），用于实现翻转等效果。
  - 内容根据需求填充，如图片、标题、段落文字、链接等。

### 3. CSS 基础样式设置
- **1. 卡片容器样式**：
  - 设置固定的宽度和高度。
  - 使用 `position: relative` 作为定位上下文。
  - 添加 `perspective` 属性，为后续的3D变换（如翻转）创建透视效果。
- **2. 卡片内层面板（.card-inner）**：
  - 设置为 `position: relative`。
  - 宽高设为100%。
  - 应用 `transform-style: preserve-3d`，使其子元素可以进行3D变换。
  - 添加 `transition` 属性，定义悬停状态变化的过渡时间和平滑曲线（如 `transition: transform 0.6s`）。
- **3. 正反面通用样式**：
  - 设置为 `position: absolute`，覆盖整个父容器。
  - 使用 `backface-visibility: hidden`，确保在背面旋转时，正面不被看到。

### 4. 悬停动画效果实现 (核心)
- **效果一：3D翻转卡片**
  - **原理**：悬停时，通过改变 `.card-inner` 的 `transform` 属性，实现Y轴（或X轴）旋转。
  - **CSS 代码示例**：
    ```css
    .card:hover .card-inner {
      transform: rotateY(180deg);
    }
    ```
  - **解释**：鼠标悬停在 `.card` 上时，`.card-inner` 在Y轴上旋转180度，显示背面内容。

- **效果二：平滑信息展开**
  - **原理**：悬停时改变卡片的高度或最大高度（`max-height`），并配合 `overflow: hidden` 和 `transition`。
  - **CSS 代码示例**：
    ```css
    .card {
      max-height: 200px; /* 初始收起高度 */
      overflow: hidden;
      transition: max-height 0.5s ease-out;
    }
    .card:hover {
      max-height: 500px; /* 悬停展开高度，需足够大 */
    }
    ```
  - **解释**：初始状态下，卡片内容被裁剪，悬停时平滑地展开到完整高度。

- **效果三：阴影与平移增强**
  - 通常与以上效果结合，增加立体感和层次感。
  - **CSS 代码示例**：
    ```css
    .card {
      box-shadow: 0 4px 8px rgba(0,0,0,0.1);
      transition: all 0.3s ease;
    }
    .card:hover {
      transform: translateY(-10px); /* 轻微上浮 */
      box-shadow: 0 12px 20px rgba(0,0,0,0.2); /* 阴影变大变深 */
    }
    ```

### 5. 完整实操步骤总结
1.  **创建HTML文件**：编写基础的卡片结构，分为前后面。
2.  **设置CSS基础样式**：为卡片容器、内层面板、正反面设置定位、尺寸和背景。
3.  **配置3D变换环境**：在父容器添加 `perspective`，在 `.card-inner` 添加 `transform-style: preserve-3d`。
4.  **隐藏初始背面**：为 `.card-back` 和 `.card-front` 添加 `backface-visibility: hidden`。
5.  **编写悬停状态**：使用 `:hover` 伪类，为目标元素（如 `.card-inner`）定义悬停时的 `transform`、`transition` 等样式。
6.  **添加附加动画**：结合 `box-shadow`、`translate` 或 `opacity` 等属性，让交互更生动。
7.  **测试与调试**：在浏览器中查看效果，调整 `transition` 时间、`rotate` 角度、阴影参数等，直至满意。

### 6. 总结
本教程通过制作一个“悬停资料卡片”实例，系统性地实践了CSS的核心交互技巧。其关键在于**利用HTML构建清晰的双层结构，并借助CSS的`transform`、`transition`以及`:hover`伪类，将简单的鼠标悬停事件转化为流畅、吸引人的视觉反馈**。无论是实现3D翻转还是平滑展开，核心逻辑都是定义元素的初始状态和目标状态，并通过过渡属性连接两者。掌握这些技术，可以广泛应用于各类网页组件的交互设计，有效提升用户界面的动态表现力和专业感。
