# 分享一个超硬核极简的 UI  视觉设计排版 Skill

## 视频概述
本视频由UP主MakerJackie分享，介绍了几个超硬核极简的UI视觉设计排版技能（Skills），旨在帮助用户快速实现一致的品牌视觉风格，提升内容排版的效率和专业性。核心工具包括MJ Adapter、MJ Writer和MJ Design，适用于微信公众文章、小红书排版、PPT设计以及品牌视觉系统构建。

## 核心技能：MJ Adapter
### 功能介绍
- MJ Adapter是一个技能工具，用于将微信公众文章转换为极简黑白银河风格的排版，生成HTML页面，保留内联样式。
- 它还可以生成小红书风格的排版，并将HTML转为图片，保持视觉一致性。

### 使用方法
1. **安装技能**：访问网站skills.makejacky.com，复制安装命令（如npm install MJ Adapter）。
2. **生成排版**：在终端或AI工具中运行安装命令后，发送一篇微信公众文章给工具。
3. **应用排版**：工具会生成HTML页面，用户可以将HTML全选复制到微信公众号后台，样式会自动保留。
4. **小红书排版**：同样生成HTML，转为图片后可用于小红书等平台。

### 效果示例
- 以UP主的“AI与互联网服务订阅费1000多块钱”文章为例，生成后的公众号文章呈现极简黑白风格，与其他文章形成鲜明对比。
- 小红书排版也通过HTML转图片实现，保持统一视觉风格。

## 其他技能介绍
### MJ Writer
- **功能**：基于数字生命卡茨克的写作技能改进，允许用户通过语音输入生成有活人感的长文，适用于自媒体内容创作。
- **应用**：提升写作效率，让文章更自然流畅。

### MJ Design
- **功能**：基于泄露的Cloud Code Design改进，用于设计PPT、UI和其他视觉材料，保持极简硬核风格。
- **示例**：UP主用MJ Design设计了介绍01MVP内容的PPT，风格一致且内容可修改（如调整文字“Hello”）。

## 工具安装指南
### 实操步骤
1. **访问网站**：打开浏览器，进入skills.makejacky.com。
2. **复制命令**：在网站上找到安装命令（例如npm install MJ Adapter），并复制到剪贴板。
3. **运行安装**：在终端（Terminal）或直接发送给AI工具，执行命令（如mpx install或npm install）。
4. **开始使用**：安装完成后，即可通过发送文章或指令来使用技能，生成排版或设计。

## 品牌视觉设计应用
### 案例一：个人品牌0MP
- **设计规范**：包括logo、字体规范、日体设计，用于线下海报、每日分享、数字名片和UI设计。
- **一致性**：整个网站都采用极简黑白风格，后续教程（如0MVBA）也遵循此风格。
- **下载功能**：提供完整品牌视觉下载包，包含SVG、PNG等格式（如logo的黑白版本）。

### 案例二：公司品牌自由维度（Unbound X）
- **主题风格**：设计为太空城主题，使用特定颜色和字体，logo体现太空弧线感。
- **应用扩展**：设计了数字名片、活动卡片、UI案例、特效（如加载动画、旋转动画）和品牌宣传片（用Remotion制作）。
- **下载资源**：同样提供完整下载文件包，方便用户获取和使用。

## 建议与最佳实践
- **设计专属风格**：建议用户为产品、PPT或个人自媒体设计独特的视觉风格，以增强品牌识别度（如看到风格就知道是Jackie的）。
- **保持一致性**：在所有输出（如文章、PPT、网站）中应用同一风格，提升专业感和用户体验。
- **利用工具效率**：通过技能工具快速生成排版和设计，节省时间并保持高质量输出。

本视频核心要点是介绍了一系列极简视觉设计技能（如MJ Adapter、MJ Writer、MJ Design），这些工具能高效地将内容转换为一致的硬核风格排版，适用于公众号、小红书、PPT和品牌视觉系统。通过具体案例（如个人网站和公司品牌）展示了如何构建和应用专属设计规范，强调设计一致性和工具使用的重要性，最终帮助用户提升自媒体和产品的视觉专业度。
