## OpenClaw 安全配置全攻略笔记

### 一、 风险认知：为什么安全至关重要
1.  **权威警告**：安全专家 Burak Agrar 警告，大多数人安装 OpenClaw 等于“不小心把自己的整个人生都交了出去”，离 GitHub 组织被伤光就差一次“提示词注入”的距离。
2.  **核心风险**：OpenClaw 是一个 **7x24小时不间断运行、具有自主权的 AI 代理**。它能自行访问你的所有文件、浏览器及各种服务，且这一切可能在你完全不知情时发生，风险远高于网页插件或本地小工具。
3.  **核心目标**：通过五个关键步骤，将这个“潜在的定时炸弹”配置成一个安全、强大的超级AI助理。

### 二、 五步深度防御体系
#### 第一步：启用沙盒（最核心的防线）
*   **作用**：为AI创建一个完全隔离的环境（“小黑屋”），将其破坏限制在容器内，不影响主系统。
*   **前提条件**：服务器必须安装并运行 **Docker**。
*   **实操配置**：
    1.  终端中编辑主配置文件：`vi OpenCode.json`
    2.  在 `Agents` -> `Default` 下，添加以下 `Sandbox` 配置块：
        ```json
        "Sandbox": {
          "Mode": "sandbox",
          "Scope": "session",
          "WorkspaceAccess": "none"
        }
        ```
    3.  **关键参数解释**：
        *   `Mode: "sandbox"`：**最安全**，强制每次互动都在隔离容器中进行。
        *   `Scope: "session"`：为每次对话创建全新的独立沙盒，实现“一次性房间”，避免交叉感染。
        *   `WorkspaceAccess: "none"`：禁止沙盒内工具访问代理的主工作目录，从根本上阻断访问。
    4.  保存配置文件后，重启网关使配置生效：`OpenCode Gateway Restart`

#### 第二步：配置主动防御（工具策略与审计）
*   **作用**：在沙盒限制爆炸半径的基础上，从源头限制AI“不允许做什么”，实现深度防御。
*   **内置安全工具**：
    1.  **运行安全审计**：在终端执行 `OpenClaw Security Audit`，系统会以红色`Critical`等级报告高危漏洞（如目录权限过于宽松）。
    2.  **自动修复漏洞**：在审计命令后添加 `--fix` 参数（如 `OpenClaw Security Audit --fix`），可自动修复常见安全问题（例如将目录权限自动设为最安全的`700`）。
    3.  **建议**：定期运行此命令，为服务器做“快速体检”。

#### 第三步：贯彻最小权限原则（管理外部连接）
*   **核心原则**：给AI的权限“需要多少就给多少，多一点都别给”。类比：只给保洁阿姨打扫房间的钥匙，而非银行卡密码。
*   **实操示例（以GitHub为例）**：
    *   创建GitHub令牌时，必须选择**细粒度令牌**。
    *   **仅勾选必需的最小权限**（例如，如果只需要读取代码，则只勾选公共代码库的**读取**权限）。
    *   **效果**：即使令牌泄露或AI被控，破坏力也极其有限。

#### 第四步：保持私密（绝对禁止的场景）
*   **强烈警告（来自安全专家）**：**永远不要**将你的个人OpenClaw机器人拉入任何WhatsApp、Telegram等群聊。
*   **原因**：这样做等于将服务器的“后门钥匙”交给了群里的每一个人，他们都能通过机器人间接在你的服务器上执行命令。请记住，它是一个拥有超级权限的终端，而非普通聊天机器人。

#### 第五步：选择更强模型（利用模型自身安全能力）
*   **建议（来自OpenClaw作者本人）**：尽量使用更高级、更强大的大语言模型（如GPT-4级别）。
*   **原因**：更强大的模型在识别和抵抗隐藏在文档或链接中的恶意指令方面能力更强，这本身就构成了一层看不见的安全防护。

### 三、 总结：核心要点
OpenClaw的安全加固是一个构建**深度防御体系**的过程：首先通过**沙盒**隔离其运行环境；其次利用**安全审计与工具策略**主动发现并限制高危行为；然后在连接外部服务时严格遵守**最小权限原则**；同时要绝对避免将其暴露于**公开群聊**等不可控环境；最后可通过选用**更强大的模型**来增强其内在的指令识别与防御能力。完成这五步，你的OpenClaw才能算得上是基本安全，让你在享受其强大功能的同时，能真正睡个安稳觉。
