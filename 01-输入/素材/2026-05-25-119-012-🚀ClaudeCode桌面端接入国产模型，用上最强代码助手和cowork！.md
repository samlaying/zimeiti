# 012-🚀ClaudeCode桌面端接入国产模型，用上最强代码助手和cowork！

> [风飞扬itzhouq01](https://space.bilibili.com/29606169) | [BV1EjoBBLE7G](https://www.bilibili.com/video/BV1EjoBBLE7G) | 时长 2:36

## ClaudeCode 桌面端接入国产模型使用指南

### 一、工具概述与优势
- **ClaudeCode** 是一款强大的AI代码助手，拥有命令行工具和**桌面客户端**。
- 核心优势：客户端版本支持接入**第三方模型**，特别是国产大模型，功能更强大。
- 视频演示使用了 **minimax 国产模型**（minimax m2.7版本）。
- 支持两种模式：**CodeWork模式** 和 **Code模式**。

### 二、安装与配置步骤

#### 第一步：下载客户端
1.  访问ClaudeCode官网，在下载页面选择对应系统（MacOS或Windows）的客户端。
2.  双击安装包进行安装。
3.  **重要提示**：不要点击初始页面的“Get Start”，否则会跳转至官方账号登录页，若无订阅会失败。
4.  **备用方案**：若官网因网络问题无法访问，UP主提供了安装包获取方式（私信获取）。

#### 第二步：开启开发者模式
1.  在客户端界面，点击菜单栏右上角的 **Help**。
2.  依次点击：`Trouble Shopping` -> **开启开发的模式**。
3.  开启成功后，菜单栏会多出一个 **`Developer`** 选项。

#### 第三步：配置国产模型（关键步骤）
点击 `Developer` 菜单，进行以下四项配置：

1.  **网关配置**：
    - 填入国产模型（如minimax）的接口签名URL。
    - 填入模型的 **API Key**（需在模型后台创建并复制）。

2.  **模型配置**：
    - 填入具体的模型名称，例如 `minimax m2.7`。
    - 如果列表中没有，点击“新增”手动添加。
    - **必须勾选**“开启上下文”以支持连续对话。

3.  **登录模式**：
    - 将 **“跳过登录模式”** 的开关打开。

4.  应用配置：
    - 完成上述三项配置后，点击 **`Apply`** 应用设置。
    - 客户端会提示重启，确认重启以使配置生效。

#### 第四步：重启与测试
1.  重启客户端后，配置即生效。
2.  **测试CodeWork模式**：直接发送消息，能收到回复即表示成功。
3.  **测试Code模式**：先打开一个代码文件夹，再发送消息，能收到回复即表示成功。
4.  在界面中可以**自由切换**已配置的不同模型。

### 三、配置要点与问题排查
- **核心配置**：网关URL、API Key、模型名称、跳过登录模式，这四项缺一不可。
- **上下文功能**：配置模型时务必开启，否则无法进行连续代码对话。
- **常见问题**：
    - **下载问题**：多数用户卡在客户端下载环节，需一定网络环境。可寻求UP主的备用安装包。
    - **配置后无响应**：检查API Key是否正确，模型名称是否准确，并确保应用配置后已重启客户端。

### 四、总结
本次教程详细介绍了如何将ClaudeCode桌面端配置为使用国产大模型（以minimax为例）的AI代码助手。整个过程分为下载安装、开启开发者模式、填写关键模型配置（URL、API Key、模型名、跳过登录）以及重启测试四步。成功配置后，用户可以在功能更强的客户端中，利用国产模型进行代码生成与对话，提升编程效率。**关键点在于开启开发者模式并正确完成四项模型配置。**
