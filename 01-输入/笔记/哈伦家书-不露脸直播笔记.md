## 一、三大平台实操手感与核心特征

### 1. 小红书

| 维度 | 关键发现 |
|------|----------|
| **受众画像** | 越来越年轻，个人起步者多，未签公司 |
| **核心需求①：活人感** | 即使用AI配音，也必须体现审美、人味（如视频中加"祝大家新年快乐"等个人化表达） |
| **核心需求②：权威崇拜** | 极度追捧外国导师IP——查理·芒格、纳瓦尔、丹蔻（推特成功学博主）；卖他们的知识库很好卖 |
| **内容偏好①：开头惊艳** | 视频开头必须瞬间命中用户心坎，否则划走 |
| **内容偏好②：捷径心理** | "一个视频带你看懂XX""XX分钟学会XX"——用极短时间让用户觉得能获得复杂技能 |
| **内容偏好③：身份幻想** | 用户未必是顶层，但渴望"看一眼顶层"的爽感，给了这种幻想就给流量 |
| **变现方向** | 最适合知识付费；电商需自有货盘，不建议盲目开店（退店不难但折腾）；接广告当前不太景气 |
| **哈伦自评（做对的）** | 每条视频奔流量最大化——选题够狠、开头够盯人、捷径心理和身份幻想全塞入；辨识度极高，英文账号里找不到同风格 |
| **哈伦自评（做错的）** | **没有及时推产品，流量最好时未完成商业化变现**——一方面心理害怕（一个人hold大流量忐忑），非常后悔 |
| **平台特性** | 一毛不拔，没有播放量分成 |

### 2. 视频号

| 维度 | 关键发现 |
|------|----------|
| **核心逻辑** | 依托微信生态，点赞会出现在朋友圈 |
| **用户心理** | 喜欢"让自己显得牛逼"的内容（显性社交价值） |
| **流量特征** | 内容质感上去后播放量均匀，基本每条过万 |
| **变现亮点①** | 停更两三个月，橱窗仍持续有人购买，佣金自动到账（长尾效应极强） |
| **变现亮点②** | 有播放量基础分成（虽然少，但比小红书好） |
| **变现亮点③** | 非常适合带货 |
| **适合人群** | 比小红书更好入局，更适合普通人轻量起步 |
| **内容策略** | 哈伦直接搬运小红书内容，未做专门运营 |
| **踩坑** | 没做好带货 |

### 3. 公众号

| 维度 | 关键发现 |
|------|----------|
| **用户核心需求** | 极看重判断精准度——"两三句话能不能一下子说到我心坎里" |
| **用户心理** | 跟随权威（与视频号一脉相承） |
| **广告业务** | 比前两个平台更适合普通人 |
| **流量特征** | **极不稳定，三个月一大换血**，平台生态如此 |
| **心理要求** | 比小红书和视频号更需要强大心脏 |
| **哈伦自评（做对的）** | 打造高端人设，所有项目奔高端去；审美本身就是商业溢价 |
| **哈伦自评（做错的）** | 变现效率跟其他创作者比不算高 |

## 二、赛道选择：三维判断法

> **核心问题：你该怎么判断自己做什么？**

### 维度一：表达力

- **测试标准**：你能不能用两三句话，精准命中目标受众的核心焦虑和需求？
- **举例**：
  - 认知类账号（如哈伦）：用户在人际关系里吃亏 → 不知道为什么吃亏 → 不知道怎么预防
  - 金融投资类：受众想把钱生钱、看哪个赛道有机会、未来10年风口
- **关键认知**：哪怕领域别人不看好，只要你能两三句话命准需求，这个赛道就有感觉
- **最根本的问题**：**你到底在赚谁的钱？**——做任何平台的自媒体都必须回答这个问题

### 维度二：优势

- **原则**：人必须顺势而为，短视频就是放大器
- **举例**：
  - 对人性敏感 → 天然适合人性赛道
  - 看东西看得准 → 适合分析/判断类
  - 能选出最赚钱的基金 → 适合投资类
  - 能把平常历史事件解读出不同味道 → 适合历史类
- **要求**：你总得有一块板子是突出的

### 维度三：商业化距离

- **核心**：你选的赛道离钱近不近，直接决定别人掏钱的欲望
- **举例**：
  - 做人情赛道 → 卖洗手液/擦脸油（严重错配，没人买）
  - 做人情赛道 → 卖"社交中为什么吃亏""穷人送礼怎么做"（天然付费缺口）
  - 兼容民生赛道 → 广告主愿意投（离钱近）

### 判断流程总结

```
能两三句话抓到别人命门？ → 是 → 进入下一步
                         → 否 → 重新审视方向

优势和所做之事匹配？     → 是 → 进入下一步
                         → 否 → 调整赛道

别人为你掏钱的欲望强？   → 是 → 确定深耕方向
                         → 否 → 优化商业化路径
```

## 三、四大变现路径详解

### 概览表

| 类型 | 核心卖点 | 适合产品形态 | 个人实力要求 | 适合阶段 |
|------|----------|-------------|-------------|---------|
| **成果类** | 你做到了，别人想复制 | 粉丝数、营收额、升学录取等硬核数字 | ★★★★★ 最高 | 已有显著成绩 |
| **判断类** | 借你的脑子看自己处境 | 一次性分析、拆解、案例库、知识星球 | ★★★☆☆ 中等 | 有独到见解即可 |
| **陪跑类** | 陪你做完一个过程 | 过程陪伴+情绪价值 | ★★☆☆☆ 门槛低但服务成本高 | 喜欢与人打交道 |
| **资源类** | 牵线搭桥、人脉整合 | 介绍品牌方、对接资源 | ★★★★☆（需积累） | 35岁以后考虑 |

### 详细说明

#### 1. 成果类
- 数字必须好看——做了多少粉丝、公司赚多少钱、考上什么学校
- 别人掏钱的本质：**想做到你这种程度**
- 天然好卖，但对个人实力要求最硬

#### 2. 判断类（哈伦自己主力赛道）
- 别人关注你 = 借你的脑子看自己的处境
- 买的是"社交分量""在社交领域少走弯路"
- **产品形态必须短、准、狠**——一篇文章带你看明白事情怎么弄
- **不能做大而全的方法课**——经验判断主观性强（"公说公有理婆说婆有理"），凭什么听你的？
- 别人必须从你的文字中**马上感受到价值**
- 一方面需要个人魅力，另一方面AI可丰富判断厚度
- 可卖：分析、一次性拆解、案例库、知识星球（角色建议）

#### 3. 陪跑类
- 本质是焦虑衍生产品
- 买的是"你陪我做一个过程"
- 真会不会是一方面，能不能让人觉得你会 + 提供情绪价值 = 关键
- **人味最强**，但服务成本高
- **不适合**：不喜欢跟人打交道、不喜欢服务、不需要团队只想自己做的人

#### 4. 资源类
- 牵线搭桥、介绍品牌方
- 需要人脉和整合能力达到一定高度
- 需积累足够多的案例和关系
- **中后期生意**——35岁以后再考虑

## 四、广告赛道判断标准

> **你是否适合接广告？**

### 适合接广告的账号必须满足：
1. **必须是活人**（AI账号则锚定不需要出镜、不需要强烈信任的品类）
2. **能立刻给出立竿见影的效果**

### 四个适合接广告的赛道举例：

| 赛道 | 逻辑 |
|------|------|
| **美妆** | 化妆视频 → 100个女性想学 → 问你用什么产品 → 广告主排队 |
| **家居** | 装修门一打开 → 别人想住你家 → 天然种草场景 |
| **科技测评** | 新品发布 → 不看你测评就不掏钱 → 广告主必投 |
| **民生兼容类** | 离钱最近，广告主最愿意投 |

### 不适合接广告的情况：
- 没有特别一技之长
- 不是功利性领域
- **建议**：老老实实卖自己的东西（利润最大化）

### 关于"卖别人的东西"vs"卖自己的东西"：
- **卖别人的**：仅适合家里开工厂、有一手货源、能拿到全国最低价的情况（本质是信息差）
- **卖自己的**：普通人首选，利润最大化

## 五、从0到1冷启动标准化流程

### 步骤拆解：

```
Step 1 → 先做公众号
         原因：有即时微收益（流量主），维持正向反馈
         
Step 2 → 快速发低价付费文章（如10元）
         目的：验证用户为哪类价值买单
         
Step 3 → 将优质内容跨平台搬运（小红书/视频号）
         目的：测试各平台反馈差异
         
Step 4 → 选定主战场深耕
         
Step 5 → 用最小成本放大已验证模型
         例如：爆款选题批量延展
         
Step 6 → 听数据反馈，不凭感觉迭代
         点击率低 → 改标题
         转化差 → 改痛点
         私信多 → 优化产品
```

## 六、爆款内容工业化生产公式

### 标题公式（四要素占其二即可发布）

| 要素 | 说明 |
|------|------|
| **熟悉感** | 用户一看就知道在说什么 |
| **风险感** | 暗示"不看会吃亏" |
| **半句不说完** | 制造悬念，勾起好奇 |
| **读者瞬间自检** | 用户立刻对号入座"这是不是在说我" |

### 开头钩子（前三句必须完成其一）

1. **对号入座**——直接戳中读者身份/处境
2. **小故事**——用具体场景引入
3. **顶层秘密**——"只有极少数人知道..."
4. **危险预警**——"你正在犯的错误..."

**拒绝长篇大论，前三句定生死。**

### 正文结构

- 每段服务一个核心判断
- 植入"**狠话金句**"——促划线转发
- **杜绝正确而温和的废话**
- 必须激发情绪：**气、笑、悟、惧**（四者至少占一）

### 结尾

- **一句震荡人心的话收束，不可软弱**

## 七、AI协同高效工作法则

### 五条核心法则：

| 法则 | 详细说明 |
|------|----------|
| **① 不许AI猜** | 必须明确告知：受众画像、期望情绪（如"让35岁职场人感到被点醒"）、预期动作（转发/关注/下单） |
| **② 一窗一事** | 标题、封面、观点、情绪反馈、付费钩子等分窗口处理，避免算力衰减 |
| **③ 掐尖筛选** | AI生成大量选题/标题后，只取1-3个最优项，主动淘汰97%的垃圾 |
| **④ 人工主导** | 标题可全托AI；**开头与正文必须人工介入润色**，确保"人味"与语感 |
| **⑤ 视AI为助理而非保姆** | 用具体反馈持续校准（如"这段太平，要让读者被戳到"） |

### 关键认知：
- AI是放大器，不是替代品
- 你给AI的输入质量 = 输出质量的上限
- 人的判断力决定方向，AI决定效率

## 八、流量与商业的底层真相

### 流量周期规律

| 周期 | 现象 |
|------|------|
| **3个月** | 一个低谷期 |
| **6个月** | 一个大拐点 |

- 爆款 = 算法与内容双向奔赴的结果，**非纯个人能力**
- 低谷期的反思比巅峰期的输出更有实操价值（低谷期无幻觉，巅峰期易自负）

### 付费与广告的平台冲突

- 付费内容会挤压广告流量
- 微信倾向圈层分发
- 二者存在平台逻辑冲突——**必须想清楚主攻哪个**

### 商业化本质

```
免费内容 → 吸引泛流量 → 建立认知
付费内容 → 服务愿掏钱的精准人群 → 真正的价值
```

商业化本质是**筛选**，不是讨好所有人。

### 心态建设

| 现金流类型 | 比喻 | 来源 |
|-----------|------|------|
| 稳定现金流 | "奶茶钱" | 流量主收益、播放量分成 |
| 爆发性收入 | "海鲜大餐" | 爆款内容、付费产品 |

**长期靠付费产品构筑护城河**，两种现金流并存是常态。

## 十、公众号数据复盘：如何从数据中找到"发财密码"

### 核心看三个指标

| 指标 | 含义 | 优化方向 |
|------|------|----------|
| **转发** | 含金量最高的动作——你说到他心坎里了，或他觉得以后用得上 | 反映内容的"社交货币"价值 |
| **推荐率** | 微信把文章放到流量市场后，大盘用户的点击率 | 点击率高 → 推荐率高；标题和封面决定 |
| **阅读时长** | 决定完读率——即使点击率高，读不完推荐也会下降 | 内容质量和结构决定 |

### 其他有价值的用户动作

| 动作 | 信号 |
|------|------|
| **收藏** | 想存着慢慢消化 |
| **私信** | 遇到类似情况想让你看，或想直接下单 |
| **付费** | 用钱投票，最真实的验证 |

### 数据复盘的核心方法

> **关键：找到"同一个动作"在"同一类内容"上反复出现的规律**

**哈伦的实操案例**：
- 社交类文章 → 转发持续极高（2000+、5000+、6000+）
- 推荐率稳定在 80%~95%
- 阅读时长持续很长
- **结论**：读者在跟着他的内容深度吸收 → 社交赛道可以深耕变现

### 高质量反馈 vs 低质量反馈

| 类型 | 特征 | 价值 |
|------|------|------|
| **高质量反馈** | 别人能从你这里带走一个操作——看明白一个问题，或明白自己该怎么干 | 真正能变现的信号 |
| **低质量反馈** | 纯情绪安慰——"鸡汤写得真好""说得太对了" | 转发高但无法变现，什么都说明不了 |

**举例**：转发很高的鸡汤文 → 能说明什么？什么都说明不了。没有可操作的价值 = 变现密码不在这里。

## 十一、数据复盘 → 确定你适合卖什么（四类对照表）

| 数据信号 | 用户行为特征 | 适合卖什么 |
|----------|-------------|-----------|
| "你怎么做的？能不能教教我？" | 想学你的方法 | **卖方法** |
| 一类文章长期转发很高，原因是"你说得很透，确实把我看明白了" | 想借你的脑子 | **卖判断**（如哈伦） |
| "我坚持不下去了，老师你能不能带着我一起做" | 需要陪伴和推动 | **卖陪跑** |
| "你认不认识谁？对某个机会了解多少？" | 需要人脉和信息 | **卖资源**（苗头出现） |

> **核心认知**：很多账号变现难，就是因为什么都想干一点，用户搞不懂你到底值钱在哪。

## 十二、产品落地：从命名到成交的完整链路

### 第一步：给问题命名（最关键）

#### 为什么要命名？

- 用户掏钱前，脑子里需要一个画面
- 你不能什么都往产品里塞 → 用户必须联想到自己
- **问题一旦被具体化，用户就会慌**："原来我吃这么久的亏，就因为这个？"

#### 好问题的三个要素（至少占两条才卖得动）

| 要素 | 说明 | 举例 |
|------|------|------|
| ① **是自己的亏** | 让他觉得你在说他自己身上的事 | 讨好型人格的人在社交中总吃亏 |
| ② **已经拖了很久** | 这个问题不是今天才有，已经持续影响他 | "这个问题跟了我好多年" |
| ③ **继续不解决有代价** | 如果不管，后果会越来越严重 | 继续讨好 → 越来越不被尊重 |

**哈伦的产品命名案例**：
- 产品名："**不动声色让人重视你**"
- 直接命名了问题 → 用户一看就知道这是不是在说他

### 第二步：免费推广内容——让用户承认自己有问题

#### 免费内容 vs 付费内容的分工

| 类型 | 目的 | 内容逻辑 |
|------|------|----------|
| **免费内容**（日更） | 让用户意识到"我有这个问题，我卡在这" | 揭示问题、提供看懂但不给完整解法 |
| **付费内容** | 带着用户做到 | 直接给解决方案，帮用户少走弯路 |

**核心逻辑**：免费让他"看懂了但不知道怎么办" → 付费才给"12345怎么做"

#### 实操案例：求人办事的推广文章

**哈伦不会直接教你怎么求人，而是这样写**：

```
标题方向：顶层的人是怎么开口的？政商圈子里那些人，求人之前做了哪些铺垫？
↓
开头：非常狠的钩子，让人身临其境
↓
激发情绪：穷人的羞耻感
↓
核心判断：你失败的原因是没搞懂你求的东西对别人意味着什么
↓
读者看完：难受，好像看懂了，但具体怎么做？→ 掏钱的理由
```

#### 推广文章的实际效果

| 类型 | 带动销量 |
|------|---------|
| 付费直发（直接甩一篇付费文章） | 100~120 单 |
| **推广文章**（5篇，按上述套路写的） | **320~340 单** |

> "人在家中写，钱从天上来"——关键是你有没有精准命名别人的问题。

### 第三步：高频推广——发少了才是最大的问题

> **"你前面两步都做好了，怎么还是卖不出去？那就是你发少了，哥们。"**

#### 核心类比

| 线下 | 线上 |
|------|------|
| 线下销售一个个去谈 | 一篇推广文章 = 一次一对多的销售谈话 |
| 触达的人越多，成交越多 | 发得越多，触发的人越多，成交越多 |

#### 推广频率

| 做法 | 结果 |
|------|------|
| 天天发 | 用户烦你 |
| **两三天推一次** | **最佳节奏** |
| 一个月才发一次 | 没人记得住你 |
| 发一篇没人买就不发了 | **最蠢的做法** |

### 第四步：听反馈照做（商学院不教的商业精华）

> "发什么内容给我带来流量，发什么内容我收钱快，我就照着模式去发，直到这个动作做不了为止。"

#### 数据诊断对照表

| 数据表现 | 问题所在 | 优化方向 |
|----------|---------|---------|
| 点击率太低 | 标题和入口不对 | **改标题** |
| 点击率高但买的人少 | 痛点没打透 | **改推广内容的痛点** |
| 买的人多但反馈一般 | 产品本身没让人感觉超值 | **打磨付费内容质量** |
| 私信多，问价格 | 用户有需求但转化路径不清晰 | **优化付费入口** |

#### 核心心法

```
点击不行 → 改标题
转发不行 → 改痛点
交互不行 → 改产品
```

**商业不是靠一篇爆款把东西拉爆，而是不断寻找最精准的入口。**

## 十三、还没开始的人怎么办？先解决地基问题

> **"别研究爆款，别学AI，浪费时间。先解决一个最基本的问题：别人凭什么停下来看你？"**

### 三个地基要素

#### ① 让人知道你是谁

- 一刷到就知道你做什么领域的
- **反面教材**：点进一个账号，看三条内容，不清楚到底想表达什么 → 完蛋
- 原因：**观众没有耐心，没有义务猜你是谁**
- 做法：明确身份——你解决什么问题，对谁说话

#### ② 让人觉得关注你有用

- 不是空泛的"带来价值"，而是具体的
- "看你的文章我就能想明白一个问题""能明白一个顶层的玩法"
- **用途越具体，停留和转化概率越高**
- 回到核心方法：用两三句话精准描述受众需求，根据需求决定提供什么

#### ③ 稳定的气质

- **信任是靠重复建立的**
- 反面教材：今天发鸡汤，明天发吐槽，后天乱搞，大后天正经分析 → 来三次每次看到的都不同 → 不信你了
- 正面做法：不需要每天发，但每次出现都让人觉得"还是那个味"

> **三个东西搞定了，才有资格谈下一步。没有这个地基，学再多爆款技巧，流量来了也接不住。**

## 十四、如果一切清零，从零来一次的完整路径

> **"如果现在一切成绩清零，我从零来一次会怎么做？"**

### Step 1：先做公众号

**为什么不是短视频？**
- 公众号发文章给钱（哪怕几分几毛）
- 花一份时间，总得有流水出来
- 短视频发再多，平台不给你钱 → 做着做着心里难受 → 不想做了

> **核心原则：先做让自己产出能给你反馈的事。**

### Step 2：迅速发付费文章

- 选择出产品
- 搞清楚东西对谁有用
- 验证用户为你的哪个价值买单
- **哪怕10块钱、两三块钱——销量本身是最好的商业反馈**
- 不卖就永远验证不了

### Step 3：公众号做到一定阶段 → 再回短视频

**为什么要这个顺序？**

| 对比 | 公众号 | 短视频 |
|------|--------|--------|
| 竞争程度 | 相对温和 | 残酷得多 |
| 起号红利 | — | 起号阶段平台愿意推你，发啥都火 |

> **红利窗口要用在刀刃上**——先在公众号验证商业模式，再用短视频放大。

### Step 4：一份内容多平台开花

- 公众号文章 → 搬去小红书 → 搬去视频号
- 一份时间产出一份价值 → 在多个地方开花
- 哪个平台反馈最好 → 在哪里扎根深耕
- 开不了花的 → 埋个种子也行

### Step 5：用最小成本验证 → 发现信号后加码

- 一旦发现"这么写卖得好" → 按这个动作加码力推
- 已经解决最难的验证问题 → 剩下的就是放大
- **不要一开始想做大，老老实实打地基**

### 额外心法：商业想法出现 → 马上动手

**哈伦的案例**：
- 做这次分享会：从做决定 → 做海报 → 发帖，**只用了半小时**
- "再拖的话这个劲儿都散了"

> **做到一个阶段之后一定要主动走出来——让受众走进你脑子，看看你怎么思考问题。受众一旦认识了你，往后卖什么都好卖。**

## 十七、AI选题实操流程

### 已经发过内容的人

```
Step 1 → 把数据最好的几篇文章拎出来
Step 2 → 喂给AI，让它自己分类（不用你规定分几类，AI会读出规律）
Step 3 → 从高表现的类目中，让AI扩充选题（每个类目30~50个）
Step 4 → 掐尖筛选——100个里可能只有1~3个符合直觉和审美
Step 5 → 剩下97个？扔了。
```

### 没发过内容的人

```
Step 1 → 明确受众画像：
         - 你要对谁说话？
         - 他看完一篇文章转发给谁？
         - 他日常搜什么问题？
         - 他焦虑什么？
Step 2 → 让AI基于受众画像生成选题
Step 3 → 同样掐尖筛选
```

### 选题筛选的三个判断指标

| 指标 | 测试方法 |
|------|----------|
| **代入感** | 你看到这个标题，能不能立刻想到一个具体的人和场景？ |
| **戳到感** | 不是"正确的道理"，而是"天灵盖被打开"的瞬间感觉 |
| **锐利度** | 标题要像针一样插进读者脑子里，不是请读者喝杯茶 |

> **好的选题不是"软绵绵让读者喝茶"——喝完茶他们撇嘴就走。**

### 核心警告

| 错误做法 | 正确做法 |
|----------|----------|
| 让AI大包大揽一次性搞定 | 分工处理，每个环节独立把关 |
| 不舍得扔AI生成的垃圾选题 | **掐尖**——只取1~3%，扔掉97% |
| 前期追求爆款 | 前期别看数据，先拿到基础反馈 |
| 凭脑子里猜 | **看后台数据** |

## 十八、AI选题的"掐尖"法则（深度展开）

### 为什么必须掐尖？

- AI生成选题天然会产生大量垃圾
- **每次让AI生成100个选题 → 真正能用的只有1~3个**
- 如果让AI全方位一次性搞定 → 绝对写不出好东西 → 平台不可能给流量

### 数据已验证过的选题如何复用

> "你的受众爱看什么，不用猜——数据已经告诉你了。"

**哈伦的方法**：
- 社交类文章 → 同一个套路、同一个选题 → 从不同角度去切
- 之前爆过的内容 → 后面还能爆
- **让AI把你已经赢过的东西，再让你重复赢**

### 前期心态建设

> **"前期别追求爆款，求求你。"**

| 阶段 | 正确心态 |
|------|----------|
| 0~初期 | 不看数据，拿到反馈就是胜利 |
| 有基础数据后 | 爆款 = 把拿过反馈的东西再放大 |
| 没有基础数据就想要爆款 | 不可能，别想了 |

## 廿一、AI选题的第三个判断指标：真东西可写

- AI生成的选题只是起点，最终能不能写下去取决于你有没有真实的经历和感悟作为底层素材

## 廿二、爆款内容生产：标题篇（深度展开）

### 核心公式（再强调）

```
标题 = 熟悉感 + 风险感 + 半句不说完 + 读者瞬间自检
```

**四要素至少占两个才能发，占一个直接扔，占越多越好。**

### 实操案例拆解

**案例标题**："别人敢反复消耗你，通常就看准了你这4点"

| 要素 | 对应 |
|------|------|
| **熟悉感** | "被消耗"——大家都经历过，一看就知道在说什么 |
| **风险感** | "是不是我哪个动作做错了，让别人觉得可以这样对我？" |
| **半句不说完** | "哪4点呢？"——不说清楚，必须点进来才能看 |
| **读者自检** | "说的是不是我？我有没有在被消耗？" |

### 标题验收三关

| 关卡 | 测试 |
|------|------|
| **一秒代入** | 读者能不能立刻想到自己？ |
| **不点心痒** | 不点进来心里痒痒的？ |
| **好奇心勾起** | 感觉里面有个认知自己没想到，真的很想看看？ |

### 标题必须反常识

- 提供别人觉得新鲜的视角
- **"正确而温和的话，在自媒体领域是特别大的缺点"**
- 标题不得罪人、不让人心头一紧 → 数据肯定推不起来

### 标题生产流程（AI协作）

```
Step 1 → 挑一个你有感觉的选题
Step 2 → 把选题 + 标题公式一起喂给AI
Step 3 → 提示词："请根据这个选题和这个公式，给我列出10篇文章的标题，
         点击率、情绪冲击力从高往低排列"
Step 4 → 如果文章已经写完 → 直接喂全文 + 公式，让AI起10个标题
Step 5 → AI给10个 → 你留2个就不错了（掐尖法则不变）
```

## 廿三、爆款内容生产：开头篇（深度展开）

### 开头的使命

> **标题把人骗进来，开头把人留住——让读者舍不得划走。**

### 四种开头范式

| 范式 | 说明 |
|------|------|
| **小故事** | 用具体场景快速引入 |
| **对号入座** | 3~5句话让读者完成身份匹配 |
| **顶层秘密** | "看完我就能掌控一个关于顶层的秘密" |
| **危险预警** | "我不看完就血亏" |

### 开头的核心原则

- **不长篇大论，稳准狠**
- 普通人做认知类公众号 → 必须给一个**狠一点的东西**
- 让读者觉得"这篇好像真的有东西，不看就亏了"
- "就像往读者头上摔小胡萝卜一样"

### 开头的高阶技巧：认知扇脸

> **"用认知扇他的脸"——一句特别难听但特别管用的话**

- 读者反应："我靠，这个人看世界的角度还真是有点东西"
- **前三行提出一个让人心里发懵的概念**
- 读者觉得"你说的好像有道理，但我3秒内想不明白你的逻辑"
- 读者悬空 → 好奇心勾起 → 必须往下滑 → **完读率就有了**

### 开头验收标准

- 前3~5句里，读者能不能感受到"不看就亏了"？
- 有没有完成对号入座 / 被小故事吸引 / 被顶层秘密勾住 / 被危险感抓住？

## 廿四、爆款内容生产：正文篇（深度展开）

### 正文的核心验收标准

| 标准 | 说明 |
|------|------|
| **每段服务一个核心判断** | 这篇文章到底想对受众说什么？ |
| **必须有划线金句** | 有没有一句话让读者觉得"我靠你说得真狠，真说到我心里去了，我要截图划线带走"？ |
| **看完能一句话复述** | 读者看完能不能用一句话说清你在讲什么？**不能 → 这篇别发** |

### 如何制造划线金句

- **写你配不上去的东西**——不要有羞耻感
- "我可以用顶层吗？我可以用富豪吗？"——别想那么多，就写
- 认知类公众号 → 你必须营造出"你确实比读者认知高"的感觉

### 正文中的情绪节奏

> **"无论你写得多封闭，中间都必须有一段话让读者觉得：这个作者是站在我这边的。"**

- 这是建立信任的关键转折点
- 让读者在被"扇脸"之后感到被理解

### 让读者不要思考

- 大部分人看公众号是来找心理安慰的
- 让他们跟你的节奏走
- 不说正确的废话，不动脑子
- "感觉模模糊糊的觉得应该是我说的那样就行了"

---

## 廿五、爆款内容生产：结尾篇

### 核心原则

> **"结尾你不能软——软了别人就记不住。"**

### 结尾要求

- 用**一句话说出全文精髓**
- 这句话必须能在别人心里**震荡很久**

## 廿八、心态篇：流量真相与心态建设（深度展开）

### 流量周期规律（实证数据）

| 时间段 | 哈伦实况 | 规律 |
|--------|---------|------|
| 10月~11月 | 连续爆款，收入可观 | 上升期 |
| 12月 | 锐减到1000多曝光量 | **低谷期（约3个月一轮）** |
| 1月下旬 | 朋友圈文章、顶层家庭文章、贾国龙文章引爆 | **爆发期** |
| 2月 | 两篇体制内文章被删、限流 | 平稳但受平台打击 |
| 3月 | 三篇大爆款（6万+9万+9万） | 回温期 |
| 4月 | 进入平淡期 | **又一轮低谷（约6个月一大周期）** |

### 平台推流逻辑的重大变化

| 过去 | 现在 |
|------|------|
| 长尾流量——文章三四天后突然爆 | **单点爆破**——一个晚上给你好几万，第二天就不推了 |
| 慢慢积累 | 更功利——要求短时间内抓住用户注意力 |
| 更像"慢功夫" | 更像"流量生产的机器" |

### 付费内容与广告流量的天然冲突

> **"你想赚广告的钱和你赚产品的钱，这两件事在平台眼里是冲突的。"**

| 现象 | 原因 |
|------|------|
| 发付费文章后大盘流量下降 | 微信只在小圈子里推，不给外面流量 |
| 广告主想要大播放量 | 但付费内容把分发锁在了私域 |
| 一边卖货一边被骂流量差 | 平台逻辑就是这样——你收费就推给你的受众，别想要大盘流量 |

### 低谷期 vs 巅峰期的价值判断

> **"如果我在低谷时告诉你什么东西能用，那这玩意就是真TM有用。如果我在巅峰期告诉你什么有用，可能只是适合我个人的。"**

- 爆发期的话是"飘的"——容易自负，觉得是自己牛逼
- 实际是平台+命运+运气+内容的**双向奔赴**
- **逆风期讲出来的东西更真实，更有普适价值**

### 流量主收入的真实预期

| 阶段 | 收入水平 | 比喻 |
|------|---------|------|
| 差的时候 | 一杯奶茶钱 | "平台请你喝个奶茶，你继续给我创作" |
| 好的时候 | 一顿海鲜大餐 | 爆款日可能几百块 |
| 稳定期 | 一个月一两千 | 起步阶段的正常水平 |

> **"不要指望流量主给你泼天的富贵，那不可能。"**

### 爆款与信任的本质区别

| 类型 | 作用 | 特征 |
|------|------|------|
| **爆款内容** | 拉人头进来 | 浏览量高、评论多、但信任沉淀弱 |
| **沉淀内容** | 建立信任 | 数据不一定好，但写得很真、很深 |

> **"爆款基本上没有什么信任——别人点进来消费一下你的内容，划走就这件事。沉淀一定是你那些数据不太行、但写得很真的东西。"**

- 两者不要混为一谈
- 不要要求同一篇内容既爆又有深度沉淀——这是不可能的

### 关于骂声与流量

**哈伦的实操案例**：

| 文章 | 评论情况 | 效果 |
|------|---------|------|
| 朋友圈观察文 | 262条评论，260条骂 | 大盘流量被拉起 |
| 顶层家庭不让孩子碰三件事 | 410条评论，350条骂 | 全网被短视频博主抄袭搬运 |
| 贾国龙相关文章 | 评论区吵翻天 | 人生最高单日稿费619元（其中600元来自这篇） |

> **"你写骂人的东西，它给你带来的是热度还是信任，你要分清楚。"**

- 有些内容适合拿流量
- 有些内容适合拿成交
- **两种内容不要混着看**

## 廿九、AI协同工作深度指南（新增细节）

### 四大常见AI写作问题及解决方案

#### 问题一：AI说的话空了吧唧

**原因**：没告诉AI这篇文章写给谁

**解决**：必须明确三点——
1. **受众是谁**（如"35~45岁职场中层"）
2. **期望的情绪**（如"被我气到""被我点醒""觉得我值得跟随"）
3. **期望的动作**（如"点关注""转发""做数据"）

> 不告诉AI这些 → AI就是废话生产器 → "不是而是""排比重复""梳理" → 跟没说一样

**修复方法**：把AI生成的废话句子单独挑出来，重新提示——"用最土的大白话，用三岁小孩都能听懂的逻辑，一个疲惫的人刷到这句话能不能迅速明白？重新说。"

#### 问题二：AI观点太温和

**原因**：AI天然设计为温和——因为大多数用户不是做自媒体的，如果AI呛他们，用户会投诉

**解决**：
- 告诉AI"这篇文章要用认知扇脸"
- 明确允许AI做出"得罪人"的举动
- 给出许可后，AI的输出就不一样了

#### 问题三：单窗口算力衰减

**原因**：一个窗口里塞太多东西 → AI算力分配衰减

**解决**：
- **一个窗口只解决一个问题**
- 哈伦一篇文章开五六个窗口：

| 窗口 | 内容 |
|------|------|
| 窗口1 | 标题 |
| 窗口2 | 封面点击率 |
| 窗口3 | 每个观点分别讨论 |
| 窗口4 | 读者看完什么感受 |
| 窗口5 | 该卖什么付费产品 |

> "你刚打开窗口时AI算力最强，慢慢往下用就钝了——跟人一样。"

#### 问题四：AI的输出没有你想要的味道

**原因**：你自己不知道自己要什么

**解决**：
- 给**具体反馈**，如："这段太平了，我想让读者看完绝对被戳到，从而更加信任我。你现在写得像个百度百科。"
- 反馈越具体 → AI调整越快
- 如果一直说不清楚 → AI就一直猜 → 猜出来的东西肯定不是你想要的

> **"如果你跟AI工作半天，产出来的东西你都想把手机摔了——说明本来你就不知道自己要什么。"**

### 与AI协作的两条铁律

| 铁律 | 说明 |
|------|------|
| **① 不要让AI猜** | 你对自己有多诚实，AI给你的结果就有多优秀。哪怕想法上不了台面，也要诚实告诉AI |
| **② 一个窗口只解决一个问题** | 不要在一个对话里塞太多任务 |

### 不同AI的分工建议

| AI | 适合场景 |
|----|---------|
| **ChatGPT** | 生活类、思维讨论 |
| **Claude** | 协作类、执行类、直接写文章 |
| **其他AI** | 搜索国际新闻、特殊信息 |
| **豆包** | 生图 |

> 这只是适合哈伦的工作流，你的AI适合帮你什么需要自己尝试。

### AI使用的终极心态

> **"不要指望AI像保姆一样给你出文章出结果——全网卖'爆款智能体'的都是骗子。"**

- 没有市面上任何一个工具可以代替你自己的判断力
- 把AI想象成助理——你是老板，AI是你的小弟
- 赋予AI一个人的身份，像跟人打交道一样工作
- 每次微调、多次确认、聊天式推进

### 润色与排版

- **润色修改是最后一步也是最重要的一步**——得让人看出来这是个人写的
- 排版要有**呼吸感**，善用回车
- 不要一堆字糊在一起
- 排版不好会打击点开后的满足率——标题和封面做得再好，排版看不下去也白搭
