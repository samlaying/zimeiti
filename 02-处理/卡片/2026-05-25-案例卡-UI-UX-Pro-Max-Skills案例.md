# UI UX Pro Max Skills案例

一个专门用于前端设计的[[Skills]]，其架构是[[Skills三层架构设计]]的绝佳实践。

**架构体现**：
- **入口文档**：包含前端设计的触发关键词、设计原则、经验教训和检查清单。
- **CSV数据存储**：将样式、配色方案（如“Steel”样式）、框架兼容性（React/Vue）等细节数据外部化存储。
- **搜索脚本**：使用[[BM25搜索脚本应用]]，根据用户输入的关键词（如“按钮”）从CSV中检索相关信息，并组合成设计系统文档供AI参考。

**价值**：该案例证明了通过架构设计，可以在一个Skills中管理和调用海量前端设计细节，确保AI生成高质量代码，同时保持高性能。
