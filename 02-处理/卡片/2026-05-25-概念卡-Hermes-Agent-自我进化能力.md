# Hermes Agent 自我进化能力

**自我进化**是 **Hermes Agent** 的核心能力，通过自动处理 **Skill** 实现。

其运作机制包括：
- **自动提炼**：完成任务后，自动从对话中提炼出可复用的 **Skill**。
- **Skill 迭代**：Skill 越被使用，就越具体、越贴合实际场景，实现本地演化的“复利系统”。
- **闭环流程**：涵盖 Skill 的**创建 -> 读取 -> 发现缺口 -> 更新**的完整自主迭代闭环。

这使得 Agent 具备了“越用越聪明，用得越久价值就越高”的特性。
