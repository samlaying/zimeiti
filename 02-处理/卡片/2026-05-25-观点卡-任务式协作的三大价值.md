# 任务式协作的三大价值

采用 [[任务式 Agent 协作]] 模式相较于直接对话，能带来三个核心价值：
1.  **异步协作**：协作变为“一步式”，用户只需发布、验收任务，无需实时盯着 Agent，极大解放了时间。
2.  **并行任务管理**：通过任务看板清晰查看所有任务状态（执行中、需验收、已完成），便于直观管理多个并行工作。
3.  **注意力升华**：这是**最有价值**的一点。用户无需介入具体执行细节，注意力可以完全聚焦在更有创造性的工作上——即“提什么任务”以及“任务的定义到底是什么”。
