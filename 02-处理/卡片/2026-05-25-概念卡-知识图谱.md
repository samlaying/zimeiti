# 知识图谱

## 知识图谱（Knowledge Graph）
在 Search Agent 数据合成方法中，**知识图谱**（如 Wikidata）是合成数据的**原始数据源**。它以结构化的方式存储实体（如人物、地点）及其之间的关系（如“出生地”、“作者”）。

### 在数据合成中的角色
知识图谱的结构化特性使得合成流程具有可控性和可追溯性：
1. 提供**子图**作为问题生成的起点。
2. 其中的实体关系链路天然定义了问题的**多跳推理路径**。
3. 为生成的复杂问题提供了**确定的、可验证的答案**，确保了数据质量（可解性）。

知识图谱是 [[Search Agent 数据合成流程]] 得以实施的基础。
