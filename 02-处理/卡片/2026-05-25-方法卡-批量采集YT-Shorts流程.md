# 批量采集YT Shorts流程

批量采集 [[YouTube Shorts]] 高热素材的典型流程如下：
1.  **环境准备**：安装 Python 及必要库（如 `requests`， `youtube-api-python-client`），或申请 [[YouTube Data API]] 密钥。
2.  **脚本编写**：定义目标（关键词/频道ID），编写函数获取视频列表并提取元数据（标题、播放量、互动率等）。
3.  **数据筛选**：基于播放量、互动率等指标过滤出“高热”素材。
4.  **数据存储**：将结构化数据导出为 CSV、JSON 或存入数据库。
5.  **自动化运行**：设置定时任务（如 `cron`）周期性执行脚本，持续更新素材库。
