# 生图模型：Gemini 3 (Nano of Nano)

这是用于驱动 [[配图 Skill]] 生成图片的底层 AI 模型，全称为 **Gemini 3 图片模型**，别名 **Nano of Nano**。

**关键信息**：
- **用途**：执行图片生成指令，是自定义 Skill 时必须指定的模型。
- **资源获取**：可以在 **ASQ6** 平台免费申请该模型的 API 额度。
- **额度**：每月提供 **10 美金**的 API 额度，基本满足个人日常使用需求。
