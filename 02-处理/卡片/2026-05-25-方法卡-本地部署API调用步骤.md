# 本地部署API调用步骤

在本地部署并调用[[NanoBanana2图像模型]]的API，步骤简单，环境冲突少。

**主要步骤如下：**
1. **下载与安装插件**：复制链接后，使用秋叶版管理的“高区选项版本管理”功能安装新拓展。
2. **注册与充值**：在AI工坊网站注册账号，首次注册会赠送点数（需充值）。
3. **获取API密钥**：在“令牌”页面添加新令牌，复制生成的API密钥。
4. **配置使用**：在已安装的插件中填入API密钥，即可通过插件调用本地API生成图像。

整个过程几乎无环境冲突，安装完成后打开流程文件即可使用。
