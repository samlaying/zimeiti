# 真正的多Agent架构

很多人以为让一个AI在对话框里扮演多个角色就是多Agent（Multi-Agent）了。但真正的多Agent架构是：
- **七次完全独立的API调用**，每个节点只接收它需要的特定字段。
- **上下文完全隔离**，避免信息污染。
- 每一步都有**JSON文件记录**，便于精确排查错误。
- 具备**可规模化**（批量运行多个方向）和**灵活性**（可随时更换底层模型）的特点。

[[Flex内容生产系统]]采用的是后者，这使得从输入到输出不到3分钟，且能实现自动化批量产出。
