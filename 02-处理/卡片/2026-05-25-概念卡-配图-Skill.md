# 配图 Skill

配图 Skill 是一种 AI 工具，能够根据用户指令（如“帮我配图”）批量生成与文本内容逻辑紧密结合的图片，尤其适用于为文章、Logo、海报、故事版等场景设计配图。

**核心特点**：
- 生成风格：手绘风格，包含图片、icon 和文字描述。
- 工作逻辑：在用户完成文章后，一句指令即可批量生成匹配各段落逻辑的配图。
- 优势：显著提升设计效率与内容的可理解性。
