# 语义化HTML构建导航栏

在构建网页导航时，应优先使用HTML5的语义化标签，这不仅有利于代码可读性，也对SEO和无障碍访问至关重要。

## 最佳实践
- 使用 `<nav>` 标签包裹整个导航区域。
- 在 `<nav>` 内部，使用 `<ul>` 和 `<li>` 来构建菜单项列表，这是最标准、最清晰的结构。
- 为导航栏容器（如`<nav class="glass-nav">`）添加有意义的类名，便于CSS和JavaScript进行样式控制与交互绑定。
- 图片元素（如头像）使用 `<img>` 标签并添加适当的 `alt` 属性。
