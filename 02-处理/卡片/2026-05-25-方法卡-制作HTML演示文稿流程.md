# 制作HTML演示文稿流程

在[[Claude Code]]中使用[[PPT Skill]]制作高级感HTML演示文稿的完整流程。

1.  **提出需求**：向Claude Code描述制作目标。例如：“我想为我的人工智能初创公司制作一份推荐演示文稿”。
2.  **需求澄清**：Skill会主动询问一系列问题，如演示文稿使用场景、幻灯片页数、是否已有内容框架、最终是可编辑还是仅供展示等。
3.  **选择风格**：它会生成3个独特的风格预览文件供选择。告知喜欢的风格编号后开始制作。**注意**：选择风格与最终生成效果可能存在差异。
4.  **微调与优化**：在生成的基础上进行调整，如通过截图或文字描述指出图标位置、Logo大小，或直接指出乱码错误让其修改。
5.  **添加高级功能**：
    - **鼠标动效**：可要求添加，并提供多种效果（如光标效果、跟随光运、卡片放大旋转、光标轨迹等）供选择。
    - **图片与链接**：要求添加图片展示，并可链接到外部网站。需将图片文件放到项目工作区目录并按其识别的格式命名，它会自动替换。
