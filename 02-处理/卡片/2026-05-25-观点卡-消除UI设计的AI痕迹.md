# 消除UI设计的AI痕迹

当前用[[AI编程工具]]生成的UI普遍存在“AI味”，即设计风格缺乏人性化和专业性，容易被一眼看出是机器生成的。

这个问题的根源在于AI的训练数据或设计逻辑缺乏对顶级人类设计规范的深度理解和应用。[[AllSang的Dieside开源项目]]通过强制AI在生成前学习并遵循专业大厂的设计规范，为解决“AI味”问题提供了一个有效方案，让AI生成的UI具备“人味”。
