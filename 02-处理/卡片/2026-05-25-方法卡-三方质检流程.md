# 三方质检流程

[[Flex内容生产系统]]中确保内容质量的全自动三方质检流程：
1.  **标准设定**：预先定义“好内容”的标准。
2.  **初审评分**：Claude按标准对稿件评分。
3.  **挑刺审核**：另一个AI（如GPT）专门寻找稿件中无聊、AI味重或可能导致用户划走的问题点。
4.  **最终裁决**：由GPT做出是否通过的最终裁决。
5.  **自动修正**：不合格的稿件会被自动打回并要求重写，直至达到标准（如从4分修正至8分）。

整个过程无需人工干预，确保输出内容质量稳定。
