# 多Agent协作四大模式谱系图

> 协作模式谱系图：路由-计划-监督-交接。

## 核心内容
基于中心化和灵活性的程度，可以将多Agent协作分为四种典型模式。简单路由是单Agent全权处理；静态计划是预先规划好顺序；动态监督由管理者Agent动态调度；同伴交接则完全去中心化，由Agent之间相互委派任务。选择哪种模式取决于任务的结构化程度和对灵活性的要求。
