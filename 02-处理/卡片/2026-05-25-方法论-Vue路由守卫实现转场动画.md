# Vue路由守卫实现转场动画

> 用Vue的router.beforeEach守卫，就能把动画和路由跳转牢牢绑定在一起。

## 核心内容
使用Vue等前端框架的路由功能可以实现无刷新跳转，完美解决动画问题。核心步骤是：1. 在全局路由前置守卫（beforeEach）中触发动画入场，并传入next函数。2. 在入场动画的setTimeout回调中延迟执行next()以完成路由跳转。3. 在页面挂载后（mounted）通过轮询document.readyState来检测新内容加载完成，随后触发动画退场。

## 原文关键段落
> this.$router.beforeEach((to, from, next) => {
>     this.$refs.loading.loadingIn(next); // 传入next函数
> });
