# 公众号转PDF方法

在[[COZE技能与SPL|COZE平台]]上，通过自然语言指令调用特定[[COZE技能与SPL|SPL]]，可将公众号文章转换为标准PDF文档。

**操作步骤**：
1.  **新建任务**：创建一个新的对话任务。
2.  **调用技能**：发送指令“请调用公众号下载为PDF”（注意与下载为HTML的技能区分）。
3.  **提供链接**：粘贴另一篇公众号文章的链接。
4.  **执行与获取**：点击发送，等待任务完成。

**效果验证**：
- **版式一致**：生成的PDF版式与原文高度一致，适合长文、含大量图片和表格的内容。
- **可编辑性**：PDF中的文字内容是**可复制**的，便于后续编辑和再利用。
