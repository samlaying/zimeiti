# Superpowers Skills

**Superpowers Skills** 是一个在 **Vibe Coding** 时代用于驾驭AI编程（尤其是命令行AI工具如[[命令行AI模式：Codex与Cloud Code]]）的强大技能框架。

- **项目热度**：GitHub Star超过7万，持续快速更新。
- **核心理念**：[[“慢就是快”的AI编程哲学]]。
- **工作原理**：当AI（如Codex）启动时，会自动从项目的`.codex/skills/`目录加载所有技能文件到上下文中，作为开发的约束和指导。
- **技能组成**：包含 `explore-before-act`、`no-yolo`、`git-workflow` 等13个核心技能，覆盖规划、编码、验证、协作全流程。

它通过一系列预定义的技能，将AI的“随机生成”转变为可控的“工程实践”。
