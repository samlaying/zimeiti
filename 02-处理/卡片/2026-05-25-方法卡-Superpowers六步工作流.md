# Superpowers六步工作流

Superpowers 的核心是一套“三列六项”的六步工作流程，将模糊需求转化为可执行任务：
1. **头脑风暴与意图澄清**：通过提问激活想法，探索备选方案，保存设计文档。
2. **制定可执行实现计划**：将设计分解为2-5分钟的小任务，包含文件路径、代码思路和验证步骤。
3. **系统优化与调试**：采用[[测试驱动开发流程]]，先写失败测试，再写最小编码。
4. **智能体协作**：通过多个AI角色分工合作，提升效率和质量。
此流程帮助开发者系统性规划，避免AI输出跑偏。
