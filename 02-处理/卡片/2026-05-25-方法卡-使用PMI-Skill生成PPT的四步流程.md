# 使用PMI Skill生成PPT的四步流程

在[[PMI工具]]中，通过一个专门的Skill生成潮流风格PPT的流程如下：

1.  **启动Skill**：在PMI工具中输入该Skill的名称。
2.  **配置报告**：根据Skill的提问（如页数、目的等），生成一份配置报告，这份报告是一段引导后续生成的文字组合。
3.  **选择风格**：从多种预设风格（如梦会Skills、Maximalism集团主义等）中选择一种。
4.  **生成与预览**：生成PPT后，利用内置的ASI预览功能查看整体结构，确认无误后即可完成。若对结构不满，可自行调整或重新选择风格。

该流程的核心优势是**无需用户事先整理文本**，直接输入一级文字（大纲或关键点）即可驱动生成。
