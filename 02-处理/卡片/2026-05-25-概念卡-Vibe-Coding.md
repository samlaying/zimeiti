# Vibe Coding

**Vibe Coding** 是一种通过向大模型提供**功能截图**和**详细功能描述**，让其“直觉式”生成产品原型的 AI 编程/设计方式。

这种方式追求快速出结果，但由于输入的上下文缺乏对现有**产品设计规范**的约束，导致输出结果容易脱离产品原有风格，出现“做的原型总不像自家产品”的痛点。
