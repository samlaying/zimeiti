# Vibe Coding

**Vibe Coding** 指的是使用AI辅助进行编码的新型开发方式。它不强调开发者需要深厚的编程基础，而是通过与AI工具（如[[Kimi]]、[[Google AI Studio]]）的协作，借助[[结构化提示词]]和视觉化操作（如[[Trae]]的选择元素功能），快速实现从设计到代码的转化与迭代。
