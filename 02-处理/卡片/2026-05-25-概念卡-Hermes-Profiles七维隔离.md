# Hermes Profiles七维隔离

一个[[Hermes Profiles核心作用|Hermes Profile]]可以隔离以下七个维度：
1. 配置
2. 绘画
3. 记忆
4. 技能
5. 人格
6. cron状态
7. 网关状态

这种隔离确保每个角色保持专业性和一致性，避免不同角色的知识和风格相互污染。
