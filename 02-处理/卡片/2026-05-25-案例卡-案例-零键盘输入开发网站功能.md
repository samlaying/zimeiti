# 案例：零键盘输入开发网站功能

一个展示 Symphony 全自动化的实际案例：

1.  **创建任务**：用户在 Linear 项目中创建一个 issue，要求将网站页角的 email 链接改为可点击（添加 `mailto:` 协议），并设置状态为 `to do` 和优先级 `high`。
2.  **自动派发**：Symphony 的轮询机制检测到新 issue，并自动派发 Agent 处理。
3.  **自动执行**：Agent 在自己的**隔离工作区**克隆代码仓库，执行代码修改，并更新 Linear issue（添加工作计划、验收标准和 token 消耗统计）。
4.  **结果验证**：任务自动完成后，用户可在 Linear issue 中查看进度和统计，并在本地 `Symphony/Workspaces` 目录下验证代码更改。

整个过程用户**键盘输入次数为零**，只需在 Linear 上管理 ticket，实现了高度自动化的项目调度。
