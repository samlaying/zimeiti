# 插件安装与飞书配置

安装并配置[[飞书多Agent协作插件]]的步骤。

### 安装插件
- **通过 Cloud Hub 安装**：执行命令 `OpenCloud Plugins in style`（具体参考插件文档）。
- **从 GitHub 安装**：参考插件的 README 文档。

### 配置飞书机器人权限（关键）
飞书机器人需开通 **“接收群组中其他机器人和用户艾特当前机器人的消息”** 权限。
1.  访问**飞书开发者后台**，找到对应的应用。
2.  **新建机器人**：该权限通常已自动添加。
3.  **对于旧机器人**：需在“权限管理”中搜索并手动勾选开通此免审权限。

### 其他注意事项
- 为兼容性，建议在 `OpenCloud.json` 配置中**关闭流失输出**。
- 确保所有参与协作的机器人已拉入目标飞书群聊。
