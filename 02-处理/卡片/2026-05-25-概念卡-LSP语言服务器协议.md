# LSP语言服务器协议

LSP（Language Server Protocol，语言服务器协议）是一个开放协议，用于在编辑器或IDE与提供代码智能特性（如自动补全、跳转定义、查找引用）的语言服务器之间进行标准化通信。可以将其理解为编程语言的“通用翻译器”，使得单一服务器能为多种编辑器提供语言智能支持，避免了为每个编辑器重复开发功能。
