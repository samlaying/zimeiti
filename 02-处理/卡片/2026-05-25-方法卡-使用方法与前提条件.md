# 使用方法与前提条件

启动 [[Video Podcast Maker]] 自动化视频制作的准备和操作步骤。

**前提条件：**
1. 需要安装 Cloud Code 或 Open Code。
2. 需要一个 Remotion 项目。
3. 需要 Azure Speech 的 API 密钥。

**启动方式：**
- 安装好前提工具后，直接在 Cloud Code 或 Open Code 中输入自然语言指令，例如：“帮我制作一个关于某某话题的视频”。
- 系统会引导你走完 [[14步全自动工作流]] 的全部流程，全程无需编写代码或剪辑技能。
