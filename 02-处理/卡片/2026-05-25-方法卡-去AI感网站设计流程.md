# 去AI感网站设计流程

一套结合AI与专业设计的网站高级感提升流程：

1.  **寻找灵感与获取素材**：从 [[Awwwards]] 等设计灵感网站搜索 `hero section`，**只截取页面中间区域**，避免边缘杂物。
2.  **AI生成与优化素材**：使用AI工具（如 [[Nada Banana 2]] 模型）将截图处理得“干净”，再使用[[King 3.0]]等模型将其生成为可循环播放的动态背景视频。
3.  **构建网站结构**：使用 [[Next.js]] 或 Vue 等前端框架，结合 [[Agent Skills]] 中的UI设计技能包，快速生成网站基础布局代码。
4.  **融入高级视觉元素**：将生成的动态视频作为背景，并最关键地——**引入专业的[[UI设计系统]]和CSS样式提示词**，用以重构网站的按钮、文字等细节，彻底去除[[AI生成感]]。
