# 复刻质朴页面案例

使用 [[原型绘制Skill]] 复刻“质朴”页面并添加新功能的具体案例：
1.  **输入**：提供质朴官方界面的截图。
2.  **输出**：Skill 生成了一个本地工程文件，复刻了原页面。
3.  **迭代**：在工程文件中新增了“登录密码”和“绑定邮箱”两个功能入口。
4.  **文档**：AI 为新增功能输出了设计说明，并生成了可查看、可复制的设计文档。

这个案例展示了从截图输入到输出带新功能的可交互原型及文档的全流程。
