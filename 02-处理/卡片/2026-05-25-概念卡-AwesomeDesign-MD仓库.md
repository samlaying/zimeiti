# AwesomeDesign-MD仓库

一个由 Volt Agent 维护的开源 GitHub 仓库，旨在收集和整理 50 多个知名网站（如 Cloud、Vercel、Notion、Stripe 等）的 [[设计规范文件 (Design.MD)]]。

- **格式**：将各设计系统的核心要素（颜色、排版、组件、阴影、Dos and Don‘ts）提炼为清晰的 Markdown 文件。
- **目的**：帮助开发者在使用 [[AI编码工具]] 生成界面时，能快速获得风格统一、像素级匹配的代码。
- **数据**：完全开源，已积累近两万 star，更新活跃。
- **附加功能**：仓库内提供 `Preview.HTML` 文件，用于直观预览颜色和组件效果。
