# Draw UI Skill: AI UI设计自动化实践

> 不是手动调参，而是把调参经验打包成Skill，让AI自动调参。

## 核心内容
UP主将上述最佳实践打包成一个名为`Draw UI`的Skill。该Skill可自动查看项目现有设计资源，引导用户提供参考图，结合内置的多种提示词策略（叙事、类比等）自动生成最优提示词，最终输出UI设计稿。这实现了从手动调试到一键智能生成的工作流提效。

## 原文关键段落
> Skill 会自动查看当前代码仓库中已有的设计资源...AI 结合参考图、代码仓库中的设计规范，以及内置的多种提示词策略...自动分析和生成最优提示词，最终输出 UI 设计稿。
