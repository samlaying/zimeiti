# Remotion Studio可视化编辑

Video Podcast Maker 最新加入的功能，允许用户在渲染前进行可视化、所见即所得的视频编辑。

**主要特点：**
- **实时预览**：打开 Studio 后可直接预览视频效果。
- **面板调整**：右侧面板支持调整颜色、字体大小、进度条样式、背景音乐音量等参数，无需修改代码。
- **交互式优化**：如果效果不满意，可返回 [[14步全自动工作流]] 中，用自然语言在 Cloud Code 或 Open Code 中进一步调整，实现“一边修改一边预览”，满意后再开启 4K 渲染，从而节约时间。

此功能对不懂代码的用户特别友好，降低了视频定制的技术门槛。
