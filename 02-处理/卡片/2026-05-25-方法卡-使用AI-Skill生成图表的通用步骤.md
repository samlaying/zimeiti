# 使用AI Skill生成图表的通用步骤

使用 [[Jetting Canvas]] 或 [[Excel Draw Diagram Generator]] 等生成图表的核心流程：
1.  **引用Skill**：在AI编程工具中引入对应的 Skill。
2.  **输入描述**：用自然语言清晰描述你想要的图表内容与目的。
3.  **生成与编辑**：Skill 生成可编辑的图表，不满意时可直接在图表上进行删减、新增或修改。

**示例描述**：“给我生成一个常见场景的基本流程图，页面流程图，目的是展示XXX流程。”
