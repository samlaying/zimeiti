# 脚本市场UI革新与信息自动化

> 用元数据自动填充插件市场描述，兼顾了信息丰富性与维护效率。

## 核心内容
新版脚本市场界面将脚本按功能分类，并提供搜索功能。点击脚本后，系统会自动从脚本文件的FrontMatter（元数据）中抓取详细描述并展示，使用户无需手动编写，信息获取非常直观。
