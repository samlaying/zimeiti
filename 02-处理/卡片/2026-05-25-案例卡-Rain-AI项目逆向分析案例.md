# Rain AI项目逆向分析案例

**案例背景**：以开源AI项目 **Rain AI** 为例，演示如何使用 [[Project Analyzer Skill]]。

**分析过程与产出**：
1.  Skill自动将所有提示词提取并存放在 `AR Analyze` 文件夹下。
2.  生成一个提示词索引文件（`index`），列出所有提示词目录。
3.  点击具体提示词可查看详细分析，包括：**文档位置、中文翻译、参数解析以及相关代码片段**。
4.  最终生成一份完整的、针对AI产品的14章PRD文档（自动调用了AI产品PRD模板）。

此案例展示了该工具在学习复杂AI开源项目结构和逻辑时的高效性。
