# 完整文档方案的弊端

基于完整的MD设计规范文档让AI生成规范，存在几个关键问题：
1.  **Token消耗大**：文档量越大，读取消耗越高。
2.  **速度慢**：处理大量文档导致生成过程延迟。
3.  **AI会“偷懒”**：即使在单轮对话中，AI在生成时仍可能遗漏已上传文档中的内容。
4.  **结构冗余**：完整文档通常包含多个模块（如页面布局+组件规范），但每次任务往往只需其中一个，读取全文是不必要的。
