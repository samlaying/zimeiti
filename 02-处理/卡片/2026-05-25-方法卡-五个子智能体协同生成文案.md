# 五个子智能体协同生成文案

**详情页文案包装智能体**通过五个子智能体协同工作，将“产品卖点”转化为“用户买点”。工作流程如下：
1.  **卖点分析师**：分析产品核心卖点。
2.  **人群分析师**：分析目标人群的生活场景与痛点。
3.  **文案策划师**：进行买点转化与内容创作。
4.  **评审专家**：对生成内容进行质量审核。
5.  **优化专家**：根据审核意见进行优化调整。
