# Remotion 两步上手方法

快速上手使用 Remotion 仅需两个步骤：
1.  **安装**：在 [[Cloud Code]] 云开发环境中，直接连接 Remotion 的 Github 仓库地址，让 Cloud Code 自动完成安装。
2.  **生成**：在 Cloud Code 中，用自然语言输入你的动画需求或代码逻辑想法，Remotion 即可自动为你生成可运行的代码和实时预览的动画，实现快速原型开发。
