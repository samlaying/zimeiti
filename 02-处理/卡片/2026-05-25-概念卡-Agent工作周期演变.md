# Agent工作周期演变

AI Agent的[[工作周期]]经历了显著的演变：
1.  **初期阶段**：以分钟为单位，进行短时任务。
2.  **发展阶段**：延长到几小时，应对更复杂的任务。
3.  **当前阶段**：达到几天（24小时无休），需要持续、稳定地运行。

这种演变对Agent的稳定性和自我提升能力提出了更高要求，使得[[记忆]]模块成为核心需求。
