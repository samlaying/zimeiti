# Agent Skills 标准

由Anthropic和Google支持的**开放标准**，旨在规范AI助手（如Claude Code, AntiGravity IDE）的技能。

- **来源**：2025年10月，由Anthropic在Claude Code中正式支持；Google的AntiGravity IDE同样支持。
- **文件结构**：一个 `.md` 文件，结构如下：
  1. **YAML元数据**：定义技能的触发条件（何时使用）。
  2. **Markdown内容**：描述技能的具体操作步骤（怎么做）。
- **价值**：能让AI助手在特定任务上的表现**提升一个量级**。
