# 生成与查询Wiki

使用已搭建好的Obsidian RAG系统的两种核心操作：
- **生成Wiki**：在Vault中打开[[Claude Code]]，指示它基于`raw`文件夹的资料或进行网络搜索，为特定主题（如“Claude Code Skills”）创建Wiki。Claude Code会自动创建主题文件夹、索引文件和相互链接的文章，并更新`master-index.md`。
- **查询知识库**：直接在Claude Code中提问。系统会依据`cloud.md`的指令，沿着 `wiki` -> `master-index.md` -> 具体主题索引 -> 具体文章的路径进行检索和回答。
