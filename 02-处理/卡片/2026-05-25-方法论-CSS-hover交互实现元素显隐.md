# CSS hover交互实现元素显隐

> 显示隐藏交互动效：`:hover` 伪类是关键

## 核心内容
利用CSS的`:hover`伪类，结合`display`、`opacity`等属性，控制元素在鼠标悬停到父容器时显示或隐藏。初始状态设为隐藏，悬停时通过`.parent:hover .child`选择器改变子元素的属性值。
