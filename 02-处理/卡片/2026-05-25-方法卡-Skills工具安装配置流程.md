# Skills工具安装配置流程

使用[[自动剪辑Skills工具]]前的安装配置步骤：
1. **获取工具**：下载自动剪辑视频Skills的压缩包。
2. **解压安装**：将压缩包解压到你的Agent对应的Workspace工作空间里，并安装其所需的Python依赖。
3. **配置API（可选）**：如需“素材理解、筛选”等高级AI分析能力，需前往`Tales Labs`申请API Key。若只想将已整理好的片段直接生成剪映草稿，可跳过此步。
4. **环境准备**：确保本机已安装[[剪映专业版]]并可正常打开，同时设置好草稿目录。
5. **环境验证**：运行测试命令，若能正常输出帮助信息，则说明安装基本完成。
