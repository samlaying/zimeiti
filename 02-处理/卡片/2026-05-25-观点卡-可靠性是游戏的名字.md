# 可靠性是游戏的名字

构建AI智能体时，**可靠性（Reliability）是游戏的名字**。我们租用的AI模型是一个不可控的黑盒，服务提供商可能随时更换底层模型（如Opus换Sonnet），资源也有限。因此，我们需要通过构建外围的[[AI Agent Harness 定义|Harness]]来确保智能体能稳定、可靠地完成预期任务，而不是依赖于不可控的模型本身。
