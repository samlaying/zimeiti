# GUI自动验收代码流程

使用 Codex Computer Use 自动验收代码的流程：
1. **指定任务**：向 AI 提交编写代码的指令，并明确开启 Computer Use 功能。
2. **AI编码与验证**：AI 首先编写代码，随后自动打开浏览器，通过操控鼠标和截图来验证网页功能、调试问题（例如自动运行游戏并测试交互）。
3. **反馈与迭代**：验证完成后，AI 会报告结果。若存在问题，它会尝试自动修复；若无问题，则通知任务完成。

此方法适用于开发者缺乏人工验收的场景，但需注意 [[Token消耗高的问题]]。
