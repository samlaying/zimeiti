# Auto Mode 内置安全分类器工作机制

> 后台智能分类器自动判断：安全放行，危险拦截并询问。

## 核心内容
Auto Mode 的核心是内置的 `auto mode classifier`（安全分类器）。它会在后台评估每个操作的安全性。对于安全操作，分类器会直接允许执行并记录日志；对于危险操作（如删除文件），分类器会拒绝执行，并主动中断任务询问用户。用户若认可拦截则无需操作，若想强制执行，可在后续指令中输入“允许该操作”或“approve”。

## 原文关键段落
> 安全操作：直接执行，日志显示 `allowed by auto mode classifier`。危险操作（如删除文件）：会被拦截（`denied`），分类器会主动中断任务并询问用户下一步指示。
