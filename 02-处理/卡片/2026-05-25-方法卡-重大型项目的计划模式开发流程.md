# 重大型项目的计划模式开发流程

对于重大型项目，应采用以计划模式为核心的迭代开发流程，确保方向正确、过程可控。

**流程：**
1.  **讨论目标，生成文档：** 使用[[计划模式]]与AI讨论，将目标、约束生成文档。
2.  **查漏补缺：** 继续使用计划模式审视文档，可选引入其他AI或人工进行[[Review]]。
3.  **文档落定，推进开发：** 文档确认后，开始实施。
4.  **阶段性迭代：** 取得阶段性进展或发现偏航时，再次使用计划模式对齐和更新文档。

关键：计划模式贯穿始终，文档作为所有工作的基础。
