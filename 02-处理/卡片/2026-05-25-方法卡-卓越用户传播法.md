# 卓越用户传播法

基于用户“喜欢自己”心理的传播策略，核心是**帮助一小部分用户成为“卓越用户”**。

1.  **用户分型**：识别具有“成长型思维模式”的用户，他们学习能力强，能利用产品做出惊艳的二次创作或作品。
2.  **定义卓越用户**：成长型用户中，其产出能成为他人羡慕目标的用户。
3.  **传播链条**：卓越用户 → 其同事、朋友羡慕其成果并询问 → 卓越用户推荐产品 → 产品得以传播。

**底层逻辑**：用户推荐产品，不是因为单纯喜欢产品，而是出于**“喜欢他们的朋友”**，希望帮助朋友也能获得提升和愉悦感。
