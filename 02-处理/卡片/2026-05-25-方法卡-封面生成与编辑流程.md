# 封面生成与编辑流程

使用 [[AirPostMaker：AI自动封面生成器]] 生成、修改并导出封面的完整流程。

1.  **生成封面**：
    - 打开文章，点击插件图标进入界面。
    - 选择 API 模型（推荐 DeepSeek V4 Flash）、目标平台和风格（可多选）。
    - 点击 **“导入当前文章”** 和 **“生成封面”**。
2.  **编辑封面**：
    - **直接修改**：在预览界面点击文字修改内容。
    - **AI修复**：点击 **“请求修改”**，输入具体指令让AI重新生成。
    - **编辑代码**：点击 **“编辑”**，直接修改 HTML/CSS 代码进行精细调整。
3.  **导出使用**：对封面满意后，点击 **“复制”**，将 HTML 代码粘贴到微信公众号编辑器中即可作为封面。
