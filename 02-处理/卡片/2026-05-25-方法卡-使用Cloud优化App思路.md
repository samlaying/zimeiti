# 使用Cloud优化App思路

通过与[[Cloud]]进行多轮对话来优化和确定App的核心思路。核心操作步骤如下：
1. **发送初始问题**：将初步的App想法描述给Cloud。
2. **批判与反馈**：AI的第一版方案可能缺乏创新，需要明确指出不足，如“这样肯定不行”。
3. **迭代优化**：AI根据反馈给出新版方案，直到满意，例如感觉“加到了我的心感深处”。
4. **理顺思路**：与AI最终确认App的核心信息和功能框架。

关键在于对话应聚焦于[[开发前必须明确App定位与创新性|创新性]]，确保AI的输出是独特的。
