# 在 Claude Web 应用中安装 Higsfield 连接器

在ClaudeAI的网页或桌面应用中安装[[Higsfield]]连接器的步骤：
1.  进入 **ClaudeAI** 的 **Settings**。
2.  选择 **Connector**，然后 **Add Custom Connector**。
3.  复制并粘贴Higsfield提供的连接器代码。
4.  点击 **Connect** 并完成身份验证登录。
5.  设置完成后，即可在Claude Web或桌面应用的聊天中直接调用Higsfield中的任何音频、视频或图像工具。
