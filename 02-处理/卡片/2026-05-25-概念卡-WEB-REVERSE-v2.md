# WEB-REVERSE v2

WEB-REVERSE v2 是一款面向前端逆向工程的AI辅助工具新版本。其核心设计理念从以“专题类型”为导向，转向**以“验收”为唯一目标**，所有规则都围绕确保任务成功交付展开。新版本增强了处理 [[VM]]、[[WebAssembly]]、[[JSYMP]] 等复杂逆向场景的能力，并新增了通过互联网主动搜索信息以辅助AI解决问题的功能。
