# 基于Git仓库的多端同步

解决不同设备（如Mac mini和MacBook Air）间Skill同步问题的方法：

1.  准备一个**Git仓库**作为同步中枢。
2.  在设备A上使用[[Skill管理器]]的**上传**功能，将本地Skill上传至仓库。
3.  在设备B上使用**下载**功能，从仓库中拉取所需的Skill到本地。

通过一个共享的Git仓库，即可实现跨设备的Skill便捷同步。
