# 张杀HTML模板库

张杀提供的并非一个Skill，而是一个位于GitHub的HTML模板库项目。

运作原理：AI将项目克隆到本地，通过交互匹配模板：先询问用户需求，再从库中选择合适模板填充内容。

核心特点：
- **风格多样化**：模板库提供大量设计选项，时尚度强。
- **设计感强**：生成效果类似艺术品，美观高级。

核心缺点：**中文适配差**（几乎不支持中文，需手动找相近风格字体改善），且采用**交互式匹配**，非全自动。
