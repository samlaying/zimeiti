# 微信收藏自动采集流程

将微信等碎片化渠道的收藏自动导入 [[Obsidian]] 知识库的工具链配置：
1.  **网页采集**：使用 **Obsidian Web Clipper** 浏览器插件，配置剪藏路径至知识库的 `raw/articles` 文件夹。
2.  **微信文章采集**：使用 **笔记同步助手** 等工具，将微信文章转发给它，并配置同步路径为 `raw/articles` 下按日期命名的子文件夹。
3.  **自动化编译**：在 [[Hermes]] 中测试手动批量编译任务，成功后创建每日定时任务（Cron Job），实现指定时间（如每天9点）自动扫描新文章并编译入 [[LLM-Wiki]]。
