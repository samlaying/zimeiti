# Sandbox与Context Hub：Skill的生产环境基石

> 智能体要跑脚本和API，需要Sandbox；管理这些文件，需要Context Hub。

## 核心内容
Sandbox为执行各类Skill（脚本、API调用）提供了安全、隔离的线上环境，是生产级智能体的必备基础设施。Context Hub则由管理Prompt升级而来，现在用于管理更广泛的上下文（Skills、Prompt、Agent Markdown等），并支持版本控制、分支和分享。两者结合，为Skill的开发和运行提供了完整的生产环境支持。
