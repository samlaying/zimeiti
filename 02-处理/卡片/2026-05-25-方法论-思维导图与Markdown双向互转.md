# 思维导图与Markdown双向互转

> 打通可视化脑图与文本笔记的壁垒，双向转换自由。

## 核心内容
思维导图可以一键导出为结构化的Markdown文件（根节点变标题，子节点变列表），也可以从剪贴板或文件导入带有层级关系的Markdown文本生成脑图，实现了可视化思维与线性文本的无缝衔接。
