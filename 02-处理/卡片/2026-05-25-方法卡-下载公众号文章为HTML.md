# 下载公众号文章为HTML

在COZE平台中将公众号文章保存为本地HTML网页文件的操作步骤：
1. **新建任务**：在COZE平台创建一个新的对话任务。
2. **调用技能**：输入指令调用“[[公众号下载为网页SPL]]”技能。
3. **提供链接**：向AI提供目标公众号文章的链接地址。
4. **执行与获取**：发送指令后，AI自动处理，生成的HTML文件将保存在平台的 `Output` 文件夹中。
