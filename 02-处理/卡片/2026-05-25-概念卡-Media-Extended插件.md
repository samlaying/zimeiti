# Media Extended插件

**Media Extended**是Obsidian的一个媒体增强插件。安装后，它支持在Obsidian内播放本地视频和音频文件，并提供关键功能：将打开的媒体文件“添加到媒体库”（Add to Media Library），自动生成一个包含该文件**唯一访问地址**的Media Lab文件。这个地址是构成可点击跳转链接的关键组成部分。
