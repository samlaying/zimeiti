# Codex 是什么？

**Codex** 是 OpenAI 的超级应用（Super App），集成了 Code（编程）与 Cowork（办公任务）功能。它被认为是控制 [[GPT-5.4]] 等 AI 代理的最佳界面，尤其适合处理复杂任务。

核心特点：
- 提供**提示界面**（Prompt Interface），适合快速生成内容和视频。
- 内置插件系统，例如 [[Remotion 是什么？|Remotion]] 插件，可直接在 Codex 内创建动效视频。
- 在一个统一界面内完成编程、文档、演示、视频等多种创作任务。
