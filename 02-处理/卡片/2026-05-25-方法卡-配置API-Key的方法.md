# 配置API Key的方法

使用该[[n8n自动化工作流]]需要配置多个API密钥：
1.  **生图/转场描述API**：默认使用“真正的AI工坊”的API。由于API调用封装在**子工作流**中，用户可**自行替换**为其他API，只需保持子工作流输入输出字段不变。
2.  **动画生成API (可灵)**：需在可灵官网单独购买API资源包，获取**访问密钥(AK)** 和**密钥(SK)** 填入主工作流。
3.  **Agent节点大模型API**：为工作流中的智能体配置一个大模型Key（如DeepSeek）。
