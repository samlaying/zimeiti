# 使用Excalidraw Skill生成讲解图

在[[Obsidian]]中利用[[Excalidraw Skill]]一键生成讲解图的操作流程：

1.  **激活Skill**：通过预设的触发词（如“帮我生成讲解图”）调用 Skill。
2.  **指定源文件**：选择需要转化为讲解图的文本文件（如视频脚本）。
3.  **自动渲染**：Skill 会先生成中间的 JSON 文件，随后由 Obsidian 和 Excalidraw 插件自动渲染成可视化的讲解图。
4.  **编辑调整**：生成的讲解图是可编辑的，用户可以精简文字、调整布局以满足需求。
5.  **预览使用**：在 Obsidian 中直接打开渲染后的文件即可查看效果，并可直接用于视频或演示。
