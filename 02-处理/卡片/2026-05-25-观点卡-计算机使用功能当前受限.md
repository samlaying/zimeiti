# 计算机使用功能当前受限

Codex 2.0 的“计算机使用功能”虽然强大，但当前仍存在限制。它无法完全操控某些应用（例如在 Chrome 浏览器中直接进行复杂操作），可能需要额外配置才能达到最佳效果。

然而，其基础的自动化操作能力（如打开应用、输入内容）已经相当出色且快速。
