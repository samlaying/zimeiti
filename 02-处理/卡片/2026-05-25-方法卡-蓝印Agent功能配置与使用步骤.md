# 蓝印Agent功能配置与使用步骤

配置并使用[[蓝印RPA的Agent功能]]（微信控制）的详细步骤：

1.  **前期准备**：
    *   在蓝印官网 `蓝印rpa.cn` 注册账号（使用邀请码可获首购九折）。
    *   访问 **DeepSeek 开发平台**，注册并获取一个 **API Key**。
    *   在蓝印的Agent配置界面中填入此API Key。
2.  **绑定微信**：配置好API后，点击绑定微信选项，使用**个人微信**扫码完成连接。
3.  **通过微信控制**：
    *   绑定成功后，微信中会出现名为 **“微信Cloud”** 的对话入口。
    *   在对话中通过**自然语言**指令进行控制：
        *   **查询**：如发送“我有多少应用”。
        *   **运行**：如发送“运行**基础教程**这个应用”。
        *   **查看状态**：系统会主动通知任务结果（如“执行失败”）。
