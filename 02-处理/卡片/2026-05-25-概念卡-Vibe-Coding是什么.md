# Vibe Coding是什么

**Vibe Coding** 是一种与 [[AI编程助手]] 进行 [[命令行交互]] 的工作方式，特别适合创造性任务（如制作演示文稿、写代码）。

其核心特征包括：
- 使用命令行界面与AI进行多轮、结构化的沟通。
- 在任务开始前，通过类似“头脑风暴”的环节（如 [[brainstorm skill]]）与AI充分对齐需求。
- 工作流灵活，允许用户在AI处理任务时（如生成HTML）并行处理其他工作。

这种方式被视为计算机领域人士的常见实践，能显著提升内容生成的效率与质量。
