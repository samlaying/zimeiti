# COZE技能与SPL

**COZE技能**是基于COZE平台的一系列预置功能单元，每个技能都封装了特定的能力（如将文章转PDF）。用户可以通过简单的**自然语言指令**来调用它们。

**SPL**（在此语境下）指代COZE平台上的一个具体技能实例，例如“公众号下载为网页SPL”或“公众号下载为PDF”。它是技能（Skill）在具体场景下的一个应用。

调用流程为：在COZE平台新建任务 → 用自然语言指令调用目标SPL → 提供必要参数（如链接） → 执行并获取结果。
