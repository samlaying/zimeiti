# 资源管理策略

管理[[Anti-Gravity Awesome Skills 项目]]技能安装以避免**上下文窗口过载**的策略。

- **社区测试数据**：
  - 50个技能：约占 **5000个token**（上下文窗口的4%）。
  - 300个技能：约占 **3万个token**（上下文窗口的23%）。
- **操作建议**：
  1. 从 **5到10个技能** 起步。
  2. 使用 **Bundle** 按需扩展，避免一次性全量安装。
  3. 了解技能采用**渐进式加载**，仅被触发的技能会完整加载（首次全量加载需10-15分钟）。
