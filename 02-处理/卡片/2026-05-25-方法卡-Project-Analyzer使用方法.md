# Project Analyzer使用方法

使用 [[Project Analyzer Skill]] 非常简单，核心只需两步：

1.  **第一步：执行分析**
    *   进入目标开源项目的路径。
    *   在终端调用Skill：`/Project Analyzer`。
    *   Skill自动提取并分析项目中的提示词，输出中间结果。

2.  **第二步：生成PRD**
    *   Skill分析完成后会有一个**确认环节**，用户需检查其翻译的提示词是否准确、有无遗漏。
    *   确认无误后，输入指令：“**请启动PRD生成**”。
    *   Skill随即生成完整的14章PRD文档。
