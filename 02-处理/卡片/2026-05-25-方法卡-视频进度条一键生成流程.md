# 视频进度条一键生成流程

使用[[视频进度条生成工具]]制作进度条的标准操作流程：
1.  **输入信息**：设置视频总时长与宽度（如默认1080P）。
2.  **添加章节**：为视频内容划分节点，并为每个节点设置自定义标题（如“开头”、“结尾”）。
3.  **一键生成**：点击按钮，工具将快速输出进度条视频文件。
4.  **应用到项目**：将生成的文件导入剪辑软件进行后续调整。
