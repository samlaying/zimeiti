# 淘宝设计功能的拆解案例

面试问题“给淘宝设计一个新功能”的拆解示范：
1. **按时间线拆**：购物前、购物中、购物后。
2. **按参与方拆**：用户侧（如一二线年轻女性、中年男性等细分人群）、平台侧、商家侧。
3. **按用户链路拆**：打开APP → 点击活动聚合页 → 查看商品详情页 → 加入购物车 → 下单支付 → 确认收货。
4. **拆关键指标**：围绕提升GMV（商品交易总额）展开。
