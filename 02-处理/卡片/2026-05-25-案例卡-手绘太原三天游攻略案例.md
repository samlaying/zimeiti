# 手绘太原三天游攻略案例

这是一个应用 [[制作手绘旅游攻略的方法]] 的完整实践案例：
- **背景与目的**：UP主计划去太原进行为期三天的旅行（含观看红灯笼体育馆演唱会），希望制作一份美观、实用的手绘风格攻略海报，用于行程规划和分享给朋友。
- **具体应用**：
  1.  创建“手绘旅游攻略”工作目录及四个素材子文件夹。
  2.  指示[[Workbody]]联网搜索太原的景区（如晋祠、红灯笼体育馆）和美食信息，为每个地点生成包含图片和TXT简介的素材包，严格遵循“[[素材收集的黄金法则]]”。
  3.  额外搜集或生成一张手绘版太原地图作为行程背景。
  4.  调用“手绘”技能，将所有素材整合成一份三页的PDF：封面、景点美食介绍、行程安排（含手绘地图背景）。
- **产出与价值**：最终生成了一份风格统一、像景区导览册的精美PDF，既可用于个人旅行规划，也是一件可分享的有趣作品。
