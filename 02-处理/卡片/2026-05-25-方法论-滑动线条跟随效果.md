# 滑动线条跟随效果

> 一个定位元素 + nth-child选择器 + hover，动态改left和颜色。

## 核心内容
通过一个绝对定位的元素（如`.underline`）作为滑动指示线条。利用`:nth-child`选择器配合`:hover`伪类，当鼠标悬停在不同的一级菜单项上时，动态改变该线条元素的`left`位置和`background-color`，并加上`transition`动画，从而实现线条跟随鼠标滑动的视觉效果。

## 原文关键段落
> /* 第2个菜单项 */
> .navbar .nav ul > li:nth-child(2):hover ~ .underline {
>     left: 240px; /* 移动到第二个菜单下方 */
>     background-color: #339933;
> }
> /* ... 其他菜单项类似 */
