# Video Podcast Maker

一个基于 Cloud Code 或 Open Code 的自动化视频制作技能，能够将任何话题自动转换为 4K 分辨率的视频播客。

它通过 [[14步全自动工作流]] 实现从话题输入到成片的全流程自动化，无需用户编写代码或具备剪辑技能。其核心功能包括 [[Azure TTS高质量配音]]、[[多音字智能校正]] 和最新的 [[Remotion Studio可视化编辑]]。

该工具开源免费，适用于 GitHub 开源项目介绍、论文解读、AI 动态、财经科普、产品评测、软件教程或热点分析等多种场景，旨在显著提升视频产出效率。
