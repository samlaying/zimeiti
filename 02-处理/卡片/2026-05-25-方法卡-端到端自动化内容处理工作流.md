# 端到端自动化内容处理工作流

这是一个使用 [[Dify]] 工作流，实现从外部平台抓取内容并进行AI优化改写的自动化流程范例。

**核心步骤**：
1.  **输入**：接收一个 [[小红书]] 文章的URL。
2.  **数据抓取**：使用`HTTP请求`节点获取页面HTML，再用`代码执行`节点（通过正则表达式）提取标题、正文、图片URL列表。
3.  **AI内容优化**：
    - 将标题和正文分别输入到`大模型`节点，通过特定的[[提示词工程]]要求模型进行优化改写。
    - 处理图片：若模型不支持生图，则使用`迭代`节点遍历图片列表，对每个图片URL调用外部[[图片处理API]]生成新图片。
4.  **输出**：汇总并输出优化后的标题、正文（Markdown格式）及新的图片链接。

> 这个方法展示了如何将数据抓取、[[AI大模型]]文本生成、外部API调用等多个环节串联成一个端到端的自动化业务流程。
