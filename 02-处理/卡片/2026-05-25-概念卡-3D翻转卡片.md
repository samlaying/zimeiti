# 3D翻转卡片

一种常见的 [[悬停资料卡片]] 交互效果。通过 CSS 3D 变换，使卡片在悬停时绕 Y 轴（或 X 轴）旋转 180 度，从而切换显示正面和背面内容。

## 实现原理
1. **HTML结构**：卡片包含 `.card-front` 和 `.card-back` 两个面板。
2. **CSS关键**：
   - 在 `.card-inner` 元素上使用 `transform-style: preserve-3d` 保持3D空间。
   - 为正反面添加 `backface-visibility: hidden`，确保旋转时背面不可见。
   - 悬停时，对 `.card-inner` 应用 `transform: rotateY(180deg)`。
