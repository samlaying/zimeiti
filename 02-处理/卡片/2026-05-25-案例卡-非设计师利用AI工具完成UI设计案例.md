# 非设计师利用AI工具完成UI设计案例

### 案例：从枯燥文档到高保真UI

一个非设计背景的用户，拥有一份完整的产品需求文档后，面临“不会设计”的困境。通过运用 [[Gemini]] + [[Figma Make]] 的AI工具链，他成功将文档转化为了精美的、像素级可交付的UI设计稿。

**核心过程**：
1.  让 [[Gemini]] 根据文档为每个页面生成精准的UI提示词。
2.  将提示词依次输入 [[Figma Make]]，自动生成首页、列表页、设置页等全部界面。
3.  对不满意之处，通过“整体重生成提示词”或“局部AI微调”进行优化。

该案例证明，AI降低了专业设计的门槛，用户只需具备基础审美和沟通能力即可完成高质量设计。
