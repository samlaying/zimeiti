# PARA与MOC：缺一不可

打造有效的笔记系统，[[PARA笔记管理法]]和[[MOC内容地图]]必须结合使用，两者解决不同问题且缺一不可。

- **PARA解决存放问题**：决定笔记放在哪个“文件夹”里，每条笔记只有一个物理位置。
- **MOC解决发现问题**：通过链接关系决定笔记如何被找到和串联，一条笔记可以出现在多个MOC中。

**缺失后果**：
- 只有PARA没有MOC：知识是碎片，难以复用和发现关联。
- 只有MOC没有PARA：文件没有清晰的存储结构，容易混乱。
