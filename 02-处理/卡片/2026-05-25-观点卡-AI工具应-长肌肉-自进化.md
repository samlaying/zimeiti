# AI工具应“长肌肉”自进化

**[[BrowserHannis]] 的一个关键优势在于其“自学习”能力，即“长肌肉”。**

- 它有一个名为 `agent-health` 的文件，初始为空。
- Agent 可以在运行中自己编写代码来填充该文件，持续积累工作经验。
- 工具本身自带了76个网站的经验（称为“A卷”），新Agent启动时即可读取，相当于“带技能上线”。

这形成了一个**正反馈循环**：使用越多，Agent积累的经验越丰富（A卷越厚），效果越好。这种经验沉淀还便于在不同Agent间分享，提升了工具的进化潜力。
