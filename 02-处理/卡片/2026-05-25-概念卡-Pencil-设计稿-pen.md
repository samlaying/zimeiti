# Pencil 设计稿 (.pen)

Pencil 是一个 UI 设计工具，其设计稿以 `.pen` 文件保存，本质是一段 **JSON 文本**。

**核心价值：**
1.  **UI 规范工具**：通过 JSON 定义颜色、间距、圆角等设计规范，确保 UI 在不同平台展示一致。
2.  **AI 友好**：AI 编程工具可以直接读取其 JSON 文本来实现还原度较高的 UI 开发。

**局限性：** 目前作为创意设计工具可能不够灵活美观，但作为定义规范的工具则非常合适。可以通过 Pencil 客户端查看和编辑设计稿。
