# 官方AI编程工具的三大痛点

以 Anthropic 的 Claude Code 为例，官方的AI编程桌面端存在三大痛点：
1.  **封号风险高**：对网络、订阅方式监管严格，用户容易踩雷。
2.  **模型与功能受限**：只能使用官方自家模型，且许多高级功能需要付费订阅。
3.  **准入门槛高**：近期增加了KYC实名认证（上传身份证件），将大量国内用户拦在门外。
这些限制催生了对更开放、灵活替代工具的需求。
