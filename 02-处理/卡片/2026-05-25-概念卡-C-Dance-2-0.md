# C-Dance 2.0

Updream平台支持的主要视频生成模型之一。

在平台的视频生成节点中，用户可以选择`C-Dance 2.0`作为生成模型。该模型支持图生视频、文生视频等多种模式，并可调整画质（720p至4K）。实测中在多个场景（广告、短剧、ASMR）下表现出色，能生成高质量、细节丰富的视频片段。
