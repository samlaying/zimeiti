# Superpowers 六步工作流程

Superpowers 是一个旨在解决AI辅助开发“跑偏”问题的项目，其核心是一套“三列六项”的结构化流程。

这六个步骤是：
1. **头脑风暴与意图澄清**：通过提问探索想法，确保理解一致，生成设计文档。
2. **制定可执行实现计划**：将设计分解为多个2-5分钟可完成的小任务，每个任务包含文件路径、代码思路和验证步骤。
3. **系统优化与调试**：采用[[测试驱动开发(TDD)]]的方法。
4. **智能体驱动开发**：通过多个AI角色（子智能体）分工协作来执行任务。
5. **分支交付**（文中提及）。
6. **系统化调试**（文中提及）。

该流程的目的是将模糊需求转化为可验证、可执行的微任务，引导AI生成结构化代码。
