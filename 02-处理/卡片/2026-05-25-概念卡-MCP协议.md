# MCP协议

MCP（Model Context Protocol）是AI Agent调用外部工具的标准协议，提供标准化接口。在[[Scrapling框架]]中，内置MCP Server，使得AI系统如Cloud、Cursor可以直接调用爬虫抓取网页数据。
