# MJ Adapter 排版方法

使用[[MJ Adapter 核心功能]]进行公众号与小红书排版的具体步骤：
1. **安装工具**：访问 skills.makejacky.com，复制安装命令（如`npm install MJ Adapter`）。
2. **运行安装**：在终端或直接发送给AI工具执行命令完成安装。
3. **生成排版**：向已安装的工具发送一篇微信公众文章。
4. **应用样式**：工具会生成一个HTML页面，将其**全选复制**到微信公众号后台，样式即自动保留。
5. **小红书应用**：同理生成HTML，再将HTML转为图片即可用于小红书发布，保持统一风格。
