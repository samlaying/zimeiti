# Auto Mode并非免费

[[Claude Code Auto Mode]] 的[[Auto Mode Classifier]]运行在独立于主模型的服务器配置上，**会产生额外的Token消耗和费用**。这不是一个免费功能。

**成本考量：**
- 单独使用 Auto Mode 会增加基础成本。
- 当 Auto Mode 与计划模式（Sub-Agent）同时使用时，分类器会在 Sub-Agent 的**启动前、运行时、完成时**三个阶段进行安全检查，成本会**显著增加**。

用户应意识到，享受其带来的便捷性和安全性，是以额外的计算成本为代价的。
