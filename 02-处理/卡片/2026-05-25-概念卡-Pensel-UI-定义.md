# Pensel UI 定义

Pensel UI 是一个基于 AI 的 UI 设计工具，能通过简单指令快速完成多项设计任务，大幅提升设计效率。

**核心能力**：
- 快速生成 UI 界面
- 克隆设计风格
- 批量创建 UI 组件
- 辅助设计（如生成详情页）
- 设计稿直接导出可运行代码

它旨在实现从[[急速出图]]到[[代码联动]]的全流程提效，是 [[AI设计工具优势]] 理念的典型体现。
