# 工作流整体流程

该[[n8n自动化工作流]]的**整体流程**分为四步：
1.  **主题风格定义与图片内容规划**：输入文字，由AI拆分内容并组装生图提示词。
2.  **生成图片**：调用[[Bunada Pro图片生成模型]]生成静态页面图片。
3.  **生成转场动画提示词**：利用多模态AI（如Gemini 3 Pro）为相邻图片生成转场描述。
4.  **生成首尾帧动画**：调用[[可灵2.6动画生成模型]]，根据图片和提示词生成转场视频。
最终，将生成的图片、视频与[[HTML演示播放器]]文件一同放置，即可播放。
