# PyWord Skills一键排版五步法

使用 [[PyWord Skills]] 实现 Word 文档自动排版的标准化流程：

1.  **打开文件**：使用 [[OpenAI Codex]] 或类似 AI 工具打开待处理的 Word 文件。
2.  **运行脚本**：执行 PyWord Skills 脚本，启动自动化处理流程。
3.  **配置参数**：根据文档规范，设置具体格式参数，例如：
    - 字体：Times New Roman（西文） + 宋体5号（中文）
    - 缩进：全角缩进两字符
    - 对齐：两端对齐
    - 行距：固定值20磅
4.  **测试功能**：验证关键功能是否正常，如检查[[参考文献目录]]的跳转、[[交叉引用]]的准确性。
5.  **完成排版**：一键应用所有设置，生成格式规范统一的最终文档。
