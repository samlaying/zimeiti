# Remotion：代码驱动的视频生成框架

**Remotion** 是一个在 GitHub 上非常受欢迎的开源项目（拥有42.6K星标）。

其核心功能是允许开发者通过**代码**的形式，来组合图片、字幕、音频等多种多媒体元素，最终聚合生成一个完整的视频文件。

它的主要特点是**代码驱动**，这使得视频制作过程变得高度可控和可定制化，是实现 [[AI视频快速生成流程（Remotion+VibeCoding）]] 的技术基础。
