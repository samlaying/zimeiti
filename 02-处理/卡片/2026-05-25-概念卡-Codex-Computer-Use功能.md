# Codex Computer Use功能

Codex Computer Use 是一种 AI 能力，允许模型通过图形用户界面（GUI）操控鼠标和键盘，模拟人类操作电脑，实现自动化任务。其更新状态可通过查看插件列表判断：列表丰富且有 Computer Use 弹窗表示已更新，并需在设置中授权使用。
