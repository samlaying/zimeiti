# 小程序从0到1的七步开发法

借助[[AI工具链协作模式]]，可将一个小程序项目从注册到上线分解为七个关键步骤，实现高效开发：

1.  **注册与准备**：在微信平台注册并获取APPID，或使用测试号学习。
2.  **聊前台需求**：从核心场景出发与AI对话，生成PRD，并用AICI和Mermaid生成可视化的界面架构图。
3.  **聊前台UI落地**：采用[[三阶段UI设计法]]，从探索（30分）到转换（80分）再到精修（90分）。
4.  **聊后台需求与落地**：基于前台确定后台功能，保持UI风格一致，开发顺序建议为“先视觉后功能”。
5.  **设计数据库和接口**：让AI帮助设计，开发者重点参与角色权限与表字段设计。
6.  **接入语音和AI能力**：接入外部服务（如腾讯云ASR、混元模型），让AI辅助完成配置与功能补齐。
7.  **部署与上线**：后端部署到服务器，前端在开发者工具中上传审核。
