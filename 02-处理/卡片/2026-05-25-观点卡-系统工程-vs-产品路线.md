# 系统工程 vs. 产品路线

[[Codex CLI]] 和 [[Claude Code]] 代表了两种AI编程智能体的工程哲学：
- **Codex CLI 是“系统工程”路线**：使用Rust构建，以内核级沙箱为核心安全支柱，工具设计少而精（核心是Shell），通过硬件隔离提供强安全性，体现类型安全、模块清晰和高性能的系统编程思维。
- **Claude Code 是“产品与应用”路线**：使用TypeScript构建，以大模型驱动的风险分析和审批流程为核心安全支柱，提供超过40个专用工具的丰富生态和企业级管理能力，侧重灵活性、用户体验和产品集成。

尽管安全机制和工具实现存在根本差异，但两者的核心智能体循环（基于流式API调用并执行工具）是相同的，所有外围工程决策都围绕此循环展开。选择取决于对安全强度、灵活性、开发语言和工具生态的不同权衡。
