# SaaS产品CLI化趋势

SaaS产品正在走向 **CLI（命令行界面）化**。以非书（飞书）开源其CLI能力为标志，这意味着产品功能可以通过命令行来调用，非常适合给 [[AI Agent]] 使用。

这一趋势带来的影响是：
1. **赋能用户**：用户拿到CLI后，可以自定义常用界面，摆脱统一界面的操作限制，实现个性化产品体验。
2. **模糊边界**：当Agent可以接管代码并直接访问产品功能时，开发与使用的边界将进一步模糊。
3. **模块化功能**：产品功能变得更模块化，用户可基于命令组合自定义工作流。
