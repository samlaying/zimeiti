# Obsidian同步飞书关键配置点

将 [[Obsidian]] 笔记同步至 [[飞书]] 云空间，需精确确认以下核心配置，任一参数错误都可能导致同步结果不符合预期：
1. **本地目录**：指定 Obsidian 库的根目录路径。
2. **飞书目标目录**：在飞书开放平台中配置对应的云空间目录。
3. **配置文件关键参数**：在脚本或配置文件中设置 API 密钥、目录映射等参数。

> 配置必须精确，这是成功同步的关键。
