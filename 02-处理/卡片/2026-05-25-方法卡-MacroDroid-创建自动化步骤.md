# MacroDroid 创建自动化步骤

在 MacroDroid 中创建自动化流程的步骤：
1. 打开应用，点击主屏幕加号，选择创建空白宏。
2. 输入宏的名称，例如“充电时杀死后台”。
3. 添加触发器：点击触发器区域加号，导航到设备 > 电池 > 充电状态，选择“已插入充电器”并确定。
4. 添加动作：点击动作区域加号，找到杀死后台应用功能（如设备或 UI 交互），选择“杀死所有应用”并确认。
5. 添加约束条件（可选）：设置时间条件，如“日落之后”，需授权定位以确定日落时间。
6. 保存或测试：插入充电器观察动作是否执行。

示例：创建“充电时杀死后台应用”流程，实现安静充电。
