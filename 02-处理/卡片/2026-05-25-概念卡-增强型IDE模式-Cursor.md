# 增强型IDE模式：Cursor

**增强型IDE模式**的代表工具是 **Cursor**。其核心特点包括：

- **强化学习训练**：内置的Tab模型基于大规模在线策略强化学习（on-policy RL）训练，持续利用用户接受/拒绝代码补全的真实反馈进行迭代优化，使其非常强大。
- **轻量级交互**：支持针对代码文件或片段进行行级别修改的对话式开发。
- **可视化差异**：拥有GUI界面，通过Source Control工具（类似Git的差异视图）可以清晰对比AI修改的代码（增删内容），便于开发者审查。

它是在传统IDE基础上，深度集成AI能力的进化形态。
