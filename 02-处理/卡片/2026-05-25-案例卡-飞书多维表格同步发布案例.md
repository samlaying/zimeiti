# 飞书多维表格同步发布案例

一个将[[飞书多维表格]]中的文章自动同步并发布到公众号草稿箱的实际应用案例。

**流程步骤：**
1. 从飞书多维表格中获取文章链接。
2. 使用 [[HTTP]] 模块获取文章内容。
3. 将内容传递给“[[公众号自动排版发布工作流]]”，映射文章内容与风格参数。
4. 工作流运行后，将输出的草稿URL等数据同步回飞书多维表格进行管理。

**案例价值**：展示了该工作流作为通用节点，与外部数据源（飞书）集成，实现内容批量、自动化发布的强大能力。
