# 小程序从0到1七步法

开发一个完整的小程序，可以将其流程系统化地分解为七个关键步骤：
1. **注册与准备**：完成小程序注册认证，获取`APPID`。学习阶段可用测试号替代。
2. **聊前台需求**：与AI对话生成PRD，并要求其用`AICI`和`Mermaid`生成可视化的界面架构图与跳转流程图，相当于绘制[[线框图]]。
3. **聊前台UI落地**：经历[[UI落地三阶段转换]]，从探索到精修。
4. **聊后台需求与落地**：基于前台功能确定后台需求，使用统一组件库和样式变量开发，并遵循[[先视觉后功能的开发顺序]]。
5. **设计数据库和接口**：在前后台功能样式确定后，让AI设计数据库与API，开发者需重点参与角色权限和表字段设计。
6. **接入语音和AI能力**：项目打通后，接入外部语音识别（如[[腾讯云ASR]]）与AI模型（如[[混元模型]]）等能力。
7. **部署与上线**：后端部署到服务器，前端通过开发者工具上传审核发布。
