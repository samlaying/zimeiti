# HTML PPT可视化编辑工具

UP主自行开发的**网页端可视化编辑工具**，用于对AI生成的HTML PPT进行精细调整。

**核心功能**：
- **文本调整**：选中任意文字，可放大、缩小字号或修改内容。
- **媒体替换**：可方便地插入图片和视频。当AI使用“占位服务”后，可通过此工具上传本地素材替换占位符，并调整大小。
- **保存**：支持“覆盖原文件”直接保存更改。

该工具弥补了AI模型在视觉细节（如字号）把控上的不足，是完成最终成品的关键步骤，其使用流程是[[插入图片与视频的策略]]的一部分。
