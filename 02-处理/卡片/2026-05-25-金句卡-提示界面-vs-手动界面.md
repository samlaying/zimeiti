# 提示界面 vs. 手动界面

> “One is a prompt interface, one is a manual interface that takes a long time to learn. And it depends on what you're working on.”
>
> （一个是提示界面，一个是需要长时间学习的手动界面。这取决于你具体在做什么。）

这句对比了 [[Remotion 是什么？|Remotion]]（基于提示）与 After Effects（基于手动操作）的本质差异，并指出选择取决于具体任务场景。
