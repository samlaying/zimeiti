# PPT-master SKILL

**PPT-master SKILL** 是一个用于 [[Cursor IDE]] 的技能包，其核心功能是**一键将结构化文本（如 Markdown）转化为美观的演示文稿（PPT）**。

**关键特性**：
-   支持从本地文件（如 .md 文件）读取内容并自动生成 PPT。
-   支持动画、音频和旁白导出等高级功能。
-   生成后可在文件夹中找到可编辑的 `.pptx` 文件以及 SVG 源文件，便于回溯和修改。

**安装方式**：从 GitHub 仓库下载并解压到用户目录的 `skills` 文件夹内，并需执行依赖安装命令。
