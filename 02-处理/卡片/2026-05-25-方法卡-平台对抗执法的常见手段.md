# 平台对抗执法的常见手段

根据“[[拼多多暴力抗法事件]]”还原，平台对抗监管的手段可归纳为一套组合拳：

1.  **拖延与隐瞒**：长时间拖延不提供数据，甚至隐瞒真实办公地点。
2.  **物理阻隔**：员工关门硬挡，争夺执法通道控制权。
3.  **肢体冲突与制造混乱**：安保人员带队冲击现场，引发推搡等肢体冲突，制造混乱。
4.  **表演性行为**：如技术总监无实质伤病的“假摔”，企图混淆视听或转移焦点。
5.  **隐蔽干扰**：如在正式问询中，通过纸条提醒同事“沉默不说”等。这些行为体现了对规则的试探，也成为了[[大平台对规则的试探已达傲慢程度]]的佐证。
