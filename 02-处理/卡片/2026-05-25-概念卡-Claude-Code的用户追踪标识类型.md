# Claude Code的用户追踪标识类型

Claude Code 不收集 MAC 地址、CPU 型号等硬件指纹，但通过以下标识进行用户追踪：
- **User ID**：永久保存的 64 位 16 进制字符串，用于跨会话追踪。
- **Anonymous ID**：设备上永久存储的备用追踪 ID。
- **Account ID**：绑定用户账号，通过 OAuth 登录关联身份。
- **邮箱**：永久存储，用于身份关联。
- **仓库哈希**：按项目仓库进行追踪。

这些标识是实现[[防止Claude Code封号的方法]]时需要清除的关键内容。
