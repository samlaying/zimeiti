# Hermes智能体并行工作案例

演示了 Codex 2.0 使用并行智能体完成多任务的能力：
1.  **任务一**：一个智能体正在构建一个生产力应用（调用了头脑风暴和前端设计技能）。
2.  **任务二**：另一个智能体（Hermes）在终端运行，并准备进行人生对话。
3.  **应用操作**：展示了 Codex 如何请求并获得权限后，快速（约12秒）自动打开计算器应用，并在5秒内输入内容（如“医生”）。

这个案例体现了 Codex 同时执行不同任务及操控本地应用的潜力，但也印证了 [[计算机使用功能当前受限]] 的观点，如对 Chrome 的直接操作遇到困难。
