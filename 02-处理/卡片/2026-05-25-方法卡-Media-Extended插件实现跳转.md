# Media Extended插件实现跳转

在Obsidian中实现点击总结文本跳转到本地视频对应位置的方法：
1. **安装插件**：在Obsidian中安装 **Media Extended** 插件。
2. **添加媒体到库**：
   - 在Obsidian中通过插件打开本地视频文件。
   - 在视频标签页右键，选择 **“Add to Media Library”**。
   - 插件会自动创建一个 `Media Lab` 文件，其中记录了该视频文件的**唯一访问地址**。
3. **链接跳转**：总结笔记中的链接格式为 `![[视频地址#t=时间点]]`，点击即可跳转。

该方法是将 [[自动化脚本替换时间戳|脚本处理后]] 的时间戳转化为可交互链接的关键环节。
