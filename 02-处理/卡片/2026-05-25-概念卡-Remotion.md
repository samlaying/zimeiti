# Remotion

**Remotion** 是一个基于 React 的视频创作框架。它允许开发者使用 JavaScript 或 TypeScript 代码来编写、定制和生成视频，将动画制作从纯设计或纯剪辑转向可编程、可版本控制、可参数化的领域。
