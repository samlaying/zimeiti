# Token Efficiency

Token Efficiency 是当前AI产业的核心发展方向，其核心要义是：

实现AGI更重要的不是造出最厉害的模型，而是**让小模型在大模型的指导下以及在Harness的配合下，胜任绝大多数的任务，从而达到节省成本、实现极致杠杆化**的效果。

它已从行业叙事转变为产业共识，各大模型厂商（如DeepSeek、千问、小米）发布大小模型组合的趋势也印证了这一点。
