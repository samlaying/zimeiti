# 串行工作流是AI Agent效率瓶颈

在AI Agent时代，让同一个Agent同时承担搜索资料、编码、查资料等多项任务，会形成**串行工作流**。这种模式既混乱又缓慢，是制约AI Agent工作效率的主要瓶颈。

[[Claude Agent Teams]] 通过引入并行工作机制，旨在解决这一根本性问题。
