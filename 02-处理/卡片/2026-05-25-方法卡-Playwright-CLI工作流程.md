# Playwright-CLI工作流程

1.  **安装工具**：在电脑上安装 [[Playwright-CLI工具]] 命令行工具。
2.  **人机交互**：用户向 AI Agent 描述任务需求。
3.  **指令生成与执行**：AI 会阅读 CLI 命令，生成操作指令，自动驱动浏览器执行一系列操作。
4.  **数据提取**：AI 识别并抓取浏览器页面中的目标信息。
5.  **流程固化**：完成一次成功操作后，AI 可以将整个操作流程总结成一个 [[Skill固化能力]]。
6.  **重复执行**：使用总结好的 Skill，可以对其他同类页面进行重复的自动化操作。
