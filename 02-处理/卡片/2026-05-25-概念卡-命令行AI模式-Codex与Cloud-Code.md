# 命令行AI模式：Codex与Cloud Code

**命令行AI模式**的代表工具是 **Codex** 和 **Cloud Code**。其核心特点是完全抛弃IDE，在命令行中工作：

- **纯粹的自然语言编程**：开发者进入项目目录后，用自然语言下达指令，AI生成代码。
- **工作流程**：开发者下达指令 -> AI生成代码 -> 开发者进行验证。
- **核心理念**：为了驾驭这种高度自动化的模式，需要一套如 [[Superpowers Skills]] 这样的规范流程来保证产出代码的质量。

它代表了当前AI辅助编程自动化程度最高的形态。
