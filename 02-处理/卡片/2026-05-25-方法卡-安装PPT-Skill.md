# 安装PPT Skill

将[[PPT Skill]]集成到[[Claude Code]]中的两种方法。

**方法一：通过Claude Code对话安装（推荐）**
1. 在GitHub上复制PPT Skill的仓库链接。
2. 在Claude Code对话框中，将链接发送给它，并指示：“帮我下载并安装这个 Skill”。

**方法二：手动安装**
1. 在GitHub页面点击 “Code” -> “Download ZIP” 下载压缩包。
2. 解压缩下载的文件。
3. 在VS Code中打开你的Claude Code项目。
4. 找到项目的 `Skill` 文件夹。
5. 将解压后的文件夹（如 `frontend slides`）拖拽到 `Skill` 文件夹内。**注意**：需去掉文件夹名的后缀（如 `-main`），确保在VS Code中调用指令时名称一致。

**安装成功验证**
在Claude Code对话框中输入斜杠 `/`，然后输入 `frontend slides`。如果能自动补全并出现相关说明，则表示安装成功。
