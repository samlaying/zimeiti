# 使用Pair Manager进行全库健康检查

这是[[Pair Manager技能]]的“全库健康检查”子命令，用于发现并报告[[Obsidian]]笔记库的潜在问题：
1.  **发出指令**：说“检查笔记库”。
2.  **AI扫描与报告**：AI会扫描并生成一份结构化报告，列出五类问题：孤立文件、断链、空目录、缺少FrontMatter、收件箱积压。
3.  **安全操作**：所有操作会先确认，不会静默修改文件。

建议每个月使用一次，进行知识库的定期维护，对应了[[Obsidian笔记管理的四大痛点]]中的“健康问题难发现”。
