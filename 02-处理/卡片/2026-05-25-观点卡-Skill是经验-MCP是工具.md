# Skill是经验，MCP是工具

Skill与MCP是“经验”与“工具”的关系。

- **MCP**：是大模型与外部工具交互的协议，相当于给大模型（大脑）配备的“手”，MCP插件是手上的“工具”。
- **[[Skill（智能体技能）]]**：是**使用工具的经验**。它规定了在什么场景下、按什么顺序、组合使用哪些工具（工具可以是MCP插件，也可以是本地Script脚本）。

**核心观点：Skill编排并驱动MCP等工具。** 给一个大学生一堆工具，他不一定能修好车，因为他缺乏经验和流程。
