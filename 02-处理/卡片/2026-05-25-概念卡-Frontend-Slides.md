# Frontend Slides

一个将文字内容直接生成为 HTML 演示文稿的AI工具。生成的文件可在浏览器中全屏播放，包含字体、动画、布局和配色。

**核心优势：**
1. **零依赖**：无需安装任何软件。
2. **响应式布局**：自动适配电脑、平板、手机。
3. **自带交互动画**：包括翻页、进度条、元素入场动画等。
4. **纯前端文件**：一个 HTML 文件即可独立运行。

它解决了传统 [[PowerPoint]] 工具依赖和灵活性的问题。
