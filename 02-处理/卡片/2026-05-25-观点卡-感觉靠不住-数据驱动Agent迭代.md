# 感觉靠不住，数据驱动Agent迭代

在Agent系统的设计与迭代中，依赖感觉判断改进效果是不可靠的。感觉不能告诉你好了多少，也不能告诉你在哪里好了，在哪里变差了。必须建立一套基于数据的质量评估方法，将“感觉好用”转变为“数据证明变好”。
