# Claude Code + MCP安装两种方法

> 两种安装路径，覆盖网页端和命令行用户。

## 核心内容
安装Higsfield MCP Server有两种主要方法。一是在ClaudeAI网页应用的设置中添加自定义连接器，通过复制粘贴代码并验证即可。二是在Claude Code终端中，直接粘贴MCP设置代码，程序会自动引导完成身份验证。连接成功后，即可在终端通过自然语言调用Higsfield的各类AI生成模型。

## 原文关键段落
> 在Cloud Code里设置MCP Server也很简单。基本上只要用自然语言说帮我设置这个MCP Server就行。
