# 跟Cloud Code学做人

> “所以这个跟Cloud Code学做人这个还是有好处的”

**语境**：在观察到`[[AI编程助手]]`在`[[brainstorm skill]]`中总是提供选项供用户选择，而非抛出开放性问题后，UP主发出的感叹。

**价值**：幽默而深刻地指出，高效、结构化、低阻力的沟通模式，是人与AI，乃至人与人之间高效协作的关键。我们可以从AI的交互设计中学到优秀的沟通方法。
