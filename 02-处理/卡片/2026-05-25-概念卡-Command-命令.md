# Command（命令）

为了方便重复输入长提示词，将其固化，并通过类似快捷键的短命令快速调用的功能。

它解决了重复输入长 [[Prompt（提示词）]] 的效率问题。例如，AI客户端（如Cloud Code）支持将常用提示词保存为文件，通过短命令一键输入。
