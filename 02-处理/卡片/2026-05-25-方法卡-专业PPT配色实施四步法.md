# 专业PPT配色实施四步法

专业PPT的配色应遵循以下流程：
1.  **先完成内容**：制作完所有PPT页面内容后，最后才考虑添加颜色。
2.  **定义整体用色规范**：确定基础色（黑白灰），选择主题色，并指定不同模块（正文、标题、注释、线条等）的颜色。
3.  **保持一致性**：确保整个文稿中，**同一内容层级**（如所有标题、所有重点标注）使用完全相同的颜色。
4.  **谨慎添加颜色**：只在必要时，为了突出关键重点（如特定数据）而添加颜色，避免大块上色。上色前需思考其与前后页面颜色的关系。
