# 使用 Awesome Design 预览并生成网站

这是一个结合设计资源与[[AI工具]]的高效网站开发流程。

**操作步骤：**
1.  **访问预览**：访问 KLN.xyz，通过滑动浏览或搜索找到心仪的设计风格，并点击大图预览确认。
2.  **获取资源**：点击跳转链接，访问该设计风格对应的仓库根部，获取文档或资源。
3.  **复制文档**：将仓库中对应风格的文档（如样式、布局代码或描述）复制下来。
4.  **AI 生成**：在 Cursor 或 Cloud Code 等AI辅助工具中，将复制的文档作为上下文，并输入提示词：“我要通过这些文件为我生成一个网站”。
5.  **获得结果**：AI 将基于文档生成一个视觉风格与所选设计高度一致的网站，无需手动调整视觉效果。

**关键优化**：网站采用了[[懒加载 (Lazy Loading)]]技术来提升加载效率。
