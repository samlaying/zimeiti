# 为“用户的用户”设计

产品设计不仅要考虑一级用户，还要考虑**“产品用户的传递链”（树状辐射模型）**。

**以AI生产力工具为例**：
1.  **产品**：AI生产力工具。
2.  **一级用户**：前端程序员（用其更快地开发网站）。
3.  **二级用户**：数学老师（使用程序员开发的LaTeX数学公式网站）。
4.  **三级用户**：数学老师的学生（通过网站学习）。

**思考价值**：通过分析二级甚至多级用户群体（如庞大的学生群体），可以反哺产品决策（如加强对LaTeX语法的训练），从而提升一级用户体验，形成良性循环，促使一级用户成为[[卓越用户传播法]]中的卓越用户并主动传播。
