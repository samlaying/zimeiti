# Skill指挥Agent调用外部工具完成复杂任务

> 用Skill为Agent编排一个复杂的‘工作流水线’。

## 核心内容
高级的Skill可以指挥Agent调用联网搜索、知识库检索等外部工具。例如一个竞品分析Skill，可以设计一个完整流程：先搜索竞品官网，再抓取页面信息，接着查询用户评价，最后生成对比报告。这使得Skill能处理更复杂的现实任务。

## 原文关键段落
> 比如一个竞品分析的skill可以这样工作：先搜索竞品官网，再抓取页面信息，再整理产品功能，再查询用户评价，再生成对比报告，再输出战略建议。
