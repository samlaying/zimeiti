# AI视频快速生成流程（Remotion+VibeCoding）

使用 AI 快速生成视频的 **VibeCoding** 方法，基于 [[Remotion：代码驱动的视频生成框架]] 实现，核心流程如下：

1.  **准备环境**：安装 Remotion 框架，并下载必要的资源（如SQL配置）。
2.  **AI对话生成**：通过 Cloud Code（AI辅助编码工具）与框架交互。向AI发送包含结构化要素的提示词，引导其自动生成视频代码。
3.  **提示词设计**：关键提示词应结构化，明确包含**配音稿、字幕、动画、图标**等部分。文案可以提前准备或让AI自动生成。
4.  **迭代优化**：通过不断对话和完善提示词，优化生成过程，最终输出包含文字动画、字幕和音频的视频。
