# 产品原型制作三阶段工作流

产品原型制作是一个系统化流程，可分为三个阶段：

## 一、画原型前
1.  **需求调研**：进行业务调研与用户调研。
2.  **需求整理**：编写[[AI辅助编写用户故事|用户故事]]。
3.  **竞品分析**：重点关注功能层面（功能点、交互布局、视觉体验）。
4.  **梳理产品结构图**：产出后续设计的基础。

## 二、画原型中
1.  选择方案：采用[[精准控制与边做边探索|精准控制或边做边探索]]方式。
2.  使用AI工具生成原型，并整理为产品说明书指导生成。

## 三、画原型后
1.  **代码管理**：使用Git管理，替代传统原型工具。
2.  **云端部署**：方便团队访问。
3.  **标注与评审**：解决[[WebCoding下的原型标注方案|标注难题]]，并支持评审现场修改。
4.  **文档与跟进**：生成PRD文档，并与研发团队持续对齐、记录变更。
