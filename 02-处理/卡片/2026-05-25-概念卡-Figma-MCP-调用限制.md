# Figma MCP 调用限制

Figma官方MCP对**免费用户**有限制：**每月最多提供6次工具调用**。这个次数基本无法满足AI在画布上进行设计的需求。

若想充分利用[[Use Figma Tool：AI直接在画布上设计]]，需要订阅 **Dev** 或 **Full** 席位。[[修改现有设计风格]]的案例就遇到了此限制。
