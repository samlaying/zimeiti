# Symphony 任务管理框架

Symphony 是一个用于实现**任务式 Agent 协作**的任务管理框架或工具。它通过任务板来组织和追踪工作流，取代传统的直接对话交互模式，为 Agent 的[[任务式 Agent 协作]]提供了基础平台支持。
