# 自适应爬虫

自适应爬虫是一种能够**自动适应目标网站结构变化**的智能爬虫框架，核心解决了[[传统爬虫的三大痛点]]中的“网站改版导致代码失效”问题。以 Scrapling 为例，它使用**相似度算法**，在网站改版后能自动定位到网页元素的新位置，无需开发者手动修改 CSS 选择器等定位代码，实现了“元素自己跟着走”，极大地提升了爬虫的稳定性和维护效率。
