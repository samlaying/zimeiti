# CSS绘制三角形指示器

> 伪元素 + clip-path polygon()，三点定一个三角形。

## 核心内容
使用伪元素（如`::before`）结合`clip-path`属性来绘制三角形。设置伪元素为绝对定位，通过`clip-path: polygon(0% 100%, 50% 0%, 100% 100%)`定义三个顶点，即可生成一个向上的三角形。三角形的颜色通常通过`background-color: inherit`从父元素继承。

## 原文关键段落
> .underline::before {
>     content: '';
>     /* ...定位和尺寸属性... */
>     clip-path: polygon(0% 100%, 50% 0%, 100% 100%);
> }
