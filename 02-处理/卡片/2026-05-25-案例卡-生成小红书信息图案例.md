# 生成小红书信息图案例

演示使用该生图Skill生成一张小红书风格的信息图，完整流程如下：
1. 在CMD中启动[[Cloud Code]]，调用生图Skill。
2. 选择“小红书信息图”模板。
3. 输入或上传需要可视化的信息。
4. 按需选择具体参数，例如：
   - **布局**：对比矩阵
   - **比例**：16:9
   - **模型**：最新的3.1 Flash
5. 提交后，系统通过[[星空AI接口]]自动生成高质量图片，速度快。

该案例验证了Skill的自动化流程和易用性，最终生成了一张符合预期的矩阵对比图。
