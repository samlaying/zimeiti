# 创建思维导图

在 Excalidraw 中创建思维导图的步骤：
1.  在画布中绘制一个矩形并选中它。
2.  运行 `Action Create Mind Map` 脚本。
3.  在弹出的可视化 UI 界面中配置：
    - **主题配色**：如彩虹色、莫兰迪色等。
    - **排版方向**：向右、向左、向下展开。
    - **连线样式**：平滑曲线（Smooth）、折线箭头（Elbow）、S型曲线（Curve）。
    - **其他**：箭头固定点、节点间距等。
4.  采用默认设置点击确认，即可将所选矩形生成为思维导图根节点。
