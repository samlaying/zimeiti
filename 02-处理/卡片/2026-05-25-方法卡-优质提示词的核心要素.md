# 优质提示词的核心要素

向AI下达生成[[PPT]]的指令时，提示词的质量决定输出质量。一个有效的提示词模板应包含五个关键要素，缺一不可：

1.  **角色设定**：我是谁（如Python讲师）。
2.  **场景**：在什么场合使用（如给零基础大学生的入门课）。
3.  **主题**：核心内容是什么。
4.  **风格要求**：具体的设计或表达风格（如轻松活泼）。
5.  **工具**：明确调用的工具，如 [[Frontend Slides]] 和 [[Tether Scale Frontend Design]]。

**核心原则**：信息越具体、越量化，AI给出的结果就越精准。避免使用“好看一点”这类模糊指令。
