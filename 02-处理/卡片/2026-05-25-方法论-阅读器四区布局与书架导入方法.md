# 阅读器四区布局与书架导入方法

> 清晰的四区布局，书籍导入仅需四步。

## 核心内容
EPUB阅读器界面分为左侧导航栏（目录、摘录、书签）、中间阅读区、顶部工具栏和右侧笔记区。导入书籍的步骤是：进入书架 -> 点击设置（齿轮）图标 -> 扫描仓库中的书籍 -> 勾选未添加的书籍 -> 点击“加入”。若书架未刷新，可手动点击刷新按钮。
