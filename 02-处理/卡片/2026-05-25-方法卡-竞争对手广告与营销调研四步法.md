# 竞争对手广告与营销调研四步法

对竞争对手的调研，核心是分析其广告内容与营销渠道，可分为四步：
1.  **识别营销渠道**：明确竞争对手主要在哪些平台（线上如淘宝、抖音；线下如商业街）进行营销。
2.  **分析广告内容**：在对应渠道下，逐一分析竞争对手的广告（如图片、文案、门店门头）。关注点包括：主体是否突出、背景是否干净、文案重点是否明确、视觉是否形成差异。例如，在淘宝分析“杭州乐园”门票广告时，发现卖得好的图片通常具备这些特点。
3.  **应用线下货架理论**：对于线下，运用[[线下竞争本质：整条街都是货架]]的观点，将整条街的店铺都视为竞品，研究其视觉呈现。
4.  **与主要对手做对比**：在调研基础上，找到自身广告和营销的差异点，为后续视觉设计提供策略方向。
