# Skill 配图实战：专栏文章配图

**应用场景**：为公众号专栏文章《AI工具的使用和安装教程》进行配图。

**操作流程**：
1. 完成文章撰写。
2. 使用 [[配图 Skill]]，发送“帮我配图”指令。
3. Skill 自动生成 **10 张**简约、美观的手绘风格图片。

**应用效果**：
- 生成的图片可直接插入文章对应段落。
- 图片与文章逻辑匹配，提升可读性。
- 可一键将配图文章推送至飞书或公众号草稿箱。
