# 配置并使用蓝印Agent

使用蓝印RPA的Agent功能需要三个核心步骤：

1.  **准备与配置**：
    *   访问蓝印官网 `蓝印rpa.cn` 注册账号。
    *   访问 **DeepSeek 开发平台**，注册并获取一个 **API Key**（可预先充值少量金额）。
    *   在蓝印Agent配置界面中，填入此DeepSeek API Key。

2.  **绑定微信**：
    *   配置API后，在蓝印客户端内点击绑定微信选项。
    *   使用**个人微信**扫码完成绑定连接。

3.  **通过微信对话控制**：
    *   绑定成功后，在微信中会出现 **“[[微信Cloud]]”** 对话入口。
    *   在此对话中，通过发送**自然语言指令**即可控制[[RPA脚本]]的运行与查询。
