# AI产品经理四大进阶学习路线

为应对新要求，一个系统性学习路线分为四个递进阶段：

1.  **零技术入门与快速上手**：建立认知，掌握[[提示词工程]]、工具实操（如低代码平台、N8N），做出简单应用。
2.  **企业级开发与核心技术框架**：掌握Python、[[RAG：检索增强生成]]、向量数据库、LangChain/LlamaIndex框架、[[Agent与协议开发]]及AI产品设计全流程。
3.  **深入底层原理与模型优化**：学习NLP、Transformer、模型微调（如LoRA）、分布式训练与部署，从“会用”升级到“懂原理，能优化”。
4.  **实战项目融会贯通**：通过六大真实企业级项目（如AI销售助手、高性能RAG系统）综合运用所学，提升解决问题的能力。
