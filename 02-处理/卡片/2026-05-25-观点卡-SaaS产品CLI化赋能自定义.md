# SaaS产品CLI化赋能自定义

SaaS 产品正在走向 CLI（命令行界面）化，这是一个重要的趋势。

**核心洞察**：
- 例如，飞书开源了 CLI 能力，允许 AI Agent 或用户通过命令行直接操作产品功能（如发消息、创建文档）。
- CLI 接口天然适合 AI Agent 调用，增强了自动化和集成能力。
- 用户拿到 CLI 后，可以自定义常用操作界面，摆脱对统一封装界面的依赖，实现个性化工作流。

**未来影响**：当 Agent 可以接管代码和直接访问产品功能时，将模糊开发者与用户的边界。
