# Pencil：AI UI设计协作工具

**Pencil** 是一款深度整合进设计流程的AI UI工具。它拥有大量官方规范，生成结果稳定、不“乱跑天”，设计过程可控。

其最大优势是**可编辑性**，生成的是带分层的原文件，类似Figma，可以直接上手修改，编辑自由度极高。这使得它**落地性强**，非常便于持续编辑和迭代，适合长期项目。
