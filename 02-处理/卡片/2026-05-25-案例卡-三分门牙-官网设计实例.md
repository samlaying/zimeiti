# “三分门牙”官网设计实例

以创建“三分门牙”官方网站为例，演示完整设计过程：

1.  **素材生成**：截取喜欢的`hero section`，使用[[Nada Banana 2]]生成干净静态图，再用[[King 3.0]]生成循环背景视频。
2.  **初版搭建**：在[[Cursor]]中初始化[[Next.js]]项目，结合下载的UI设计[[Agent Skills]]，生成包含首页、AI工具栏、门牙课堂板块的初始布局代码（AI味较重）。
3.  **核心提升**：
    - 将背景视频放入`public/assets`文件夹，通过提供路径方式让AI引入。
    - 从专业设计网站提取字体、CSS样式等设计系统提示词，提供给Cursor，**重构网站按钮、文字等细节**，这是去除[[AI生成感]]的关键。
4.  **细节微调**：调整背景视频明亮度以突出视觉效果；将顶部导航栏与背景视频融合，避免悬浮感；统一字体；控制液态玻璃等效果强度。

最终得到一个背景动态、布局干净、细节精致、视觉高级的官方网站。
