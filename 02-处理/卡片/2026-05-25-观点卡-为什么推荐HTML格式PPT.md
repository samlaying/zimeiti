# 为什么推荐HTML格式PPT

在生成演示文稿（PPT）时，推荐使用 **HTML（网页）格式**，而非 Microsoft PPT 或 Beamer 等格式。其优势在于：

1.  **零依赖，易分享**：不需要安装特定软件（如 PowerPoint 或 LaTeX 编译环境），生成的 HTML 文件可以在任何设备的浏览器中打开和查看。
2.  **代码友好，易修改**：PPT 的内容和样式由代码（HTML/CSS）定义，这对于习惯使用代码的编程者来说，修改起来比操作 GUI 软件更直接、更精确。
3.  **结构清晰，利于协作**：整个演示文稿封装在一个文件中，便于版本控制和内容管理。

这种格式选择与 [[Vibe Coding]] 的工作流高度契合，让内容创作回归到代码构建的本质。
