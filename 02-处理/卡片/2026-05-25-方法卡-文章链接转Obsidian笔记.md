# 文章链接转Obsidian笔记

这是一个通过 [[Claude Code Skill]] 自动将网页文章链接转为结构化 Obsidian 笔记的方法。

**核心步骤：**
1.  **粘贴链接**：向 Skill 提供文章 URL（如微信公众号文章）。
2.  **自动抓取与分析**：Skill 调用如 DeFado 等工具抓取网页正文，提取标题、作者等元数据。
3.  **结构化生成**：基于内容分析，按预设模板生成笔记，自动推断标签、阅读时间、核心观点等。
4.  **自动打开**：生成的笔记文件在 Obsidian 中自动打开。

**输出效果：**
- **属性区**：包含规范、完整的元数据（标题、作者、来源、标签等）。
- **正文**：按主题分段整理，配有小标题和要点总结，保留核心内容，阅读清晰。
