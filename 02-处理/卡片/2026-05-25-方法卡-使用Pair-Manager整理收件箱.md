# 使用Pair Manager整理收件箱

这是[[Pair Manager技能]]的“整理收件箱”子命令的操作流程：
1.  **发出指令**：直接对AI说“整理收件箱”或“处理inbox”。
2.  **AI操作**：AI会扫描 `00-inbox` 文件夹，找到所有待处理笔记，逐个展示摘要并询问归档位置。
3.  **用户确认与操作**：用户回复一个选项（如对应领域），文件即被移动到正确目录。操作完成后，MOC和首页会自动更新。

此命令是日常笔记流管理的关键，建议每天使用以清空收件箱。
