# Superpowers Skills 框架

**Superpowers Skills** 是一个旨在驾驭[[Vibe Coding]]时代软件开发的技能框架，已在GitHub获得超7万星标。
- **加载方式**：AI（如Codex）启动时，会自动从项目`.codex/skills/`目录下加载所有技能文件到上下文中，作为开发的约束和指导。
- **核心理念**：**“[[慢就是快]]”**。不急于让AI直接生成最终代码，而是先让AI进行周密的规划与思考，避免错误和反复，整体上更快。
- **作用**：通过一系列预定义的技能，强制AI遵循“先探索、后规划、再小步实现并持续验证”的开发范式，将不可预测的“随机生成”转变为可控的“工程实践”。
