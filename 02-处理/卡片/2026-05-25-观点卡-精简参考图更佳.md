# 精简参考图更佳

在保证 UI 一致性的前提下，**精简参考图策略**比丰富参考图更优：
- 仅提供必须保留的框架元素截图（如干净的侧边栏），右侧内容留白。
- 优点：在确保框架一致的基础上，给予 AI 最大发挥空间，生成设计感更强、排版更精致、配色更舒服的 UI。
- 效果：例如某次生成被评价为“排版紧凑又不过密，配色舒服”，带来全新灵感。
