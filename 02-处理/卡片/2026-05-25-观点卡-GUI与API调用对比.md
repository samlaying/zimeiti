# GUI与API调用对比

在 AI 执行任务时，存在两种主要方式：
- **GUI 方式**（如 Computer Use）：模拟人类操作，更直观，但非常消耗 [[Token消耗高的问题]]，效率相对较低。
- **API 调用方式**（如 OpenCloud）：直接调用底层工具接口，更省 Token，效率更高，是更优的交互解。

[[Codex Computer Use功能]] 中的鼠标操控能力可能强于其内容生成能力（如代码编写）。
