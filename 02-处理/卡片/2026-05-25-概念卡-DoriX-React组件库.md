# DoriX React组件库

DoriX是一个面向前端开发者的开源React组件库，提供了110多个高质量的UI组件。组件覆盖文本动画、动画边框、交互式菜单、3D背景等高级感设计效果，旨在快速提升项目视觉效果和交互体验。基于React框架，无需复杂配置即可使用。详细使用方式可参见[[使用DoriX组件库的方法]]，其核心价值见[[DoriX项目的核心价值]]。
