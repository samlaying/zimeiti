# 产品驱动增长：无引流却自然涨星

一个开源项目的核心增长动力，可能并非来自主动引流，而是源于其独特的产品价值。

PPT-Master 作者没有专门引流，主要通过 B 站分享视频，但认为 B 站流量对 GitHub 用户影响有限。项目在一个月内从 2200 star 涨到 7200 star，作者认为这更多源于[[PPT-Master 设计初衷]]——工具本身对用户的吸引力。

这反映了“产品驱动增长”（Product-Led Growth）的逻辑：当产品本身具备足够的差异化和实用价值时，用户（尤其是开发者社区）会自发地使用和传播。
