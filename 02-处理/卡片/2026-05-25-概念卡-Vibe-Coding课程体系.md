# Vibe Coding课程体系

一个为期60天的[[Vibe Coding]]系统化学习课程框架，分为三大阶段：
1.  **工具与基础**：学习必要的工具使用和基础知识。
2.  **进阶（全栈开发）**：掌握包含前后端的全栈开发能力。
3.  **整合上线**：学习项目上线、数据维护等独立开发者关键能力。

每日学习单元包含学习大纲、具体知识、实践作业（结合正在进行的项目）、参考资料和可保存的学习笔记，实现“学做结合”。
