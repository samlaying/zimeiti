# 用AI构建AI工具的开发循环

构建 [[Claude Code]] 追踪系统的过程本身就是一个“用AI构建AI工具”的循环：
1.  使用 **Perplexity AI** 进行快速研究，生成结构化的需求文档（`changelog app.md`）。
2.  将需求文档和相关 API 文档（如 [[Gemini API]] 文档）喂给 **Claude Code**，通过提示工程指导它编写和迭代应用程序。
3.  利用生成的工具（更新追踪系统）来更好地理解和使用 **Claude Code** 本身。
这个循环展示了AI工具如何赋能个人开发者快速构建复杂的自动化解决方案。
