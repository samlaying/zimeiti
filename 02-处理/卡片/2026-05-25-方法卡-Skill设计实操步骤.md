# Skill设计实操步骤

使用[[Skill全能反推提示词]]进行设计的完整步骤：
1. **寻找参考案例**：找到喜欢的参考图像或设计案例。
2. **生成反推提示词**：为参考图像生成详细提示词，描述核心视觉元素。
3. **修改提示词**：根据需求修改提示词中的元素，如主视觉元素。
4. **生成新设计**：基于修改后的提示词生成新设计方案。
5. **进一步编辑**：编辑文字、颜色等细节以完善设计。
6. **探索更多方案**：生成不同方案并尝试个性化调整。
