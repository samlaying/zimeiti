# 手动添加MCP

在 `Builder with MCP模式` 中手动添加 MCP 服务的标准流程：
1.  **获取配置**：从 MCP 服务商（如火山引擎 MCP 市场）处获取对应工具的 **JSON 配置代码**。
2.  **打开管理界面**：在 IDE（Cursor/Tree）的 MCP 管理界面，选择“手动添加”。
3.  **粘贴确认**：将复制的 JSON 数据粘贴到输入框并确认。

**关键点**：不同 IDE 的 JSON 字段结构完全一致，都遵循 MCP 的标准格式校验规则。此方法适用于需要精细控制或市场未收录的 MCP。
