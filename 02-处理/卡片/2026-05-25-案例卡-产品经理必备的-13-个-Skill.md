# 产品经理必备的 13 个 Skill

## 产品经理 Skill 包实例
基于 [[基于职业自动生成工作技能包]] 方法，并经过实践总结，开源的 13 个产品经理核心 Skill 包括：

- 需求文档编写
- 需求评审团
- 需求优先级划分
- 需求路线图撰写
- 需求数据分析
- 原型图绘制（三种方法）：
  1.  **截图 -> HTML**：输出可运行的 HTML 原型。
  2.  **截图 -> Pencil 设计稿**：强烈推荐，尤其使用“Image to Pencil”插件效果很好。
  3.  **URL -> Next.js 项目**：输出一个本地项目。

这些 Skill 旨在提升产品经理的工作效率，或为其他职业者提供产品经理的能力。
