# MCP集成赋予AI眼睛

通过内置MCP Server，[[Scrapling框架]]让AI Agent能够直接调用抓取网页数据。例如，AI说“帮我抓取这个页面的价格”，Scrapling自动执行并返回结构化数据，使AI Agent拥有了自己的“眼睛”，实现更智能的数据交互。
