# 迭代节点+外部API处理图片

当Dify内置模型不支持所需功能（如图片生成/修改）时，可通过**迭代节点配合外部API调用**的方案解决。

**以批量处理图片列表为例**：
1.  **输入**：包含多个图片URL的变量（如 `image_list`）。
2.  **迭代**：使用 `迭代` 节点遍历列表中的每一项。
3.  **调用API**：在迭代循环内，添加 `HTTP请求` 节点（POST方法），调用外部图片处理API（如文心一言）。
   - **请求配置**：需在请求头中携带 `Authorization: Bearer <API Key>`，并在请求体（JSON）中包含 `model`、`messages`（含原始图片URL和修改提示词）等参数。
4.  **收集结果**：迭代完成后，汇总每次调用返回的新图片URL，输出为新列表。

**注意事项**：需注意API密钥安全，并关注外部API的调用频率与稳定性。
