# 硅葬Skill（HTML方式）

硅葬的PPT Skill，原理是使用HTML方式生成动态网页格式的演示，属于“暴力确认技能”。

核心特点：
- **轻量且快速**：生成和修改速度快。
- **支持动效**：可加入视察动效（如动画），让PPT更灵活。
- **可集成图片**：支持插入[[Image 2模型]]生成的图片或AI写的SVG矢量图。

缺点是**模板化风格**（所有生成PPT基于固定排版布局，个人风格强，如杂志风），且**适用性有限**（不适合国企、事业单位等正式场合）。
