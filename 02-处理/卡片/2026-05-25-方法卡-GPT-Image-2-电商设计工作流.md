# GPT Image 2 电商设计工作流

使用 [[GPT Image 2 工具定义]] 生成跨境电商图片的标准工作流如下：
1.  **上传产品图**：准备一张清晰的产品原图（如服装、鞋子）。
2.  **一键执行**：在工具界面点击“整组执行”按钮。
3.  **自动输出**：系统自动生成12张主图和10张详情页。
4.  **自定义调整**：如需改变风格，可更换配色，但产品核心特征（颜色、面料）保持不变。
