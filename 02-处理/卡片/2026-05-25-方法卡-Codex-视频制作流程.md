# Codex 视频制作流程

使用 Codex 的 HyperFrames 插件制作视频的完整四步流程：

1.  **准备内容**：将用于讲解的文档（如公众号文章）拖入 Codex 对话框，下指令让 AI **生成一个适合做视频的、详细的讲解脚本**。
2.  **录制音频**：按照生成的脚本，使用录音软件录制个人讲解音频并保存为文件。
3.  **生成视频**：将录制好的音频文件拖入 Codex 对话框，下指令给 [[HyperFrames 插件]]，提出具体的视频制作要求（如保留所有内容、有动画效果、带字幕等）。
4.  **等待输出**：让 Codex 和插件自动处理，最终生成一个嵌入了你声音和匹配字幕的视频文件。
