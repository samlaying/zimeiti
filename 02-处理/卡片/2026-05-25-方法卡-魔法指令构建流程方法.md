# 魔法指令构建流程方法

使用[[影刀RPA魔法指令]]创建自动化流程的标准步骤：
1.  **准备环境**：在影刀客户端新建“网页自动化”应用，并安装对应浏览器插件。
2.  **描述指令**：将“魔法指令”组件拖入编辑区，选择“网页自动化”类别，在输入框中用自然语言**详细描述任务目标**（如“在小红书搜索AI并抓取前10篇笔记的标题和作者”）。
3.  **辅助框选**：根据工具提示，框选网页上的关键区域（如搜索框、结果列表），以帮助AI精确定位元素，提升流程稳定性。
4.  **优化与生成**：使用“优化提问”功能让AI规整指令描述，然后发送指令，AI会自动生成完整的RPA流程。
5.  **调试与完善**：首次运行可能出错，可使用“智能修复”功能让AI尝试自动纠正。最后检查流程，如缺少“打开网页”等前置步骤需手动补全。
