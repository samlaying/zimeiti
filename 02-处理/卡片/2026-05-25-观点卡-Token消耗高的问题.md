# Token消耗高的问题

Codex Computer Use 功能会消耗大量 Token。原因在于，每次操作都需要截图，并将视觉信息转换为“视觉 Token”。视觉 Token 的数量远多于文字 Token（因需切片、向量化），导致 Token 快速耗尽。

因此，该功能更适合关键性验收任务，需谨慎权衡使用成本。
