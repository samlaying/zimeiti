# Python模块

模块是Python中预先定义好的代码文件（.py格式），包含函数、变量等，便于代码复用和项目管理。

## 模块的类型
1.  **内置模块**：安装Python时自带，如 `math`、`sys`，可直接使用[[import]]导入。
2.  **外部库**：需要使用 `pip` 等工具额外安装的模块，如 `pygame`、`numpy`。

模块通过 `import` 关键字导入使用。
