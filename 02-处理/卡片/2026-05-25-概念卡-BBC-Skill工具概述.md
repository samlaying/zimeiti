# BBC Skill工具概述

**BBC Skill** 是一个开源的 Python 命令行工具，专为 **[[B站UP主]]** 设计。

*   **核心痛点**：解决创作者面对海量视频评论（数百至上千条），人工阅读、筛选、分析效率极低的问题。
*   **解决方案**：一键拉取视频下的全部评论（包括顶级评论、楼中楼、置顶评论）及视频元数据，并将其整理为结构化数据（`comments.csv` 和 `summary.json`），专门用于交给 **[[AI Agent]]** 进行深度分析。
*   **核心特性**：
    *   **零依赖**：仅使用 Python 3.9 标准库，无需 `pip install`。
    *   **标准化输出**：输出文件专为 AI 读取分析而设计。
    *   **开放兼容**：遵循开源 Agent Skills 规范，原生兼容主流 AI 编程助手及 SkillsMP 包管理器。
