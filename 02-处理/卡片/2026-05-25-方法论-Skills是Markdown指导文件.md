# Skills是Markdown指导文件

> Skills是Agent时代的可执行程序。

## 核心内容
Skills（技能）本质上是指导大模型行为的文本文件，通常采用Markdown格式。它通过结构化的指令和约束，将原本可能由大模型自由发挥的复杂工具调用链（如获取位置-获取日期-获取天气）固化下来，从而提高其执行任务的准确性和效率。

## 原文关键段落
> 那么Skills呢它其实在我们创建Agent或者我们使用Agent或者说ClauseCode这种级别的Agent工具的时候呢那么其实它本质上最核心的它还是一个文本文件就是一个Demarkdown文件。
