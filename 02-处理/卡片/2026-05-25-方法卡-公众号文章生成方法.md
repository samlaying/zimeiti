# 公众号文章生成方法

## 方法：AI生成并上传公众号文章
使用[[AI Skills 定义|AI Skills]]，将视频脚本自动转化为Markdown格式的公众号文章，并上传至草稿箱。

**实操步骤**：
1. 将视频脚本发送给AI，AI生成Markdown格式的公众号文章。
2. AI生成文章预览，你确认内容并提出修改意见。
3. 确认无误后，指示AI上传至公众号。
4. AI会自动完成上传，并设置你预设好的标题、封面图和文末签名。

**前置条件**：此方法需要在公众号后台完成配置（获取App ID、App Secret、设置IP白名单），具体教程需参考UP主相关视频。
