# Flex系统工作流

[[Flex系统]]的核心工作流程如下：
1.  **设置画像**：在JSON文件中填入个人的[[创作者画像]]（内容赛道、目标受众、语言风格等）。
2.  **智能选题**：`趋势情报官`（Perplexity）实时抓取热点并评分，`选题总监`结合画像从候选中选出最合适选题。
3.  **脚本生成**：`Claude`生成视频骨架并扩写为逻辑严密的口播稿；[[情绪导演]]（GPT）添加钩子、情绪起伏，去除书面语。
4.  **三方质检**：系统按预设标准自动评分（Cloud评分、Germ挑刺、GPT裁决），不合格稿自动打回重写，直至通过。
5.  **自动输出**：整个过程无需人工干预，从输入到最终输出不到三分钟。
