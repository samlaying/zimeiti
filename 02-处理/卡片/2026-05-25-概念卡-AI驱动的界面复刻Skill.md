# AI驱动的界面复刻Skill

这是一款运行在 **Cursor** 编辑器中的 AI Skill，由 **GPT 5.3 Codex** 模型驱动。

**核心功能**：
- 自动化复刻网站界面。
- 生成结构化的、可定制的**设计文档**。
- 支持在已有框架上通过自然语言或图片描述新增页面功能。

**开源与定制**：
- 该项目已在 GitHub 开源，鼓励社区协作优化。
- 用户可基于自己负责的产品，复刻工程文件并持续扩展功能。

它代表了 AI 辅助设计和快速原型开发的一种实现方式。
