# 设置Windows计划任务

将同步用的 JS 脚本集成到 Windows 计划任务中，可以实现定时自动同步，无需手动操作。

- **触发方式**：可以设置按固定时间间隔（如每小时）自动同步，或通过按键触发。
- **窗口设置**：定时任务执行时会弹出CMD窗口，可以在任务属性中设置为隐藏，避免干扰日常使用。
