# 代码联动

将 AI 生成的设计稿直接导出为可运行的 HTML、CSS 和 JavaScript 完整文件，实现设计到前端开发的无缝衔接。

**操作步骤**：
1. 将设计稿导出指令发送给 Pensel UI。
2. 工具将当前设计图导出为 HTML + CSS + JS 的完整文件包。

**应用场景**：前端开发人员可直接使用或基于此代码进行优化，减少手动编码时间。此方法是 [[Pensel UI 定义]] 中“导出可运行代码”能力的核心体现。
