# 五步工作流程

使用[[原型绘制Skill]]进行原型绘制的核心工作流程分为五步：

1.  **提供输入**：发送网页地址或截图。
2.  **复刻页面**：AI请求URL并截图，像素级复刻网站页面。
3.  **本地启动**：在本地启动复刻后的项目文件。
4.  **优化对话**：与AI对话优化工程文件，调整不符合要求的地方。
5.  **输出文档**：基于工程文件，新增功能或修改页面时，输出代码和设计文档。

这个流程通过 Cloud 的 Cowork 操作 Chrome 浏览器来实现。
