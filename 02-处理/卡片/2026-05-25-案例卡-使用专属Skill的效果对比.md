# 使用专属Skill的效果对比

通过实践对比，展示了应用[[专属AI Skill]]前后的效果差异：
- **未使用**：AI基于通用模板生成，结果在颜色、布局、交互上均可能偏离原有产品风格。
- **使用后**：AI会**优先套用它学习到的产品专属UI和交互风格**进行创作。最终生成的原型在整体布局、风格上与现有产品高度一致，实现了“像你家的产品”。

这个案例证明了该方法能有效解决[[AI原型的“形似神不似”]]的痛点。
