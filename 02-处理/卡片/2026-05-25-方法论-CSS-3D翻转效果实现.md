# CSS 3D翻转效果实现

> 最经典的CSS交互效果之一，仅需一行代码触发。

## 核心内容
实现卡片的3D翻转效果，核心是利用`:hover`伪类改变元素的`transform`属性。当鼠标悬停在卡片容器上时，对其内层面板应用`transform: rotateY(180deg)`（绕Y轴旋转180度），即可显示出预先用`backface-visibility: hidden`隐藏的背面。

## 原文关键段落
> .card:hover .card-inner {
>   transform: rotateY(180deg);
> }
