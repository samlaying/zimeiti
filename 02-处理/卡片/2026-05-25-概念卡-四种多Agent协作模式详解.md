# 四种多Agent协作模式详解

系统实现了四种协作模式：1. **简单路由**：将用户问题转交给一个Agent全权处理。2. **静态计划**：将问题预先拆分成计划表，依次安排不同Agent执行。3. **动态监督**：由管理者Agent判断任务是否执行完毕，灵活安排下一步，依赖大模型。4. **同伴交接**：任务下发给一个角色，然后由角色相互委派，摆脱中心化控制。> 来源：[[2026-05-25-052-学习多 Agent 协作，我做了一个系统 demo.md]]
