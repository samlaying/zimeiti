# 创建Skill的四种方法

从日常工作中提炼并创建[[AI Skill]]，主要有四种方法：

1.  **反对法**：从理想的**交付物**（如PRD、周报）入手，让AI学习该范本，产出格式、风格一致的内容。
2.  **流程法**：从一个完整的**工作流**（如播客处理）入手，先为每个子步骤创建独立Skill，运行顺畅后再合并为一个大Skill。
3.  **归纳法**：翻看与AI的**聊天记录**，找出重复性的指令（如“取一个吸引人的标题”），将其固化为Skill。
4.  **经验提炼法**：将个人的**隐性知识**（如写作风格、团队规范）显性化，建立Skill并持续迭代优化。
