# Figma MCP安装方法（Kursor/CC/Codex）

在不同MCP客户端中安装Figma MCP的步骤：
1.  **Kursor**：进入设置中的 `Marketplace`，搜索 `Figma` 并安装，然后验证身份。
2.  **Claude Code (CC)**：通过命令行安装官方Figma MCP，安装后需验证身份。
3.  **Codex**：先添加各个 `Skills`，然后在设置中添加 `MCP Server`，找到Figma服务器并安装认证。

安装完成后，即可使用包括[[Use Figma Tool：AI直接在画布上设计]]在内的所有工具。
