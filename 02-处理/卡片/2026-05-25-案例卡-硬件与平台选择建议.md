# 硬件与平台选择建议

针对不同角色，选择本地部署AI硬件与平台的策略（购买API服务永远是首选）。

| 角色 | 推荐硬件顺序 | 核心理由 |
| :--- | :--- | :--- |
| **普通程序员/提效用户**（预算约3万） | Mac (MLX) > 4090 48GB > GB10 > AMD | Mac本地跑大模型体验好；4090多功能性。 |
| **创业者/技术探索者** | GB10系列 > 4090 48GB > Mac | 必须在**CUDA平台**上验证[[Token Efficiency]]和杠杆效应，GB10是更好的学习平台。 |
| **有预算的老板/生产环境** | 必须使用多卡设备（如Pro 6000） | 前提是已用API跑通经济模型，明确Token转化价值的效率。 |

**软件平台**：对于创业者角色，应主攻**VLLM和SGLang**（对新模型支持好），使用Linux系统。
