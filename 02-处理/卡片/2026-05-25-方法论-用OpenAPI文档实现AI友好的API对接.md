# 用OpenAPI文档实现AI友好的API对接

> 让AI读懂你的后端接口，一键生成对接代码。

## 核心内容
OpenAPI（如openapi.json）是一种AI友好的API文档格式，它以结构化的JSON或YAML格式定义API的名称、版本、路径、请求和响应数据结构。提供这种格式的文档，能让AI编程工具更准确、高效地完成后端API的接入代码。

## 原文关键段落
> OpenAPI文档，AI友好的API文档格式，常见为openapi.yaml或openapi.json文件。内容：包括API名称、版本、访问路径、请求/响应数据结构。作用：方便Codex读取和实现后端API接入。
