# Skills管理的两大痛点

在使用AI Agent（如Cloud Code, Codex）时，本地管理下载的Skills存在两大核心痛点：
1.  **组织混乱**：成组的Skills下载后被平铺，失去了层级关系，数量增多后难以分辨归属。
2.  **资源占用**：本地Skills会占用与AI Agent对话的**上下文空间**，过多可能影响性能和对话质量。

这两个问题催生了对具备[[Skillflow的开关控制功能]]的整理工具的需求。
