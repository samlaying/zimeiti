# Skill四模块：说明书、资料、脚本、资源

一个标准的Skill包含四个核心目录结构：
1.  **Skill文件**（必须存在）：核心任务说明书，定义AI的基本功能、输入输出信息、执行逻辑。
2.  **Reference**（资料手册）：存放参考资料，如规则、流程、数据库，用于查询和确认信息。
3.  **Scripts**（工具脚本）：存放可执行脚本或工具，用于调用外部接口（如API）。
4.  **Assets**（参考文件）：存放非必要资源，如图片、示例文件，用于提供参考。

这种结构实现了配置、逻辑、数据和资源的分离。
