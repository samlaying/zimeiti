# HTML演示文稿可一键部署分享

[[PPT Skill]]的一个核心价值在于，它制作的HTML格式演示文稿**支持一键部署上线，生成长期可分享的链接**。
- **部署平台**：可直接在Skill内选择部署到[[Vercel]]或GitHub Pages。
- **分享优势**：生成的链接可直接访问，方便将演示成果分享给他人，突破了传统PPT文件传输的限制，实现了“在线演示”。
- **注意事项**：如果在插件模式下配额用完，建议在Claude Code的终端中进行部署操作，以获取完整能力。
