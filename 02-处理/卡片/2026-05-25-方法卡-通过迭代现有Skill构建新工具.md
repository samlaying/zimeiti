# 通过迭代现有Skill构建新工具

创建如 [[Project Analyzer Skill]] 这样的定制化工具，其核心方法是**组合与迭代现有的Skill**。

**构建逻辑示例**：
1.  **基础Skill一**：用于拆解开源项目提示词的Skill（如How Prompt）。
2.  **基础Skill二**：用于生成AI产品PRD的Skill。
3.  **内容框架**：PRD章节参考了权威课程中对Agent产品PRD内容的定义。
4.  **模板适配**：内置传统项目与AI产品两套PRD模板，根据分析对象自动调用。

最终，通过将两个功能不同的Skill组合，并融入新的需求框架，创造出能解决特定复杂场景（开源项目逆向分析）的新工具。
