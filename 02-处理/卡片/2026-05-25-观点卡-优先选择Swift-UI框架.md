# 优先选择Swift UI框架

在进行iOS应用UI原型开发时，应**优先选择Swift UI框架**。

**核心建议**：当AI工具（如[[ReCloud概念|ReCloud]]）询问技术选型时，应尽可能选择Swift UI，遵循“能用Swift UI的话绝对不用其他”的原则。这不仅是因为Swift UI的现代性和效率，也因为在此过程中，应用的UI基调和设计语言就此确定，后续应保持一致性，确保产品体验统一。
