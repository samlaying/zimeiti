# Figma MCP 旧工具区别：从导入到创造

Figma MCP 新旧工具的核心区别：
- **旧工具（Generate Figma Design）**：主要将网页HTML代码**转化**为可编辑的Figma图层，是“导入”模式。
- **新工具（Use Figma）**：允许AI**直接在画布上操作**，基于设计系统进行创作，是“创造”模式。

这标志着AI在设计工具中角色的根本性转变，新工具是[[Use Figma Tool：AI直接在画布上设计]]的核心。
