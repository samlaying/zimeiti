# 章节导出为Markdown

在[[Weave EPUB阅读器]]中，可以将当前阅读的章节**导出为Markdown文件**。

**功能特点**：
- **支持图片提取**和语法转换，生成的MD文件兼容Obsidian。
- 导出后的文件可以**发送给AI进行分析**，辅助深度阅读和理解。
- 也可以自行编辑，作为独立的阅读笔记保存。

此功能为从阅读内容到[[知识管理]]的转化提供了便捷途径。
