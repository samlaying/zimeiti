# AI Agent Harness 定义

**AI Agent Harness** 是**包围在AI模型周围的一切外部系统**，旨在为黑盒、不稳定的AI智能体提供一个**稳定、可控**的现实执行环境。

**核心类比**：
- **登山安全绳**：将登山者锚定在稳定处，防止偏离路线或坠落。
- **遛狗牵引绳**：控制宠物的行动范围，防止其乱跑。

其本质是将智能体“锚定”在一个可控的工程环境中，是实现**可靠性**的关键。
