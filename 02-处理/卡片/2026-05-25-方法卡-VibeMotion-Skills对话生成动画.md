# VibeMotion Skills对话生成动画

`VibeMotion Skills` 是 [[VibeMotion：AI编程制作精准动画|VibeMotion]] 的标准化组件，用于通过对话生成动画。

**核心特点：**
- **安装与使用**：安装后，可通过与 AI（如“[[龙虾]]”）的对话来生成视频。
- **低消耗**：**几乎不消耗 Token**。
- **开源愿景**：用户可以将自己创作的动画作为 “Skill” 上传，形成一个开源社区，让全球的 AI 都能复用这些动画效果。
