# Excalidraw Skill安装步骤

在本地[[Obsidian]]中安装并配置[[Excalidraw Skill]]的完整步骤：

1.  **前提准备**：确保已安装 Obsidian 并配置好[[Excalidraw插件]]。
2.  **获取文件**：从指定来源下载 Excalidraw Skill 的 JSON 格式文件。
3.  **导入目录**：将下载的 Skill JSON 文件直接拖入 Obsidian 的本地工作空间目录中。
4.  **配置路径**：在安装过程中，为 Skill 指定一个存放生成文件的目录。
5.  **设置触发词**：配置一个简洁的触发词（如“帮我生成讲解图”），以便快速调用 Skill 功能。
