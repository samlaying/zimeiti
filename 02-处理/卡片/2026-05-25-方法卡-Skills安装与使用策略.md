# Skills安装与使用策略

使用AI进行前端设计时，技能安装与使用有明确的策略：
1.  **切忌同时安装多个同类技能**。这会导致AI调用混乱，设计风格不稳定。建议**单场景安装一个最合适**的技能。
2.  **按场景选择技能**：
    - 做**官网**可选 `Frontend Design Pro` 或 `UI UX Pro Max`。
    - 做**后台管理系统**可选 `Web Design Guidelines` 或 Material Design 风格类技能。
    - 需要**暗色特效**可选 `Frontend Design Pro`。
3.  **提供详细提示词**：必须提供**约3000字的详细需求文档**，明确网站类型、功能、风格要求，避免模糊词汇（如“高端”），以防AI做出错误的风格联想。
