# 学会读代码验证技术可行性

当研发声称需求“实现不了”时，产品经理应掌握基本的代码验证方法，避免因信息不对称而被误导。

**关键步骤：**
1.  **前往代码仓库**：如研发使用的 Git 仓库。
2.  **搜索 API 入口**：通过关键词（如 “AzureController”）定位相关功能的代码模块。
3.  **浏览关键文件**：重点查看 Controller、Service 等文件中的方法名和注释，快速理解代码结构和功能概要，以此判断技术可行性。
