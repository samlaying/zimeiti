# MCP（Model Context Protocol）

**MCP（Model Context Protocol）** 是一套标准协议，允许 AI 助手（如在 IDE 中）连接到外部工具和服务，以获取特定数据或执行特定任务，从而扩展 AI 的能力边界。

它是实现 `Vibe Coding` 中[[Builder with MCP模式]]的关键。配置 MCP 是让 AI 能够操作 Figma、查询特定 API 等外部世界的桥梁。
