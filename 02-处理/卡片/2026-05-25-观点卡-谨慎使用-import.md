# 谨慎使用 import *

在[[导入模块的方法]]中，`from module_name import *` 这种导入全部内容的方式应**谨慎使用**。

因为它会将模块中所有的函数和变量导入到当前命名空间，极易引发**命名冲突**，降低代码的可读性和可维护性。明确的导入（如导入特定函数或使用别名）是更清晰、更安全的选择。
