# 要求代码模块化与生成README

> 模块化+README，让复杂项目可维护、可协作。

## 核心内容
为避免单一大文件带来的上下文管理和阅读修改困难，应要求AI将代码按功能拆分到不同文件中实现模块化，并生成一个README文件说明每个文件的用途。README如同“字典”，便于快速定位修改文件，且模块间通过明确输入输出接口交互，内部修改不影响其他部分。

## 原文关键段落
> 我们要求模型把代码尽可能的节奏开然后把不同功能的代码不同功能的代码拆分到不同的文件单中并在readme中说明每个文件的用处这样对于人也好对于模型也好整个readme就形成了一个直男
