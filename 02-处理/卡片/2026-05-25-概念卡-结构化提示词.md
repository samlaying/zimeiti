# 结构化提示词

在[[Vibe Coding]]中，为了获得更精准、定制化的AI生成结果，需要向思考型AI（如[[Kimi]]、ChatGPT）提供一份**结构化提示词**。它应包含：
1.  **用户身份与需求**（如“我是前端工程师兼UP主”）。
2.  **核心设计原则**（如“大留白、有高级感、动画有叙事目的”）。
3.  **视觉风格**（色彩系统、字体）。
4.  **页面结构与具体内容**。将生成的详细提示词交给[[Trae]]，可进行更专业的修改。
