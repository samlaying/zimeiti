# 自动化卡片布局

**自动化卡片布局**是[[Weave EPUB阅读器]]中将摘录自动组织成可视画布（Canvas）的方法。

**核心操作**：
1. 点击自动化按钮，**新建Canvas文件**或关联已有文档。
2. 在阅读器中选中文本，点击工具栏按钮，文本会自动添加为Canvas中的**新卡片节点**。
3. 可调整节点添加方向（**向下**或**向右**）。
4. **支持截图功能**：框选后自动添加为图片卡片。

所有卡片都支持[[源链接跳转]]，点击即可定位回原文。
