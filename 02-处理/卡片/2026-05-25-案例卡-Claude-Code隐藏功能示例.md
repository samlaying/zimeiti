# Claude Code隐藏功能示例

Claude Code 源代码中预埋了多种未公开的隐藏功能和彩蛋，部分已可测试，部分仅限内部使用：
- **实体贴纸**：通过命令启动，浏览器自动打开贴纸页面。
- **极简模式**：启动时添加参数，只保留三个基础工具。
- **Compute Use 电脑操控**：全自动电脑控制功能，已可演示。
- **伴侣精灵系统**：一个完整的收藏型宠物系统，包含 18 种宠物，尚未开放。
- **隐身模式/投机执行**：仅 Anthropic 内部员工可用。

这些功能展示了 Claude Code 作为“AI 助手操作系统”的潜力，当前公开版本只是冰山一角。
