# 故障排查：系统完全恢复

当操作后新旧机器人都无法回复，需要将 [[OpenClaw]] 系统恢复至初始状态的步骤：
1.  **停止网关**：停止OpenClaw的运行。
2.  **删除工作区**：在飞书中删除新建的机器人关联的工作区文件。
3.  **恢复配置**：执行恢复命令（如将备份文件重命名）或使用备份文件覆盖当前配置。
4.  **重启网关**：重新启动系统，即可恢复至最初状态，然后可重新操作。
