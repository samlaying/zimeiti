# API接口清单编制方法

> 接口清单能清晰定义接口状态，避免开发混淆。

## 核心内容
API接口清单基于系统流程图，列出接口提供方、调用方、名称和状态（已有、新增、已有修改），明确对接责任和开发工作量。

## 原文关键段落
> API接口清单基于系统流程图，明确列出系统间需要对接的接口...状态：已有、新增、已有修改。
