# 文章转笔记流程

使用 [[Article to Note]] [[Claude Code Skill]] 将文章链接转化为笔记的标准流程：

1.  **输入链接**：向 Skill 提供目标文章的 URL（例如微信公众号链接）。
2.  **自动抓取**：Skill 调用 [[DeFado]] 等工具抓取网页正文，并提取标题、作者等元数据。
3.  **分析与生成**：AI 分析文章主题，依据预设模板（包含属性区与正文结构）生成结构化笔记。
4.  **输出与预览**：笔记生成后自动在 Obsidian 中打开，供用户查看和编辑。

整个过程高度自动化，仅需几秒钟即可完成。
