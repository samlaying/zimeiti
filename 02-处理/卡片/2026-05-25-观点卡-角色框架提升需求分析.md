# 角色框架提升需求分析

**Product Manager Skills** 的本质是一套[[Product Manager Skills|角色框架]]，其核心价值在于**显著提升需求分析场景的结构化与专业性**。

它不仅是产品经理的工具，更是一套可复用的方法论集合。运营人员可用它做用户洞察，创业者可用它梳理产品定位，项目经理可用它管理需求池。这说明，通过为AI注入特定领域的专业框架，可以使其跨越角色限制，成为各类决策者的结构化分析搭档，从而提升决策效率与专业度。
