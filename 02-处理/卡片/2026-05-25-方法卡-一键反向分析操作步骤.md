# 一键反向分析操作步骤

使用[[反向分析插件定义]]的通用步骤如下：
1. **安装插件**：在浏览器（如Chrome）应用商店搜索并安装插件。
2. **定位目标**：在网页中浏览，找到想要分析的视频或图片。
3. **一键分析**：激活插件（通过点击工具栏图标或右键菜单），选择“反向分析”功能。
4. **获取结果**：插件自动运行分析算法，几秒内生成结构化的AI提示词。
5. **复制使用**：将提示词复制到AI绘画或视频生成工具（如[[Midjourney]]、[[Stable Diffusion]]）中进行创作或微调。
