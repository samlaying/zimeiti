# Vue路由生命周期钩子控制

在Vue SPA中控制转场动画的关键生命周期与API：
- **动画退出**：无法使用`window.onload`。需在根组件的`mounted`钩子中，用`setInterval`轮询`document.readyState`，当其为`“complete”`时，调用动画组件的退出方法。
- **动画进入与路由跳转**：利用Vue Router的全局前置守卫`beforeEach`。在守卫中调用动画组件的进入方法，并将`next`函数作为参数传入。在进入方法内，播放入场动画后，延迟执行`next()`以完成路由跳转，实现“动画播完再跳转”的效果。
