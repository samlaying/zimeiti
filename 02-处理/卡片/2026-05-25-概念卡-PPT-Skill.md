# PPT Skill

一种集成在[[Claude Code]]中的AI能力插件（Skill），专门用于通过对话交互的方式生成具有高级设计感、丰富动效的HTML格式演示文稿。
- **来源**：小红书博主张杂拉（Zara）发布于GitHub，项目获星标超1.4万。
- **核心优势**：
  1. 设计风格简洁，有高级感。
  2. 交互与动效酷炫丰富。
  3. 生成HTML格式，体验新颖，区别于传统PPT。
  4. 支持将传统PPT格式转换为HTML格式。
  5. 生成的HTML文件支持在线编辑修改。
  6. 可一键部署至[[Vercel]]或GitHub Pages，生成长期可分享链接。
