# 五步完整工作流

全真模拟人类专家PPT设计流程的五步法：
1. **需求确认**：AI搜集资料后主动向用户提问，确保深度需求“聊透”。
2. **大纲规划（便利贴法）**：将每页核心内容做成“数字便利贴”，可视化地调整逻辑结构。
3. **资料检索**：将大纲标题逐个交给搜索能力强的工具（如[[Grok]]）进行信息搜集整理。
4. **策划出稿**：借鉴[[策划师流程]]，AI先完成单页内容与版式的“策划”，再交由“设计”完成视觉风格。
5. **设计执行（卡片式布局）**：使用精确的提示词，指导AI采用[[卡片式布局]]生成SVG代码。
