# 是参考实现，不是产品

OpenAI 明确表示，**Symphony 不是产品**，他们不会将其作为产品来维护。它的定位是：
- 一个**开源参考实现**。
- 旨在展示 **Codex App Server 接入 workflow 工具**的潜力。

该参考实现基于 Elixir 语言编写，并提供了完善的文档以帮助用户快速上手理解其工作方式。
