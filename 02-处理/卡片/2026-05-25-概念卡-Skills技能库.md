# Skills技能库

Skills是Fone Cloud或Cloud Code中的一套可定制的“技能库”，其核心目的是**教AI理解你的特定业务逻辑和需求模式**。

用户可以通过预设来告诉AI，例如“我是一名AI矛盾玩具卖家”，并设定其拓展方向（如“还能做什么产品”）。一个关键能力是，Skills中可以**设定AI必须主动询问用户进行信息补充**，以交互方式完善调研所需细节。

这实质上是将个人或团队的业务Know-How沉淀为一个AI可理解、可调用的标准化工作流，从而让后续的AI任务执行更精准、高效。
