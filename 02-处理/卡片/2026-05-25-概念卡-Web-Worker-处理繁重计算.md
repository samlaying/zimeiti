# Web Worker 处理繁重计算

**Web Worker** 是浏览器提供的一个多线程解决方案，用于在后台线程执行繁重的计算任务（如视频压缩、转码），**完全不阻塞主线程的UI响应**，从而确保用户体验的流畅性。它运行在独立的CPU核心上，与主线程只能通过消息传递进行通信。
