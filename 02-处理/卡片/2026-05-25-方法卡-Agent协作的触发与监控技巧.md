# Agent协作的触发与监控技巧

在飞书群中使用[[飞书Agent协作插件]]时，掌握以下技巧能优化协作流程：
1.  **触发协作**：在群聊中提及关键词如“群内协作”、“一起做一下”，插件会自动发现并启动协作。
2.  **监控角色**：可以指定一个Agent（如产品）作为监控者，设置定时任务（如每5分钟）检查其他Agent状态，避免任务卡死。
3.  **手动干预**：当Agent未及时回复时，可手动艾特并引用回复特定Agent，提供指导或纠正方向，实现实时监控。
4.  **流程设计**：可选择主动参与（避免跑偏）或完全自动化（让Agent自主推进，通过监控确保完成）。
