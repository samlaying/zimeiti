# Pensel UI

Pensel UI 是一个基于AI的UI设计工具，能通过简单的文字指令（如“咒语”）快速完成UI界面生成、[[风格一键克隆|风格克隆]]、[[组件批量生成|组件批量创建]]、辅助设计和[[代码联动|代码导出]]等功能，大幅提升设计效率。
