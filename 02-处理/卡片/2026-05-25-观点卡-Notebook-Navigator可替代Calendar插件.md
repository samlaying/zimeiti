# Notebook Navigator可替代Calendar插件

[[Notebook Navigator插件]] 内置了显示日历的功能，点击后可以集成 Daily Note 功能。这一特性使其能够完全替代 [[Obsidian]] 专门的 Calendar 插件，实现了文件管理与日期导航的一体化。
