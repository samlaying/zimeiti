# Agent Harness的典型模块

> 工具、模型、守卫栏、循环、验证——Agent Harness的五大标准模块。

## 核心内容
一个典型的Agent Harness通常由五个核心模块构成：1. 工具注册表（智能体可调用的工具集合）；2. 模型（Harness调用的核心AI模型）；3. 守卫栏（预设的规则限制，如最大迭代次数）；4. 智能体循环（驱动任务逐步执行的核心逻辑）；5. 验证步骤（任务完成后进行的确定性检查，如运行测试）。
