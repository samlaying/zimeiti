# 伊朗局势多维简报

[[总结一下鸭小程序]]中的“伊朗局势简报”模块，是处理复杂时政信息的一个优秀案例。它通过多维度、结构化地梳理信息，帮助用户快速建立全局认知。

**简报核心构成：**
1.  **投资者增量信息**：用“升级/新增”标签标注局势变化要点（升温/降温）。
2.  **关键博弈板块**：总结近期热门文章，信息模块化，可溯源。
3.  **态势总结**：分别梳理伊朗、美国及世界各方态度。
4.  **预测市场数据**：整合Polymarket等平台数据。
5.  **社媒动态**：分类整理关键人物发言、社交热点，提供原文与翻译。
6.  **事件跟踪与自信度排序**：跟踪多信息源，**为每条信息标注确定性等级**，帮助筛选高可信度信息。
