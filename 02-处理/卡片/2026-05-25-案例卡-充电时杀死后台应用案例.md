# 充电时杀死后台应用案例

案例：在 [[MacroDroid 工具简介]] 中设置自动化，当手机插入充电器时，自动杀死所有后台应用，实现安静充电。

具体设置：
- 触发器：设备插入充电器（充电状态）。
- 动作：杀死所有后台应用。
- 约束条件：可设置为日落之后，系统需定位授权。
- 测试：插入充电器，观察动作执行。

扩展应用：类似思路可用于短信处理（读取短信内容后通过 HTTP 请求发送或邮件转发）、UI 交互抢票等场景。
