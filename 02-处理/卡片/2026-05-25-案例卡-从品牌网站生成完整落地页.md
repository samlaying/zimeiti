# 从品牌网站生成完整落地页

**目标**：基于多米诺骨牌风格网站，创建“小鸟嗷桐理发店”的Landing Page并生成可运行代码。
**模型**：GPT-5.4 (Medium 推理模式)。
**关键步骤**：
1.  **提取设计系统**：提示AI在Figma中基于参考网站创建完整的设计系统和主件库。
2.  **生成高保真设计**：使用[[Use Figma Tool：AI直接在画布上设计]]生成单页Figma设计并根据反馈优化。
3.  **生成代码**：将设计链接提供给新Agent，调用 `get design context` 编写React代码。
4.  **优化资源**：AI调用图片生成工具替换远程资源，并添加动效。
**结果**：最终生成了一个完整、可运行、风格鲜明的理发店网页，体现了[[AI降低设计门槛]]和流程提效。
