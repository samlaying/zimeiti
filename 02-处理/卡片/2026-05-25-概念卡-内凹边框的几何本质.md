# 内凹边框的几何本质

CSS 中创建**内凹（凹角）** 边框的核心几何心法是 **“从盒子（矩形）中减去一个位于角落外部的圆形”（box - circle）**。

- **凸角（convex）**：`border-radius` 创建向外的圆角，曲线远离盒子生长。
- **凹角（concave）**：我们需要相同的曲线向内凹陷，即曲线向盒子内部生长。
- **CSS 限制**：没有直接的 `inset` 或 `concave` 属性来实现此效果。
- **通用方法**：所有实现技术都是在用不同的方式（阴影、遮罩、裁剪）执行这个“盒子减圆形”的减法操作。

理解这个基础几何模型，是掌握所有后续[[CSS内凹边框实现技术]]的关键。
