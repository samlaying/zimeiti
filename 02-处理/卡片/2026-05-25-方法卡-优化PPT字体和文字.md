# 优化PPT字体和文字

使用[[AI PPT设计工具组合]]优化字体和文字：
1. **字体分析**：选择喜欢的字体风格，用豆包分析其特征生成提示词。
2. **文字生成**：将提示词放入吉梦生成类似风格的文字图片。
3. **转换为可编辑文件**：使用在线工具将图片转为SVG文件，导入PPT后点击“转化为形状”，即可自由编辑（如换颜色、加渐变）。
4. **重要性**：合适的字体能显著提升PPT的专业感和视觉质量。
