# Claude Code Skill

**Claude Code Skill** 是用户在 Claude Code 平台上创建和运行的自定义技能。

它允许用户通过定义提示词或代码，封装特定的任务流程，实现自动化的复杂操作。

在演示中，它被用来自动化处理信息，将外部内容（如网页文章、视频）转化为结构化的 [[Obsidian 笔记]]，是实现 AI 提效的载体。
