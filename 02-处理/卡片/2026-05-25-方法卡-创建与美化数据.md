# 创建与美化数据

使用 [[CPAHelper Excel Agent]] 自动生成并美化表格数据的基本流程：
1.  **选择区域**：在 Excel 中选中一个空白区域（如 A1:D10）。
2.  **输入提示**：向 Agent 发出自然语言指令，如“模拟一组销售数据，填写在当前的区域”。
3.  **数据生成**：Agent 会读取区域信息并自动生成一组示例数据填充其中。
4.  **优化格式**：可进一步下达指令优化表格，例如：“日期列的格式有问题，加个好看点的样式，以及一个合计行（用公式）”。Agent 会自动调整格式、添加边框、设置对齐，并插入带公式的合计行。

这个过程展示了 Agent 理解自然语言意图并调用内置工具（如读取、写入、设置格式）完成任务的自动化能力。
