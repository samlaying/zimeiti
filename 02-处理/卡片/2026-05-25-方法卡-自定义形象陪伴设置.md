# 自定义形象陪伴设置

为Obsidian添加自定义形象陪伴的完整步骤：
1. 使用抠图软件或相册自带功能，将目标形象（如宠物、偶像照片）抠图。
2. 将抠好的图片上传至图床，获取可访问的图片链接。
3. 在插件设置中，输入图片链接，并执行“+P”命令重新加载。
4. 重启Obsidian软件，形象即出现在工作区陪伴你记笔记。

此方法旨在为笔记环境注入个性和情感，提升[[情绪价值也是价值|情绪价值]]。
