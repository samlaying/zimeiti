# 精简参考图案例

案例：使用精简参考图（仅包含侧边栏截图）生成 Dashboard 页面。
- **过程**：右侧内容留白，让 AI 自由发挥。
- **结果**：生成多张 UI 设计，设计感强，排版精致，配色舒服。例如某版被评价为“排版紧凑又不过密，配色舒服”，带来全新灵感。
- **启示**：跳出现有设计框架后，AI 能产出更惊艳的设计，帮助突破创作瓶颈。
