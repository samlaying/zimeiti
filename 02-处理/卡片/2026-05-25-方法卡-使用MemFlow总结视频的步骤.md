# 使用MemFlow总结视频的步骤

使用 [[MemFlow一键总结功能]] 的完整流程分为以下四步：
1.  **开启字幕**：在 B 站视频播放页面，手动确保视频的字幕功能已开启。
2.  **触发导出**：点击浏览器右上角的 MemFlow 智能导出按钮。
3.  **等待解析**：插件会自动处理并显示解析进度，通常等待几秒钟。
4.  **完成保存**：解析完成后，生成的 Markdown 格式笔记会自动保存，无需额外操作。
