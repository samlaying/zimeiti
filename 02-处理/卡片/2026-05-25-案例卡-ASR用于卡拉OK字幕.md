# ASR用于卡拉OK字幕

利用[[ASR]]（自动语音识别）技术制作卡拉OK式字幕（字随声动、逐字高亮）是一个典型的增值应用案例。其关键在于不仅需要识别语音，还需要精确到每个字的**毫秒级起止时间**。

实现该效果依赖于**千问三ASR配套的Forced Aligner（强制对齐模型）**。该模型能输出精确的时间轴，再结合ASR生成的字幕文本，即可集成到剪辑软件或自定义系统中，生成样式丰富的动态字幕。这展示了如何通过组合多个专用模型（ASR + 强制对齐）实现超越基础功能的效果。
