# Skill使用流程

使用 [[自定义Skill排版工具]] 的步骤如下：
1. **准备模板**：提供一个预先设置好样式的 Word 文档作为排版模板。
2. **输入文件**：准备需要转换的 Markdown 原文件。
3. **执行指令**：将原文件、模板和 Skill 指令输入 AI 工具（如 GPT-5.5），一键输出排版后的 Word 文档。此过程依赖模板样式，确保排版精准。
