# PARA笔记管理法

PARA 是一种由 Tiago Forte 提出的笔记管理方法，旨在按**行动导向**而非主题来组织笔记，解决笔记“存放”问题。

它包含四个桶：
1.  **[[Projects项目]]**：有明确目标和截止日期的工作。
2.  **[[Areas领域]]**：你持续负责、无截止日期的领域。
3.  **[[Resources资源]]**：感兴趣或未来可能用到的参考资料。
4.  **[[Archive归档]]**：不再活跃但需有序保存的历史内容。
