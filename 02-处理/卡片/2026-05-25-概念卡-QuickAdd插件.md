# QuickAdd插件

**QuickAdd** 是 Obsidian 中一个强大的自动化插件，常被称为“Obsidian 最难插件”（因其掌握门槛较高）。它像一个“**胶水插件**”，核心功能是将 Obsidian 的原生命令、Templater、DataView 插件，乃至外部 API 和大语言模型等集成起来，用以构建高度自动化和智能化的知识管理系统。

其主要功能分为三部分：[[模板功能]]、[[捕获功能]] 和 [[宏功能]]。还有一个“多选”功能，用于分类管理这些命令。
