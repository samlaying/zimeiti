# 用户隔离是Agent商业化的关键

要让Agent服务被商业公司接受并面向普通用户，**用户隔离的沙箱系统**是关键的安全与隐私保障。

### 为何关键
- **降低门槛**：用户无需自备24小时在线设备或安装复杂环境，服务端封装后，用户只需客户端。
- **安全隔离**：每个用户拥有独立环境（存储、任务、会话），进程级隔离（如NSJ）控制权限，用户无法访问他人内容。
- **数据一致性**：通过`user lock`（写入锁）机制，防止同一用户的多Session并发读写同一文件导致冲突。
