# 导入模块的方法

在Python中导入和使用[[模块]]有多种灵活方式：

1.  **整体导入**：`import module_name`，使用时需加模块名前缀，如 `module_name.function()`。
2.  **导入特定成员**：`from module_name import function_name`，可直接使用 `function_name()`，无需前缀。
3.  **别名导入**：`import module_name as alias`，为模块设置简称，之后使用 `alias.function()`。
4.  **导入全部内容**：`from module_name import *`，谨慎使用，易引起命名冲突。
