# 从 OpenClaw 无缝迁移至 Hermes Agent

**Hermes Agent** 支持将 **OpenClaw** 中积累的用户数据一键迁移，无需复杂配置。

**迁移内容**：包括记忆 (Memory)、技能 (Skill)、MCP 等所有核心数据。

**操作步骤**：
1. 安装 Hermes Agent 时，安装向导会提示是否从 OpenClaw 迁移数据。
2. 用户只需选择 **“迁移”** 选项。
3. 系统会自动从 OpenClaw 中提取所有相关数据并导入到 Hermes Agent 中。

迁移完成后，Hermes Agent 能立即识别用户信息（如 GitHub 账号、工作偏好、编程规则等），实现平滑过渡。
