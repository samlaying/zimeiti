# CodeWork与Code模式的区别与测试

> 两种模式，两种测试方法，确保配置无误。

## 核心内容
ClaudeCode桌面端有两种核心模式。CodeWork模式可直接对话测试。Code模式需要先在客户端内打开一个代码项目文件夹，再进行对话，以验证其在具体项目环境下的代码生成与理解能力。配置后应分别测试这两种模式。

## 原文关键段落
> 首先测试一下这个cork...随便发送一个消息它来回复就是成功了...然后这个地方可以切换模型这个测试环节的话就是code模式也是打开一个文件夹然后发送消息能收到回复就说明成功了。
