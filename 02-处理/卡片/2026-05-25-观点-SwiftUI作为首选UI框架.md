# SwiftUI作为首选UI框架

> iOS开发UI的第一选择：能用SwiftUI就别用别的。

## 核心内容
在确定iOS应用的UI开发框架时，应尽可能选择SwiftUI。UP主给出了一个非常明确的建议：如果能用SwiftUI，就绝对不要使用其他框架。这为后续的开发一致性（UI基调）和效率奠定了基础。
