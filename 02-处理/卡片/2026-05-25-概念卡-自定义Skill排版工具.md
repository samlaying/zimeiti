# 自定义Skill排版工具

一个自定义的 [[Skill]]，用于将 Markdown 文件转换为 Word (Docx) 文档，并能直接按照指定的 Word 模板进行出版级排版，而不仅仅是格式转换。其核心能力包括处理多级标题样式、代码块与分段、缩进、列表、表格、页眉页脚及页码，还能将 Markdown 中的 Mermaid 图表渲染为图片嵌入文档。
