# 构建响应式导航栏的完整步骤

构建一个带有[[汉堡菜单动画]]的[[响应式导航栏]]，主要分为四个步骤：

### 1. 环境准备与基础设置
- 创建项目文件夹和基础HTML、CSS文件。
- 进行CSS基础重置（`padding: 0; margin: 0; box-sizing: border-box;`）。
- 引入字体（如Google Fonts的Poppins）。

### 2. HTML结构搭建
- 使用 `<nav>` 作为主导航容器。
- 用 `<ul class="nav-links">` 和 `<li>` 包裹导航链接。
- 创建 `.burger` 容器，内含三个 `<div>` 模拟汉堡菜单线条。

### 3. CSS样式实现
- **桌面端**：使用 `flex` 布局（`justify-content: space-around`）排列链接。
- **移动端核心样式**：
  - 将 `.nav-links` 定位为 `absolute`，并用 `transform: translateX(100%)` 移出屏幕隐藏。
  - 设置链接初始 `opacity: 0`。
  - 显示 `.burger` 容器并设置线条样式。
  - 定义 `@keyframes navLinkFade` 动画（淡入+滑入）。
  - 创建 `.nav-active` 类（`transform: translateX(0%)`）和汉堡菜单动画的 `.toggle` 类样式。

### 4. JavaScript交互逻辑
- 获取汉堡按钮、导航容器和所有链接元素。
- 监听汉堡按钮点击事件：
  - 切换 `.nav-active` 类名以显示/隐藏导航。
  - 遍历链接，利用列表项索引（index）动态设置不同的动画延迟（如 `index / 7` 秒），实现[[索引驱动的动画延迟优势]]。
  - 切换 `.toggle` 类名以实现汉堡菜单动画。
