# 实体模糊化

## 实体模糊化（Entity Obfuscation）
在 Search Agent 数据合成中，**实体模糊化**是将一个明确的实体名称替换为更模糊、更泛化描述的技术。例如，将“埃隆·马斯克”模糊化为“一位知名企业家”。

### 核心作用
它是驱动模型生成**多跳推理**问题的**关键步骤**。通过对初始问题中的实体进行模糊化，迫使后续的推理必须经历多个搜索步骤才能定位到最终答案，从而将简单的事实性问题转化为需要复杂检索链路的问题。

### 在数据合成中的位置
在 [[Search Agent 数据合成流程]] 中，它发生在“初始问题生成”之后、“复杂问题生成”之前。
