# 使用Gemini生成UI设计提示词

### 方法：用Gemini为UI设计生成提示词

1.  **准备输入**：将完整的 [[产品需求文档]] 内容复制下来。
2.  **交互指令**：将文档内容发送给 [[Gemini]]，并添加明确指令：“帮我生成UI界面设计的提示词，我用Figma Make来制作UI”。
3.  **获取结果**：Gemini会为文档中的每个页面（如首页、列表页）分别生成详细的UI设计提示词。
4.  **备用**：将生成的提示词保存到文档中，供下一步使用。
