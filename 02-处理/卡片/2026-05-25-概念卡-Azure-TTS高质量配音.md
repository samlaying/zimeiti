# Azure TTS高质量配音

Video Podcast Maker 使用的文本转语音服务，提供高质量的配音。

**核心特性：**
- **支持中英文混读**：脚本中的中文和英文可以随意混排，系统会自动识别并切换到对应的语音引擎进行朗读，实现无缝衔接。
- **发音校正**：如果遇到读错的字，可以用拼音标注进行手动调教（见 [[多音字智能校正]]），确保发音准确。
