# 公文格式自动调整方法

通过一个[[Cloud Code Skill]]实现公文格式的自动化调整，具体步骤如下：
1.  **获取项目**：在GitHub搜索“official.com”并下载对应项目。
2.  **环境准备**：在VS Code中打开解压后的项目文件夹，并确保已安装Cloud Code插件。
3.  **执行指令**：将杂乱公文的文件路径复制，在Cloud Code中输入“请调用公文格式要求”并确认，等待Skill自动处理完成。
4.  **获取结果**：处理完成后，规范格式的公文会自动生成并保存。
