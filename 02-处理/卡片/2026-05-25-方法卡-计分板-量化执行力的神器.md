# 计分板：量化执行力的神器

一种用于客观追踪[[关键行动]]执行情况、避免自我感觉良好的工具。

## 目的
- **客观提醒**：每周提醒自己该完成的核心任务。
- **量化衡量**：用完成率（如21%、0%）替代模糊的感觉，清晰了解执行力水平。
- **避免高估**：人容易高估自己的付出，计分板用数据说话，提供真实反馈。

## 使用方法
1.  每周初列出所有待办的[[关键行动]]（数量可能随阶段变化）。
2.  每周末核对，记录已完成的关键行动数量。
3.  计算完成率（完成数/总数）。
4.  允许波动（如因旅行、过年完成率低），核心是持续记录与反思，而非追求100%。
