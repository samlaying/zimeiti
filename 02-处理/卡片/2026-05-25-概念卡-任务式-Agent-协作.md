# 任务式 Agent 协作

一种区别于直接对话的人机协作模式。在此模式下，用户的核心动作是**定义、发布与验收任务**，而无需实时监控或介入 Agent 的具体执行过程。它依赖于像 [[Symphony 任务管理框架]] 这样的工具，将协作过程结构化为一系列可追踪的任务状态。
