# 组件批量生成

用一句话描述需求，自动生成一批 UI 组件（如按钮、卡片），并可复用以快速生成新页面，极大提升 UI 库建设效率。

**操作步骤**：
1. 将描述组件需求的指令发送给 Pensel UI。
2. 工具自动批量生成对应的 UI 组件。
3. 生成的组件可在页面中直接调用，快速生成新页面。
4. 若生成结果有错位，可在工具内直接进行微调。

此方法是 [[Pensel UI 定义]] 中“批量创建 UI 组件”能力的具体实践。
