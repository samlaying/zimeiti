# 社区UI组件库助力AI绘图

目前AI通过脚本生成UI原型的效果尚不理想，一个关键的优化方向是社区协作。

**核心观点**：社区需要形成一套标准的**UI组件库**供AI学习。只有当AI“知道怎么画”这些标准化组件时，才能生成质量更好的设计作品。

**现状**：社区对[[Affinity脚本管理器]]中脚本功能的探索仍处于**早期阶段**，组件库的建立是提升效果的关键。
