# Edit参数：剪辑监听源

`Edit`参数控制剪辑的监听源，决定工具根据什么进行自动剪辑。
- **Audio模式**：适用于以说话为主的视频（如Vlog、播客），自动剪除无声音段落。
- **Motion模式**：适用于录屏内容（如游戏、软件教程），自动剪除画面无变化的片段。
