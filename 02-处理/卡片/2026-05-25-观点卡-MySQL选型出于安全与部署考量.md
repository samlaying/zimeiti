# MySQL选型出于安全与部署考量

在选择技术方案时，**业务需求和安全考量应优先于开发便利性**。

以本项目为例，虽然微信云开发更便捷，但因后端需部署到公司服务器，且涉及敏感的客户信息，最终选择了**MySQL**数据库。这一决策平衡了部署环境要求和数据安全需求，是更稳妥的选择。
