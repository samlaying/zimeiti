# 如何用brainstorm skill与AI沟通

在使用 [[AI编程助手]] 执行创造性任务（如生成PPT）前，可以通过一个名为 **brainstorm skill（头脑风暴技能）** 的功能，与AI进行高效的结构化沟通，确保需求明确。

**具体步骤与技巧：**
1.  **启动技能**：在与AI的对话开始时，明确要求激活 brainstorm skill。
2.  **提供初始需求**：清晰地描述你的核心目标（例如：“生成一个40分钟的博士开题PPT，包含两个已完成工作和一个进行中idea”）。
3.  **结构化回答**：AI会主动提问以细化需求。**关键技巧是避免开放性问题**，尽量提供多个选项（如A、B、C）让AI选择，例如：
    - 观众是谁？（选项：A. 评审委员会 B. 同学...）
    - PPT核心主题？（选项：A. 介绍我的研究 B. 申请资助...）
    - 时间如何分配？（选项：A. 侧重已完成工作 B. 三者均衡...）
    - 希望什么格式？（选项：A. Microsoft PPT B. HTML...）
4.  **确认与保存**：完成所有问答后，AI会生成一份设计计划。**务必使用“save”等命令保存此计划**，防止对话中断丢失上下文。

这种方法能大幅减少后续修改，提升 [[Vibe Coding]] 的整体效率。
