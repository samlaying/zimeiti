# Flexbox实现水平导航布局

> Flex布局的`space-around`是均匀分布子元素的利器。

## 核心内容
给导航容器（如`<nav>`）设置`display: flex`，并使用`justify-content: space-around`（或`space-between`）属性，可以让内部的导航链接项在水平方向上均匀分布。再配合`align-items: center`可以确保所有项在垂直方向上居中对齐，这是构建桌面端水平导航栏的经典方法。

## 原文关键段落
> nav { 
>     display: flex; 
>     justify-content: space-around; 
>     align-items: center; 
> }
