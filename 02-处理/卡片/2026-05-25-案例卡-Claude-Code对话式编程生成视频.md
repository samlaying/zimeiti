# Claude Code对话式编程生成视频

实操案例展示了如何使用[[Claude Code]]，结合自定义的[[标准化Skills体系]]进行视频开发：

1.  **输入**：将定义好的Skills文件和[[Remotion]]官方最佳实践提供给Claude。
2.  **对话**：向AI提出简短的视频需求（如“帮我生成产品介绍视频”）。
3.  **生成**：AI自动分析需求，告知将使用哪些分段和现成组件进行开发，并快速生成完整的Remotion项目代码。
4.  **结果**：生成的产品风格统一，开发过程高效。
