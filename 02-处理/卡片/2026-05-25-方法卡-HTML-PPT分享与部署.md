# HTML PPT分享与部署

完成HTML PPT制作后，有两种主要的分享方式：

**1. 文件方式分享**
- **无外部素材**：若PPT未使用外部图片/视频，直接分享单个`.html`文件即可，接收方用浏览器打开。
- **有外部素材**：若使用了图片、视频等，必须将**整个项目文件夹**打包（如压缩为.zip）发送。接收方解压后，打开其中的HTML文件。

**2. 在线链接分享（推荐）**
- 推荐平台：**Netlify**。免费，部署后生成的链接在国内可访问。
- 部署步骤：注册Netlify账号，选择“Upload a folder”，上传打包好的整个项目文件夹。部署完成后，为项目设置一个易记的网址名称，即可生成可直接分享的链接。

在线链接方式避免了文件传递的麻烦，便于跨设备访问，是[[AI辅助HTML PPT核心工作流]]的最终交付环节。
