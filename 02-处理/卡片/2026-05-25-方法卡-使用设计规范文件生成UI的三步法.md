# 使用设计规范文件生成UI的三步法

一套使用 [[AwesomeDesign-MD仓库]] 中的规范文件，指导 AI 生成高质量 UI 的标准操作流程：

1.  **选择规范文件**：访问 AwesomeDesign-MD 仓库，浏览并选择你喜欢的网站设计规范（如 Visa 或 Cloud 的 `Design.MD` 文件），复制该文件内容。
2.  **集成到项目**：将复制的 `Design.MD` 文件放置到你的项目根目录中，作为 AI 的上下文参考。
3.  **指导AI生成**：打开 AI 编码工具，输入明确的提示词，例如：“**参考项目里的 Design.MD，给我做一个像 XXX 风格的页面**”。AI 将根据文件中的设计 Token 自动生成风格一致的界面代码。
