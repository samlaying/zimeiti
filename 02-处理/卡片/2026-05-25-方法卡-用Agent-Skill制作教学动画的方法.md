# 用Agent Skill制作教学动画的方法

利用专注于视频生成的 [[Agent Skill：AI智能体的可执行能力包|Agent Skill]]（如Hagent公司的Hyper Frames）来制作教学动画，其流程和产出与传统视频不同，具有独特优势。

**核心流程与产出：**
1.  **工具选择**：使用一系列专注于视频生成的Skills（如Hyper Frames）。
2.  **产出形式**：产出并非传统视频文件，而是**HTML（用于动画）加上音频（可克隆教师声音）** 的组合。
3.  **应用**：将此Skill应用于具体教学场景，例如用[[泰坦尼克号DNN教学动画案例|深度神经网络分类泰坦尼克数据集]]的案例来生成讲解动画。

**关键优势（对比传统方式）：**
- [[Agent Skill让教学动画修改极其简便|修改极其简便]]。
- [[Agent Skill让教学动画修改极其简便|高度个性化]]，能结合具体数据集讲解。
