# AI深度访谈法生成需求

通过让AI采访用户，系统化挖掘并生成详尽需求文档。

## 问题
直接让AI写代码，结果往往与预期不符，陷入反复修改的循环。

## 操作步骤
1.  向AI发出指令，明确要求它使用 `ask user question` 工具进行提问。例如：“我想开发一个家庭点菜网站，请使用ask user question工具对我进行深度访谈。”
2.  根据AI提出的问题进行选择或补充回答。AI会主动引导并补全你未考虑到的需求点。
3.  访谈结束后，AI会自动生成一份详尽的需求文档（如 `Size.md`）。

## 进阶
如果需要修改方案（如更换数据库），可以直接在文档中手动修改，或让AI协助修改。
