# Skill完整工作流程

以Cloud Code为例，Skill从安装到执行的完整流程：

1.  **安装**：将Skill文件夹放入AI客户端的指定目录。
2.  **触发**：用户在聊天框输入指令（如“帮我写个简历PDF”）。
3.  **识别**：AI客户端加载所有本地Skill文件夹中的 [[Metadata（元数据）]]，发送给大模型。
4.  **路由**：大模型根据元数据判断应使用哪个 [[Skill（智能体技能）]]，并返回指令。
5.  **加载**：AI客户端将目标Skill文件夹的**主提示词文件**加载为系统提示词，发送给大模型。
6.  **执行**：大模型根据提示词指示，让AI客户端按需**读取参考文件**或**执行脚本**（如生成PDF）。
7.  **返回**：大模型基于所有信息生成最终结果，返回给用户。
