# Claude Code 封号机制分析

Claude Code 存在**基于多种标识的用户追踪机制**，用于可能的封号，但**不直接收集硬件指纹**（如 MAC 地址、CPU 型号）。

**追踪标识类型**包括：
- 永久保存的随机 `User ID`（跨会话追踪）
- 永久存储的 `Anonymous ID`（备用追踪）
- 绑定账号的 `Account ID`
- 永久存储的 `邮箱`
- 按仓库追踪的 `仓库哈希`

因此，封号风险主要来自这些软件标识的积累。防止被封号的核心策略是定期清除这些标识，并禁用非必要的遥测数据上传。具体的防护操作可参考 [[查看与删除Claude Code追踪标识]]。
