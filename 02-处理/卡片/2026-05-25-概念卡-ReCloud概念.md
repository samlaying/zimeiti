# ReCloud概念

**ReCloud** 是 [[Cloud]] 的一个化身或替代方案，主要用于AI辅助开发。其主要特点包括：
- **无需网络设置**：解决了原版Cloud可能存在的网络访问问题。
- **额度灵活**：使用Max照号直连，额度管理更高效，可以根据使用情况选择专项或拼车方式，灵活性高。
- **持续开发**：特别适合在遇到Codex等工具的额度限制时，作为接力工具继续进行开发。

使用流程简单，通常在终端执行一条授权命令即可启动，然后将已有的项目文件夹（如[[使用Codex初始化项目|Codex初始化的项目]]）直接丢给它即可继续工作。
