# brainstorm skill使用流程

## 目标
在生成创造性内容（如PPT）前，与`[[AI编程助手]]`进行高效、精准的需求对齐。

## 关键步骤
1.  **激活技能**：在命令行中召唤 `brainstorm skill`。
2.  **输入初始需求**：明确说明要生成什么（如“40分钟的博士论文开题PPT”），并提供所有参考资料（如PDF论文）。
3.  **回答结构化提问**：AI助手会依次提问，只需从其提供的ABC选项中选择。例如：
    - 确认观众（如“评审委员会”）。
    - 确认核心主题。
    - 调整时间分配（如“更关注已完成工作”）。
    - 确定已有工作的详细程度（如“brief概要”）。
    - 选择输出格式（如“HTML”）。
4.  **保存设计**：沟通完成后，指令AI将生成的结构化设计计划保存下来，防止丢失上下文。

## 技巧
避免向AI提出开放性问题，应主动提供选项让它选择，这能显著降低沟通成本。
