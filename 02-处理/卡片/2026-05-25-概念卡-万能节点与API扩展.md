# 万能节点与API扩展

**万能节点**是一毛AI画布的核心扩展组件，旨在解决AI模型迭代快、接口繁杂的痛点。
- **作用**：允许用户通过提供**API地址**，直接接入**任意模型**的接口，包括需要长时间等待的**异步任务**。
- **AI智能配置**：对于代码小白，内置的AI助手可根据自然语言描述或直接粘贴API文档，**自动写好配置、填好参数**。
- **变量系统**：支持一键设置变量，变量可在不同节点间自动连线，实现图文数据在[[工作流一键狂飙]]中的自动化流转，无需用户懂代码。
- **本质**：它赋予了用户接入**外部世界无尽算力**的能力，是插件强大扩展性的根基。
