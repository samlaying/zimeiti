# API现状与生态采用

> 主流生产力应用已开始原型探索，这预示着该API将成为未来Web图形UI的重要标准。

## 核心内容
截至2026年5月，HTML-in-Canvas API处于“Origin Trial”实验阶段，可在Chrome中通过注册试用。尽管如此，它已吸引了Figma、Adobe、Miro、JetBrains等知名应用进行原型设计和探索。Chrome团队正在与其他浏览器合作，推动该API成为开放标准，预示着广阔的行业前景。
