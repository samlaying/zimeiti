# Remotion：React驱动的视频创作框架

**Remotion** 是一个使用 **React** 创建视频的全新框架，实现了“代码即视频”的创作模式。

其五大核心特性包括：
1.  **动画**：支持复杂的动画效果。
2.  **组件化**：可复用的组件。
3.  **高质量输出**：支持4K 60帧的专业级视频。
4.  **代码即视频**：用代码定义视频。
5.  **集成AI**：可使用Claude等AI进行[[对话式编程]]。

通过与[[标准化Skills]]和[[标准化组件库]]结合，Remotion成为实现高质量、风格统一AI视频生成的生产力工具基础。
