# 灵感知识库自动化四步法

将手动拆解视频的5个步骤压缩并自动化为4步，形成一套自动化[[Skill：自动化工作流的载体]]：

1.  **自动下载视频**：输入视频链接，输出视频文件和标题。
2.  **智能获取逐字稿**：优先提取视频自带字幕（快速准确）；若无，则调用微信进行语音转写。
3.  **AI多维度分析**：将逐字稿拆解为5个独立任务（视频分类、标题分析、内容结构拆解、爆款公式提炼、金句提取），每个任务使用专用模板，产出结构化结果。
4.  **自动写入飞书多维表格**：将所有分析结果一键写入飞书表格，支持批量处理。

这套方法大幅提升了效率（从手动30分钟到自动5分钟）并实现了结果的可存储与可筛选。
