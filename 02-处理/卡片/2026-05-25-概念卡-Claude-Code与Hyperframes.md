# Claude Code与Hyperframes

**Claude Code** 是一个AI驱动工具，核心功能是**根据用户提供的文案，自动生成视频内容**。它负责处理视频制作的核心环节，如剪辑、动画、字幕和效果生成。

**Hyperframes** 是一个辅助工具，通常与Claude Code配合使用，可能专注于**视频编辑或动画渲染**，用于提升制作效率。

这两个工具的组合，实现了从文案到视频成品的自动化流水线，是实现[[文案驱动AI视频制作]]的基础。
