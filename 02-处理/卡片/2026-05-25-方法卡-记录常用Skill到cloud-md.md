# 记录常用Skill到cloud.md

将常用的、需要全局遵守的[[skill]]规则记录在一个`cloud.md`文档中，以便AI在所有交互中遵守。

**操作**：
1. **创建文档**：建立一个名为`cloud.md`的文档。
2. **记录规则**：将需要AI时刻遵守的规则写入，例如：“每次完成文章时，都要根据我的[[审稿标准]]和[[写作风格]]来。”
3. **AI遵守**：通过文档约束，确保AI的行为保持一致。

**好处**：为AI提供了一个稳定的“记忆”和“规范”，确保长期执行标准统一。
