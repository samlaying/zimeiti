# AI驱动的UI设计工具Pencil

**Pencil** 是一个基于[[AI Agent]]的[[MCP]]插件，它可以集成到 [[Cursor]]、VS Code 等主流代码编辑器中。其核心功能是**通过AI对话，直接生成高保真、可编辑的UI界面**，操作体验和界面布局与 **[[Figma]]** 非常相似。

它实现了**AI生成能力与类Figma设计体验的深度融合**，将AI驱动的设计生成无缝嵌入开发环境。
