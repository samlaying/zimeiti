# SOLO模式：复杂任务自动化Agent

**SOLO模式**是TRAE AI推出的一种**Agent**产品形态。

它与普通Chat产品不同，专为处理**复杂、高负担任务**而设计。其核心目标是自动化完成从**调研**到**写稿**的全流程。

用户仅需提供一个链接（如论文链接），[[SOLO全流程自动化工作流]]便能自动跑完全流程，最终输出结构专业、图文并茂的成品文档（如DocX文件）。
