# ClawHub管理工具

ClawHub 是 OpenClaw 的集中管理平台，用于搜索、安装和更新技能（Skill）。它通过 NPM 安装 ClawHub Cloud 后启用，支持自然语言搜索、安装指定 Skill 以及一键批量更新所有已安装的 Skill。懒人方法可直接对 OpenClaw 发出指令，它会自动执行命令。
