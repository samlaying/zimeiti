# Tether Scale Frontend Design

一个被称为“品味外挂”的AI指令或工具（Scale）。其功能是为AI注入高级审美约束，指导其生成设计感更强的作品，避免AI默认的保守设计（如白底黑字加蓝色标题）。

**具体的设计约束示例：**
- 留白要大。
- 圆角12px。
- 禁止使用默认蓝色。
- 间距使用8的倍数（如4， 8， 12， 16， 20）。

它的作用是显著提升 [[AI生成]] 内容的视觉质感和专业性。
