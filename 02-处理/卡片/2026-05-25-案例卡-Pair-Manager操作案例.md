# Pair Manager操作案例

通过自然语言指令操作[[Pair Manager]]的典型案例：

- **整理收件箱**：说出指令“整理收件箱”，AI会扫描`00-inbox`文件夹，逐条展示笔记摘要并询问归档位置。用户回复后，文件即被移动，且[[MOC]]和[[首页]]自动更新。
- **归档项目**：说出“归档项目 My Project”，AI展示确认信息（项目名、文件数、目标路径）。确认后，项目文件夹移至`40-Archive`，相关链接标记为已归档。
- **检查笔记库**：说出“检查笔记库”，AI输出报告，列出孤立文件、失效链接等问题。所有操作前均会确认，避免误改。
