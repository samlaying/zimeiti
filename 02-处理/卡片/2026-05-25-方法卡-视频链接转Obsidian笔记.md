# 视频链接转Obsidian笔记

这是一个通过 [[Claude Code Skill]] 自动将视频链接转为结构化 Obsidian 笔记的方法。

**核心步骤：**
1.  **输入链接**：向 Skill 提供视频 URL（如 B 站视频）。
2.  **获取元数据与音频**：使用 YT-DLP 获取视频元数据（标题、时长、作者）并下载音频文件。
3.  **语音转录**：若视频无字幕，使用 Whisper ASR 进行语音识别。为提高效率，会将音频切分成多个片段并行转录。
4.  **结构化生成**：基于转录文本生成视频笔记，通常包含章节划分、核心观点和个人思考。
5.  **自动打开**：生成的笔记文件在 Obsidian 中自动打开。

**输出效果：**
- **属性区**：包含完整的视频来源信息。
- **正文**：按视频章节整理成小节，提炼观点。
- **附件**：原始转录文本完整保留在笔记底部，方便回溯。
