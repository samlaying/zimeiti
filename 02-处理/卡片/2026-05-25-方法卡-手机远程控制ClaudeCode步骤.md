# 手机远程控制ClaudeCode步骤

此桌面端支持无需持续启动服务的跨Session手机远程控制，操作步骤如下：
1.  在电脑端通过命令行工具选择工作目录（如 `tempo`）。
2.  在手机端（演示中使用飞书）选择目标项目。
3.  在手机对话框中直接下达任务指令（如“你是什么模型？有哪些能力？”）。
4.  模型将在指定目录下工作并返回结果。

此功能与[[设置ClaudeCode定时任务]]结合，可构建强大的远程自动化工作流。
