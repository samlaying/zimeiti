# 从零开始的Agent质量评估

如果当前没有任何评估体系，可以从成本最低的 **Andre Capacity 三个指标** 零成本起步，这些指标可以从已有历史记录中直接统计：

1.  **Diff干净度**：从Git History统计不必要改动的行数比例。
2.  **重写次数**：从Code Review记录统计因过度复杂化而重写的次数。
3.  **问题提出时机**：从对话记录统计问题是在实现前还是实现后提出。

这些指标能立即量化Agent的工程成熟度，是将评估能力内化到团队的第一步。
