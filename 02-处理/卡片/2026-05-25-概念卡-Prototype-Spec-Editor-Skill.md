# Prototype Spec Editor Skill

**Prototype Spec Editor Skill** 是一个基于 AI 的技能，用于自动化生成和维护 [[一体化原型]]。

- **主要功能**：将现有原型页面升级，为其自动生成结构化的需求说明，并支持后续的编辑与版本管理。
- **核心目标**：通过技术手段将原型与需求文档“合二为一”，替代传统的“原型 + PRD”交付模式，从而提升产品团队的交付效率和准确性。

它代表了 AI 工具在产品设计流程中的深度应用，旨在解决文档与原型同步维护的行业难题。
