# 对话式编程：用自然语言驱动开发

**对话式编程**是指使用AI编程助手（如**Claude Code**），通过自然语言对话来生成代码的工作方式，是实现“一句话出视频”的技术支撑。

**在本场景中的应用流程：**
1.  将定义好的[[标准化Skills]]文件和Remotion官方最佳实践提供给AI。
2.  向AI提出简短的视频需求（一句话描述）。
3.  AI自动分析需求，告知将使用哪些分段和组件（通常是[[标准化组件库]]中的现成组件）进行开发。
4.  AI快速生成完整的Remotion项目代码。

此方法将AI从“容易产生人工智障感”的生成者，转变为遵循明确规范的“高效执行者”。
