# 设计规范统一AI生成UI

使用 AI 编码代理（如 Cloud Code、Cursor）生成界面时，核心痛点是风格不统一、颜色和组件不协调。手动向 AI 描述设计规范既繁琐又易遗漏细节。

**核心洞察**：提供一份标准化的 [[设计规范文件 (Design.MD)]]，是解决 AI 生成 UI “AI味十足”、专业度不足的关键。将设计系统（Design Token）预先结构化，可以让 AI 严格按照规范输出高质量、一致的界面代码，从而提升开发效率和产品专业性。
