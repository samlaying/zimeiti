# 查看与自定义 Auto Mode 规则

> 用命令行查看和评审规则，支持深度自定义。

## 核心内容
用户可以通过 `claude automode` 子命令管理规则。常用命令包括：`claude automode defaults` 查看默认规则；`claude automode config` 打印当前配置；`claude automode critique` 评审当前规则。更详细的自定义规则需要参照官方文档，在本地配置文件中进行，这些自定义规则会与线上分类器互动，共同决定操作权限。
