# Pair Manager的智能联动设计

[[Pair Manager技能]]的一个核心设计亮点是其命令之间的**智能联动**。大多数操作完成后会自动触发下游更新，实现一次指令完成多步操作。例如：
- “[[使用Pair Manager整理收件箱|整理收件箱]]”后，会自动“同步MOC”并“更新首页”。
- “[[使用Pair Manager归档项目|归档项目]]”后，同样会触发相关MOC链接的更新。

这种设计将简单的用户指令转化为连贯、准确的自动化流程，极大地提升了管理效率。
