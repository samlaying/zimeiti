# Draw UI Skill 工作流

自动化工具 Draw UI Skill 的工作流：
1. **安装**：通过 Git 仓库链接或终端命令安装 Skill。
2. **配置**：配置第三方图像生成服务的 API Key（如 ZenMax）。
3. **自动学习**：Skill 自动查看代码仓库中已有的设计资源。
4. **参考图引导**：AI 主动引导用户提供系统内代表性页面截图（如 Dashboard）。
5. **智能生成**：结合参考图、设计规范和内置提示词策略（叙事、信息布局等），自动分析并生成最优提示词，输出 UI 设计稿。
6. **迭代优化**：支持对设计不满意时重新调整。
