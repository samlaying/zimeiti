# Skill Creator

**Skill Creator** 是一个可以安装在本地的GitHub项目，它本身也是一个“[[Skill]]”。

它的核心功能是**帮助设计和搭建其他Skill**。可以理解为一个“元技能”或“技能工厂”，在构建[[个人文风Skill]]等复杂系统时，由它来协助规划和生成核心的[[Skill文档结构]]。
