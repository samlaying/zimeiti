# 官方ClaudeCode的四大痛点

官方ClaudeCode桌面端存在四大核心痛点，是催生开源替代方案的主要原因：
1.  **封号严重**：对网络环境、订阅方式监管严格，用户极易被封号。
2.  **模型限制**：只能使用官方自家模型，无法自定义。
3.  **功能受限**：核心功能需要付费订阅才能解锁。
4.  **KYC实名认证**：最近要求上传身份证、护照等证件，国内用户基本被拦在门外。
