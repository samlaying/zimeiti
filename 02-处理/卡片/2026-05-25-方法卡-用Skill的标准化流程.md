# 用Skill的标准化流程

安装和使用[[Skill]]的步骤通常大同小异，以在Cloud Code中为例：

**安装Skill（任选其一）**：
- 使用`mix`命令安装。
- 使用自然语言指令，如“请帮我安装XX文件夹下的技能”。
- 将Skill文件夹复制到`Cloud Code Skill`目录下。
- 上传Skill压缩包（支持在Tree等工具中）。

**使用Skill步骤**：
1.  在终端中进入项目路径（`CD`到项目路径）。
2.  启动Cloud Code（如希望减少确认操作，可在命令后添加特定参数）。
3.  执行Skill：输入`/技能名`（例如`/SkillTranslator`），即可运行该技能完成特定任务。
