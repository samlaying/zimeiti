# 暗色模式设计要点

进行**暗色模式**设计时，需要调整以适应深色背景：
1.  **降低对比度**：高亮边框在暗色背景下可能过于刺眼，需适当调低。
2.  **营造层次感**：不依赖阴影，而是通过**让卡片比背景更亮**来区分层级。
3.  **调整组件**：降低过亮标签的饱和度和亮度，将对比度重点用于文字。
4.  **颜色灵活性**：深紫色、深红色、深绿色在暗色模式下效果良好，不仅限于藏青蓝或灰色。
