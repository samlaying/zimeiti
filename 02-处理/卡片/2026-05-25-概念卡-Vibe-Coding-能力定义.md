# Vibe Coding 能力定义

**Vibe Coding能力**，对于AI产品经理而言，并非指凭感觉、口喷式的随意编程。它指的是遵循专业开发规范，利用AI编码工具高效将产品想法落地的综合能力。其核心是**技术理解力**与**动手实操能力**的结合，绝非随心所欲的编码。
