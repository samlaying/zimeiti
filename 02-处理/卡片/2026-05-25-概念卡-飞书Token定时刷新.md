# 飞书Token定时刷新

指通过脚本定期自动获取并更新 [[飞书]] API 访问令牌（Token）的机制。

**为什么重要**：飞书 Token 具有时效性，过期后所有基于 API 的操作（如 [[JS脚本同步Obsidian到飞书]]）都会失败。必须设置定时刷新，才能保障自动化流程稳定、持续运行。
