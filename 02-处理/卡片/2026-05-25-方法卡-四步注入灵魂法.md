# 四步注入灵魂法

让AI生成的APP“起死回生”的核心四步流程：
1.  **[[快速生成功能原型]]**：使用[[Cursor]]等AI工具快速生成基础功能。
2.  **[[定义产品调性]]**：与[[Claude]]等AI对话，明确产品的**情感目标**和**[[视觉隐喻]]**。
3.  **[[收集灵感与搭建情绪版]]**：搜索视觉隐喻的实物图片，利用AI工具二次创作，整合成设计规范。
4.  **[[实现高级感界面]]**：将完整的设计方案交给代码生成工具，重新生成或修改代码，获得有质感的最终应用。
