# MCP工具配置与运行

配置和运行[[MCP工具]]以启用[[搜索机制]]的步骤如下：
1.  **工具准备**：获取SpinSearchGetHub等MCP工具及配置文件。
2.  **切换模式**：使用CC switch等工具切换到MCP模式。
3.  **添加配置**：在MCP配置中添加指定的配置行内容。
4.  **启动服务**：运行MCP服务，在本地开启搜索接口。
5.  **使用搜索**：在AI工具中调用预设的搜索功能（如WebGangSearch）即可。**注意**：必须先运行MCP服务才能进行搜索。
