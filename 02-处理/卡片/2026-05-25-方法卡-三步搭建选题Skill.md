# 三步搭建选题Skill

搭建个人专属选题 Skill 的具体步骤，形成一个可迭代的“选题手册”：

1.  **整理个人说明书**：向 Skill 提供包括个人履历、具体账号定位、用户画像等详细信息。这是 Skill 进行个性化判断的**基准**，能增进工具对用户的了解。
2.  **提供对标博主选题方向**：将对标博主的选题方向输入给 Skill。这能帮助它发现用户可能存在的**盲区**，为选题判断提供具体参考。
3.  **提供自发布视频选题**：输入自己已发布的视频选题列表。让 Skill 判断当前选题类别，与对标博主比较，找出用户选题的局限或需要扩充的方向。

持续往这三个维度更新信息，可以使 [[Claudian+Skill 定制化选题助手]] 越来越“懂你”。
