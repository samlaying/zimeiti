# AI闪卡生成与复习

用于长期学习任务（如法考）的AI辅助复习方法：
1. 在笔记界面生成基于当前内容的闪卡
2. 核对并保存闪卡
3. 切换到闪卡界面复习
4. 基于IbisHouse记忆曲线，系统自动安排复习计划
