# API接口清单的制定与管理

基于[[系统流程图]]，制定[[API接口清单]]是明确开发工作的直接依据。清单应包含以下关键字段：

- 接口提供方
- 接口调用方
- 接口名称
- **接口状态**：
    - **已有**：可直接使用。
    - **新增**：需要新开发，有工作量。
    - **已有修改**：需要调整开发，有工作量。
