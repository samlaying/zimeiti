# AwesomeDesign.md 设计系统项目

一个托管在 GitHub 上的项目，收录了众多知名品牌的官方设计系统（如 Spotify、Anthropic Cloud 等）。

### 主要用途
- **解决 AI 审美问题**：ChatGPT 默认生成的设计可能较差，通过在提示词中指定一个设计系统的链接，可以约束 AI 生成符合该品牌设计规范（如色彩、字体、组件风格）的 PPT。
- **提升设计质量**：为生成的 PPT 提供一个专业、统一且美观的设计框架。
