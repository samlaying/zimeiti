# Pair Manager的推荐使用节奏

为了高效管理[[Obsidian]]笔记库，建议将[[Pair Manager技能]]融入日常工作流，遵循以下节奏：
- **每天**：使用“[[使用Pair Manager整理收件箱|整理收件箱]]”，清空Inbox。
- **项目迭代完成后**：使用“[[使用Pair Manager归档项目|归档项目]]”。
- **每个月**：使用“[[使用Pair Manager进行全库健康检查|检查笔记库]]”，进行一次健康检查。

这个节奏有助于实现定期、自动化的知识库维护。
