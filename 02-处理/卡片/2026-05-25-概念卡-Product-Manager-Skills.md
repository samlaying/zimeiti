# Product Manager Skills

**Product Manager Skills** 是一个开源的AI Skill（技能包），它将真实的产品经理方法论拆解为Claude等AI能理解的模块。

其核心是集成46个产品管理框架，如 [[RICE优先级排序应用|RICE]]、用户洞察、用户故事等。安装后，AI便如同学习过这些方法论的[[角色框架提升需求分析|资深产品经理]]，能主动按专业框架引导用户工作，解决了传统AI不懂产品经理专业工作流的问题。
