# QuickAdd模板的自动化设置

QuickAdd的模板功能通过一系列设置实现高度自动化，核心要点如下：
1. **集成双链**：启用后，在当前文档通过模板创建新笔记时，会自动在原文档添加指向新笔记的[[双链]]引用。
2. **动态参数**：在模板中使用 `{{title}}`、`{{date}}` 等[[内置函数]]，创建笔记时可自动填充。
3. **单页输入弹窗**：将多个参数（如书名、日期）的输入集成到一个弹窗中完成，提升效率。
4. **保存路径与行为控制**：精细设置笔记的默认保存位置、命名冲突处理、打开方式（如新标签页打开、自动聚焦）等。
