# CSS 3D变换环境配置

在制作类似 [[3D翻转卡片]] 的效果前，必须在 CSS 中正确配置3D变换环境。

## 关键步骤
1. **创建透视容器**：为卡片的最外层容器（如 `.card`）添加 `perspective: 1000px;`（值可调整），定义3D元素距视图的距离，产生透视感。
2. **保持3D空间**：为直接包含翻转内容的元素（如 `.card-inner`）设置 `transform-style: preserve-3d;`，使其子元素能够进行3D变换。
3. **隐藏背面**：为卡片的正反面（`.card-front` 和 `.card-back`）都添加 `backface-visibility: hidden;`，确保在旋转到背面时，正面不被渲染。
