# Pair Manager

**Pair Manager**是集成于 **Cloud Code** 中的一个Obsidian笔记管理技能。其设计目标是通过AI自动化，解决[[Obsidian笔记管理四大痛点|笔记管理的四大核心痛点]]。它提供六个子命令，覆盖从笔记整理到项目归档的完整工作流，用户可以通过自然语言指令驱动操作，实现笔记库的自动化维护。
