# 下拉动画核心CSS

> 隐藏用 scaleY(0)，显示用 scaleY(1)，加上 transition 就有动画。

## 核心内容
实现二级菜单平滑下拉的核心技术是CSS的`transform`与`transition`。默认使用`transform: scaleY(0)`将二级菜单容器沿Y轴缩放至0实现隐藏，并设置`transform-origin: 50% 0`让动画从顶部中心展开。通过`:hover`伪类将`scaleY`改为1来显示，并配合`transition: all 0.6s`属性实现平滑过渡动画。

## 原文关键段落
> transform: scaleY(0); /* 沿Y轴缩放至0，隐藏 */
> transform-origin: 50% 0; /* 缩放中心点为顶部中间 */
> transition: all 0.6s; /* 过渡动画 */
> .navbar .nav ul > li:hover ol {
>     transform: scaleY(1);
> }
