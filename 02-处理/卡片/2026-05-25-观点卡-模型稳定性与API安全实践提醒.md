# 模型稳定性与API安全实践提醒

在构建AI工作流应用时，需要关注两个关键实践问题：

1.  **模型稳定性**：某些外部模型（如案例中提到的文心模型）在高频或多次调用后可能出现不稳定现象。在应用设计时需考虑**替换备用模型或实现重试机制**。
2.  **API密钥安全**：在`HTTP请求`节点中使用API Key时，务必注意保密。在演示或共享工作流时，应隐藏或使用变量来管理密钥，避免泄露。

> 这些是确保AI应用在生产环境中可靠运行的重要考量。
