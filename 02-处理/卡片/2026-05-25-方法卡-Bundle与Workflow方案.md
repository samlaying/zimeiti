# Bundle与Workflow方案

解决“面对1431个技能无从下手”问题的两种组织方案：

- **Bundle（技能包）**：按**角色**或**场景**打包的技能推荐方案，解决“装什么”的问题。
  - 示例：
    - 做全栈产品：`Essentials` + `Fullstack Developer` + `QA Testing` 三个Bundle。
    - 加固安全：使用 `Security三件套`。
    - 做开源项目：使用 `Essentials` + `OSS Maintainer` Bundle。
- **Workflow（工作流）**：定义技能的**有序执行流程**，解决“怎么用”的问题。
  - 示例：“Shipping Assess MVP” Workflow会按顺序调用：头脑风暴 → 架构设计 → 前端开发 → 测试 → 安全审计 → 提交PR。
  - 比喻：不仅给了你“工具箱”，还给了你“施工图纸”。
