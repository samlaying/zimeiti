# 参考图控制一致性

解决 UI 一致性的有效方法是使用参考图：
- 提供系统现有元素的截图作为参考，AI 会保持参考图中已有的元素（如侧边栏）完全不变。
- 新生成的部分会自动沿用参考图的设计风格（如图标样式、配色）。
- 提示词只需强调“保持侧边栏与参考图完全一致”，并描述新页面内容即可。
