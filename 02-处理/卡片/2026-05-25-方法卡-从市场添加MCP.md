# 从市场添加MCP

在 `Builder with MCP模式` 中从市场添加 MCP 服务的标准流程：
1.  **浏览市场**：在 MCP 管理界面，选择“从市场添加”。
2.  **选择服务**：浏览并选择所需的 MCP 服务（如 Figma MCP）。
3.  **填写凭证**：根据提示填写必要信息（如 API Key），系统会自动生成配置。
4.  **确认添加**：完成配置。

**典型案例**：添加 Figma MCP 时，需要获取 [[Figma Personal Access Token]] 作为凭证。这是连接设计与开发的关键步骤。
