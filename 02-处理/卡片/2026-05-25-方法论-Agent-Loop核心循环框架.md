# Agent Loop核心循环框架

> 为系统设计提供了核心循环框架，理解它就理解了Agent的运行机制。

## 核心内容
Agent Loop是系统处理用户请求的核心循环，包含请求输入、流式输出、工具调用与结果回传等环节。循环在任务完成、需用户授权或达到最大轮次时停止，避免死循环。

## 原文关键段落
> Agent Loop是系统的核心，处理用户请求的循环流程：- **请求处理**：用户输入进入系统，流式输出结果，包括工具展示和输出界面。- **工具调用**：系统执行各种工具（如飞书、搜索、文件操作），工具结果回传到循环中。- **停止条件**：循环在任务完成、需要用户授权或达到最大轮次时停止，避免死循环。
