# 认知资源节省策略

用户的大脑认知资源宝贵，应尽量节省，避免因过多选项让用户“被水淹没”而放弃。

**四种核心解决方案**：
1.  **设计符合直觉**：如左键拖拽是框选，让用户一看即懂。
2.  **设置最优默认值**：让大部分用户无需额外调整即可开始使用。
3.  **信息隐藏**：将复杂设置面板折叠，在用户需要时再展开。
4.  **信息排序**：将常用功能放在显眼位置，不常用功能置于深处。

**目标**：将用户的注意力集中在他们认为重要的事情上，降低初始使用门槛。
