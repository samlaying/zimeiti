# Use Figma Tool：AI画布设计新范式

Figma MCP 新增的核心工具 **Use Figma Tool**，实现了 AI 能力从“导入”到“创造”的质的飞跃。

### 核心功能
与旧工具 `Generate Figma Design`（将 HTML 转为 Figma 图层）不同，`Use Figma Tool` 允许 AI Agent **直接在 Figma Canvas 上创建、修改和操作系统化的设计资源**。

### 关键能力
- **创造设计**：AI 可基于设计系统，直接在画布上生成 UI。
- **系统化操作**：可创建设计 Token、颜色变量集、组件库等。
- **自我迭代**：结合 [[Skills（技能）]] 的自我修复循环，通过截图验证和迭代来优化设计。
