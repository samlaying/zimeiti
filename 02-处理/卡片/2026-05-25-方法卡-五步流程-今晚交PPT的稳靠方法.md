# 五步流程：今晚交PPT的稳靠方法

如果急需交出一版高质量的PPT，遵循此[[五步流程]]容错率最高，可避免方向性错误和无效返工：
1. **定义唯一目标**：用AI明确PPT给谁看、希望对方做什么、只留哪三个主结论。
2. **[[每页写一句主旨]]**：用ChatGPT先输出每页的标题、核心观点和所需证据，确保逻辑推进而非页面堆砌。
3. **[[取第一版样张]]**：用Gamma或Beautiful.ai只做封面、目录、核心页等关键样张，确认方向后再铺开。
4. **[[视觉统一]]**：用Canva或Figma Slides补全封面、关键图示、品牌元素，提升完成度。
5. **[[自己念一遍]]**：从头到尾口头讲述，删减重复、冗余及不服务结论的页面，确保能讲、能用。
