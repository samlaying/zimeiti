# 协作配套Skills

为配合[[飞书Agent协作插件]]使用，作者编写了六个优化协作的Skills：
1.  **协作手册Skill**：让机器人了解群内协作规则。
2.  **中断协作Skill**：用于中断当前任务。
3.  **协作模式切换Skill**：切换不同协作模式。
4.  **总结汇总Skill**：帮助汇总结果。
5.  **状态查询Skill**：查询工作状态。
6.  **任务分解Skill**：分解任务。

在协作过程中，可通过特定触发词调用这些Skill，以提升协作效率和规范性。
