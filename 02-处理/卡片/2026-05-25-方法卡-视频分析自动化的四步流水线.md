# 视频分析自动化的四步流水线

将手动拆解视频的五步压缩并自动化为四步流水线：
1.  **自动下载视频**：输入视频链接，自动下载视频文件及获取标题。
2.  **智能获取逐字稿**：优先提取视频自带字幕（快且准），若无则自动调用语音转写。
3.  **多维度AI分析**：将逐字稿拆解为多个独立分析任务（见[[多任务AI分析模板]]），每个任务有专门模板，产出结构化结果。
4.  **自动写入飞书多维表格**：所有分析结果自动一键存入表格，支持批量处理，便于后期筛选查找。

**成效**：分析一个视频仅需约5分钟，成本约两毛钱，全程无需手动操作。
