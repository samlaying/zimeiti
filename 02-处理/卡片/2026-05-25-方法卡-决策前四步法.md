# 决策前四步法

使用[[AI决策辅助工具Steel|Steel]]进行重大决策辅助的操作步骤：

1. **启动时机**：面临重大决策（如投资、转行）且感到纠结时启动工具。
2. **接受挑战**：工具会通过尖锐提问（如要求忽略[[七种自欺谬误|沉默成本]]）挑战你的假设。
3. **识别谬误**：诚实面对工具帮助列出的七种常见自欺心理，反思自己的思维偏差。
4. **评估方案**：基于工具的分析，重新思考决策，聚焦于当前现实，评估最优行动方案。

**核心原则**：方向不对努力白费。决策前先把脑子搞清楚，确保基于当前现实而非心理谬误做出选择。
