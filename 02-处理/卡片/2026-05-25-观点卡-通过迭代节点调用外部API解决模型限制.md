# 通过迭代节点调用外部API解决模型限制

当[[Dify]]内置的[[AI大模型]]节点不直接支持某项功能（如图像生成）时，一个强大的替代方案是：**使用迭代节点配合外部API调用**。

**实现逻辑**：
1.  使用`迭代`节点处理一个数据列表（如图片URL列表）。
2.  在每次迭代中，添加一个`HTTP请求`节点，向支持该功能的外部服务（如百度的文心一言图片API）发送请求。
3.  请求体中包含要处理的数据（如原始图片URL）和具体的修改指令（提示词）。
4.  接收并处理API的返回结果（如新图片URL）。

> 这种方法将平台限制转化为技术灵活性，通过组合自身节点和外部服务能力，扩展了工作流的边界。需要注意外部API的稳定性和调用成本。
