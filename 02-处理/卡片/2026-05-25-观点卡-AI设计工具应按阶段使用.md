# AI设计工具应按阶段使用

AI 设计工具已覆盖 [[传统产品设计六步]] 的每个阶段，但大多数人用法错误：跳过前序步骤直接使用工具。**问题不是工具不好用，而是用户跳过了信息架构和线框图，未想清楚结构，AI 无法替代思考。**

正确做法是明确每个工具对应的设计阶段：
- **信息架构**：Relume（一句话生成站点地图和页面流程）。
- **线框图**：Uizare（手绘转数字线框）、UXpilot（提示词生成线框）。
- **视觉设计**：Google Stitch、V0、Lovable（语音/文本生成UI）。
- **交互原型**：Figma（行业标准，集成AI功能）。
- **交付**：Locify（Figma转React）、Claude Code + 设计Skill库（设计稿生成高质量前端代码）。
