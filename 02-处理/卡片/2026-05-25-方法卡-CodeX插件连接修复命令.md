# CodeX插件连接修复命令

在使用[[CodeX]]的[[Figma]]插件时，若测试生成页面失败（无法正常写入），需要输入命令进行修复：

**命令**：`CodeX MCP MedFigma`

**操作流程**：
1.  按照CodeX的提示输入上述命令。
2.  根据建议，重新开启一个线程。
3.  再次进行测试，此时通常能成功连接并写入。

这是解决插件初始化或连接问题的关键步骤。
