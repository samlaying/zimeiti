# Prompt方法与Skill方法的核心区别

两种去除文章AI味的核心方法对比：

- **Prompt方法**：临时性指令，AI仅基于即时提示写作，容易留下AI痕迹。类似于临时给新员工交代公司规定。
- **Skill方法**：创建一份个人风格的[[写作手册]]，记录表达偏好、写作习惯、常用词和风格边界。AI每次写作前参考手册，实现风格一致性。更像一份写给新员工的录制手册，可通过迭代更新优化效果。优势：更持久、可积累，AI逐渐“懂”用户风格。
