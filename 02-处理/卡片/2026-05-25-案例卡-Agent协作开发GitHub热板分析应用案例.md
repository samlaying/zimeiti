# Agent协作开发GitHub热板分析应用案例

使用[[飞书Agent协作插件]]，作者指挥四个AI机器人（产品、前端、后端、测试）协作开发了一个“GitHub热板分析”网页应用。

**协作流程**：
1.  产品机器人作为协调者，分配任务。
2.  各机器人回复各自方案（前端方案、技术方案、测试方案）。
3.  产品汇总方案，生成MVP功能清单，并继续分配任务。
4.  机器人自主协作，完成开发、测试、验收。
5.  产品设置定时任务监控进度，确保工作推进。
6.  最终生成测试报告和使用手册，完成产品交付。

整个过程就像现实中团队协作，实现了自动化的产品交付。
