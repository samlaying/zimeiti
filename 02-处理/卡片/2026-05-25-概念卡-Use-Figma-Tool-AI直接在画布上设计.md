# Use Figma Tool：AI直接在画布上设计

Figma MCP 新增的核心工具，允许AI Agent直接在Figma画布上进行**创建和编辑**操作，实现了从“导入”到“创造”的跨越。AI可以基于你的[[设计系统]]，通过MCP客户端（如Claude、Codex）来生成和修改设计资源。

这是[[Figma MCP 旧工具区别]]中提到的重大升级点。
