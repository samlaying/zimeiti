# 交互优化：pointer-events

> 加上 pointer-events: none;，让装饰层“透过去”。

## 核心内容
当使用绝对定位元素（如滑动线条）覆盖在可交互元素（如菜单链接）之上时，为避免该覆盖层阻挡鼠标事件，可以对其应用`pointer-events: none;`样式。这会使该元素“穿透”，鼠标事件会直接作用于其下方的元素。

## 原文关键段落
> pointer-events: none; /* 避免鼠标事件干扰 */
