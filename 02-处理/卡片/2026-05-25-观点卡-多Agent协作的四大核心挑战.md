# 多Agent协作的四大核心挑战

多Agent协作面临四大挑战：1. **上下文过多导致核心指令失效**：对话历史过多使Agent忽略核心任务。2. **工具误匹配**：模型选择看似相关但业务无关的工具。3. **“只分析不执行”问题**：过度强调输出格式导致注意力分散，忽略执行。4. **任务异常终止**：Agent输出任务终止但未完成，或中止后未执行下一步。这些挑战影响多Agent协作的实用性和可靠性，需要[[优化多Agent协作的策略]]。> 来源：[[2026-05-25-052-学习多 Agent 协作，我做了一个系统 demo.md]]
