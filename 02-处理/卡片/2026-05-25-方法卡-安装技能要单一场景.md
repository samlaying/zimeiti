# 安装技能要单一场景

切忌同时安装多个同类[[前端设计技能(Skills)|Skills]]，这会导致AI调用混乱，设计风格不稳定。

**正确做法**：坚持“一次只装一个”原则，根据具体设计场景选择最合适的一个技能。
- 做**官网**可选 `Frontend Design Pro` 或 `UI UX Pro Max`
- 做**后台**可选 `Web Design Guidelines` 或组件库类技能
- 需要**暗色特效**可选 `Frontend Design Pro`
