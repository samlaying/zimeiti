# Skills：本地Agent的能力扩展包

**Skills** 是指一套可以被本地Agent（如OpenCloud Malabot）调用的功能包，用于扩展Agent的具体能力。在本案例中，它是一套自动视频剪辑Skills，能将Agent的能力延伸到视频素材分析和剪辑规划领域，实现从素材到剪映草稿的全流程自动化。
