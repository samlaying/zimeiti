# Hermes Agent 核心定位

**Hermes Agent** 是一个开源的 AI Agent 项目，在 GitHub trending 榜登顶第一。

它旨在成为 **OpenClaw** 的替代品，其核心价值在于三大能力：
1. **自我进化**：通过自动提炼和迭代 `[[Hermes Agent 自我进化能力|Skill]]` 实现“越用越聪明”。
2. **安全防御**：经过测试达到 B 级安全评级。  
3. **无缝迁移**：支持从 OpenClaw 迁移记忆、Skill、MCP 等数据。

它并非一个简单的聊天工具，而是一个完整的 Agent 平台，适用于多种自动化场景。
