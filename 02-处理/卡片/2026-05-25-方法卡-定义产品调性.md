# 定义产品调性

通过与AI对话，为产品注入情感与风格的方法：
1.  **选择工具**：强烈推荐使用[[Claude]]（或豆包）作为头脑风暴伙伴。
2.  **关键提问**：
    -   目标用户是谁？
    -   希望用户产生什么样的感觉？（如：模拟真实触感、拒绝科技焦虑）
3.  **核心提示词**：“哪些现实世界的模拟事物可以作为UI提供灵感？”（即询问[[视觉隐喻]]）
4.  **获取产出**：AI会提供情感关键词和具体的视觉隐喻列表，为后续设计奠定基调。
