# 本地知识库构建流程

构建本地知识库的完整操作流程如下：
1.  **获取并启动工具**：获取本地知识库工具包，确保[[一键运行]]成功。
2.  **输入视频源**：复制在线视频链接，或选择本地视频文件。这是[[视频转笔记]]的起点。
3.  **生成结构化笔记**：工具自动处理视频内容，生成详细笔记。
4.  **导出与整合**：
    - 基于笔记，自动生成[[多元化学习材料生成|Anki记忆卡片、思维导图和测试题]]。
    - 将生成的笔记及材料，导入到你的本地知识库中，完成[[知识库集成]]。
5.  **进行学习与测试**：使用Anki记忆，通过导图回顾，利用题目自测，并对不理解处进行[[AI互动问答]]。
