# Frontend Slides：零依赖HTML演示工具

> 一个HTML文件就是你的整个PPT，随时随地，在任何设备上演示。

## 核心内容
Frontend Slides是一款通过编写内容直接生成HTML演示文稿的工具。其核心优势是零依赖（无需安装软件）、响应式布局（适配多端）、自带交互式动画，且生成的是一个纯前端的HTML文件，便于分享和播放。
