# Codex 2.0

Codex 2.0 是由 ChatGPT 和 OpenAI 打造的超级自动化应用，旨在让用户构建并实现几乎任何任务的自动化。

它类似于一个桌面应用程序，核心特性包括：
1.  **项目管理**：创建并管理任务项目。
2.  **计算机使用功能**：能够操控电脑上的所有软件（浏览器、应用），实现查看、点击和输入操作。
3.  **技能管理**：允许管理各种技能（如[[技能管理：头脑风暴与前端设计]]），并根据任务自动调用。
4.  **文件管理**：新增处理PDF、电子表格等文件的能力。
5.  **后台自动化**：支持任务在后台常驻并自动运行。
