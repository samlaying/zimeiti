# Search Agent数据合成整体流程

> 一个清晰的二阶段框架：用图谱合成复杂轨迹数据，然后用SFT训练Search Agent。

## 核心内容
该方法的核心是利用知识图谱作为数据源来合成Search Agent的训练数据。整体分为数据合成与模型训练两大部分。数据合成阶段，通过生成需要多步推理才能回答的复杂问题及其推理轨迹，产出高质量的监督微调数据。模型训练阶段，直接用合成数据进行SFT。
