# Sahil 极简创业七步法

创业不是从想法开始，是从社区开始。每一步建立在前一步之上，跳步是失败主因。

1.  **找社区**：加入或找到你已属于的社区（Discord、Reddit 等），搞清人们每天的抱怨。
2.  **手动验证**：手动为 3 个人解决问题，不要搭系统、不写代码。验证他们愿意付钱（哪怕 9.9 元），只有真金白银才能验证真需求。这比学会用任何 AI 工具都重要。
3.  **流程化**：将手动处理过程写成一张“魔法纸条”，清楚到任何人照做都能完成。这是产品化前的最后一道闸。
4.  **AI 产品化**：将“魔法纸条”交给 Cursor 或 Claude Code，让它把人肉流程翻译成可重复运行的产品。此步建立在前三步跑通的基础上，否则速度越快死得越快。
5.  **卖给 100 人**：采用“同心圆销售法”，先卖给亲友，再卖给社区认识的人，最后才是陌生人。
6.  **开始营销**：内容触达、产品内功能拉新、社区长期运营三条线一起跑。注意：**前五步没跑通就推广，等于将资源投入漏桶**。
7.  **保持盈利**：不再追加获客，而是用 [[NDR（净收入留存率）]] 让老客户复利增长。
