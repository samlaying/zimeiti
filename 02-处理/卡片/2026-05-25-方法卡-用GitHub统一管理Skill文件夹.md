# 用GitHub统一管理Skill文件夹

使用**GitHub**来管理和同步你的[[skill]]文件夹，确保跨设备的一致性和版本控制。

**操作**：
1. **创建仓库**：将存放所有skill的文件夹作为一个Git仓库。
2. **提交到GitHub**：将本地仓库推送到GitHub远程仓库。
3. **跨设备同步**：在任何一台新电脑上，克隆该仓库即可获得完全相同的skill文件夹。

**好处**：
- **版本控制**：可以回溯skill的每一次修改。
- **跨设备无缝切换**。
- **便于协作与分享**（如果需要）。
