# CLI是AI交互的趋势

GUI（图形界面）是人类擅长的交互方式，但并非 AI 最优解（如 AI 难以理解时钟，且视觉 Token 消耗高）。

未来的趋势是 AI 将更依赖 **CLI（命令行界面）** 或直接 API 调用。因为 CLI 更高效、可溯源、可回滚，是 AI Agent 与操作系统沟通的更佳方式。

这意味着传统 GUI 交互方式将逐渐过时，软件行业（如提供 [[软件是AI的手和脚]]）的角色也将向为 AI 提供可调用的工具接口演进。
