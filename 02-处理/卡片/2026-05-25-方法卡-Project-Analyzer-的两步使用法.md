# Project Analyzer 的两步使用法

使用 `Project Analyzer` [[Skill]] 生成逆向PRD文档的过程仅需两步：

1.  **第一步：执行分析**
    - 进入目标项目的路径。
    - 在终端输入命令 `/Project Analyzer` 调用Skill。
    - Skill开始自动提取和分析项目中的提示词及代码结构。

2.  **第二步：生成PRD**
    - Skill分析完成后会提示用户确认翻译和提取的提示词是否准确、无遗漏。
    - 确认无误后，输入指令：“**请启动PRD生成**”。
    - Skill随即生成包含完整14章节的PRD文档。
