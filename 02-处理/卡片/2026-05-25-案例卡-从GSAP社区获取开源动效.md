# 从GSAP社区获取开源动效

这是一个完整的[[VibeMotion]]实操案例：从网页上获取开源动效代码。

1.  **定位动效**：在 **GSAP 动画社区**中找到喜欢的动效，例如 `RollingTags`。
2.  **导出代码**：进入该动效的 **CodePen** 页面，点击右下角的 `Export` -> `Export ZIP`。
3.  **本地准备**：将下载的ZIP文件解压，得到一个包含动效代码的文件夹。
4.  **编辑打开**：使用 **VSCode** 或 **Cloud Code** 等代码编辑器打开这个文件夹，即可获得可供AI魔改的原始代码。
