# 本地账套查询与分析

这是一个展示 [[CPAHelper Excel Agent]] 与本地数据库交互的完整案例：查询并分析导入的财务账套数据。

**流程概要**：
1.  **前提准备**：需要在 CPAHelper 专页提前导入本地账套文件（如 .txt 格式）。
2.  **调用专用Scale**：在 Agent 对话框中输入 `/`，选择用于查询账套的 Scale（如“账套的viewer”）。
3.  **下达查询指令**：用自然语言提出需求，如“看一下账套里面的管理费用的情况”。
4.  **AI执行查询**：Agent 会自动编写 SQL 语句，从本地数据库中查询相关明细数据（如39个明细科目及金额）并汇总返回。
5.  **结果写入Excel**：可进一步指令“把结果写入当前的Sheet”，Agent 会将查询结果整理并填入指定的 Excel 工作表中。

此案例体现了 Agent 的核心价值：将“数据查询（SQL） -> 数据整理 -> 结果呈现”的繁琐流程完全自动化。
