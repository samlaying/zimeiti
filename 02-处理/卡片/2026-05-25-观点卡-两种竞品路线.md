# 两种竞品路线

AI技能聚合项目存在两种不同的产品路线，对应不同的用户需求：

1.  **Voltagen的Awesome Agent Skills（精选路线）**：
    - 特点：只收录官方团队技能，**少而精**。
    - 适用场景：想要**最靠谱**的那一小撮核心技能。
2.  **Anti-Gravity Awesome Skills（广度路线）**：
    - 特点：收录1431个技能，包罗万象，**大而全**。
    - 适用场景：想要一个覆盖开发、测试、安全等各方面的**军火库**。

**选择建议**：取决于你更看重“少而精的可靠性”还是“大而全的覆盖度”。
