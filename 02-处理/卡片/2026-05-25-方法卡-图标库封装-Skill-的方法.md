# 图标库封装 Skill 的方法

将本地[[图标库]]封装成可执行[[本地图标库 Skill]]的流程如下：
1.  **规范导出**：首先将图标设计规范导出为 PDF 格式。
2.  **工具输入**：将规范 PDF 输入到特定的设计或自动化工具中。
3.  **执行与输出**：执行工具，其结果（如处理后的图标）会自动写入 3D 等目标文件，形成可执行的 Skill。
4.  **人工干预**：在 AI 执行过程中，必须进行人工检查、修改和取用，以弥补自动化的不足。
5.  **最终应用**：将 Skill 部署到特定系统，使用时输入想要的图标名称即可调用。
