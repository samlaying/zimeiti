# Agent Skill 个性化助教的三样产出

一个设计良好的个性化学习助教 [[Agent Skill]] 可以为学生提供三样核心产出：

1.  **IPython Notebook**：一个可运行的文档，包含清晰的步骤、关键发现、代码、运行结果，以及所有问答卡点的记录和反思。
2.  **专属学习报告**：面向人类阅读，包含任务摘要、Checkpoint（记录的问题）、每个步骤需掌握的知识点及学生表现，以及可分享给教师的学习画像与课上补讲建议。
3.  **个性化教程**：针对学生在问答中暴露的卡点和错误答案，提供针对性的讲解、代码示例和练习题。
