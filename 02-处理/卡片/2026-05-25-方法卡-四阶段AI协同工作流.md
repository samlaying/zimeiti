# 四阶段AI协同工作流

将AI协同开发过程分为探索、规划、实现、提交四个阶段，确保逻辑清晰。

## 核心理念
像装修房子一样，遵循“勘测构思 -> 出图确认 -> 严格施工 -> 验收记录”的流程。

## 在Claude Code中的实践
1.  **探索**：明确问题，可结合 [[AI深度访谈法生成需求]] 生成 `Size.md`。
2.  **规划**：**切换到 `Plan mode`**（通过`Shift+Tab`）。在此模式下，AI不会写代码，仅专注于根据需求生成技术方案。
3.  **实现**：讨论、调整方案直至满意后，**切回 `Normal mode`**，让AI严格依据已确认的方案施工（写代码）。
4.  **提交**：整理并提交成果。

## 注意
对于微小问题，可让AI直接修复以节省Token。
