# 实操：从创建到后台自动化

使用 Codex 2.0 构建应用或自动化的典型流程如下：
1.  **创建新项目**：点击“开启新项目”，创建一个新文件夹。
2.  **输入指令**：在项目内输入具体需求，例如“创建一个精美好玩且有趣的生产应用”。
3.  **设置权限与模型**：
    -   设置权限（完全访问或默认权限）。
    -   选择要使用的AI模型（如GPT5.4、GPT5.3 Codex等）。
    -   根据任务复杂度调整推理级别。
4.  **执行与构建**：运行任务，可切换模型测试效果，或添加图片等素材丰富内容。
5.  **启用后台自动化**：通过自动化功能设置任务，让 Codex 在后台自动运行。
