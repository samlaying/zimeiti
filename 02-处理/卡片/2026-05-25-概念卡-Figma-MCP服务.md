# Figma MCP服务

MCP（Model Context Protocol）是Figma提供的一种服务，用于连接外部AI工具（如ChatGPT）。

**关键点**：
- 需要Figma**交易版或付费版本**才能开启此服务。
- 它实现了[[ChatGPT与Figma的自动化连接]]，使得AI生成的内容能直接导入Figma，是[[从位图到矢量图的设计稿工作流]]的技术基础。
