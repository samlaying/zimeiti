# JS事件监听与类切换模式

> `classList.toggle`是控制UI状态切换的优雅方式。

## 核心内容
为交互元素（如汉堡菜单按钮）添加`click`事件监听。在回调函数中，使用`classList.toggle('className')`方法来为目标元素（如导航容器）切换一个预定义的CSS类。这个类通常包含了改变元素状态（如位置、可见性）的样式，从而实现组件的显隐切换，实现逻辑清晰且与样式解耦。

## 原文关键段落
> burger.addEventListener('click', () => { 
>     nav.classList.toggle('nav-active'); 
>     burger.classList.toggle('toggle'); 
> });
