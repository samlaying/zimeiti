# 飞书多Agent协作插件

插件名称为 **Fayshul Boat Chat Plugin**，旨在实现飞书群聊中多个AI Agent（机器人）之间的互相艾特与协作，以模拟团队工作流程，自动完成产品交付。

### 核心功能
- 支持群内多个Agent（如产品、前端、后端、测试）互相通信、分配任务。
- 通过关键词（如“群内协作”）触发，自动发现并启动群内所有Agent的协作流程。

### 依赖条件
1. OpenCloud 版本大于 4.12。
2. 已安装飞书官方插件 OpenCloud-LARC。
3. 已配置多个Agent机器人绑定到飞书。
