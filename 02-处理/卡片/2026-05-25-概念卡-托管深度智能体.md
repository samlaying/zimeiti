# 托管深度智能体

**托管深度智能体 (Hosted Deep Agents)** 是 [[LangSmith]] 平台提供的一项新能力，允许开发者直接将深度智能体（Deep Agent）托管到线上运行。

其关键特性在于支持 **`Profiles`**，即允许开发者通过提供配置文档（如 `agent.md`、技能文件、MCP配置等）而非完整代码来定义和托管智能体，这显著降低了部署门槛。

同时，平台推出了更简化的 **`Deep Agents API`**，让开发者能将更多注意力放在应用层（如 [[Sandbox 与 Context Hub]] 和 Skills）的开发上。这使得智能体的开发、部署到运行变得更快捷、更符合直觉。
