# C端产品经理

C端产品经理负责设计用户直接使用的功能（产品的“壳子”）。

**职责**：设计用户界面和交互流程。

**以微信为例**：
- 设计聊天、朋友圈、群聊等功能。
- 决定聊天框位置、群聊消息提醒方式等。
- 因用户和环境变化，功能不断迭代（如聊天功能从文字、语音扩展到视频、表情包、红包、绿包、礼物等）。
