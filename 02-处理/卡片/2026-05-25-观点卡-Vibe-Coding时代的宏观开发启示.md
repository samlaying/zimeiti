# Vibe Coding时代的宏观开发启示

[[Superpowers Skills 框架]]所代表的理念，适用于所有AI编程场景，其宏观启示是：
- **[[小步快跑，模块化]]**：将大任务拆解为小模块，逐步验证，控制风险。
- **意图明确，有据可依**：让AI基于明确的需求文档（`requirements.md`）和现有代码规范工作，而非模糊的指令。
- **[[验证为王]]**：每个步骤都尽可能进行自动化验证（如测试、静态检查），将问题扼杀在早期。
在[[Vibe Coding]]时代，驾驭强大AI工具产出高质量软件的关键，在于用一套严谨的**流程和规范**来约束和引导AI的行为，而非完全放任其自由生成。
