# 插件使用五步法

使用视频/图片反向分析插件的标准操作流程：
1.  **安装插件**：在浏览器应用商店搜索并安装。
2.  **定位目标**：在网页中找到想要分析的图片或视频。
3.  **一键分析**：点击插件图标或右键菜单激活功能。
4.  **获取结果**：插件自动生成结构化的AI提示词。
5.  **复制使用**：将提示词复制到AI绘画或视频生成工具中进行创作。
