# AI动态设计生成工作流

利用AI生成动态设计图的**核心工作流**如下：
1.  **准备视频素材**：录制一个清晰展示你想要的**动态效果**（如加载动画、悬停效果）的短视频。
2.  **上传至视频托管服务**：将视频上传至如 [[Amazon CloudFront]] 这类CDN服务，获取一个**稳定、公开可访问**的链接。这是成功的[[视频托管链接稳定性]]关键前提。
3.  **嵌入链接到提示词**：在提示词中明确指定AI从该链接的视频中学习何种动态风格，并将其用于生成何种新设计。例如：“请参考这个动态效果 [视频链接]，为我生成一个具有类似风格的动态登录按钮。”
4.  **发送并获取结果**：将包含视频链接的完整提示词发送给AI模型，最终获得可预览的动态设计图或代码片段。
