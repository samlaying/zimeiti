# 蓝印RPA Agent功能

蓝印RPA近期上线的 **Agent** 功能，其核心是通过**个人微信**作为交互入口，实现对蓝印客户端上所有已部署[[RPA脚本]]的**远程控制**（如启动、停止任务）。

它旨在提供一种轻量、便捷的调度能力，用户无需守在电脑前或登录专门的网站控制台，即可通过微信管理自动化流程。
