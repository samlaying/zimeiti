# 选择工具的决策树

根据项目需求和个人角色选择合适的工具：

- **选择 [[Google Stitch]] 的情况**：
  - **赶时间**，需要快速出效果。
  - 希望**门槛低**，生成质量高。
  - 采用“生成 → 复制代码 → 交付开发”的快速工作流。

- **选择 [[Pencil]] 的情况**：
  - **专业开发者**或负责**长期项目**。
  - 虽然有安装等入门门槛，但看重**过程的可控性**。
  - 需要设计稿能够**持续迭代和精细调整**。
