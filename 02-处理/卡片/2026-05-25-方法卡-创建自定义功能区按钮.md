# 创建自定义功能区按钮

将常用宏命令添加到 Word 功能区，形成一键式按钮，操作步骤如下：

1.  进入自定义设置：`文件` -> `选项` -> `自定义功能区`。
2.  **新建选项卡**：在右侧窗格点击 `新建选项卡`，并将其重命名（如“我的工具”）。
3.  **添加宏命令**：在左侧 `从下列位置选择命令` 下拉菜单中选择 `宏`，找到你的宏（如 `化学室数字小写`），点击 `添加`，将其移入右侧你新建的选项卡下的组中。
4.  **个性化设置**：可以重命名按钮，并选择一个更直观的图标。

完成后，功能区会出现你的自定义选项卡，点击按钮即可运行对应宏。
