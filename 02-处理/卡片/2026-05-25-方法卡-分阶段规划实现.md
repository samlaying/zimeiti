# 分阶段规划实现

遵循“探索 -> 规划 -> 实现 -> 提交”的四阶段工作流，可以将无序的[[Vibe Coding]]转变为可控、清晰的AI协同开发流程。

**在Claude Code中的应用**：
1.  **探索与规划阶段**：在需求文档（如 `Size.md`）生成后，通过 `Shift+Tab` 切换到 `Plan mode`。此模式下AI专注于制定详细技术方案，不会编写代码。
2.  与AI讨论、调整方案直至满意。
3.  **实现阶段**：切回 `Normal mode`，让AI严格依据已确认的方案编写代码。
