# 视频剪辑工作流演示

UP主使用[[罐头的Snapflow]]剪辑一个38分钟远程webcoding教程的完整工作流案例：

1. **拼接**：将5段分录视频通过节点一键拼接。
2. **去静音**：使用[[去除静音片段]]节点（阈值-18dB），时长从38分钟压缩至20分钟。
3. **变速**：使用[[视频变速]]节点（1.25倍速），时长进一步压缩至16分钟，提升听感。
4. **识别与校对**：通过[[Whisper模型]]本地生成字幕，再经[[AI字幕校对]]自动纠错（示例中修正34处）。
5. **交互编辑**：在独立页面微调字幕，删除句子会同步删除视频片段。
6. **输出**：一键生成烧录字幕视频、带封面视频、B站章节参考及多平台文案。

**核心价值**：通过工作流自动化，将繁琐的手工剪辑变为可配置的流水线，显著提升长视频处理效率。
