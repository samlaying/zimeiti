# 有意识采集优质信息到素材库

建立高质量选题素材输入的第一条路径：日常浏览社交媒体（如推特、YouTube）时，有意识地采集有价值内容。

**操作步骤**：
1. 使用浏览器插件，一键将有用内容采集到本地笔记工具（如Obsidian的素材库）。
2. 插件可自动将内容归类到相应文件夹，便于管理。
3. 采集后，利用AI工具（如[[Obsidian+Claude工具组合]]中的Claude）直接分析素材，提取可用选题点或值得沉淀的优质信息。
4. 将筛选出的“火素材”存入库中，待需要时直接调用。

这个方法确保了持续、有意识地从海量信息中筛选并存储高质量素材。
