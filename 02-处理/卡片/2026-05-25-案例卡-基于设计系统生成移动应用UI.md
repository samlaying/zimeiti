# 基于设计系统生成移动应用UI

**目标**：基于一份设计系统文档，为可折叠手机应用生成全新的Figma设计页面。
**模型**：Claude Opus 4.6。
**关键步骤**：
1.  使用**计划模式**，AI生成六阶段执行计划。
2.  依次使用 `Create a new file` 和 [[Use Figma Tool：AI直接在画布上设计]] 在画布上创建设计Token、颜色变量、屏幕。
3.  创建后AI会截图验证，并利用[[Skills：AI设计的自我修复循环]]进行优化。
**结果**：一次性生成了四个遵循设计系统的页面（包括内屏、外屏折叠态等），设计淡雅，配色和排版受到好评。
