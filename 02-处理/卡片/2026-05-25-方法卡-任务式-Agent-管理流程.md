# 任务式 Agent 管理流程

基于 [[Symphony 任务管理框架]] 实现 [[任务式 Agent 协作]] 的标准工作流程：
1.  **发布任务**：用户在任务板中创建新任务（如“帮我出一份DeepSeek V4的分析报告”）。
2.  **自动执行**：Agent 自动获取任务并开始执行。
3.  **状态更新**：任务完成后，Agent 将状态改为 `human review`，提示需要用户介入。
4.  **用户验收**：用户检查任务产出，确认质量。
5.  **任务完成**：验收通过后，用户将任务标记为 `done`，流程结束。
