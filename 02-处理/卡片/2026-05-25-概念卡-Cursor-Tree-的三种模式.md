# Cursor/Tree 的三种模式

以 Cursor、Tree 为代表的 [[Vibe Coding]] 工具，通常提供三种工作模式，对应不同的能力等级：
1.  **Chat**：仅具备**读取**项目文件的能力，不会进行任何修改。
2.  **Builder**：既能读取，也具备操作和修改本地文件的功能。
3.  **Builder with MCP**：在 Builder 基础上，增加了用户自行配置**各种 MCP 服务**的能力，从而连接外部工具。

产品经理的核心能力跃升始于熟练使用 **Builder with MCP** 模式。
