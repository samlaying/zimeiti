# 急速出图

使用一句“咒语”（特定指令）快速生成一个完整的 UI 界面，适用于快速原型设计或初稿制作。

**操作步骤**：
1. 编写或获取用于描述目标 UI 的“咒语”。
2. 将咒语直接发送给 Pensel UI 工具。
3. 工具自动生成一个完整的 UI 界面。

此方法是 [[Pensel UI 定义]] 中“快速生成 UI 界面”能力的具体实践，体现了 [[AI设计工具优势]] 中的“快”。
