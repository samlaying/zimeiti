# Routines 工作原理

Routine 是任务生成器，每次执行时会自动创建一个新的 issue，分配给指定的 agent（如 CTO、CEO），然后立刻唤醒该 agent 来处理任务。这类似于 cron job，但创建的是 issue 而非脚本，更集成于 AI 工作流。
