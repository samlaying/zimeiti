# Figma插件一键复刻网页

在Figma中使用 [[rtml.to.dx插件]] 复刻网页的两种方法：

**方法一：在Figma内直接输入网址**
1.  在Figma中打开 `rtml.to.dx` 插件。
2.  在输入框中粘贴目标网页的完整URL地址。
3.  点击操作按钮，在弹出的配置弹窗中（一般保持默认，注意避开“Pro”付费选项），点击确认。
4.  等待插件分析并插入网页内容。

**方法二：通过浏览器扩展下载文件后导入**
1.  确保已在Chrome浏览器安装对应的 `rtml.to.dx` 扩展。
2.  在Chrome中打开目标网页，点击扩展图标，选择第一个选项抓取页面。
3.  将分析后的页面保存为一个 `.h2d` 格式的文件到本地。
4.  回到Figma的 `rtml.to.dx` 插件界面，将 `.h2d` 文件拖入。
5.  点击确认按钮，插件自动解析并导入。
