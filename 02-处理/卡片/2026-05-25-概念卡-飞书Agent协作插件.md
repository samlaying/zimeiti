# 飞书Agent协作插件

一个名为 **Fayshul Boat Chat Plugin** 的开源插件，其核心目的是实现飞书群聊中多个Agent机器人之间的互相艾特和通信，从而模拟团队工作流程，自动完成产品交付。它依赖于OpenCloud（版本>4.12）和飞书官方插件OpenCloud-LARC。
