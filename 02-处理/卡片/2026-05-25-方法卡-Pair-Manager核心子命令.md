# Pair Manager核心子命令

[[Pair Manager]]提供六个核心子命令，用以自动化管理Obsidian笔记库：
1.  **整理收件箱**：扫描`00-inbox`文件夹，逐条引导将笔记归档到正确位置。
2.  **同步MOC**：自动检测目录变化，更新所有[[MOC]]文件的链接列表。
3.  **更新首页**：根据笔记库当前状态，重新生成`Home`页的导航内容。
4.  **归档项目**：一键将项目文件夹移入归档文件夹，并同步更新所有相关链接。
5.  **新建条目**：在对应的PARA目录下创建新条目并自动生成模板。
6.  **全库健康检查**：扫描孤立文件、断链、空目录等五类问题，生成结构化报告。
