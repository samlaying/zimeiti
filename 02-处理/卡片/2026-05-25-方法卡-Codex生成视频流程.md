# Codex生成视频流程

使用[[Codex AI工具]]和[[HaperFrames插件]]生成动态视频的流程：
1. 下载并安装Codex，登录GBA账号。
2. 在Codex中安装HaperFrames插件。
3. 输入详细提示词，控制视频内容、时长、尺寸和风格。
4. 发送生成，Codex自动搜索资料、生成内容并更正问题。
5. 耗时约10分钟，完成后查看效果，无需预先模板。
