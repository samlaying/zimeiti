# OpenPose控制模特姿势

使用[[OpenPoseEditor插件]]与[[ControlNet]]来控制或修改AI生成的模特姿势：
1. **提取原图姿势**：在ControlNet单元中上传原图，选择“OpenPose”预处理器并生成，即可提取出原图的**骨骼图**。
2. **编辑目标姿势**：将骨骼图发送到`OpenPoseEditor`插件，在插件中**手动拖动、旋转骨骼点**，调整至想要的新姿势（如调整手部、腿部位置）。
3. **姿势约束生成**：
   - 将编辑好的姿势图发送回ControlNet。
   - **关键调整**：若同时启用多个ControlNet单元（如Canny控制细节+OpenPose控制姿势），需通过调整**权重**来平衡影响。通常可调高OpenPose的权重（如0.7），适当调低Canny的权重（如0.6）。
4. **问题处理**：如生成结果出现畸变，可开启**ADetailer插件**进行自动修复。
