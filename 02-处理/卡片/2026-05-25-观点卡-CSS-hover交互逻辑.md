# CSS :hover交互逻辑

使用纯CSS的`:hover`伪类，而非JavaScript，是实现鼠标悬停交互的轻量化和高效方式。其核心逻辑在于父子选择器。

## 设计模式
- **原理**：将需要显示的子元素（如菜单`.menu`、头像`.avatar`）默认隐藏。
- **触发**：当鼠标悬停在父容器（如`.glass-nav`）上时，通过选择器 `.glass-nav:hover .menu` 重新定义子元素的样式，将其设为可见。
- **优势**：代码简洁，无需脚本，性能开销小，适合基础交互效果。
