# 复刻GitHub网页到Figma

以复刻 `https://github.com` 网页为例，演示 [[rtml.to.dx插件]] 的使用流程：

1.  在Figma中打开插件，在地址栏输入GitHub的网址并点击抓取。
2.  在弹出的配置弹窗中保持默认（注意避开付费选项），点击确认。
3.  插件分析完成后，GitHub的页面内容（包括图片、文字、部分布局）被完整插入到Figma画布中。
4.  插入的页面中，所有文字和图片都是独立可编辑的对象，可以直接点击修改。
5.  如果页面高度不足，可以手动拉伸图层组调整至完整页面高度。
