# Anthoropic Skill 作为评测基准

GitHub 近 10 万热度的 [[Anthoropic Skill]]（推荐使用其 **Front End Design** 子 Skill）因其高效与精致，适合作为评测 Design Skill 的基准。

它的核心优势：
1.  **设计质量高**：产出干净、精致，字体使用克制，大标题高亮设计利于用户扫读，减少认知负荷。
2.  **极致高效**：其 Design Scale 仅 **43 行**，在控制力和产出质量间取得了优秀平衡。

基于此，在 [[Vibe Coding 评测方法|评测方法]] 中，任何低于此标准的 Skill 产出被认为没有下载必要。
