# Claude Code 三层记忆架构

Claude Code 的记忆系统是一个**三层渐进的知识管理体系**，用于支持长期记忆和团队协作。

1.  **会话记忆**：存储单次会话中的临时信息。
2.  **持久记忆**：长期保存用户偏好和项目知识，以 markdown 文件格式存储，包含四种类型：用户角色/目标/知识背景、工作方式指导、项目进展/目标/事件、外部系统指针。
3.  **团队记忆**：支持团队协作，共享知识和进展。

持久记忆的前两种类型与 [[OpenAI 的记忆系统]] 类似，但 Claude Code 的实现更加结构化。这种分层设计是 Claude Code 能进行有效 [[Claude Code 四层上下文压缩算法]] 的基础。
