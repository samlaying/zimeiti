# 蓝印RPA的Agent功能

蓝印RPA近期推出的一项新功能，其核心是通过**个人微信**，实现对蓝印客户端上已部署的所有脚本应用的**远程控制**（如启动、停止任务）。它旨在提供类似企业版控制台的调度能力，但通过更普及的微信作为入口，使用更为便捷。

与[[影刀RPA企业版控制台调度]]的对比：实现相似效果，但交互入口更轻量。
