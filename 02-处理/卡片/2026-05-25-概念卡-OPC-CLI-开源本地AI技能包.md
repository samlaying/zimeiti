# OPC CLI 开源本地AI技能包

**OPC CLI** 是一个开源的本地AI技能包，其核心价值是将多种常用的[[多模态模型]]组合成命令行工具，旨在帮助用户（尤其是一人公司）在本地环境运行以提升效率和节省成本。

- **核心功能**：集成了[[TTS]]（文字转语音）、[[ASR]]（自动语音识别）等模型，提供如剪辑口播视频、生成特效字幕、语音合成等能力。
- **配置要求**：Mac需M系列芯片，PC最低6G显存。
- **使用方式**：可将Github代码库交给AI Agent（如“小龙虾”）自动使用；或手动配置模型下载目录的环境变量，通过`UVSync`自动安装依赖。
- **深层定位**：它不仅是一个实用工具，更是一个用于培养用户对模型能力边界和技术洞察力的训练场。
