# LangExtract

LangExtract 是谷歌开源的一个 Python 工具库，其核心功能是利用大模型从海量的非结构化文本（如网页、TXT文件）中，精准地提取出清晰、且**来源可溯源**的结构化信息。

它解决的核心痛点是：在【合同审核】、【医疗文本处理】、【法律文书分析】等场景中，需要从大量非结构化文本中可靠地提取关键数据。
