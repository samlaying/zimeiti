# 星星收集系统设计

该游戏的[[星星收集系统]]设计体现了几个关键理念：

1.  **稀缺性与独特性**：每颗星星都是“独一无二”的，且一旦错过可能无法再次获取。这创造了收集的紧迫感和珍贵感。
2.  **探索驱动**：星星位置“藏得比较隐蔽”，线索“藏在角落中”，强制玩家进行细致的环境观察和探索，而非线性推进。
3.  **辅助引导**：使用“罗盘”根据场景给出提示，平衡了探索的挑战性与避免玩家完全卡关。
4.  **叙事与玩法融合**：收集星星不仅是收集品，其主题“寻找属于你的那颗星星”本身也呼应了游戏的故事内核。
