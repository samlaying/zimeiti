# 公众号转HTML方法

在[[COZE技能与SPL|COZE平台]]上，通过自然语言指令调用特定[[COZE技能与SPL|SPL]]，可将公众号文章保存为本地HTML网页文件。

**操作步骤**：
1.  **新建任务**：在COZE平台中创建一个新对话。
2.  **调用技能**：发送指令“请调用公众号下载为网页SPL”。
3.  **提供链接**：粘贴目标公众号文章的链接地址。
4.  **执行与获取**：点击发送，任务完成后，生成的HTML文件会保存在 **`Output`** 文件夹中。

**效果**：生成的HTML文件能基本复刻原文的图文排版。
