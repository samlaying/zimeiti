# 查看与删除Claude Code追踪标识

Claude Code 通过多种永久标识（如 User ID、Account ID、邮箱）追踪用户。若担心封号，可按以下步骤清除这些标识，使客户端如同新用户：

1.  **查看当前标识**：运行诊断命令（如 `claude --diagnostics`）查看当前存储的追踪 ID。
2.  **删除追踪标识**：执行特定脚本删除存储 User ID、Anonymous ID 等标识的文件内容。下次启动时将生成全新 ID，断开历史关联。
3.  **清空遥测数据**：运行三条命令清除本地缓存的遥测内容，防止数据上传分析。
4.  **清除OAuth关联**：通过脚本查看并清空 OAuth 信息，断开账号关联。
5.  **完全重置（可选）**：执行一系列命令完全重置 Claude Code 保存的所有信息，等同于全新安装。
