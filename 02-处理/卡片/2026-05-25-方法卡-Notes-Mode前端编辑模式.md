# Notes Mode前端编辑模式

一种借鉴Notion“Block”概念的功能，允许在网站前端直接以可视化、模块化的方式编辑和调整页面内容，从而大幅降低频繁更新网站的技术门槛和出错率。

**核心操作流程**：
1.  **进入模式**：在网站上启用“Notes Mode”。
2.  **可视化编辑**：页面中的各个内容板块（文本、列表、概念卡等）变为可交互的“块”。
    -   **调整布局**：可以自由拖拽、移动各个“块”的位置。
    -   **增删内容**：可以新增不同格式的内容块，或删除不需要的板块。
    -   **修改文案**：直接在前端修改文字内容。
3.  **应用修改**：编辑完成后，退出时选择应用修改，系统会自动将前端调整同步更新到网站后端。

**主要优点**：
-   **降低技术门槛**：无需直接修改代码，避免因格式错误导致的崩溃。
-   **提升迭代效率**：特别适合需要频繁更新、填充内容的“活”网站。
-   **支持双语同步**：一次前端编辑可同步更新多语言版本，减少UI错配。
