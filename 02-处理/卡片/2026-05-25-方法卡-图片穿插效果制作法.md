# 图片穿插效果制作法

当看到“图片放入建筑内”的创意效果时，可通过以下两种方法学习制作：

### 方法一：利用[[AI工具]]（如豆包）辅助
1.  将包含该效果的灵感图保存到桌面。
2.  向豆包提问：“请问图中的图片穿插效果怎么做的？”
3.  豆包会识别图片并给出**制作步骤参考**（通常包括分层：主场景背景、照片下半部分、建筑主体抠出）。

### 方法二：在PPT中手动制作（基础版）
1.  **插入形状**：点击插入 -> 形状 -> **任意多边形**。
2.  **抠出遮挡物**：用任意多边形工具沿着需要遮挡的物体（如建筑）边缘绘制形状。
3.  **相交生成遮挡层**：复制物体图片，与绘制的形状进行**相交**操作，得到仅包含物体局部的形状。
4.  **调整图层顺序**：将原物体图片置于底层 -> 需“穿插”的图片置于中间层 -> 生成的**遮挡形状**置于顶层并精准对齐。
5.  **（进阶）模拟渐变透明**：使用插件（如OKPLUS）为图片下半部分添加渐变透明；或插入一个**黑色渐变填充的矩形**（调整透明度），颜色从场景图片中吸取，组合到图片下方模拟效果。
