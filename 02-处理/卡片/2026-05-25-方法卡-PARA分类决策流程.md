# PARA分类决策流程

面对一条新笔记时，通过以下顺序提问来决定其放入[[PARA笔记管理法]]的哪个桶：
1.  与当前某个[[Projects项目]]直接相关吗？是 → 放入 Projects。
2.  是持续负责的[[Areas领域]]吗？是 → 放入 Areas。
3.  是知识或[[Resources资源]]参考资料吗？是 → 放入 Resources。
4.  都不是 → 删除或放入 [[Archive归档]]。
