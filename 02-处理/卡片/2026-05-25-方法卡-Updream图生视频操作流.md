# Updream图生视频操作流

在[[Updream平台概述|Updream]]中，将静态图像转换为动态视频的标准操作流程。

**步骤**：
1.  **创建图像节点**：在[[无限画布]]上右键，选择添加图片节点。配置模型（如C-Dream 5.0）、比例，并输入提示词生成初始图像。
2.  **添加视频节点**：从图像节点拖出“+”号，选择“视频生成”模式。
3.  **配置视频节点**：
    - 子模式选择“图生视频”。
    - 模型选择`C-Dance 2.0`。
    - 设置画质（如1080P）和比例（如16:9）。
    - 输入详细的提示词，描述动作、镜头、声音等。
4.  **运行生成**：点击发送，[[节点式工作流]]将自动连接并执行，生成视频。
