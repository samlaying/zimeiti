# AI工具开发四步法

通过自然语言与AI agent对话开发个人工具以建立信心的方法：
1. **定义需求**：从一个简单需求入手（如：建立一个AI新闻获取工具）。
2. **AI咨询**：向[[AI聊天机器人]]询问“什么新闻源比较好”，获取建议。
3. **描述给Agent**：在agent（如cloud code）中清晰描述你的具体需求。
4. **需求细化**：若不知如何描述，可先向AI求助“我想做一个新闻搜索工具，但不知道怎么描述需求，帮帮我”，通过对话理清思路。
5. **开发与迭代**：让agent基于清晰需求开发工具，并持续测试优化。
