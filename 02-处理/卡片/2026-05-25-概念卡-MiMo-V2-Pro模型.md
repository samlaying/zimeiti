# MiMo-V2-Pro模型

MiMo-V2-Pro是小米推出的多模态AI大模型，属于 [[小米MiMo系列模型]] 中的性能版本。

**核心特点**：
- **前端UI生成能力强**：在多种应用场景（如原型图、小游戏、管理系统）下能生成细节丰富、逻辑正确的界面。
- **支持真实代码转化**：生成的原型图可轻松转化为实际应用代码。
- **响应高效**：设计上注重效率，能用简单提示词实现复杂输出。
