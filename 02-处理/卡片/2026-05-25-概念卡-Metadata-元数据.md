# Metadata（元数据）

每个独立的系统提示词文件开头的一段简短描述信息，用于描述该文件的功能和使用条件。

## 解决的问题
- 将不同场景的 [[System Prompt（系统提示词）]] 拆分为独立文件，避免全部塞入一个文件。
- 避免一次性加载所有文件内容消耗大量Token。

## 工作流程（路由）
1. 用户输入请求（如“帮我写个简历”）。
2. AI客户端将所有文件的**元数据**发给模型（Token消耗少）。
3. 模型根据元数据判断所需场景，返回指令。
4. AI客户端加载对应文件的**完整内容**作为系统提示词，再发送给模型生成内容。
