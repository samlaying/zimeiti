# Skill制作四法：从隐性到显性

系统化的四种将个人能力与工作流程转化为标准化[[Skill]]的方法。

**四种方法**：
1.  **样板法**：将自己最满意的交付物（如文档、方案）交给AI，让其学习并提炼成可复用的Skill。
2.  **流程法**：写下清晰的工作步骤，将其中可串联、自动化的部分提炼为Skill。
3.  **归纳法**：回顾与AI的聊天记录，找出反复让AI执行的同类型高频任务，将其补全为正式Skill。
4.  **经验法**：将个人的隐性知识（如设计方法、写作技巧）显性化，写成文档或搜索通用方法，封装成Skill。
