# Remotion：React视频创作框架

**Remotion** 是一个基于React的视频创作框架。它允许开发者使用熟悉的Web技术（如React组件）来编写、定制和生成高质量的动画与视频。其核心优势在于将视频内容生产代码化，实现了**可编程、可版本控制、可参数化**的动画制作流程。
