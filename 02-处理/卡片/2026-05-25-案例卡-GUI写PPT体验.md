# GUI写PPT体验

通过 Codex Computer Use 功能，以 GUI 方式（非 API 调用）创建 PPT 的体验：
1. **任务描述**：要求 AI 以“AI发展历史”为主题，通过 GUI 操作制作一个美观的 PPT。
2. **操作流程**：AI 先分析 PPT 中的可交互元素（文本、按钮、区域），然后通过模拟鼠标点击应用设计、插入图片（可生成）、调整格式。其打字方式为一段一段输入，而非逐字。
3. **能力观察**：AI 能识别 PPT 元素，部分得益于与微软的深度合作（如 Office API），仅靠截图识别成功率较低。

该案例主要为能力演示，展示了 AI 在 [[GUI与API调用对比]] 中的进步，但实际应用可能有限。
