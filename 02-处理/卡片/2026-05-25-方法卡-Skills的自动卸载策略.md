# Skills的自动卸载策略

在对话轮次结束后，自动卸载已加载的Skills内容，以保持[[LLM]]上下文清洁，防止注意力下降。具体操作：

1.  **缓存加载**：检索到的Skills文件内容先暂存于一个`cache`（缓存）中。
2.  **注入上下文**：将`cache`内容包装成`System Prompt`插入当前轮次的上下文。
3.  **清空卸载**：当本轮对话结束（模型输出最终回答后），立即清空`cache`。
4.  **效果**：在下一次对话轮次开始，重新注入上下文时，`cache`为空，从而自动实现了上一轮Skills内容的“卸载”。
