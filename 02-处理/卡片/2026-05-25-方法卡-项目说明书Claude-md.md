# 项目说明书Claude.md

通过 `/init` 命令在项目根目录生成 `Claude.md` 文件，为项目创建一份持久的“说明书”，以防止AI在长对话中“失忆”而忘记你设定的规则和约定。

**文件作用与建议内容**：
-   **作用**：作为持久的项目上下文，每次新对话AI都会优先读取，用于记录基础设定、约定和规范。
-   **建议写入**：
    -   非常规的文件命名约定。
    -   测试要求、代码风格指南。
    -   明确指定某些文件夹或文件不要动。
    -   使用 `important` 标记提醒AI注意特殊约定。
