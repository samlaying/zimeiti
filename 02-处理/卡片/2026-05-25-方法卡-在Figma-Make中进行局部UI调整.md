# 在Figma Make中进行局部UI调整

### 方法：在Figma Make中局部调整UI组件

当AI生成的UI设计稿中某个组件（如按钮、卡片）不满意时，可以进行局部调整：

1.  **选中组件**：在 [[Figma Make]] 生成的设计稿中，直接选中需要修改的UI组件。
2.  **输入指令**：在弹出的AI助手窗口中，输入具体的调整指令，例如：“UI，给我重新调整一下颜色，简单一点”。
3.  **执行修改**：发送指令，AI将仅对该选中的组件进行局部修改，不会影响整体设计。
