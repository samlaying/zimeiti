# GPT Image 2 UI 设计挑战

使用 [[GPT Image 2]] 进行 UI 设计面临双重挑战：
1. **创造力**：如何激发 AI 设计出美观、有设计感的 UI，而不是局限于现有风格。
2. **一致性**：如何确保生成的新页面 UI 与现有系统设计风格（如侧边栏、顶部栏）保持一致，避免破坏现有框架。
