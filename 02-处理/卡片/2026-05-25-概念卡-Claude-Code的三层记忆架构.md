# Claude Code的三层记忆架构

Claude Code 采用三层渐进的知识管理体系来管理记忆：
1. **会话记忆**：存储单次会话中的临时信息。
2. **持久记忆**：长期保存用户偏好和项目知识，以 Markdown 文件格式存储，包含用户角色/目标/知识、工作方式指导、项目进展和外部系统指针四类信息。
3. **团队记忆**：支持团队协作，共享知识和进展。

其持久记忆系统比 OpenAI 的记忆系统更结构化，与[[Claude Code的四层上下文压缩算法]]协同，共同提升长上下文能力。
