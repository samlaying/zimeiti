# 工具安装通用指南

安装并使用MakerJackie分享的系列Skills（如[[MJ Adapter 核心功能]]）的通用方法：
1. **访问网站**：打开浏览器，进入 skills.makejacky.com。
2. **获取命令**：在网站上找到对应技能的安装命令（例如`npm install MJ Adapter`）并复制。
3. **执行安装**：在电脑的终端（Terminal）或直接发送给AI工具（如Claude）来运行该命令（如`mpx install`或`npm install`）。
4. **开始使用**：安装完成后，即可通过发送文章或指令来调用技能，生成排版或设计。
