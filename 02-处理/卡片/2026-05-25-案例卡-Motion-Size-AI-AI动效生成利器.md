# Motion Size AI：AI动效生成利器

这是一个专注于AI动态视觉生成与灵感的网站。
**核心特点**：
- **零门槛使用**：内置大量高级视觉动效案例，用户无需自己设计。
- **资源库丰富**：拥有高质量的“线程提示词库”，包含“高级暗黑风”、“液态玻璃”等热门风格。
- **生成高效**：喜欢某个风格后，可复制其提示词到 **Lovable Dev** 平台，几分钟即可一键生成精美动态视觉页面。
