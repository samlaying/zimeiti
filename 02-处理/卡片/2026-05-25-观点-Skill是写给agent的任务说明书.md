# Skill是写给agent的任务说明书

> Skill的本质不是提示词，而是Agent的操作指南。

## 核心内容
很多人误以为Skill等同于系统提示词，其实它更准确的定义是写给AI Agent看的任务说明书。它告诉Agent在处理特定类型任务时，应该如何思考、按什么步骤执行、以及最终输出什么格式，是一套标准化的操作指南。

## 原文关键段落
> 更准确的说Skill是写给agent看的任务说明书，用来告诉agent在某类任务中应该怎么工作。
