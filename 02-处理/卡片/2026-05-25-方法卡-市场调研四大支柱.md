# 市场调研四大支柱

市场调研是品牌策划的起点，旨在了解市场、竞争对手和消费者，包含四个基本层面：
1.  **行业趋势洞察**：分析行业规模、特点、人群画像，并运用[[PESTEL宏观环境分析模型]]进行宏观分析，预测未来趋势，计算自身所处行业位置。
2.  **竞争对手调研**：重点是分析其广告内容与营销渠道，可采用[[竞争对手广告与营销调研四步法]]。
3.  **消费者调研**：通过[[消费者调研三大支柱]]构建目标客户画像，洞察其行为习惯。
4.  **市场营销调研**：分析区域GDP与消费水平，并通过[[ROI计算与营销问题诊断]]评估渠道效果。
