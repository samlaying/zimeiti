# C-Dance与VibeMotion的分工定位

在AI生成视频的工作流中，`[[C-Dance]]` 能够完成大约 **70%** 的常规视频任务。而剩余的30%中，尤其是需要**算法驱动**和**精准控制**的 **25%** 部分（例如复杂的[[科普动画]]），则是其不擅长的。这部分任务应交给 **`[[VibeMotion]]`** 来完成。两者是**互补**关系，共同构成了更完整的AI视频制作能力矩阵。
