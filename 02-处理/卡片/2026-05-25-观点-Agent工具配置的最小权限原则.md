# Agent工具配置的最小权限原则

> 防止Agent技能污染：最小权限原则。

## 核心内容
在多Agent系统中，常出现模型选择与任务看似关键词匹配、但业务逻辑完全无关的工具（工具误匹配）。解决方案是遵循“最小权限原则”：仅为每个Agent配备其当前职责所必需的工具集，避免因工具过多造成的“技能污染”，从而提高工具调用的精准度。
