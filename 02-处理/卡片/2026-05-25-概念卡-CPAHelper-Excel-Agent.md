# CPAHelper Excel Agent

CPAHelper Excel Agent 是一款国内可直接使用的免费 Excel 插件，它通过自然语言交互，让 Excel 能够“长出手来”，自动执行数据处理、分析和报告生成等复杂任务。

该插件在 Excel 的“通用”选项卡中以“智能助手”形式出现，内置了超过 20 个 Excel 专用工具，支持读取/写入数据、格式设置、创建图表/透视表、执行公式计算、查询本地数据库、运行 VBA/Python 代码等。其核心价值在于将重复、繁琐的 Excel 操作流程自动化，并通过 [[使用Scale固化工作流]] 功能实现工作流的沉淀与复用，显著提升财务、数据分析等岗位的办公效率。
