# Skillflow管理Skills四步法

使用 [[Skillflow的开关控制功能]] 管理Skills的完整流程：
1.  **发现与导入**：通过推荐浏览或主动搜索（关键词/GitHub链接）找到目标Skill仓库。
2.  **下载与预设**：下载仓库时，可预设希望导入的Agent（Cloud Code, Codex, Open Code），下载后Skills将自动存入对应目录。
3.  **查看详情**：在首页点击仓库，可查看其文件树、ReadMe介绍及单个Skill.md的具体内容。
4.  **开关使用**：在首页或详情页，通过[[Skillflow的开关控制功能]]精细控制Skills的启用/禁用状态，之后在对应的AI Agent中即可使用。
