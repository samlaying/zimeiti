# Remotion：代码驱动的动画工具

Remotion 是一个**通过代码直接制作动画的开源工具**，而非传统的拖拽式剪辑软件。

它特别适合制作高度**定制化**的动画，例如：
- 动态字卡（从四面八方跳出）
- 定制化的片头片尾
- 模拟社群媒体贴文样式
- 在屏幕录制视频中插入动画片段

其核心工作流是：在终端安装后，通过 `npm run dev` 开启 Remotion Studio 后台（`localhost:3000`）进行动画预览和导出。可以输出普通视频，或带透明背景的视频（用于剪辑合成）。
