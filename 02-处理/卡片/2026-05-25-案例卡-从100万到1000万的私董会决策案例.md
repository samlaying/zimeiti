# 从100万到1000万的私董会决策案例

**案例背景**：UP主“黄叔”作为个人运营商（OPC），年收入100万，考虑组团队扩大到年收入1000万。

**虚拟顾问团构成**：选取了南怀瑾、纳瓦尔、老毛（毛泽东）、张一鸣、乔布斯、芒格等6位思想家作为示例。

**关键辩论与决议**：
1.  **核心分歧**：[[AI杠杆]]能否替代团队；发展策略应先聚焦还是先找主要矛盾。
2.  **共识**：100万到1000万不是同一件事做更多，而是要**换一件事做**。
3.  **行动建议三步走**：诊断现状 → 聚焦并应用[[AI杠杆]] → 验证后再考虑团队。
4.  **核心洞察**：团队是放大器，不是发动机。发动机（商业模式/个人核心能力）没变，放大器加再多也没用。
