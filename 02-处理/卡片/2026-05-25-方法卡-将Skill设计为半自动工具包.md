# 将Skill设计为半自动工具包

## 核心升级
[[Skill]] 不仅能指导 Agent “怎么说”（文档），还能指导它“怎么做”（执行），升级为**半自动工具包**。

## 实现方式
在 Skill 中集成可执行的程序脚本（如 Python），形成“**文档指导 + 程序执行**”的混合能力。

## 应用示例
- **PDF 处理 Skill**：调用 Python 脚本提取表格数据、分析 SQL、生成图表。
- **数据清洗 Skill**：调用脚本执行数据清洗（文中称“消炎自断”）。

## 设计建议
将 Skill 设计为文档与代码的混合体，通过集成脚本实现对重复性任务的自动化执行。
