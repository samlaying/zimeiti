# Skill可以指挥Agent调用外部工具

## 核心观点
[[Skill]] 的作用域不应局限于本地文件。更高级的理解是，Skill 可以指导 [[Agent]] **调用外部工具**，以完成更复杂的任务。

## 工具调用示例
Skill 可以规定 Agent 在何种条件下、如何调用以下工具：
- 联网搜索
- 读取知识库
- 调用 GitHub 仓库

## 应用场景
以“竞品分析 Skill”为例，其工作流可定义为：
1. 搜索竞品官网
2. 抓取页面信息
3. 整理产品功能
4. 查询用户评价
5. 生成对比报告
6. 输出战略建议

## 意义
这极大地扩展了 Skill 的功能边界，使其能够指挥 Agent 利用多种工具实现高度自动化。
