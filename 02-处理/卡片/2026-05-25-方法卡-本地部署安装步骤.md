# 本地部署安装步骤

本地部署[[NanoBanana2]]并通过API调用的操作步骤简明，几乎无环境冲突。

**核心步骤**：
1. **下载与安装插件**：在秋叶版本管理中，使用“高区选项版本管理”功能安装提供的新拓展。
2. **注册账号**：访问AI工坊网站注册，首次注册会赠送点数。
3. **生成API密钥**：登录后，在“令牌”页面添加新令牌，提交后复制生成的API密钥。
4. **配置与使用**：将API密钥填入已安装的插件中，之后打开提供的流程文件即可调用本地API进行图像生成。
