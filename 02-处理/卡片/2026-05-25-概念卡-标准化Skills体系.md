# 标准化Skills体系

标准化Skills是一套**固化个人风格与开发规范**的指令集，用于指导AI（如Claude Code）进行视频开发。其核心是将个人设计系统“告诉”AI，避免AI凭空想象。

Skills通常包含：
1.  **设计理念**：强制AI遵循的视觉风格（如“Apple级简方影视季风的质感”）。
2.  **组件匹配规范**：优先使用已有的[[自定义组件库]]。
3.  **设计标准**：如颜色系统、设计Token，用于在没有现成组件时指导AI创造新组件。
4.  **行为与禁用规范**：如何寻找资源、禁止AI使用非标准素材等。
