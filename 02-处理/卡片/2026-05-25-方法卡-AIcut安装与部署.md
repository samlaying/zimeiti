# AIcut安装与部署

部署 [[AIcut：AI视频剪辑工具]] 的步骤相对简单，AI 自动化程度很高。
1.  **获取项目**：前往 Github 搜索并克隆 AIcut 项目到本地文件夹。
2.  **自动部署**：将项目克隆后，AI 会自动执行一系列部署命令（如环境配置），用户只需等待其完成。过程中 AI 会处理未完成的步骤，无需手动干预。
3.  **启动项目**：部署完成后，使用 `cd` 命令进入项目目录，运行 `LongApp` 命令启动项目。启动初期可能较慢，需耐心等待。
