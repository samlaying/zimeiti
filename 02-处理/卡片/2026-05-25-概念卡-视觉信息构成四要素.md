# 视觉信息构成四要素

PPT的[[视觉信息]]主要由四个部分构成：
1.  **主体**：需要重点展示的核心图片或视觉元素，应放大处理以确保视觉冲击力。
2.  **背景**：用于营造**立体感**和层次，常用技巧包括使用**透视关系的线条**和**渐变色块**。
3.  **氛围**：通过**颜色**、**元素**（如图标、抠图素材）、**字体**共同塑造页面的整体感觉（如清新、高端）。
4.  **装饰**：指页面边角等处的元素，用于平衡版面、提升精致度。
