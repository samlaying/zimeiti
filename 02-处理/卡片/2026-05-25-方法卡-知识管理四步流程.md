# 知识管理四步流程

基于 [[Hermes]] + [[Obsidian]] + [[LLM-Wiki]] 的知识管理闭环包含四个关键步骤：
1.  **初始化**：在 [[Obsidian]] 中创建一个独立仓库，通过指令让 [[Hermes]] 生成标准化的知识库目录结构。
2.  **导入与编译**：将原始资料（网页、PDF）放入指定文件夹，或通过链接采集。[[Hermes]] 会自动提取概念、实体等，编译到知识支持层，并更新日志。
3.  **检索与复用**：基于知识库直接向 [[Hermes]] 提问。有价值的问答结果会被自动补充回知识库，实现知识复利。
4.  **定期体检**：通过定时任务（Cron Job），让 [[Hermes]] 定期基于 `index` 和 `schema` 检查知识库健康状态，进行更新或清理。
