# 地图动画需接入真实地图API

制作旅游路线、航线图等**[[地图动画]]**时，不能只靠AI“脑补”地图，必须接入真实的**地图底图API**作为“外挂”。

- **国内推荐**：使用**高德地图API**，更稳定。
- **海外推荐**：使用**Mapbox**，免费额度通常足够。

**操作步骤**：
1. 前往对应地图的开发者后台，申请一个**API Key**。
2. 在与AI的提示词中明确说明需要接入地图。
3. 将API Key提供给AI，其余配置工作交给AI完成。
