# 字幕整理捉虫方法

## 方法：AI字幕校对流程
使用[[AI Skills 定义|AI Skills]]校对剪映自动生成的字幕，专用于纠正专有名词错误（如 Obsidian、Code Codex）。

**实操步骤**：
1. 在**剪映**中生成自动字幕并导出字幕文件。
2. 将字幕文本发送给配置了相应Skill的AI工具（如[[腾讯元宝/Workbody]]）。
3. AI先列出可能有问题的地方并与你确认。
4. 确认后，AI修改字幕并生成标准的SRT文件。
5. 在剪映中选择“导入本地字幕”，用AI校对后的SRT文件替换原始字幕。

**优势**：AI可以联网搜索信息，大大提高专业词汇的校对准确性。
