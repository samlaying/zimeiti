# Agent质量三维度评估法

评估Agent系统质量需从三个独立的维度入手，每个维度有对应的评估方法：

1.  **行为正确性**（关注**过程**）：衡量Agent在压力下是否遵守规则。采用**压力测试**，量化其在N次压力场景中的遵守比例。
2.  **结果质量**（关注**输出**）：衡量输出是否符合预期。必须转化为一系列**可机器验证**的客观验收条件。
3.  **可靠性**（关注**一致性**）：检验同一输入多次执行是否一致。采用**最差情况优先**策略，每个场景运行多次，用最差结果代表可靠性，以捕获偶发故障。
