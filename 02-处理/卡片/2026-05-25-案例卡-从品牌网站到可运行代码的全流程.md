# 从品牌网站到可运行代码的全流程

这是一个完整的端到端案例，展示了如何利用 Figma MCP 从零构建一个可运行的网页。

### 目标
基于多米诺骨牌风格网站，为“小鸟嗷桐理发店”生成 Landing Page 及可运行代码。

### 关键步骤
1.  **提取设计系统**：提示 AI 基于参考网站，在 Figma 中创建设计系统、组件库，并拉取真实 Logo、字体和颜色。
2.  **生成高保真设计**：AI 生成 Figma 设计，并通过手动微调优化（如调整不协调的橘色卡片）。
3.  **生成可运行代码**：将 Figma 设计链接提供给新的 AI Agent，调用 `Get Design Context` 工具生成 React 代码。
4.  **资源优化与定稿**：针对代码中的远程图片资源，调用图片生成工具生成配图并替换，最后添加动效。

### 核心价值
该案例证明了 Figma MCP 能打通**设计到开发**的链路，实现从品牌灵感、UI设计到可部署代码的全流程 AI 辅助。
