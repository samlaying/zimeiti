# Frontend Design Pro 暗色专家

`Frontend Design Pro` (Anthropic官方出品) 是**顶级视觉专家**，效果精致，极具设计感。它尤其擅长**暗色模式、液态玻璃、毛玻璃**等现代风格，视觉冲击力强。但要注意，其暗色模式应用场景相对受限，且更适合做官网或落地页，而非通用后台。
