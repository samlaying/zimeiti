# Figma设计稿转网页实战

一个完整的 `Vibe Coding` 实战案例：使用 Figma MCP 将设计稿转为静态网页。

**步骤**：
1.  **添加服务**：通过`从市场添加MCP`的方式配置 Figma MCP，并填入 [[Figma Personal Access Token]]。
2.  **下达指令**：在 `Builder with MCP模式` 下，提供 Figma 设计稿链接，要求生成网页。
3.  **AI执行**：AI 会调用 Figma MCP 读取设计稿，获取布局、颜色等信息，并尝试下载图片资源。
4.  **问题处理**：可能遇到**图片下载失败**等问题。处理策略包括：
    - 耐心观察，看模型是否自行调整。
    - 更换不同能力的模型（如从 Claude 换成豆包）。
    - 给出更明确的指令（如“使用MCP提供的图片下载功能”）。
    - 随时使用对话历史的“返回”按钮进行**版本回滚**。
