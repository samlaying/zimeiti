# Routines 应用案例

具体应用案例包括：
- CTO 每天早上 9 点自动执行 code review。
- CEO 每周一自动出一份周报。
- 当 GitHub 仓库有新的 Push 时，自动触发测试。
这些场景展示了 Routine 如何支持多样化自动化需求。
