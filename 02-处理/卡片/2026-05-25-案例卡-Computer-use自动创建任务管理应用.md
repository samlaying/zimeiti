# Computer use自动创建任务管理应用

案例展示了原代码中缺失的Computer use功能如何在桌面端实现自动化任务：
- **任务**：创建一个任务管理应用。
- **过程**：在可视化界面创建任务并选择技术栈后，Agent自动执行完整流程：
  1. 制定修改计划。
  2. 调用相应的Skills。
  3. 编写代码。
  4. 最终完成任务。
- **结果**：全程自动化，类似CLS（命令行服务）的体验，但通过[[可视化UI提升操作效率]]使操作更直观方便。
