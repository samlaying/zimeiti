# Search Agent 数据合成流程

## Search Agent 数据合成的完整流程
该方法源自论文《OpenSeeker》，利用 [[知识图谱]] 合成用于训练搜索代理（Search Agent）的复杂问答轨迹数据。

### 七步实操流程
1. **数据准备**：获取并加载知识图谱（如Wikidata）作为原始数据源。
2. **子图提取**：从图谱中随机选择一个节点，提取其局部有向子图，子图天然包含相关实体。
3. **初始问题生成**：基于子图中的实体，生成一个简单的、直接的问题。
4. **实体模糊化**：将实体名称替换为更模糊的描述（例如，“埃隆·马斯克” → “一位知名企业家”），这是驱动模型生成多跳问题的关键。
5. **复杂问题生成**：利用模糊后的实体和初始问题，通过提示大语言模型，生成需要多步推理的最终问题。
6. **质量筛选**：从生成的问题池中筛选出**难度足够**且**有解**的问题，并记录其对应的图谱路径或答案。
7. **数据组装**：将筛选后的问题、答案以及模拟的推理轨迹组装成可用于SFT的训练数据。

该流程的核心在于将结构化知识图谱转化为需要 [[多跳推理]] 的复杂问题数据。
