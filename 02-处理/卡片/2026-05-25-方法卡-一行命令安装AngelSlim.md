# 一行命令安装AngelSlim

AngelSlim 设计为易于上手，通过简单的命令和代码调用即可使用。

- **安装**：一行命令 `pip install AngelSlim`。
- **调用**：通过 Python 代码直观调用压缩功能。
- **兼容性**：压缩后的模型可直接部署到 **vLLM** 和 **SGLang** 两个主流推理框架，无需额外适配。
- **数据支持**：项目仓库提供了大量精度与速度的基准测试对比表格，便于用户评估效果。
