# 跨会话的手机远程控制流程

实现对AI编程工具的手机远程控制，且无需电脑端持续运行服务：
1.  **在电脑端**：通过命令行工具（如飞书应用）清除历史会话，激活项目选择功能。
2.  **在手机端**：从最近的工作目录列表中选择一个项目目录（如 `tempo`）。
3.  **下达任务**：在手机端直接输入指令（如“你是什么模型？有哪些能力？”）。
4.  **接收结果**：模型在指定的工作目录下执行任务并返回结果。
该流程突破了官方工具需要持续启动服务的限制，实现了真正的**跨Session远程控制**。
