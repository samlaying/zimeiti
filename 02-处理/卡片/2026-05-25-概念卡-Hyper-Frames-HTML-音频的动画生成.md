# Hyper Frames: HTML+音频的动画生成

**Hyper Frames** 是由 Hagent 公司推出的一系列专注于视频生成的 [[Agent Skill]]。

它的产出形式不是传统视频，而是 **HTML 加上音频**（音频可克隆教师声音），这为教学内容的创建和修改提供了新的可能性。
