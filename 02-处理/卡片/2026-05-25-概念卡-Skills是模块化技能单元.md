# Skills是模块化技能单元

在AI协作中，**Skills** 是将一个完整、庞大的规则文档（如设计规范MD文档），根据具体任务场景拆解成的独立的、模块化的技能单元。

其核心目的是为了让AI能专注于单一、明确的任务上下文，从而提高输出的准确性和效率，避免处理无关信息。例如，一份设计规范可以被拆分为“组件规范Skills”和“页面布局Skills”。
