# PWA 获得原生APP体验

**PWA（渐进式Web应用）** 让网站具备类似原生手机应用的体验。主要优势包括：可“安装”到设备桌面一键启动；具备离线能力，在网络恶劣时核心应用仍可启动；支持推送通知（受限于平台）。PWA 是使 Web 应用突破浏览器限制、提供可靠体验的关键技术，尤其在对抗恶劣网络环境时至关重要。
