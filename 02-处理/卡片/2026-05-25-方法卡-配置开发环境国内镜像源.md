# 配置开发环境国内镜像源

为解决访问官方源速度慢的问题，可以为 Python (pip) 和 Node.js (npm) 配置国内镜像源，如清华大学源。

**核心步骤**：
1.  **Python (pip)**：执行 `pip install -i https://pypi.tuna.tsinghua.edu.cn/simple` 命令，或使用 `pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple` 永久配置。
2.  **Node.js (npm)**：执行 `npm config set registry https://registry.npmmirror.com` 命令。
3.  使用 `pip config list` 或 `npm config get registry` 命令验证配置是否成功。

**意义**：确保后续安装依赖包（如 [[PPT-master SKILL]] 所需的依赖）时快速、稳定。
