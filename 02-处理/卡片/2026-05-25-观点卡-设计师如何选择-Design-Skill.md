# 设计师如何选择 Design Skill

选择 Design Skill 的核心建议：

1.  **关注核心设计原则**：优先考虑**字体克制**、**视觉层次清晰**、**风格一致**的 Skill。
2.  **注重用户体验**：选择能优化用户阅读体验、降低认知负荷的设计（如合理使用高亮）。
3.  **参考评测基准**：以 [[Anthoropic Skill 作为评测基准|Anthoropic Skill]] 的 43 行标准作为质量底线。
4.  **警惕不稳定工具**：避免使用如 [[Pro Max Skill 的问题案例|Pro Max Skill]] 这样质量不稳定、细节处理不佳的 Skill。

最终应根据项目的具体需求（如是否需要 [[Impossible Skill 的设计亮点|大胆风格]] 还是稳妥的基准设计）做出选择。
