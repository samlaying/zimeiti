# RICE优先级排序法

安装了[[Product Manager Skills Skill]]后，可以使用Claude进行结构化的需求优先级排序。

**操作步骤**：
1.  向Claude提供多个产品需求。
2.  指导Claude按照RICE框架进行分析。
3.  Claude会基于四个维度逐个打分：
    - **R**each (覆盖)
    - **I**mpact (影响)
    - **C**onfidence (信心)
    - **E**ffort (努力)
4.  最终给出综合排序结果，并明确指出哪些是“Quick Win”（快速获胜），哪些是“FitBit”（适合匹配），帮助快速识别高价值需求。
