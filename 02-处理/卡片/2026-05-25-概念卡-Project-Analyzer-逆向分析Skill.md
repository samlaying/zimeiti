# Project Analyzer 逆向分析Skill

这是一个专为分析已有项目（如开源项目或老项目）而设计的 [[Skills]] 工具，其核心功能是进行**逆向工程分析**，自动生成一份结构化的 [[产品需求文档|PRD]]。

它通过拆解项目代码和提示词，输出一份包含**14个章节**的详细文档，内容涵盖产品定位、用户故事、Agent流程等。同时，它还能提取并分析项目中所有提示词的存放路径、中文翻译、参数解析等。
