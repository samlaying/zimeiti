# Skill Creator：构建技能的核心工具

**Skill Creator**是Anthropic官方推荐的核心技能，专门用于构建其他[[Skill]]。它本身是一个元技能，能帮助用户将自己的工作流程、知识或专业角色封装成可复用、可执行的AI技能。

**核心价值**：
- **流程固化**：将手动、复杂的学习或工作流程自动化。
- **能力封装**：把个人或团队的最佳实践变成可分享的技能。

**应用场景**：
- 在[[做Skill的完整步骤]]中作为创建新技能的起点。
- 在[[学Skill的两种模式]]中，用于将分析Skill的流程固化，实现自动化学习。
