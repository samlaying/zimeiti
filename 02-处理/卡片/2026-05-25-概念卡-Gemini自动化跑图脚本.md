# Gemini自动化跑图脚本

这是一个基于[[油猴插件]]（Tampermonkey）的自动化脚本，由UP主“小熊猫的pandy”开发。

其核心功能是**自动化批量处理**Google Gemini的图像生成任务。用户只需将多张图片及其任务（如去除人物、更换背景）加入队列，脚本便会自动完成：发送请求 -> 接收结果 -> 下载高清原图 -> 处理下一个任务，全程无需人工干预。
