# 设置任务依赖关系

在Microsoft Project中设置任务依赖关系（最常见为FS）的操作步骤：
1. 在“前置任务”列输入依赖的任务编号（如“1”）。
2. 多个前置任务用英文逗号隔开。
3. 复杂逻辑可用“FS+1”表示隔一天开始。
4. 设置完成后，通过“网络图”或勾选“甘特图”中的“关键路径”来检查逻辑。
