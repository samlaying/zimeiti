# Anthropic产品问题：封闭式托管

Anthropic的Cloud Managed Agents采用封闭式托管方案，虽然旨在降低工程门槛，但导致企业控制权丢失，主要问题包括：
1.  **[[Harness黑盒化]]**：企业只能使用结果，难以检查和改造Agent本身。
2.  **[[记忆锁定]]**：企业无法管理和使用历史对话、长期记忆等关键资源。
3.  **模型选择受限**：只能使用平台提供的模型，缺乏灵活性。
4.  **可观测性边界**：企业只能看到平台暴露的事件，难以进行自主的审计、复盘和合规检查。

这与Expert平台的企业自主可控理念形成鲜明对比。
