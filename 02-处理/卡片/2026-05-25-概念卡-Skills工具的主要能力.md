# Skills工具的主要能力

该工具的核心能力包括：
1.  **内容理解与筛选**：识别视频中的主体、动作、场景、情绪，根据用户提示词自动筛选匹配主题的片段。
2.  **结构化剪辑规划**：根据提示词生成结构化的剪辑方案。
3.  **自动化后期处理**：可自动配置字幕、转场、滤镜和BGM。
4.  **最终输出**：生成一个完整的[[剪映专业版]]草稿工程，以及可复用的剪映配置文件。
