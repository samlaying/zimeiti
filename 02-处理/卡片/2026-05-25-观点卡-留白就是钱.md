# 留白就是钱

**高级感 = 内容密度的倒数。** 密度越低，高级感越强。

这就像在高级餐厅，盘子很大但菜量精致。应用到PPT设计上：
- **每页只讲一个核心观点。**
- **主动留出约40%的空白区域**，给观众的视觉“呼吸”空间。

不要试图在一页上塞进所有信息。在通过 [[Frontend Slides]] 和 [[Tether Scale Frontend Design]] 生成内容时，应将“大留白”作为一条核心约束。
