# MCP工具

MCP (Model Context Protocol) 工具在此上下文中特指用于实现[[搜索机制]]的本地服务端工具（如视频中提到的SpinSearchGetHub）。它通过在本地开启一个接口，让AI工具能够调用并获取互联网上的结构化搜索结果，从而辅助解决逆向工程中的问题。使用前必须先运行MCP服务。
