# DOM操作实现无限循环滑动

> 移动元素而非创建新元素，实现高效无限循环。

## 核心内容
通过JavaScript的`appendChild`和`prepend`方法移动现有DOM元素，而非创建新元素。点击“下一个”按钮时，将列表中的第一个元素移到末尾；点击“上一个”按钮时，将最后一个元素移到开头。结合CSS动画，实现视觉上无缝的无限循环滑动。

## 原文关键段落
> When the user presses next I use the following function append child with the input value being item 0 item 0 is the item in the first position append child is the function that moves the item to the end of the row and listHTML is the class containing all items... use prepend to move it up to the first position
