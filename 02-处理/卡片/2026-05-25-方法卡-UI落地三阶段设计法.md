# UI落地三阶段设计法

将UI从想法落地到产品，可分为三个逐步深化的阶段，此方法适用于[[小程序从0到1的七步开发法]]中的UI设计环节。

1.  **探索期（AI Studio）**：使用AI工具（如AI Studio的3.1模型）探索UI效果。初期不纠结细节，先用文字方案确定方向，再生成小Demo，目标是满足30分的大致风格。
2.  **转换期（技术栈适配）**：将设计稿转换为目标技术栈（如从React转换为UniApp），进行整体调整，将界面效果提升至80分。
3.  **精修期（开发者工具）**：导入真实的开发环境（如微信开发者工具），在真实场景下微调细节，达到90分效果。整个过程遵循“先粗后细”的迭代思路。
