# 将节点流程封装为Skill的四步法

在UPdream中，将已验证的节点流程封装为可复用[[Skill]]的步骤如下：
1.  **封装流程**：在画布中将搭建好的节点**打组**并**重命名**。
2.  **与Agent沟通**：在对话框中告知Agent，希望将当前制作逻辑保存为Skill。
3.  **生成代码**：Agent分析节点逻辑后，生成Skill文本内容（代码块）。
4.  **发布Skill**：复制文本，前往“技能秀区”->“我的工作台”->“创建技能”，粘贴内容并按指引发布。经官方快速审核后即可使用。
