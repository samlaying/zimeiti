# 使用Skill进行设计的核心步骤

使用[[Skill全能反推提示词]]进行设计的通用流程如下：
1. **寻找参考案例**：找到喜欢的参考图像作为基础。
2. **生成反推提示词**：使用Skill为参考图像生成详细的[[反推提示词]]。
3. **修改提示词**：根据需求，修改提示词中的核心视觉元素（如改为水果、建筑等）。
4. **生成新设计**：基于修改后的提示词，生成新设计方案。
5. **进一步编辑**：对生成的设计进行文字、颜色等细节调整。
6. **探索更多方案**：生成不同方案以供选择，并进行个性化调整。
