# 产品经理高频使用Skill推荐

从产品经理的视角，高频使用的[[Skill定义：AI的技能说明书|Skill]]主要集中在需求与设计阶段：
1.  **需求完善**：优化和细化产品需求。
2.  **下拉生成**：生成下拉菜单或类似UI组件。
3.  **流程图**：自动生成业务流程图。
4.  **用例/用户故事类**：辅助生成用户故事或用例。
5.  **数据埋点**：帮助设计和规划数据埋点（使用频率相对较低）。
6.  **查找SQL**：辅助查询数据库（使用频率相对较低）。

其中，前四项是更常用的需求文档提效工具。
