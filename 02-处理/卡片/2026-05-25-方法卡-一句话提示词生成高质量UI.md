# 一句话提示词生成高质量UI

测试者使用简单的提示词（甚至仅一句话）驱动 [[MiMo-V2-Pro模型]]，即可生成高质量、逻辑清晰的前端UI原型图。

**流程**：
1. 准备一个清晰、简洁的需求描述（如“做一个人事系统原型图”、“模拟上海地铁运行”）。
2. 将提示词输入MiMo-V2-Pro模型。
3. 模型生成可直接评估或转化为真实代码的UI界面。

这种方法极大地提升了前端原型设计的效率。
