# 查看与管理Auto Mode规则

可以使用 `claude automode` 子命令来查看和管理 [[Auto Mode Classifier]] 的规则。

**常用命令：**
- `claude automode defaults`: 查看所有**默认的内置安全规则**（哪些操作会被允许/拒绝）。
- `claude automode config`: 打印当前的 Auto Mode 相关设置。
- `claude automode critique`: 对当前规则集进行评审。

**自定义规则：**
规则可以在本地进行配置。配置方法详见官方文档的“Configuring Auto Mode”页面，涉及目录结构、字段格式和条目写法。自定义的规则会与线上的分类器互动，共同决定操作是否被允许。
