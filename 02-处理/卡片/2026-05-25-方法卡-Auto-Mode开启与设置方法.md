# Auto Mode开启与设置方法

要使用 [[Claude Code Auto Mode]]，需确保满足条件（版本 ≥ 2.1.83，非PRO套餐，使用Sonnet 4.6/Opus 4.6+等SOTA模型）。开启方式如下：

### 1. 临时切换（当前会话）
在 Claude Code 界面底部状态栏，使用快捷键 **`Shift + Tab`** 循环切换模式：`Default` -> `Accept Edit` -> `Plan` -> `Auto`。

### 2. 设置为默认模式（所有新会话）
1.  输入命令 `/config` 进入配置界面。
2.  找到 `default permission mode` 设置项。
3.  使用空格键选中 `auto`。
4.  **（推荐）** 确保 `Use Auto Mode During Plan` 已开启，以保障计划模式（Sub-Agent）中的操作也能自动执行。
5.  确认后，所有新会话将默认以 Auto Mode 启动。
