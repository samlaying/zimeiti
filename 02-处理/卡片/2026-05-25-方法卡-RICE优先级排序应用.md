# RICE优先级排序应用

使用 [[Product Manager Skills]] 中的RICE框架对产品需求进行优先级排序。

**操作步骤**：
1. 向Claude提供多个待排序的产品需求。
2. Claude会依据RICE框架的四个维度为每个需求打分：
   - **R**each (影响范围)
   - **I**mpact (影响力)
   - **C**onfidence (信心)
   - **E**ffort (所需努力)
3. AI会给出综合排序结果，并明确标识出哪些是能快速实现的**Quick Win**（快速获胜），哪些是匹配度高的**FitBit**（适合匹配）。

此方法将主观的需求判断转化为结构化的多维度分析，帮助快速识别高价值需求。
