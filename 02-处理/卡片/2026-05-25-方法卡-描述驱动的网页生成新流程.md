# 描述驱动的网页生成新流程

基于 [[Kimi K2.6 核心能力]] 的新型网页开发流程，其核心是“描述驱动”：
1.  **创意描述**：用户用自然语言（一句话）描述期望的网页效果和需求。
2.  **AI生成**：工具自动调用生图、生视频等能力，生成包含动态视觉元素的网页原型。
3.  **细节调整**：用户可对生成结果进行微调，如修改3D元素的样式。
4.  **上线发布**：利用工具的后端能力存储数据并完成网站上线。

该方法极大地简化了传统前端开发中视觉动效实现的复杂度。
