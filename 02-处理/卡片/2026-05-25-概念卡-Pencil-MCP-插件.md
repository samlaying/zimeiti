# Pencil MCP 插件

**Pencil** 是一个近期热门的 **MCP（模型控制协议）插件**，它可以集成到 **Cursor**、**VS Code** 等主流代码编辑器中。

其核心功能是**通过自然语言对话，生成高保真、完全可编辑的UI界面**，操作窗口和设计模式与 **Figma** 高度相似。

它为UI/UX设计师和前端开发者提供了一个在编码环境中直接进行AI驱动设计的强大工具。
