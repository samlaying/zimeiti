# Vue无刷新转场实现

使用Vue 2实现无刷新转场动画的步骤：
1.  **组件化动画**：将转场动画抽离为独立组件（如`<Loading>`），在根组件`App.vue`中引入。
2.  **动画退出**：在`App.vue`的`mounted`钩子中，用`setInterval`轮询`document.readyState`，为`“complete”`时调用`this.$refs.loading.loadingOut()`。
3.  **动画进入与路由跳转**：
    - 在`App.vue`的`mounted`钩子中配置`router.beforeEach((to, from, next) => { ... })`。
    - 在守卫中调用`this.$refs.loading.loadingIn(next)`，将`next`函数传给动画组件。
    - 在`Loading`组件的`loadingIn(next)`方法中，播放动画后，用`setTimeout`延迟1秒再执行`next()`。
