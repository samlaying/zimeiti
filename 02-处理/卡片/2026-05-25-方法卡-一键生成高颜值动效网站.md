# 一键生成高颜值动效网站

利用AI工具，1分钟内完成高颜值动效网站制作并上线的完整流程：

1.  **获取提示词**：访问提供审美UI设计的[[提示词资源库]]，找到喜欢的页面，复制其详细提示词。
2.  **生成设计稿**：打开[[谷歌Stitch]]，粘贴提示词，生成首页设计稿。
3.  **生成详情页**：在Stitch中使用Gemini模型，基于首页风格自动延展生成下方的详情页面。
4.  **替换与发布**：点击导出至[[谷歌AI Studio]]，替换为自己的内容信息，最后点击“一键发布上线”。
