# Skill文档结构

一个[[个人文风Skill]]的文档结构主要由两部分组成：

1.  **`Skill.md` 文档**：主文件，包含Skill的核心指令。内容构成为：
    *   **个人写作DNA**（如文风、节奏）
    *   **规则清单**（应该怎么做）
    *   **禁止清单**（不应该怎么做）
2.  **`Reference` 文件夹**：参考资料目录。内容构成为：
    *   **个人语料库**：你原有的优质文章。
    *   **[[对比案例库]]（关键）**：存放“AI写的版本”和“你亲自优化后的版本”的对比文件。
    *   可放置一个**模板文件**，规范输出格式。
