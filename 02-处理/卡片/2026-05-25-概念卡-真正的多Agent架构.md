# 真正的多Agent架构

[[Flex系统]]的核心特性是其**多Agent架构**。这并非让一个AI大模型扮演多个角色，而是指七个独立API调用。

- **上下文隔离**：每个节点只接收其需要的字段，上下文完全隔离。
- **问题追踪**：每一步都有JSON文件记录，便于精确排查错误。
- **可规模化**：可批量运行多个方向（如10个方向自动跑出10套物料）。
- **灵活性**：节点可更换模型（如从Claude切换到其他AI），修改配置无需改代码。
