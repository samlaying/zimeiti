# 开启与设置 Auto Mode 方法

> 一键切换或永久默认，两种方式按需设置。

## 核心内容
开启 Auto Mode 有两种方式：一、临时切换，在当前会话中用 `Shift + Tab` 循环切换至 Auto 模式；二、设为默认，输入 `/config`，在 `default permission mode` 中选择 `auto`，并建议同时开启 `Use Auto Mode During Plan`。设置后，所有新会话将默认以 Auto Mode 启动。
