# Agent是OS，Skills是App

> Agent是OS，Skills是App。

## 核心内容
可以将Agent系统类比为一个操作系统（OS），而各种Skills则像是运行在这个操作系统上的应用程序。例如，“获取天气”Skill就像一个天气助手软件，“发邮件”Skill就像一个邮件客户端。这个类比生动地体现了模块化、可扩展的Agent设计理念。

## 原文关键段落
> 比如说我们的Agent就像是一个操作系统然后我们的Skills呢这一片内容呢就说这一片内容呢就很像我们写了一个新时代的程序那么这个程序加载到OS当中OS去运行它就会得到现在的结果
