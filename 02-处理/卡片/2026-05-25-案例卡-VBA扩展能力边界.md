# VBA扩展能力边界

当 [[CPAHelper Excel Agent]] 的内置工具无法满足特定需求时，可以通过让 AI 编写并执行 VBA 代码来扩展其能力边界。

**案例演示：批量设置打印格式**
*   **问题**：Excel 中多个工作表的打印区域可能分散在多页，手动逐一设置效率低下。
*   **解决方案**：向 Agent 输入提示：“使用VBA帮我设置一下打印格式，每一个Sheet都只有一页”。
*   **AI执行**：Agent 会自动生成一段 VBA 代码，该代码能遍历所有工作表，并将它们的打印区域调整为单页。
*   **用户确认**：执行时，系统会请求运行权限，用户确认后，所有工作表的打印设置将被一键修改。

这个案例表明，AI Agent 不仅能调用现有工具，还能生成并执行新的代码来解决更具体、甚至预设工具未涵盖的个性化问题，实现了能力的动态扩展。
