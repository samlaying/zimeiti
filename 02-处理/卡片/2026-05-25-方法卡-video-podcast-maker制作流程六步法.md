# video-podcast-maker制作流程六步法

使用 [[video-podcast-maker]] 制作视频的标准流程如下：

1.  **获取与安装**：从官网或应用商店下载，确保系统兼容。
2.  **项目创建与设置**：新建项目，设置分辨率、帧率等参数。
3.  **素材导入与编辑**：在时间轴上排列、剪辑素材，并利用AI（如自动生成字幕）进行编辑。
4.  **AI功能应用**：运用其核心AI能力，如[[语音转文字]]、[[音频优化]]、画质增强。
5.  **预览与调整**：实时预览并进行微调，如调整字幕、添加背景音乐。
6.  **导出与分享**：设置导出参数并渲染，或直接分享到社交平台。
