# 脚本采集网页到 Figma

通过脚本方式将网页转换为 Figma 设计稿：
1. 获取官方提供的 FakeMilam CP 脚本代码。
2. 在网页上打开浏览器控制台（Console）。
3. 粘贴脚本代码并回车执行。
4. 等待脚本自动滚动页面并采集所有元素。
5. 完成后，内容已复制到剪贴板，粘贴到 Figma 中。

脚本方式支持整体或局部采集，可以采集特定元素进行设计还原。这是实现 [[网页转 Figma 设计稿]] 的一种灵活方法。
