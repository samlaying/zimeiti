# 个人网站的核心价值

制作个人网站的核心价值在于四点：
1.  **求职升级**：将静态纸质简历变成可交互的[[作品集 (Portfolio)]]，极大提升竞争力。
2.  **成为[[AI Native]]**：体验从设计、[[Vibe Coding]]到部署的全流程，掌握AI工具。
3.  **自我整理**：通过创作过程回顾自己的成就、优势与不足。
4.  **展现与乐趣**：即使最终未用于求职，过程本身也极具趣味性，完成后还能用来“装酷”。
