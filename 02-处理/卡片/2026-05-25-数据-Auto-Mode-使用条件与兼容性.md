# Auto Mode 使用条件与兼容性

> 不是所有套餐和模型都能用，高版本和 SOTA 模型是门槛。

## 核心内容
使用 Auto Mode 需满足特定条件：软件版本需 Claude Code 2.1.83 或更高；套餐方面，PRO 套餐不支持，其他套餐和标准的 Anthropic API 连接方式均可；模型方面，必须使用 SOTA 模型，如 Sonnet 4.6 或 Opus 4.6/4.7，较弱或较老的模型不支持。
