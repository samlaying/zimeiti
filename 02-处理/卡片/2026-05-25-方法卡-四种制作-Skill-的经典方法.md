# 四种制作 Skill 的经典方法

## Skill 制作方法论
除了 [[基于职业自动生成工作技能包]]，总结了四种系统化制作个人 Skill 的方法：

1.  **样板法**：将你最满意的交付物（如文档、方案）提供给 AI，让其学习并提炼成可复用的 Skill。
2.  **流程法**：写下清晰的工作步骤，将其中可以串联、自动化的部分提炼成 Skill。
3.  **归纳法**：回顾与 AI 的聊天记录，找出反复让 AI 执行的高频任务，并将其补全为正式的 Skill。
4.  **经验法**：将个人的隐性知识（如设计方法、写作技巧）显性化，封装成 Skill；也可以整合网上的通用优秀方法。
