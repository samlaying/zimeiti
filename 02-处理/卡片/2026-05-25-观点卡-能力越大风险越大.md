# 能力越大风险越大

Skill 的本质是赋予 AI 操控电脑的新能力，因此**安全至关重要**。[[ClawHub]] 生态中，能力越大的 Skill（如 [[Shell]]），潜在风险也越高。

**核心安全原则**：
- **来源可信**：只安装来自可信来源的 Skill
- **选择标准**：优先选择**下载量高**、**仓库公开**的 Skill
- **权限管理**：
    - 高权限 Skill（如 Shell）**强烈建议开启确认模式**
    - 需要 OAuth 授权的 Skill（如 Gmail、Github）务必**确认权限范围**
    - 不用的 Skill 或授权，**及时撤销**
- **定期更新**：安全补丁包含在更新中
