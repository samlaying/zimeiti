# 多Agent管理核心机制

[[OpenClaw]] 多Agent系统的管理核心在于两点：
1.  **自动化**：通过Skill自动创建和配置机器人，包括文件备份和工作区隔离。
2.  **可恢复性**：**自动备份机制**和清晰的日志查看、手动调整建议，为用户提供了可靠的故障恢复路径，确保能顺利解决常见问题。
