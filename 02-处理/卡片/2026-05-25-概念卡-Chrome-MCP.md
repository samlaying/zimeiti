# Chrome MCP

Chrome MCP 是一种浏览器工具或插件，连接后，用户可以像「搬运工」一样复制目标网站的内容、结构和设计，实现快速复刻。它是降低网站复刻技术门槛的关键工具，使得[[复刻网站无需专业技能]]成为可能。

**隐含操作步骤**：1. 安装并连接Chrome MCP。2. 访问目标网站。3. 使用工具复制并生成复刻版本。
