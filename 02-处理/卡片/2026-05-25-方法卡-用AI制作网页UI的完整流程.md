# 用AI制作网页UI的完整流程

从一份简单文档（如简历）生成高级感网页UI的标准化流程，核心是**先规划后执行**的迭代循环。

### 操作步骤
1.  **定义框架与视觉风格**：遵循“[[AI复刻能力优于创造能力]]”的原则，在写代码前，先确定页面布局框架，并找到风格参考图。使用工具（如Nano Banna）生成和谐的配色方案，让AI在代码中提前统一定义。
2.  **AI生成初版**：使用[[AI编程工具（如Cursor）]]，上传文本或模板，让AI根据要求生成基础网页。
3.  **高效沟通与细节微调**：遵循“[[截图优于Prompt沟通]]”的原则，通过“截图标注 -> AI修改”的循环，进行多轮精细调整。
4.  **总结与优化**：重复上述循环，直到达到满意效果。
