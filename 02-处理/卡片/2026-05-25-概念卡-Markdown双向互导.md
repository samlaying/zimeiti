# Markdown双向互导

Excalidraw 的思维导图功能支持与 Markdown 进行双向转换：
- **导出到 Markdown**：选中脑图运行 `Action Export Mind Map to MD`，根节点变为二级标题，子节点变为无序列表。
- **从剪贴板导入**：复制一段包含层级关系的列表文本，运行 `Action Import Mind Map from Clipboard` 即可生成脑图。
- **导出到剪贴板**：运行 `Action Export Mind Map to Clipboard`，可将脑图内容以 Markdown 格式写入剪贴板。
这打通了可视化思考与结构化笔记之间的壁垒。
