# Video to Note

**Video to Note** 是一个 [[Claude Code Skill]]，用于将视频链接（如 B 站）自动转化为包含视频转录和结构化摘要的 [[Obsidian 笔记]]。

它整合了视频元数据获取、音频下载、[[Whisper ASR]] 语音转录和内容分析，解决了将动态视频内容沉淀为静态可搜索笔记的需求。
