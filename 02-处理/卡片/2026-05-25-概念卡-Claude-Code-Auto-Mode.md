# Claude Code Auto Mode

Claude Code 的 **Auto Mode（自动模式）** 是一种编辑模式，旨在解决开发者在执行长程任务时，因频繁的权限确认提示而导致工作流中断的问题。

它通过在后台运行一个[[auto mode classifier]]（安全分类器）来自动评估操作的安全性，从而实现**连续数小时不间断**的编程工作。相较于不安全的[[YOLO模式]]，它提供了**内置的安全检查**，是在效率与安全性之间取得平衡的长任务保障方案。
