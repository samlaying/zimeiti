# System Prompt（系统提示词）

优先级高于用户输入提示词（User Prompt）的指令，能显著提升大模型的指令遵循效果。

## 用户接入方式
一些AI软件（如Cursor的`CursorRule`， Cloud Code的`Cloud.md`）允许用户通过特定文件来写入“系统提示词”，其效果通常优于在输入框直接输入的 [[Prompt（提示词）]]。
