# 使用Scale固化工作流

**Scale** 是 [[CPAHelper Excel Agent]] 中可保存和复用的提示词模板，用于将固定的工作流程标准化，实现一键执行。

**创建新 Scale**：
1.  **调用创建器**：通过输入 `/` 召唤出所有 Scale 列表，选择内置的 `Scale Creator`。
2.  **描述流程**：输入指令，如“把刚刚的流程写为一个‘分析销售费用’的Scale”。
3.  **生成模板**：Agent 会基于之前的操作历史，自动生成一个包含详细步骤（如美化表格、创建透视图、编写说明）的 Scale 文件。

**使用 Scale**：
*   通过输入 `/` 并选择已保存的 Scale（如“分析销售费用”），Agent 就会按照模板定义的步骤自动执行整个工作流。

这个功能的核心价值在于**工作流的沉淀**，让复杂、重复的任务可以标准化、自动化地执行。
