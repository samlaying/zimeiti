# 飞书机器人关键权限配置

为了让[[飞书Agent协作插件]]工作，飞书机器人必须开启特定权限：**接收群组中其他机器人和用户艾特当前机器人的消息**。

**配置方法**：
1.  访问飞书开发者后台，找到对应应用。
2.  **新建机器人**：创建时该权限通常已自动添加。
3.  **已有机器人**：需在“权限管理”中搜索权限Key（如“获取群组中其他机器人和用户艾特当前机器人的消息”），然后勾选开通。此权限免审，可直接开通。
4.  将配置好的机器人拉入目标飞书群聊。
