# AI动态设计图生成工作流

使用 [[Google Stitch]] 等AI工具生成动态设计图的核心工作流：

1.  **准备视频素材**：录制一个清晰展示目标动态效果（如加载动画、按钮悬停效果）的短视频文件。
2.  **上传至视频托管服务**：将视频上传至如 **Amazon CloudFront** 等稳定、公开可访问的托管服务，获取一个可靠的网络链接。这是确保成功的 [[稳定视频链接是前提]]。
3.  **整合链接并发送**：将获取的视频链接嵌入到提示词中，构建如“请参考这个动态效果 [视频链接]，为我生成一个具有类似风格的动态登录按钮”这样的精确指令，即 [[提示词需明确指向]]。
4.  **生成与获取**：AI模型会处理请求，输出基于所学动态风格的可交互设计图或代码片段。
