# 高含金量Vibe Coding的两大核心

有求职含金量的Vibe Coding能力包含两大核心：

1.  **熟悉并遵循开发工作范式**：不是凭感觉编程，而是遵循如[[规范驱动开发（SDD）]]这样的专业流程。
2.  **能够产出前后端分离的产品**：后端是产品逻辑的承载者，有能力利用AI工具实现从**想法（Idea）到产品落地**的全流程，快速输出可交互的前端界面并跑通**MVP（最小可行产品）**。
