# CSS内凹角技术选择决策树

> 一张清晰的决策图，帮你根据场景快速选择最合适的实现方案。

## 核心内容
根据设计需求和背景类型选择最合适的实现技术：1. 纯色背景，首选最简单的box-shadow技巧。2. 渐变、图片或毛玻璃背景，使用径向渐变遮罩。3. 需要自定义非圆形曲线，使用clip-path: shape()（需接受边框阴影被裁剪）。4. 面向Chromium内核应用（如内部工具、Electron应用），直接使用新兴的corner-shape属性。兼顾兼容性与未来的方式是使用@supports查询，以box-shadow为基线，为支持的浏览器提供原生升级。
