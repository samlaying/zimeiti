# 工作流Skills化方法

**核心方法**：将工作中重复出现的工作流，封装成独立的 [[Skills]]，交给 Claude 执行。

**执行步骤**：
1.  **识别**：发现工作中频繁重复的工作流。
2.  **整理**：清晰地梳理该工作流的**规则、标准和上下文**。
3.  **封装**：将整理好的规则**封装成一个独立的Skills**。
4.  **调用**：后续遇到相同任务时，**直接调用该Skills**，让 Claude 代为执行。

**最终效果**：通过将重复工作流Skills化，设计师可以将规则明确、重复性强的任务高效、准确地交给AI处理，从而专注于创造性工作。
