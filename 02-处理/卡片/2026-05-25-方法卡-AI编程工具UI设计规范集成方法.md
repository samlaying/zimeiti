# AI编程工具UI设计规范集成方法

通过以下步骤，将[[AllSang的Dieside开源项目]]的UI设计规范集成到你的AI编程工作流中：

1.  **获取规范**：在GitHub搜索并下载该项目，解压后得到包含59个站点规范的文件夹。
2.  **配置路径**：复制该文件夹的绝对路径备用。
3.  **修改提示词**：在你的AI编程工具（如Trace 3.0的Code模式）中，使用或修改提示词。**关键一步**：在提示词中加入明确指令，要求AI在开发前**深入分析**并**严格遵守**该路径文件夹内的所有设计规范文档。
4.  **执行与交互**：AI会先分析所有规范文档，然后询问你想要的设计风格（如Figma风格），确认后即可开始自动生成符合规范的UI。
