# 小白一键安装Hermes Agent

对于不懂配置环境的小白用户，安装Hermes Agent极其简单：
1.  使用支持的AI Agent工具，如OpenClaw、Codex或国产Coldwork。
2.  复制Hermes Agent官方仓库链接。
3.  在工具中输入自然语言描述，例如：“阅读Hermes官方文档中安装和配置说明并严格按照步骤为我在本地安装”。
4.  点击发送，AI Agent将自动完成安装、配置及数据迁移。

如果不会配置大模型API，也可以将API key提供给上述AI Agent，让它们自动配置。
