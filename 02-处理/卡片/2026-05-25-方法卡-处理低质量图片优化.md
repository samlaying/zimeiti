# 处理低质量图片优化

针对甲方指定的低质量图片，使用[[AI PPT设计工具组合]]优化：
1. **上传图片**：将低质量图片上传到豆包。
2. **去除干扰元素**：让图片更干净。
3. **裁剪留白**：增加文字放置空间。
4. **压暗背景并突出主体**：输入提示词调整，使图片具备制作PPT封面的实力。
