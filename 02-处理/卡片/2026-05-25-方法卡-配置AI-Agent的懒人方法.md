# 配置AI Agent的懒人方法

对于不熟悉AI模型配置（如API key输入、模型选择）的用户，有一个简单的“懒人方法”：将所需的API key提供给一个已能正常工作的AI Agent（如OpenClaw、Codex或CodeWork），并用自然语言描述你的需求（例如“用这个API key帮我配置minimax m2.7high speed模型”），让该AI Agent自动完成配置操作。这体现了[[小白一键安装Hermes Agent]]中提到的AI思维。
