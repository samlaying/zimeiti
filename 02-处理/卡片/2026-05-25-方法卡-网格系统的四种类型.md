# 网格系统的四种类型

## 四种基本网格类型

1. **手稿网格**：单栏满版，用于书籍正文、长篇散文。
2. **列状网格**：多栏并列，尺寸可计算，是期刊、报纸的语法基础。
3. **魔术网格**：横列乘纵行的方格矩阵，海报和目录的利器，瑞士国际主义最具识别度的类型。
4. **层级网格**：不规则但有逻辑，用于复合排版、杂志专题、网页布局。

**共同前提**：所有距离都是可计算的。
