# 做Skill的完整步骤

使用[[Skill Creator：构建技能的核心工具]]技能可以创建自己的Skill，流程如下：

1.  **描述需求**：清晰说明Skill的输入、输出、处理过程。
2.  **提供参考**：提供格式、示例文件等要求。
3.  **生成与优化**：先在Cloud Code中生成初稿，再用其他AI编程工具（如Cursor、Tree）进行优化。可要求AI分析技能功能并提出优化建议。
4.  **测试与调试**：在Cloud Code中通过**软链接**方式安装开发中的Skill，方便在项目路径下直接进行测试。

**输出文档结构建议**：生成以`Skill名.z`命名的文件夹，包含整体技能介绍（技能总数、定义场景、使用方法）和**应用演示**（清晰的用户输入示例和模型预期输出）。
