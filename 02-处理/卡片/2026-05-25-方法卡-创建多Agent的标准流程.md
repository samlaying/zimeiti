# 创建多Agent的标准流程

使用[[OpenClaw多Agent系统]]的[[Skill]]创建新Agent的完整实操流程：
1.  **安装Skill**：在终端执行安装命令。
2.  **创建机器人**：在飞书平台创建新机器人应用。
3.  **配置机器人**：向现有机器人发送描述，提供新机器人的APPID和APPsecret，系统自动创建新[[工作区隔离|工作区]]。
4.  **获取白名单**：给新机器人发送一条消息，触发OpenID白名单的获取与配置。
