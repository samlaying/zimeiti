# Product Manager Skills Skill

这是一个能让Claude AI化身为资深产品经理的[[技能(Skill)]]。

**核心内容**：由一位PM开源在GitHub上，名为`Product Manager Skills`。它包含了总共46个真实的产品管理框架，如[[RICE优先级排序]]、用户洞察、用户故事等。

**实现方式**：这些经典方法论被拆解成Claude能够理解的skill模块，安装后，Claude就像读过这些方法论的专家，会主动按照这些框架引导用户工作，解决AI不懂专业工作流的问题。
