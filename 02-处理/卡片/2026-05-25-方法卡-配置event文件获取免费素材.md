# 配置event文件获取免费素材

为了优化 [[AIcut：AI视频剪辑工具]] 的使用效果，建议配置 `event` 文件以接入免费资源。
1.  **获取API Key**：前往免费素材库网站（如 Pix）和模型网站，注册登录后，在对应的 API 管理页面复制你的 Key。
2.  **修改配置文件**：将复制的 Key 粘贴到项目 `event` 配置文件的对应位置。配置后即可使用更多免费资源和模型，提升剪辑效果。
