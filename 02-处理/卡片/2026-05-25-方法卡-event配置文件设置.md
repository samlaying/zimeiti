# event配置文件设置

`event`文件是AIcut的关键配置文件，通过配置它可以接入免费资源，提升工具能力。

**主要配置步骤**：
1.  **获取API Key**：前往指定的免费素材网站（如Pix）和AI模型网站（如Sentu）注册并登录。
2.  **复制与粘贴Key**：在网站的API管理页面复制你的专属Key，然后将其粘贴到项目`event`文件的对应位置。

**配置目的**：接入免费的素材库和强大的AI模型，使工具能够更好地完成剪辑任务。虽然不配置也能使用，但配置后功能会更强大。
