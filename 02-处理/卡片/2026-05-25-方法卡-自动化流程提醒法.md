# 自动化流程提醒法

通过多维表格的自动化流程功能，设置外部触发条件，并将提醒发送到指定渠道，适用于跨平台通知。

**操作步骤：**
1.  **选择触发条件**：例如“到达记录中指定时间”，并指定表格和日期字段（如“下次跟进时间”）。
2.  **设置触发时间**：例如设置为每天上午9:00或9:30（上班时间），确保及时提醒。
3.  **配置消息渠道**：支持多种通知方式，需提前配置：
    - WPS协作群消息、邮件
    - **企业微信**（需配置 webhook）
    - **钉钉**（需配置 webhook）
    - 短信
4.  **保存设置**：系统将在指定时间自动检查并发送提醒。

这是实现跨平台、主动推送的[[自动化提醒功能]]的核心方法。
