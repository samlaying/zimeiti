# Skill管理五大痛点

在使用AI编程助手时，Skill管理普遍面临五大痛点：
1.  **查找困难**：不知道已安装的Skill存放在何处。
2.  **管理混乱**：缺乏统一管理手段。
3.  **版本迭代难**：缺少版本控制，难以回滚。
4.  **多端同步不便**：不同电脑间的Skill无法便捷同步。
5.  **功能冗余冲突**：安装了功能相似的Skill，导致AI助手调用混乱。

[[Skill管理器]]正是为解决这些痛点而生。
