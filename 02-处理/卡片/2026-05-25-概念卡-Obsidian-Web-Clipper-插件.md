# Obsidian Web Clipper 插件

Obsidian Web Clipper 是一个浏览器扩展，用于从网页剪藏内容到 Obsidian 笔记软件，辅助构建个人知识库或第二大脑。它通过浏览器工具栏固定图标方便使用。
