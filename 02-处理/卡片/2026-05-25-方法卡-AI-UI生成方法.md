# AI UI生成方法

使用 [[Pencil MCP 插件]] 生成UI界面的步骤：

1.  **安装插件**：在编辑器拓展市场搜索 **“Pencil”** 并安装。
2.  **启用能力**：进入插件设置，**务必勾选**你所使用的编辑器（如 `Cursor`, `Cloud Code`）对应的选项。
3.  **验证连接**：在对话框中询问AI“你能不能看得到Pencil的MCP？”，AI会列出所有可用工具。
4.  **生成设计**：通过自然语言下达指令，例如：“使用Pencil绘制一个**微信**的APP，但要采用**Instagram的设计风格**来绘制，并输出多张方案”。
5.  **等待与调整**：AI执行过程约10分钟，最终输出多张设计方案。如发现细节问题（如文字显示错误），可继续通过对话指令修改。
