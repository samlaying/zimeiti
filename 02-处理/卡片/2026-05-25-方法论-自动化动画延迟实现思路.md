# 自动化动画延迟实现思路

> 用JS索引驱动CSS动画延迟，实现自动化、可扩展的列表项依次动画。

## 核心内容
使用JavaScript遍历目标列表项，在为每项添加CSS动画时，将列表项的索引值(index)作为动画延迟(delay)的计算因子（如 index/7 秒）。这样，动画会依次执行，且当新增或删除列表项时，动画会自动适应，无需手动调整延迟时间，实现了高度的可扩展性和定制性。

## 原文关键段落
> const navLinks = document.querySelectorAll('.nav-links li'); 
> 
> navLinks.forEach((link, index) => { 
>     link.style.animation = `navLinkFade 0.5s ease forwards ${index / 7}s`; 
> });
