# 进度条生成四步法

使用[[视频进度条生成器]]制作进度条的标准操作流程：

1.  **输入基础参数**：设置视频的总时长和宽度（例如默认1080P）。
2.  **添加章节信息**：根据视频内容结构，添加多个章节，并为每个章节设置具体的时间节点和可自定义的标题（如“开头”、“中间”、“结尾”）。
3.  **一键生成**：点击生成按钮，工具会基于输入信息快速输出一个独立的进度条视频文件。
4.  **拖拽使用**：打开生成的文件位置，将进度条视频直接拖拽到常用的视频剪辑软件中，并调整其位置（如下移）使其与主视频节奏对齐。
