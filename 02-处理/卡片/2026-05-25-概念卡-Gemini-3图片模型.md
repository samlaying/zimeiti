# Gemini 3图片模型

**Gemini 3图片模型**，也被称为 **“Nano of Nano”**，是实现配图功能所依赖的核心AI模型。

- **必要性**：在创建或使用 [[配图Skill定义|配图Skill]] 时，必须在 API 配置中指定使用此模型。
- **特点**：它能理解自然语言指令并生成符合要求的、高质量的图片，是 Skill 能够“看懂”文章内容并生成匹配配图的技术基础。
