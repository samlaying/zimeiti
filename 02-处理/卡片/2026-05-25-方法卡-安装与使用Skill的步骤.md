# 安装与使用Skill的步骤

在AI编程工具（如Cloud Code）中安装和使用Skill的步骤：

**安装（方法任选）：**
- 使用 `mix` 命令。
- 用自然语言指令（如“请帮我安装XX文件夹下的技能”）。
- 将Skill文件夹复制到工具指定目录。
- 上传Skill压缩包。

**使用：**
1. 在终端进入项目路径。
2. 启动AI编程工具（可加参数减少确认）。
3. 执行Skill：输入 `/技能名`（例如 `/SkillTranslator`）。
