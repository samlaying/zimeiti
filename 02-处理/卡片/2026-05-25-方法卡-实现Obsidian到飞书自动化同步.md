# 实现Obsidian到飞书自动化同步

基于 JS 脚本实现 [[Obsidian]] 至 [[飞书]] 的自动同步，可通过以下两步实现无人值守：

1. **设置 Windows 计划任务**：将 JS 脚本集成到系统计划任务中，可按固定时间间隔（如每小时）或按键触发自动同步，并可设置隐藏弹出的 CMD 窗口。
2. **确保 [[飞书Token定时刷新]]**：飞书访问令牌具有时效性，需配置定时刷新脚本，自动获取并更新 Token，防止同步中断。

> 可接入Windows计划任务，设置定时自动同步，无需手动操作。
