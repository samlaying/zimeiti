# 配置多Agent工作流程

使用 [[OpenClaw]] Skill 创建新机器人的核心流程：
1.  **提供描述**：向默认机器人发送描述（如“搞钱探子”），系统会**自动创建备份**并生成一个新的、[[独立工作区]]。
2.  **提供凭证**：提供飞书APPID和APPsecret，完成配置。
3.  **发送消息**：在飞书中给新机器人发送一条消息，以触发系统**自动获取其OpenID白名单**并写入配置。
4.  **重启生效**：配置完成后，通常需要重启网关使配置生效。
