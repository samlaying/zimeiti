# 触发Agent群内协作

使用[[飞书多Agent协作插件]]启动并管理Agent协作流程的技巧。

### 触发协作
在群聊中提及关键词，如 **“群内协作”** 或 **“一起做一下”**，插件会自动发现群内所有Agent并启动协作流程。

### 监控与干预技巧
1.  **设置监控角色**：指令某个Agent（如产品机器人）定时检查其他Agent状态（如“每5分钟监控一次”），避免任务卡死。
2.  **实时人工干预**：如果机器人未及时回复，可以**引用其消息并手动艾特**它，提供指导或纠正方向，防止流程跑偏。
3.  **协作模式选择**：
    - **人工参与模式**：用户随时介入，确保方向正确。
    - **全自动模式**：依赖Agent自主推进和定时监控，理论上可持续工作直至任务完成。
