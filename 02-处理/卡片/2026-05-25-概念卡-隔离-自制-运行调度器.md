# 隔离·自制·运行调度器

**OpenAI Symphony** 是一个开源项目，其核心定义是将项目工作转化为一组**互相隔离、自制的任务执行**。它不是直接编写代码的 Agent，而是一个**任务调度器**，负责派发和管理由其他 Agent（如 Codex App Server）执行的任务。

其三个核心关键词决定了它的本质：
- **隔离**：每个任务在独立的工作区运行，互不干扰。
- **自制**：任务由 Agent 自主决策如何执行。
- **运行**：通过调度器管理任务的整个生命周期。
