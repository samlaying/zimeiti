# n8n文字生成动态演示工作流原理

该自动化工作流的核心原理分为四步：

1.  **主题风格定义与图片内容规划**：
    *   输入用户文字内容和图片数量。
    *   由AI Agent拆分内容，第一页固定为封面页，后续页面按语义逻辑拆分，并**尽量遵循原文**。
    *   将拆分后的内容与预制的**主题提示词母版**（包含基础风格、封面/内容页构图逻辑）组合，生成完整的图片提示词。

2.  **生成图片**：
    *   使用 [[Bunada Pro 模型]] 生成图片，该模型能处理复杂提示词并精准还原中文。

3.  **生成转场动画提示词**：
    *   使用多模态模型（如Gemini 3 Pro）为每两张相邻图片生成过渡提示词。

4.  **生成首尾帧动画**：
    *   使用 [[可灵2.6生成首尾帧动画]]，根据提示词生成转场动画视频。该模型在动画过程中中文文字崩坏效果最小。
    *   生成的图片和视频按 `1.png`, `1-2.mp4` 等规则自动命名。

最终，所有素材与一个HTML播放器文件同放一个文件夹，双击HTML即可像PPT一样通过键盘控制翻页播放。
