# Agent Loop核心框架

> Agent的心脏跳动循环：请求、工具、停止。

## 核心内容
Agent Loop是处理用户请求的核心循环，其流程包括：接收用户输入并流式输出结果、执行工具调用并将结果回传、以及在任务完成、需用户授权或达到最大轮次时停止。整个过程注重流式响应以提升用户体验。

## 原文关键段落
> Agent Loop是系统的核心，处理用户请求的循环流程：请求处理、工具调用、停止条件、流式输出。
