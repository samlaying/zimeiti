# 用 AI 工具自动安装配置 Hermes Agent

对于没有编程或环境配置基础的小白用户，可以借助其他 **AI Agent 工具**自动完成 **Hermes Agent** 的安装与配置。

**推荐工具**：OpenClaw、Codex、Coldwork (国产)。

**核心方法**：使用自然语言提示词，让 AI 工具阅读官方文档并自动执行。

**示例提示词**：
`“阅读 Hermes 官方文档中安装和配置说明并严格按照步骤为我在本地安装”`

**操作流程**：
1. 复制 Hermes Agent 的官方仓库链接。
2. 在上述任一 AI 工具中，输入包含仓库链接的自然语言描述。
3. 发送指令，AI Agent 便会自动完成从拉取代码到环境配置的全过程，并可同步处理从 OpenClaw 的 `[[从 OpenClaw 无缝迁移至 Hermes Agent|数据迁移]]`。

**关键思维**：将复杂的环境配置工作交给 AI 完成，是 AI 时代应有的“AI 思维”。
