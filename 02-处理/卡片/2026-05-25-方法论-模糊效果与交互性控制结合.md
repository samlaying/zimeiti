# 模糊效果与交互性控制结合

> `opacity`与`pointer-events`组合，实现视觉和功能的双重隐藏。

## 核心内容
通过组合CSS属性控制元素的可见性和交互性：使用`opacity: 0`使其透明（但仍占据布局空间），并搭配`pointer-events: none`确保鼠标无法与之交互，两者结合实现了元素的“视觉隐藏”和“功能禁用”。

## 原文关键段落
> In initial state all content in the detail class will be hidden i will use opacity0 because the purpose of opacity is to make everything blurry but it doesn't really disappear so combine it with pointer events none to make the mouse pointer never touch it
