# 截图优于Prompt沟通

与AI沟通时，**用截图传递视觉信息**，比用文字描述（Prompt）更精准、高效。

AI模型对视觉信息的理解和复现能力更强。当你需要调整UI细节（如圆角、配色）时，直接截图标注，比描述“让文字边角更圆润”等文字指令，传达的内容更清晰，AI的“秒懂”率更高。

这个沟通技巧是执行[[用AI制作网页UI的完整流程]]第三步“高效沟通与细节微调”的核心。
