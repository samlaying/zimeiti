# 技能集成与自定义

为了让[[Frontend Slides Skill]]在生成PPT时能自动调用[[Beautiful HTML Templates Skill]]中的精美模板，需要对技能进行集成和个性化修改。

**操作步骤**：
1.  下载`Beautiful HTML Templates`项目文件并解压。
2.  将文件夹拖入AI Agent（如Cline）窗口，并发送指令“帮我安装这个skill”。
3.  **关键步骤**：将`Beautiful HTML Templates`文件夹中的`agents.md`文件拖入Agent，并附上指令，如：“生成PPT时，请直接调用这个文件夹里的模板”。
4.  AI Agent会自动修改`Frontend Slides`技能，实现模板集成。

**此方法的意义**：避免了先用技能生成内容再手动换模板的麻烦，实现一键式美化。这也是一种通用的、对下载到的各类技能进行个性化定制的思路。
