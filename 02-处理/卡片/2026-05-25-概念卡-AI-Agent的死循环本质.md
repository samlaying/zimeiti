# AI Agent的死循环本质

**AI Agent的本质是一个死循环**，其运行流程为：组装上下文 → 调用大模型推理 → 执行工具 → 将结果塞回上下文 → 进入下一轮。

这个循环如果次数过多，会导致上下文过长、截断，可能使Agent目标偏移，从而引发执行风险，这是引入[[沙箱是隔离环境]]的核心原因。
