# LangSmith Agent Builder 升级

**LangSmith Agent Builder** 是 [[LangSmith]] 平台的**无代码/低代码智能体构建工具**，其核心交互是通过对话式交互来创建和调试智能体。

结合 [[Sandbox 与 Context Hub]] 和 [[托管深度智能体]] 能力后，它获得了关键升级：
1. **沙箱集成**：创建的智能体可以直接使用 Sandbox 执行技能。
2. **预置模板**：提供一批官方预制的智能体模板（如用于客户调研的 CIM 系统）。

这些升级使其从“玩具”升级为具备生产级能力的工具，类似于一个**企业版的 OpenClu**，开箱即用，无论是程序员还是非技术同学都可以使用。
