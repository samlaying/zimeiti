# Prompt（提示词）

让AI执行任务的最简单的自然语言指令。

## 问题
简单的自然语言提示词，AI输出的结果可能不符合预期（如格式、内容）。

## 解决方案
使用 [[结构化提示词]]，即详细规定步骤和规则的提示词，以提升指令遵循效果。
