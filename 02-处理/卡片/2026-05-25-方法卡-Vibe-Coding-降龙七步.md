# Vibe Coding 降龙七步

## 概述
面对大厂AI代码题目（如使用AI进行Web编码实现功能）的通用解法流程。其核心在于通过清晰的步骤划分、与AI的交互模式和持续的过程控制，解决[[AI上下文限制]]导致的[[目标漂移]]问题。

## 七步流程
1.  **[[问题定义]]**：与AI共同梳理输入、输出、交付物、约束、要求与风险。
2.  **[[确定方案]]**：与AI讨论多个技术方案的优缺点，选定一个并记录。
3.  **[[搭建核心架构]]**：基于方案，定义模块、目录、API、数据结构等。
4.  **[[分拆任务与完成最小闭环]]**：将任务拆解，循环完成“最小可运行版本 -> 验证”的闭环。
5.  **[[异常测试]]**：以测试工程师视角，让AI补全边界案例、异常输入等。
6.  **[[验收]]**：根据问题定义、方案和架构进行全流程检验。
7.  **[[交付材料]]**：让AI生成README、设计文档等最终交付物。

## 核心理念
整个流程强调将关键决策和文档沉淀到MD文件中，而非仅停留在对话里，便于[[上下文管理]]和后续回溯。这实质上是一个由人类持续驱动和检查的[[项目管理]]过程。
