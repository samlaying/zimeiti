# Sub Agents与Agent Team的生动比喻

区别[[Sub Agents（子代理）是什么|Sub Agents]]与Agent Team，可以用一个生动的比喻：

- **Sub Agents**：想象成**老板直接管理多个高管**。老板与每个高管紧密接触并跟踪进度，高管之间独立工作，不相互交流（单会话工作）。
- **Agent Team**：则是**老板通过一个CEO与各个高管对接**。高管之间可以跨会话交流，适用于复杂项目的并行持续迭代（多层级结构）。
