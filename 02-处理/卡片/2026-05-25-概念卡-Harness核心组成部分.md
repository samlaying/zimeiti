# Harness核心组成部分

一个典型的**Agent Harness**通常包含以下核心模块：

1.  **工具注册表**：智能体可调用的外部工具集合（如读写文件、执行命令）。
2.  **模型**：Harness所调用的核心AI模型。
3.  **守卫栏**：预设的规则和限制，用于控制智能体行为边界（如最大步骤数、上下文长度）。
4.  **智能体循环**：驱动智能体逐步执行任务的核心逻辑。Harness是包裹在此循环外的一层。
5.  **验证步骤**：任务完成后的确定性检查，以验证结果是否真正成功（如运行代码检查、测试）。
