# AI Skill 的定义与结构

**Skill** 是AI的技能，可以理解为一个外挂，它将常用信息（如提示词、规则、脚本）组合成一个标准文件，作为AI执行特定任务的说明书。

它的核心结构（目录）包括四个部分：
1.  **Skill文件（核心）**：AI的任务说明书，定义功能、输入输出、执行逻辑。这是必须存在的。
2.  **Reference（资料手册）**：存放参考资料，如规则、流程、数据库，用于查询和确认信息。
3.  **Scripts（工具脚本）**：存放可执行的脚本或工具，用于调用外部接口（如API）。
4.  **Assets（参考文件）**：存放非必要资源，如图片、示例文件，用于提供参考。

理解 Skill 就像写一篇结构化的[[智能点奶茶的Skill拆解案例|论文]]，它通过分离配置、逻辑、数据与资源，使结构清晰，并能提高查询速度、降低Token成本。
