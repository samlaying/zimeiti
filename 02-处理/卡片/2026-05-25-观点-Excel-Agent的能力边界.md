# Excel Agent的能力边界

> Excel Agent能力包罗万象，从算账到写代码都能干。

## 核心内容
现代Excel Agent插件的能力远超基础读写。它不仅能完成数据操作、格式美化、创建图表透视表等原生功能，还能通过内置计算器解决AI算数不准的问题，并能查询A股财报、连接本地数据库执行SQL、甚至编写并执行VBA和Python代码，从而极大扩展了Excel的功能边界。
