# 子Agent是执行脏活的手脚

核心设计哲学：使用昂贵的、强大的模型（Opus）作为主脑进行规划和总结，使用廉价、快速的模型（Haiku）作为“手脚”去执行具体的、可能含有不可信内容的脏活。**所有潜在的脏活和不可信内容都隔离在子Agent里**。

这句话精准概括了[[主Agent与子Agent隔离设计]]中主次Agent的角色分工与隔离理念。
