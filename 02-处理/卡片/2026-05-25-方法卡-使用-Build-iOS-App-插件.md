# 使用 Build iOS App 插件

在使用 Codex 开发 iOS 应用时，推荐添加 **Build iOS App** 这个 OpenAI 官方插件。

**插件包含的关键资源 (Skills)：**
1.  **SwiftUI UI Partials**：一系列 SwiftUI 代码模板。
2.  **Liquid Glass 液态玻璃设计指南**：指导 AI 实现 iOS 26 的液态玻璃 UI 效果 (如边框、触感动画)。

**使用建议：**
- 在规划阶段就引入此插件，可以引导 AI 生成更规范、美观的 UI 代码。
- 配合 [[Pencil 设计稿 (.pen)]] 使用，能提升 UI 还原度。
