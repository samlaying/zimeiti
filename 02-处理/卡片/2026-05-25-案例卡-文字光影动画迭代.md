# 文字光影动画迭代

视频中详细演示了如何从一个简单的静态文字设计图，通过多次[[提示词]]迭代，生成一个带有可控光影效果的文字动画。

**迭代过程**：
1.  **初始需求**：生成带有“白色发光效果”的文字动画。
2.  **发现问题**：AI生成的发光效果溢出文字边界，不符合预期。
3.  **优化指令**：修改[[提示词]]，要求“移除发光效果，改为让字母后面的投影产生动画，模仿光线从左到右悬停在文字上方”。
4.  **添加控制**：进一步要求AI“让用户能够在编辑器里调整模糊强度和投影的不透明度”，从而在预览界面获得了可手动微调的滑块。

此案例体现了通过清晰的问题描述与指令，逐步引导AI优化产出的过程。
