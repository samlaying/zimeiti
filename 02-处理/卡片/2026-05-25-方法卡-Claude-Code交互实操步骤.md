# Claude Code交互实操步骤

1. **环境设置**：创建项目文件夹和[[Markdown规范文档定义]]。
2. **启动与学习**：打开Claude Code，导航到文件夹，让AI先学习MD文档。
3. **执行任务**：提出具体指令，如“帮我把单行输入框的所有状态补充完整”，AI基于文档自动在Figma中创建组件。
4. **输出与优化**：AI生成组件和修改日志，检查结果，如有问题在对话中指出，并让AI将规则更新到MD文档，逐步迭代。
