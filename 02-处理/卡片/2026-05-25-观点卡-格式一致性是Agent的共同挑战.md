# 格式一致性是Agent的共同挑战

在所有[[AI Agent]]的学术[[Word]]场景测试中，一个普遍存在的挑战是**格式一致性**。无论是填写表格、论文还是合同，即使内容填充正确，多个Agent（如Kimi、Claude、GLM5）都出现了字体与模板不一致、未正确处理加粗/非加粗、破坏原有排版等问题。这表明当前AI Agent在理解并精确复现文档的**格式细节**方面仍有较大改进空间，用户在使用后需要做好人工校对和格式微调的准备。
