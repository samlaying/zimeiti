# VibeMotion Skills: 动画技能库与开源生态

**VibeMotion Skills** 是 `[[VibeMotion]]` 生态中的一个核心组件，它是一个**动画技能库**。

**核心功能**：安装后，用户可以通过与AI（如“龙虾”）的**自然语言对话**来生成动画视频，并且这个过程**几乎不消耗 Token**。

**开源愿景**：用户可以将自己创作好的动画效果作为“Skill”上传到仓库，从而形成一个全球性的开源社区。目标是让全世界的AI助手（龙虾）都能复用这些动画效果，使其成为 `[[VibeMotion]]` 开源社区的发源地，潜力巨大。
