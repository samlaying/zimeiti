# 创建自定义Skill的流程

使用 **Skill Creator** 技能可以创建自己的Skill：
1. **描述需求**：清晰说明Skill的输入、输出、处理过程。
2. **提供参考**：给出格式、示例文件等要求。
3. **生成与优化**：先在工具中生成初稿，再用其他AI编程工具（如Cursor）优化，可要求AI分析功能并提出建议。
4. **测试与调试**：在开发工具中通过**软链接**方式安装Skill，方便直接在项目路径下测试。

创建的Skill应包含完整的[[Skill输出文档示例]]。
