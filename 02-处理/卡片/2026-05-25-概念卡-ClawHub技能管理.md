# ClawHub技能管理

**ClawHub** 是 OpenClaw 的技能（Skill）集中管理平台，类似于应用商店。

**核心功能**：
1. 使用自然语言搜索所需 Skill
2. 安装指定的 Skill
3. **一键批量更新**所有已安装的 Skill

**基础操作流程**：
1. 通过 NPM 安装 `ClawHub Cloud`
2. 安装完成后，终端将增加 `ClawHub` 命令
3. 可通过命令行或直接向 OpenClaw 发出指令（如“帮我装某个Skill”）进行管理
