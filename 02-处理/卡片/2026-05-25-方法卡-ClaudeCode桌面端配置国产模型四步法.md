# ClaudeCode桌面端配置国产模型四步法

将ClaudeCode桌面端配置为使用国产大模型的流程可分为四步：

1.  **下载安装客户端**：从官网下载MacOS或Windows客户端并安装。
2.  **开启开发者模式**：在客户端内通过 `Help` -> `Trouble Shopping` 路径开启，开启后菜单栏会新增 `Developer` 选项。
3.  **填写模型配置（关键）**：在 `Developer` 菜单中完成四项配置：
    - **网关配置**：填写国产模型的接口URL和 [[API Key]]。
    - **模型配置**：填写模型名称（如`minimax m2.7`），并**必须开启上下文**以支持连续对话。
    - **登录模式**：打开 **“跳过登录模式”** 开关。
4.  **应用与测试**：点击`Apply`应用配置并重启客户端，随后分别在[[CodeWork模式]]和[[Code模式]]下测试模型回复是否正常。

成功后即可在客户端内自由切换已配置的模型。
