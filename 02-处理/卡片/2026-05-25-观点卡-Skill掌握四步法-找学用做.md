# Skill掌握四步法：找学用做

掌握AI编程工具中的Skill能极大提升工作效率，其核心方法论是 **“找、学、用、做”四步法**。这是一个系统化、可复制的流程，能帮助用户快速构建起强大的个人AI工具箱。

1.  **[[找Skill的三种途径]]**：通过特定网站或技能高效找到所需。
2.  **[[学Skill的两种模式]]**：利用工具或对话快速学习。
3.  **[[用Skill的标准化流程]]**：在IDE中通过标准化命令安装和使用。
4.  **[[做Skill的完整步骤]]**：借助[[Skill Creator：构建技能的核心工具]]将自身流程固化，做出专属技能。
