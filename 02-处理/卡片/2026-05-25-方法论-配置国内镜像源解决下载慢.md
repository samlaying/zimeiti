# 配置国内镜像源解决下载慢

> 解决国内开发者安装依赖慢的通用方法。

## 核心内容
当开发者没有稳定的国际网络访问环境时，可以分别配置 Python 的 pip 和 Node.js 的 npm 使用国内镜像源（如清华源），以显著提升安装依赖包的速度。安装完成后可通过相应命令验证配置是否生效。

## 原文关键段落
> 配置清华镜像源（解决无梯子访问慢问题）：执行命令后，使用 `pip config list` 验证，输出中应显示 `index-url` 为清华源地址。
> 配置国内镜像源：执行 `npm config set registry https://registry.npmmirror.com` 并验证。
