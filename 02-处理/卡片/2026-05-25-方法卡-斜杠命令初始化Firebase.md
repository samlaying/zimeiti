# 斜杠命令初始化Firebase

使用斜杠命令 `/firebase init` 初始化Firebase项目的步骤：
1. **切换分支**：运行 `git switch -c firebase-setup` 创建并切换到新分支，确保安全。
2. **执行命令**：输入 `/firebase init`，选择允许使用工具。
3. **选择服务**：启用 [[Firebase核心服务]] 中的 **Cloud Firestore** 和 **Firebase Authentication**。
4. **配置规则**：部署Firestore安全规则时，选择**保持在测试模式**。
5. **设置认证**：启用**密码认证**，根据指南在控制台设置其他方法。
6. **提交合并**：提交更改到新分支，然后合并到主分支。
