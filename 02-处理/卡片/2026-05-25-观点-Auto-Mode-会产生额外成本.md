# Auto Mode 会产生额外成本

> 高效有代价：自动模式安全审核需要额外付费。

## 核心内容
Auto Mode 并非免费功能。其安全分类器运行在独立的服务器配置上，会产生额外的 Token 消耗和费用。当与计划模式（Sub-Agent）同时使用时，由于分类器会在 Sub-Agent 的启动前、运行时和完成后三个阶段进行检查，成本会显著增加。用户需意识到这是提升效率和安全性的一项付费投资。

## 原文关键段落
