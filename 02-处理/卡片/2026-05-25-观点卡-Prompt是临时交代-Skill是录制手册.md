# Prompt是临时交代，Skill是录制手册

用传统**提示词（Prompt）** 去除AI味，如同临时给一个新员工交代公司规定，效果短暂且不稳定。

而构建**个人写作Skill**，则像是给新员工一份详细的、录制好的操作手册。AI每次工作前先学习手册，犯错后可以将纠正案例更新到手册中，使手册越来越准确，AI也越来越“懂”你。

这种方法的优势在于：**更持久、可积累**，AI能通过学习逐渐掌握你的个人风格，实现风格一致性。
