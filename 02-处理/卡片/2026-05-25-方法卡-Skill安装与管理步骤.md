# Skill安装与管理步骤

管理 OpenClaw Skill 的流程：
1. 使用 NPM 安装 [[ClawHub管理工具|ClawHub Cloud]]。
2. 通过 ClawHub 的自然语言搜索功能查找所需 Skill。
3. 安装指定的 Skill。
4. 定期运行一键批量更新所有已安装的 Skill，以获取安全补丁。
懒人方法：直接对 OpenClaw 发出指令（如“帮我装某个Skill”），它会自动执行相关命令。
