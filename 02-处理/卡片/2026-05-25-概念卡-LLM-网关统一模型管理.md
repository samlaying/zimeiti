# LLM 网关统一模型管理

**LLM 网关（模型网关）** 是 [[LangSmith]] 平台新增的统一接入和管理各种大模型供应商服务的模块。

除了基本的模型连接能力，它还提供了重要的增值能力：
- **全链路 Tracing**：所有通过网关的模型调用都可被监控追踪，与 [[LangSmith Engine：自闭环智能体运维]] 的能力无缝集成。
- **成本与安全监控**：实时监控 API 支出，并包含敏感信息、密钥的检测与脱敏等安全能力。

这相当于将类似创业公司 AGUI（Copilot）提供的增值服务（如追踪、安全加固）集成到了平台内部，为企业级落地提供了更完整的保障。
