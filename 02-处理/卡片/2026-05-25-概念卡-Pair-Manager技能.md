# Pair Manager技能

Pair Manager是一个集成于[[Cloud Code]]中的[[Obsidian]]笔记管理技能，旨在通过AI自动化解决笔记管理中的手动维护、信息过时、操作繁琐和健康隐患四大痛点。它提供六大子命令，覆盖从整理到归档的完整流程，核心价值在于命令间的[[智能联动]]，将一条指令转化为一系列连贯的后台操作。
