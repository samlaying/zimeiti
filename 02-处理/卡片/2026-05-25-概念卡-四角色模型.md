# 四角色模型

推荐新手使用的多代理团队结构，映射真实工作流程，易于上手。包含四个核心角色：
1. [[协调者角色|协调者]]：门面与项目经理
2. [[研究专家角色|研究专家]]：信息守门人
3. [[作家角色|作家]]：内容雕琢者
4. [[工程师角色|工程师]]：技术执行者

他们之间形成协作流程：协调者接受任务 → 拆解分配 → 研究专家收集信息 → 作家组织内容 → 工程师实现方案 → 协调者汇总交付。
