# Landbook：按行业筛选的高级UI灵感库

Landbook是一个集合了大量审美高级、提供丰富灵感的UI设计网站。

## 核心特点
强调实用性与可操作性，是获取高质量设计参考的常用工具。

## 主要功能
- **高效筛选**：支持按行业、风格甚至颜色进行筛选，快速定位所需设计。
- **完整范例**：提供非常完整的UI设计范例，包含PC端和移动端的适配效果。
- **配色利器**：提供网页的具体配色卡，用户可以直接借鉴使用，保证色彩一致性。
