# 用正则从HTML提取结构化数据

在Dify工作流中处理网页数据时，常需从HTML中提取结构化信息，方法如下：
1.  **发送请求**：使用[[HTTP请求]]节点，GET方法获取目标URL的完整HTML页面内容（存入 `response.body`）。
2.  **代码提取**：使用[[代码执行]]节点（如Python），编写正则表达式逻辑。
3.  **解析内容**：通过正则匹配从HTML中分别解析出 `title`（标题）、`content`（正文）、`image_list`（图片URL列表）等结构化数据。
4.  **输出变量**：将提取出的多个字段分别定义为输出变量，供后续节点使用。

**关键点**：此方法将非结构化的网页HTML转化为可被后续AI处理流程使用的标准化数据。
