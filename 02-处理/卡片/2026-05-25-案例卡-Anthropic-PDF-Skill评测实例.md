# Anthropic PDF Skill评测实例

Anthropic的[[Skill]]（PDF处理）评测案例，展示了评测如何将失败转化为可迭代资产：

- **问题发现**：在处理**非填充式表单**（需将文字精确放置到坐标）时失败。若无评测，此问题可能靠用户偶然上报。
- **评测价值**：评测将此失败模式**隔离出来**并**可复现**。
- **解决过程**：修改定位逻辑，改为基于提取出的文本坐标进行锚定。
- **验证闭环**：修改后重新运行评测，确认该失败模式被解决。

**核心启示**：评测的目标并非打分，而是**把失败变成可复现的对象**。
