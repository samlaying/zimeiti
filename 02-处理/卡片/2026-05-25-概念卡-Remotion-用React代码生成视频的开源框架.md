# Remotion：用React代码生成视频的开源框架

Remotion 是一个开源框架，它允许开发者**使用 [[React]] 代码**来程序化生成动画和视频内容。

**核心特点**：
- **内容可控**：视频的每一个部分都由代码定义，确保精确、可预测和一致。
- **高可扩展性**：作为基于React的框架，它拥有庞大的生态和灵活的组件化能力。
- **社区认可**：在GitHub上已获得超过42k star，是一个成熟且广受欢迎的开源项目。
