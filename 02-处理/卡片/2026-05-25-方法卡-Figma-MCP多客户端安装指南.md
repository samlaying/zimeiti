# Figma MCP多客户端安装指南

Figma MCP 可在多个主流 AI 编程/设计客户端中安装，安装后需验证 Figma 身份。

### 安装步骤
1.  **在 Cursor 中**：进入设置 -> Marketplace -> 搜索 `Figma` -> 安装。
2.  **在 Claude Code (CC) 中**：通过命令行安装官方 Figma MCP。
3.  **在 Codex 中**：
    - 先添加各个 [[Skills（技能）]]。
    - 在设置中添加 MCP Server，找到 Figma 服务器并安装认证。

### 注意事项
免费用户每月仅有 **6 次** 工具调用额度，充分使用需订阅 Dev 或 Full 席位。
