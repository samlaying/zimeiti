# CSS悬停阴影与上浮

> 通过组合变换与阴影，让UI反馈更生动、有质感。

## 核心内容
为了增强悬停时的立体感和视觉反馈，可以同时改变元素的`transform`和`box-shadow`。常见组合是悬停时`translateY`产生轻微上浮效果，同时`box-shadow`的扩散和透明度增大，模拟阴影变大变深的效果，两者都需要配合`transition`实现平滑过渡。

## 原文关键段落
> .card:hover {
>   transform: translateY(-10px); /* 轻微上浮 */
>   box-shadow: 0 12px 20px rgba(0,0,0,0.2); /* 阴影变大变深 */
> }
