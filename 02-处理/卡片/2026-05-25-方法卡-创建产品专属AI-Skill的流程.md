# 创建产品专属AI Skill的流程

为了训练AI理解你产品的“设计DNA”，可以遵循以下流程创建[[专属AI Skill]]：
1.  **输入**：提供你产品的现有代码库。
2.  **分析**：AI学习并判断产品的形态（如Web、移动端、大屏）。
3.  **提炼**：从源码中提取所有稳定的设计元素，包括：
    - 主题色（如 `#00313D`）
    - 布局架构（导航栏、侧边栏、内容区结构）
    - 公共组件（按钮、卡片、表单的通用样式）
    - 交互模式（页面跳转、表单提交、弹窗逻辑）
    - 代码规范
4.  **输出**：一个为你产品量身定制的原型设计Skill。

这个流程体现了[[先学习，后创造]]的核心思想。
