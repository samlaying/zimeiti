# 快速安装使用Zvec

Zvec 提供开箱即用的体验，开发者无需部署复杂服务器集群即可快速上手。

**操作步骤：**
1. **安装**：通过包管理器执行一句安装命令（如 `Keep install Zwec`，具体以官方文档为准）。
2. **构建原型**：从安装到成功构建出第一个语义搜索原型，整个过程设计得非常迅速。

其优势在于原生支持全平台，并提供多种编程语言的 SDK，实现零门槛快速接入，极大地降低了本地向量检索应用的开发门槛。
