# Markdown规范文档定义

Markdown规范文档用于定义Figma组件创建的规则，包括：
- **页面布局规范**：底部复用标题、内容小标题、模块小标题的层级和命名，以及变体对齐规则。
- **组件变体规范**：变体名称命名规则、单个状态的变量引用（文本变量、颜色变量、圆角变量等），以及尺寸和间距具体规则。
