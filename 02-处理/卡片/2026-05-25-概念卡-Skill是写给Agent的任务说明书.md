# Skill是写给Agent的任务说明书

## 核心定义
[[Skill]] 并非普通的提示词，而是写给 [[Agent]] 看的**专项任务说明书**。它旨在告诉 Agent 在执行某类特定任务时，应遵循的工作流程与规范。

## 关键特征
- **系统性**：它规定了从判断到执行再到优化的完整步骤（如：先判断平台→分析开头钩子→优化节奏→检查结尾转化）。
- **目的性**：旨在确保 Agent 在特定任务上执行的一致性。

## 认知要点
应将 Skill 视为一份可执行的**操作指南**，其作用是规范和优化 Agent 在特定领域的工作流程，避免与通用的提示词概念混淆。
