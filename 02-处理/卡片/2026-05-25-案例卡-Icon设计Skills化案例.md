# Icon设计Skills化案例

**场景**：需要频繁绘制符合特定视觉规范的Icon。

**Skills化做法**：
1.  **制定规范**：将Icon设计规范整理成文档，包括固定尺寸、宽度、不同形状需保证视觉占比均衡、圆角值等。
2.  **交给AI封装**：将规范文档提供给[[Claude]]，让它生成一个专门用于“画Icon的Skills”。
3.  **调用与应用**：之后需要画Icon时，直接调用此Skills，Claude即可基于内置规范生成一套符合要求的Icon。
