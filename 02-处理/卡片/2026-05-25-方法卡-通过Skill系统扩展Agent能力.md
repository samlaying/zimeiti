# 通过Skill系统扩展Agent能力

Agent的能力通过**Skill系统**进行模块化扩展，这是实现灵活功能的核心方法。

### 扩展步骤与设计
1.  **利用内置工具**：系统提供开箱即用的工具（如Python环境、飞书、Google Workspace）。
2.  **编写自定义Skill**：用户可创建自定义Skill，优先级高于共享工具，编写逻辑类似OpenClaw。
3.  **约束工具类型**：区分安全工具（如搜索）和风险工具（如删除），后者需要[[human-in-the-loop]]授权。
4.  **处理复杂任务**：对于复杂任务，可拆分为[[子Agent（Subagent）]]执行，它们拥有独立上下文，完成后结果回传主循环。
