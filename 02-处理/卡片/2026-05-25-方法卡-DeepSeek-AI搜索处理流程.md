# DeepSeek AI搜索处理流程

DeepSeek处理用户查询的标准流程如下：
1. **问题拆分**：系统将复杂问题拆分为多个可并行处理的子问题。
2. **网页搜索**：每个子问题独立搜索，获取多个网页。
3. **内容读取与判断**：系统只读取网页的部分内容（如标题或前几个Token）进行判断。
4. **决策与生成**：判断当前内容是否足够回答问题。若足够，则生成答案；若不足，则进入[[循环补充搜索]]。
