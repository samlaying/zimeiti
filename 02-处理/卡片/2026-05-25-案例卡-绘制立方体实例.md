# 绘制立方体实例

一个具体的[[PPT VBA]]绘图应用案例：绘制[[立方体]]，常用于论文中的三维结构示意图。

**操作步骤**：
1.  获取或编写用于绘制立方体的VBA代码。
2.  按照[[PPT VBA绘图工作流]]，在PPT的VBA编辑器中插入新模块并粘贴该代码。
3.  将光标置于函数名内部，运行代码。
4.  幻灯片上即会自动生成一个立方体。可以通过修改代码中的参数（如位置坐标）来定制其外观。
