# 1700个Skill的筛选实践

一位用户在安装 OpenClaw 后，面对 [[ClawHub]] 中超过 1700 个 Skill 感到无从下手。经过数天实践筛选，最终提炼出 **15 个经过验证的实用 Skill**。

**筛选逻辑**：
- 遵循 [[OpenClaw分层安装策略]] 进行系统性筛选
- 从核心底座开始，逐步构建能力
- 根据真实工作场景验证效率工具的实用性
- 在熟练使用后探索进阶功能

**筛选结果**：筛选并非追求大而全，而是找到一个能覆盖从基础任务到个性化需求的、**真正有用的起点**。
