# SDD规范驱动开发流程

## 流程步骤
高含金量的[[Vibe Coding的正确定义|Vibe Coding]]能力，其核心是遵循**SDD（Specification Driven Development，规范驱动开发）** 流程：
1.  **制定规范**：编码前，必须产出清晰的**PRD（产品需求文档）** 和**原型**。
2.  **方案定稿**：PRD与原型必须经过确认并定稿。
3.  **正式开发**：在规范体系下，按顺序开发：
    *   **先前端**：开发并确认前端界面。
    *   **再后端**：基于确认的前端进行后端逻辑开发。
4.  **质量保障**：开发流程中必须包含**自动化测试**环节。

**产品经理的角色**：产品经理需遵循此规范来驱动和管理开发过程，即使使用AI编码工具也是如此。
