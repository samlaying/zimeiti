# AI设计规范文档

`design.md` 是 Google 开源的 [[Google Stitch 开源项目]] 中的核心文件，是一种 **专门给AI看的设计规范文档**。

它并非 Figma 或 JSON 等复杂格式，而是一个 **Markdown 文档**。其核心内容是描述产品的视觉风格，包括配色、字体、按钮、卡片、布局原则等，甚至包含给AI的提示词写法，用以指导AI生成页面。
