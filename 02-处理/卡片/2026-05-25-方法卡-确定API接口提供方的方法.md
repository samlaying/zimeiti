# 确定API接口提供方的方法

根据[[系统流程图]]确定接口提供方，遵循以下两个核心规则：

1.  **箭头指向规则**：**箭头指向谁，谁就提供接口**。通常是核心系统（如订单系统）主动发起调用。
2.  **数据源规则**：**谁是数据源的上游方，谁就有义务将数据推送给下游**。例如，订单系统（上游）需向WMS系统（下游）推送发货单。
