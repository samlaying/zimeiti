# Vibe Coding 评测方法

测评多个 Design Skill 可靠性的核心方法：

1.  **Prompt 一致性**：使用同一段提示词分别对每个 Skill 运行多次（如三次），确保公平比较。
2.  **设计师视角评分**：从**设计细节**（字体、间距、对齐等）、**风格设计**（整体调性、大胆性、舒适度）和**综合评分**三个维度进行评价。
3.  **设立审核基准**：选定一个高质量的 Skill 作为基准（例如 [[Anthoropic Skill]] 的 43 行 Design Scale），任何产出质量低于此基准的 Skill 被认为不值得下载使用。
