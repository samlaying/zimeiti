# 账号直连，免API配置

> 无需API密钥，登录账号即用，零配置上手。

## 核心内容
该画布工具直接关联用户的即梦官方账号，使用画布时的登录信息和积分消耗与即梦平台完全同步。用户无需进行任何API密钥的申请和配置操作，彻底省去了技术配置环节，使得普通用户能直接上手使用。

## 原文关键段落
> 这里关联的也是自己的即梦账号...跟API是一点关系都没有。
