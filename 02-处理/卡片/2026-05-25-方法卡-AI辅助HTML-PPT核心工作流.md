# AI辅助HTML PPT核心工作流

利用AI Agent（如Cline）配合特定技能（[[Frontend Slides Skill]]和[[Beautiful HTML Templates Skill]]）从零开始制作HTML演示文稿。

核心流程四步走：
1. **准备与配置**：安装核心Agent和两个关键技能，并通过修改`agents.md`文件实现[[技能集成与自定义]]。
2. **生成初稿**：向Agent提供内容（如文案），通过交互确定页数、风格等，AI会先生成风格示例页供选择，再一次性生成全部内容。
3. **精细调整**：使用[[HTML PPT可视化编辑]]工具调整字体、大小，并根据[[插入图片与视频的策略]]处理图片和视频素材的插入与替换。
4. **分享交付**：通过文件打包或免费的在线平台（如Netlify）进行[[HTML PPT分享与部署]]。
