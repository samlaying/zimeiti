# API输入输出定义规则

> 输入由提供方定，输出由调用方定，简单准则。

## 核心内容
输入数据由接口提供方的业务需求决定，输出数据由接口调用方的数据需求决定；在具体场景中，如前端查询页面，输出由页面展示内容决定，输入由查询条件决定。

## 原文关键段落
> 输入由接口提供方决定...输出由接口调用方决定
