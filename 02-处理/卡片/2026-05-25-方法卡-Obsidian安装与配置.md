# Obsidian安装与配置

安装并配置 [[AirPostMaker：AI自动封面生成器]] 插件的步骤。

1.  **前置条件**：在 Obsidian 设置中关闭“第三方插件”的**安全模式**。
2.  **安装BRAT插件**：从官方市场安装 BRAT 插件，用于从 GitHub 安装 Beta 版插件。
3.  **安装AirPostMaker**：
    - 启用 BRAT 后，点击左侧新增的 **BRAT 笑脸图标**。
    - 点击 **Add Plugin**，粘贴 AirPostMaker 的 GitHub 仓库地址。
    - 选择 `latest` 最新版本并添加，等待安装成功。
4.  **配置插件**：进入插件设置页面。
    - **添加API**：填入 DeepSeek V4 Flash 模型的 API Key。
    - **平台管理**：可添加或修改生成封面的目标平台及尺寸。
    - **风格管理**：管理内置或自定义的风格提示词。
