# Agent Builder升级：企业版OpenClu

> 聊天就能造智能体，还能直接上线跑，像个企业版的OpenClu。

## 核心内容
LangSmith Agent Builder是无代码/低代码智能体构建工具，通过对话式交互创建智能体。本次升级集成了Sandbox和Context Hub，并提供官方预制模板（如客户调研系统）。这使其从“玩具”升级为具备生产级能力的工具，可视为一个企业版的OpenClu。
