# Project Analyzer Skill

**Project Analyzer** 是一个专门用于对开源项目或已有老项目进行**逆向工程分析**的[[Skill]]。

**核心功能**：自动生成一份包含14个章节的详细PRD文档。

**主要输出**：
1.  **结构化PRD**：涵盖产品定位、用户故事、Agent故事、Agent流程、提示词设计策略等14章。
2.  **Agent故事详情**：每个故事会明确其所需的上下文、输入、输出和工具依赖。
3.  **提示词拆解分析**：自动提取项目中的所有提示词，生成索引，并提供每个提示词的详细位置、中文翻译、参数解析及关联代码。
