# 创建专属HTML Skill的完整流程

创建一个专属HTML Skill主要分为三步：

1.  **明确风格需求**：确定颜色、字体、布局等个人偏好，整理成提示词。
2.  **生成与筛选**：使用提示词在多个AI平台（如MainTi、小米Memo等）生成多版HTML，筛选出最满意的版本并保存。
3.  **创建与优化Skill**：利用支持Skill的Agent（如[[Workbody]]）将满意的HTML转化为可复用的Skill，并通过迭代调试解决各类问题。

整个过程的核心是“提炼个人偏好 -> AI生成 -> 人工筛选 -> 沉淀为自动化模板”。
