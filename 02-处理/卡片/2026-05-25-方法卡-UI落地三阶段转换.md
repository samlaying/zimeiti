# UI落地三阶段转换

将产品想法最终视觉化、技术化落地的过程，可分为三个阶段迭代：
1. **探索期（AI Studio）**：使用AI工具探索UI方向。先让AI用文字提供方案，选定方向后生成小Demo。此阶段目标是确定大致风格，只需达到30分效果，目的是省Token并快速验证方向。
2. **转换期（技术栈适配）**：将AI生成的设计（如React）转换为目标技术栈（如`[[UniApp]]`），并进行整体调整，达到80分左右的效果。这一步会进行技术栈的转换。
3. **精修期（开发者工具）**：导入微信开发者工具等真实环境，微调细节，达到90分左右的效果。每一步迁移可能有少量UI失真，但因使用同一套CSS，总体影响不大。
