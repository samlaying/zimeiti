# Agent Skills：AI角色认知换挡

**Agent Skills**是2025年AI行业的重要技术成果之一，其核心是将AI切换到具体角色（如CEO、设计师、研究分析师），以实现“**认知换挡**”，从而更有效地完成特定任务。

其设计思路受Y Combinator现任掌门人Gary Tan开源的**G-Stack**项目启发。G-Stack包含32个日常使用的Cloud Skills，旨在让AI适配不同工作场景。

在专业PPT制作中，可应用`PPT Classify`（分类）、`PPT Research`（研究）、`PPT Narrative Review`（叙事评审）等Skills。
