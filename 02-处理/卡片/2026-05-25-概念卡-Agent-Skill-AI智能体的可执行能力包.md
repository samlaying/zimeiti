# Agent Skill：AI智能体的可执行能力包

**Agent Skill（智能体技能）** 指的是赋予AI智能体（Agent）特定能力、任务或工作流程的模块化组件。

- 它本质上是一段“你和AI的交互”，将特定的知识、规则和步骤交给了AI，使得AI能够执行特定的任务。
- Skill以文件形式存在（如Zip包），可以被安装到支持Agent功能的客户端（如Tray、HermesCloat Code）。
- 它具有**可修改性**，用户可以根据自己的需求，用自然语言对Skill文件进行添加、修改和删除，从而创建高度个性化的Agent。
