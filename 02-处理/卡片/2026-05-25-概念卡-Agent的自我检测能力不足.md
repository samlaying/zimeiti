# Agent的自我检测能力不足

Agent通常擅长在被告知问题后快速解决问题，但**无法自行检测问题**。这是因为它缺乏两个关键认知基础：
1. **锚点（Anchor）**：agent依赖什么来知道任务是否正确。
2. **基准线（Baseline）**：用于判断什么是问题的客观标准。

如果没有外部给定的指标和基准线，agent无法识别自身输出的偏差，导致其认为“没问题”的内容可能完全不符合用户期望。
