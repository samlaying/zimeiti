# 资源可控是端侧部署通行证

**资源可控是 Zvec 走进端侧（如嵌入式设备）的通行证。**

在资源受限的环境中，向量系统必须为内存和 CPU 划定明确边界。Zvec 从架构层面内置了资源约束机制，确保在有限资源下依然能够稳定运行。这一特性使其成为满足数据安全、隐私和低延迟需求的本地化部署方案的关键。
