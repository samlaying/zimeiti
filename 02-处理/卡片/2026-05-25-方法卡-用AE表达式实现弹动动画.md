# 用AE表达式实现弹动动画

为文字添加自然弹动出场效果的操作步骤：
1.  **准备**：在AE中输入文字并选中图层。
2.  **展开属性**：在图层属性中找到并展开“变换”。
3.  **添加动画器**：点击“动画制作工具”旁的下拉箭头，选择“位置”。
4.  **设置初始位置**：将“位置”属性的Y坐标值调整为 `120`（此值定义弹动起始点）。
5.  **添加表达式**：按住`Alt`/`Option`键点击“位置”前的秒表，粘贴**弹动表达式**代码。
6.  **完成与调整**：按空格预览效果，并可通过调整代码中的`freq`、`decay`等参数控制弹动细节。
