# 编程 Agent 比模型更重要

在 AI 辅助编程领域，**编程 Agent (如 Codex) 的重要性可能超过底层大模型本身**。

**原因：**
- **任务拆解与规划**：Agent 能将复杂的大任务拆解为多个子阶段，并逐步执行、自我修正。
- **工具使用**：Agent 能调用各种工具 (如搜索网页、读取日志、使用 [[Codex 的 Sub-Agents 并行审查]]) 来解决问题。
- **长上下文执行**：相比单次问答模型，Agent 能长时间专注于一个任务。

单纯的聊天模型是一次一次的回答，而编程 Agent 是一个能够自主规划、执行和迭代的“工作者”。
