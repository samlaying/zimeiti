# AI采访生需求文档

通过向AI发出明确指令，使用其自带的 `ask user question` 工具进行多轮访谈，可以深度挖掘并生成一份结构化的需求文档（如 `Size.md`）。这能有效避免直接让AI写代码导致的预期不符和反复修改。

**操作步骤**：
1.  明确指令AI使用该工具进行提问，例如：“我想开发一个家庭点菜网站，请使用ask user question工具对我进行深度访谈。”
2.  根据AI的问题选择或补充回答，AI会主动引导并补全未考虑到的需求点。
3.  访谈结束后，AI会自动生成详尽的需求文档。
