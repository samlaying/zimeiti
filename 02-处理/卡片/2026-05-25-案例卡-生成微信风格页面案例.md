# 生成微信风格页面案例

**案例背景**：用户向 [[Pencil MCP 插件]] 提出复杂要求，希望用Instagram的设计风格来绘制微信APP的界面，并输出多张方案。

**执行过程**：
1. AI开始执行，过程涉及资料查询和设计生成。
2. 整个执行过程约**10分钟**。

**产出结果**：
- 最终输出了**五张**不同风格的设计方案。
- AI对每种设计风格都进行了解释。
- 生成的设计是**高保真**且**可编辑**的，但部分文字显示有问题，后通过对话指令修改字体后得到解决。
