# AI UI工具选择指南

选择AI UI生成工具应基于项目阶段和核心需求：

- **选择 [[Google Stitch：AI UI生成工具|Stitch]] 的情况**：
  1. 赶时间，需要快速出效果。
  2. 追求低门槛和高生成质量。
  3. 工作流是“生成 → 复制代码 → 交付开发”。

- **选择 [[Pencil：AI UI设计协作工具|Pencil]] 的情况**：
  1. 你是专业开发者或负责长期项目。
  2. 看重设计过程的可控性。
  3. 需要设计稿能够持续迭代和精细调整。
