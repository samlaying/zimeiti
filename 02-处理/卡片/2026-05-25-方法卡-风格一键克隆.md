# 风格一键克隆

通过上传一张参考图，将它的视觉风格（如配色、图标风格）一键迁移到另一张目标图上，确保设计的一致性。

**操作步骤**：
1. 上传作为风格参考的图（图一）。
2. 让 Pensel UI 处理并识别该风格。
3. 将该风格应用到另一张需要设计的目标图（图二）上。

此方法是 [[Pensel UI 定义]] 中“克隆设计风格”能力的具体实践，体现了 [[AI设计工具优势]] 中的“统一”。
