# Agent Loop核心循环机制

**Agent Loop** 是用户沙箱Agent系统的核心运行机制，是一个处理用户请求的循环流程。

### 主要组成
- **请求处理**：用户输入进入系统，以SSE（[[流式输出]]）方式返回结果。
- **工具调用**：系统在循环中调用各种工具（如飞书、搜索），并将工具结果回传至循环。
- **停止条件**：循环在任务完成、需要[[human-in-the-loop]]或达到最大轮次时停止，防止死循环。
