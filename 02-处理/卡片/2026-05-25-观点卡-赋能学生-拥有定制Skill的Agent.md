# 赋能学生：拥有定制Skill的Agent

AI时代个性化学习的定义，不是所有学生用同一个 Agent，而是**每个学生拥有自己定制的、运行着独特 [[Agent Skill]] 的 Agent**，以此处理自己的问题。

这是引领学生适应AI时代、实现**真正个性化学习**的关键途径。学生获得的 [[Agent Skill]] 文件，可以根据自己的需求，用自然语言进行修改、添加和删除，从而拥有一个处理自己问题的个性化工具，并可举一反三扩展到其他课程和学科。
