# 标签体系

## 卡片类型标签

| 标签 | 用途 |
|------|------|
| `#卡片/概念` | 核心术语、工具定义 |
| `#卡片/方法` | 具体操作步骤 |
| `#卡片/观点` | 有传播价值的判断 |
| `#卡片/案例` | 具体实践故事 |
| `#卡片/金句` | 适合直接引用的表达 |

## 主题标签

| 标签 | 适用内容 |
|------|----------|
| `#主题/AI工具` | Claude Code, Codex, GPT, Gemini 等工具测评和用法 |
| `#主题/Skills` | Agent Skills, MCP, Workflow 相关 |
| `#主题/编程` | Vibe Coding, 前端开发, 独立开发 |
| `#主题/产品经理` | 产品思维, PRD, 需求分析, API |
| `#主题/设计` | UI设计, CSS, HTML, Figma, 审美 |
| `#主题/PPT` | PPT制作, 文档自动化, 排版 |
| `#主题/自媒体` | 内容创作, 博主成长, 视频制作 |
| `#主题/知识管理` | Obsidian, Hermes, LLM-Wiki, 笔记方法 |
| `#主题/浏览器自动化` | Playwright, RPA, 网页操作 |
| `#主题/视频` | 视频制作, 剪辑, TTS, 动画 |

## 素材用途标签

| 标签 | 用途 |
|------|------|
| `#素材/观点` | 可直接输出的观点 |
| `#素材/教程` | 可转化为教程内容 |
| `#素材/热点` | 跟当下热点相关 |
| `#素材/金句` | 可直接引用的高光句 |
| `#素材/案例` | 可作为论据的案例 |

## 来源标签

| 标签 | 来源 |
|------|------|
| `#来源/bilibili` | B站视频逐字稿 |
| `#来源/播客` | 播客转录 |
| `#来源/文章` | 外部文章摘录 |
| `#来源/课程` | 课程笔记 |

## 标签使用规则

1. 每张卡片**必须**有一个 `卡片/*` 类型标签
2. 每张卡片**必须**至少一个 `主题/*` 标签
3. `素材/*` 标签用于标记可直接用于发布的内容
4. 标签写在 frontmatter 的 `tags` 字段中
5. 同一张卡片可以有多个主题标签（交叉主题）
