# 03-分发 — 平台适配规则

## 定位

每个平台有自己的内容规则。内容从输入层经过 AI 处理后，按这里的规则适配到各平台。

## 平台规则文件

| 平台 | 文件 | 状态 |
|------|------|------|
| 公众号 | `公众号.md` | 待补充 |
| 小红书 | `小红书.md` | 待补充 |
| Twitter/X | `twitter.md` | 待补充 |
| 朋友圈 | `朋友圈.md` | 待补充 |

## 每个平台规则文件应包含

```markdown
# 平台名称

## 内容格式
- 字数限制
- 结构要求（标题/正文/结尾）
- 配图要求

## 语气风格
- 正式/口语化
- 人称（第一人称/第三人称）

## 标签策略
- 话题标签规则
- 关键词密度

## 发布节奏
- 建议频率
- 最佳发布时间

## 适配示例
- 同一内容在该平台的示例
```

## 适配逻辑

AI 在生成输出草稿时，会：
1. 读取目标平台的规则文件
2. 按规则调整内容格式、语气、结构
3. 生成符合该平台规范的草稿
4. 放到 `04-输出/` 对应平台目录
